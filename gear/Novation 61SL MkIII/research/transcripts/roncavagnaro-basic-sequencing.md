Ron Cavagnaro · Basic Sequencing Pt. 1 // Novation Remote SL Mk3 Tutorial · 2021-06-03

A focused, concise step-sequencing tutorial. Notable for this rig: Ron sequences a Roland Boutique JU-06 **through the Hologram Microcosm** — i.e. the SL driving a synth into a granular/texture pedal, which is squarely the rig's aesthetic. Best for the generative/aleatoric direction controls demoed cleanly.

## Step entry
- **Steps view**: step 1 counts up to 8, then 9–16. Add a note by choosing a note and pressing the step you want it on. Preview by play.
- Add notes across the grid (e.g. step 1, halfway, and another). **Change a note**: hold the step, press the note to remove, press the new note instead.

## Pattern timing
- **Options → Pattern → sync rate**: 1/16 default → 1/32 (faster) → 1/8 (slower). When he drops to 1/8 he wants notes only on four spots, so he **Duplicates** (hold Duplicate, press source, press destination, release) to copy a note to the right steps, and **Clears** the unwanted ones.

## Direction + generative controls (the aleatoric toolkit)
- **Direction**: forward → backward → **ping-pong** (plays through then reverses) → **random** (plays steps all over the place).
- **Start / end position**: trim the active range. Combined with **chance** he limits which notes will play ("limit the chances of what notes are going to be playing") for improvised, evolving results — and suggests **recording the results into an external sequencer**.
- He plays with cutoff/resonance on the synth live while the random sequence evolves.

## Octave / transpose / scale
- **Shift + Up/Down** changes octave.
- A **Scale** is set up; **Options** lets him change the transposition.

## Patterns + chaining
- Move to **pattern 2** (clean slate), add notes, set scale.
- **Chain patterns**: select multiple patterns so it plays across e.g. 4 patterns; or select a single pattern to isolate it. Per-pattern sync rate again — a chained pattern can run at a different speed.
- **Duplicate** a pattern across slots, then fill the rest, so a sequence starts at one pattern and steps through each.
