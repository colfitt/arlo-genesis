# InControl setup — Ableton Live 12 (the best-integrated DAW)

## Setup steps (Live → Preferences → Link/Tempo/MIDI)
1. Live menu → **Preferences** → **Link Tempo MIDI** tab.
2. **Control Surface** = **SL MkIII**.
3. **Input** and **Output** = **"Novation SL MkIII InControl / Port 2"**.
4. In **MIDI Ports**: for **"Novation SL MkIII (SL MkIII MIDI)"** set **Track**, **Sync**, **Remote** = **On** for **both** Input and Output. Also set **Track = On** for **SL_MkIII Input** and **SL MkIII Output**.
5. Press **InControl** on the SL.

## What you get (Ableton-specific)
- **Selection ring** (red rectangle, Session view) of 8 tracks shows track names on the SL's screens.
- **Track Left/Right** = move selection; **Shift + Track L/R** = jump 8 tracks (selected track keeps relative position). **Up/Down arrows** = move scene up/down. **Soft keys** under screens = jump straight to track 1–8 in the ring.
- **Pads (8×2) = Clips** (Ableton-exclusive). Arm a track → empty slots go red, press to record; recorded clip pulses green / shows track colour to launch. **Scene launch** via the green ▶ at each pad row. **Hold Shift** → lower pad row + scene-launch button turn red = **stop clip / stop all clips**.
- **Grid** button → pads become a **Drum Rack** layout (C1 bottom-left … D#2 top-right; green up/down to change octave). Press Grid again → back to clips.
- **Faders** = the 8 selection-ring track volumes (fader LEDs brighten/dim to show true value since physical position may not match after banking).
- **8×2 soft buttons** = **Mute (top/yellow) / Solo (bottom/blue)** by default; **green down arrow** → bank 2 = **Monitor (Auto/In/Off)** + **Record Arm**.
- **Options** button → **Device Control**: top screens show the selected track's **device chain**; click a **pink pad** to select a device → its **first 8 params on the encoders**; **Up/Down arrows** bank through more param pages. Soft key **"DevceSlct"** returns to the chain; yellow **Pan** + green **Sends** views also available (bank sends with Up/Down).
- **Shift** (in InControl) → first three soft buttons = **Undo / Redo / Click (metronome)**; a **Capture** button appears if you just played un-recorded MIDI (grabs it into a clip).

## Rig use
This is the configuration to run **soft synths (Kontakt/Komplete 15, Arturia V Collection 11) in Live 12** with full hands-on: keybed plays the armed track, encoders ride device params, faders mix the ring, pads launch backing/texture clips. Pair with the standalone sequencer only by choosing one transport at a time (mode toggle is hard — see dossier §4).
