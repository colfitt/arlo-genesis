# Pattern view — start/end, direction, sync rate, shift, chains

Two ways to reach pattern-level settings:
- **Steps view → Options → Pattern** soft button (start/end/direction/sync/shift).
- **Patterns view** (Patterns button) for selecting/chaining the 8 patterns per Part.
- Pattern changes **take effect when playback reaches the end of the current pattern.** Novation explicitly recommends "playing around here to achieve happy accidents."

## Pattern options (Steps → Options → Pattern)
- **Start Position** (rotary above 'Start Position'): moves the start step → changes pattern length. The start step pad lights **yellow**. Steps that fall outside and have notes light **red**.
- **End Position:** moves the end step → changes length. **End can come *before* start.** Out-of-range note steps light red.
- **Direction:**
  - **Forward** (default) — start → end.
  - **Backwards** — end → start (reverse).
  - **Ping-Pong** — start→end, then end→start; **end steps repeat.**
  - **Random** — continually picks steps between start and end at random → repeated notes & general chaos. (The most experimental / generative option.)
- **Sync Rate** (rate of steps vs BPM, internal or external clock): **1/32T, 1/32, 1/16T, 1/16 (default), 1/8T, 1/8, 1/4T, 1/4** note values.
- **Pattern Shift:** value **0–15** — simultaneously nudges the start AND end points; shows how many steps shifted since created. (Also settable in Patterns view → Options → 3rd screen.)

## Patterns view (Patterns button)
- 8 patterns per track laid out **horizontally** across the 8×2 pads; each row takes the colour of its Part. Up/down arrows scroll patterns per Part; 5th-screen bottom shows which Parts each row maps to.
- **Select a new pattern:** press a pad. If transport stopped → starts when transport starts. If running → new pattern starts **when the current pattern ends.**
- **Instant pattern switch / audible blend:** **hold Shift + select a pattern** → switches instantly (a way to "combine" patterns).
- **Reset a pattern to default:** hold **Clear** + press a pattern pad (removes all note + automation data).
- **Copy a pattern:** hold **Duplicate** + select pattern, keep holding + press destination pad(s). Can paste multiple times; can copy patterns **between Tracks** (automation does NOT copy across tracks).

## Pattern Chains (arrangement)
- **Press two or more pattern pads together** → creates a Chain from the left-most to right-most pressed pad. Patterns launch **consecutively** (does NOT destructively merge them) → longer musical ideas / song sections.
- **Expand view:** **Shift + Patterns** → arranges each Track's patterns vertically (good for 2-pattern chains / seeing more tracks). Up/down for more patterns. Press Patterns again to return to horizontal. Far-right screen blocks show the chain + current pattern.
- Tip (Novation): copy a drum pattern to a bass track and vice-versa for "happy accidents."
