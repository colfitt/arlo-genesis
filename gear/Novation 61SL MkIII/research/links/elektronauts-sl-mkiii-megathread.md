The long-running Elektronauts owner thread — directly rig-relevant because it's Elektron users sequencing Digitakt / Octatrack / Analog Rytm from the SL. Distilled practical content:

## Sequencer / patterns
- Sequencer engine is **adapted from the Circuit**. Core gripe at launch: "only 16 steps on the sequencer" with no native 64/128-step mode. The intended answer is **pattern chaining** to build longer arrangements — repeatedly called out as the way to make it work alongside an Octatrack.
- A user notes "32 steps is possible" (per a Sonic State demo) — this is the **gate length** going to 32 steps (firmware 1.2), not 32 sequencer steps; a common point of confusion.

## DAW integration / HUI gripes
- Deep dedicated scripts for **Ableton Live, Logic Pro, Reason**; everything else (Cubase, Pro Tools, Studio One, **Reaper**) falls back to **HUI**.
- User **themixtape** (June 2019) calls HUI "WAY too outdated" for Reaper and wants Mackie Control instead. Resolution for Reaper users in-thread: the third-party **DrivenByMoss 4** control-surface script (July 2019) is a markedly better integration than stock HUI. (Relevant if Logic/Ableton scripts ever feel limiting — DrivenByMoss also covers Bitwig/Reaper.)
- Missing vs Ableton: no equivalent of Live's one-button "capture MIDI" trigger.

## Mode-switching friction (confirms dossier §4)
- themixtape: "I hate how holding **shift + the mode** enables the mode. It should just be the button WITHOUT shift." — the day-to-day annoyance of flipping between standalone sequencer world and InControl.

## Control-count limits
- Only **2 banks of 8 = 16 encoders** addressable per template; power users wanted 8 pages / 64. Same complaint about fader count. Pattern-button brightness/feedback called inconsistent.

## Components template friction (confirms dossier §4)
- themixtape: "It's a lot of assigning in the Components software… but you get a lot of bang for the buck if you put the time in." Translation: every CC map for the VG-800 / Chase Bliss / pedals is a Components session, but the payoff is real once built.

## Sessions praise (counterpoint to Save-Lock risk)
- themixtape (June 2019): "the ability to instantly recall any Session, or save (and rename later, in Components)… tremendous." Recall-by-pad is genuinely loved; the overwrite risk (see Save-Lock file) is the flip side of how fast saving is.

## Power-user tips surfaced (cross-checked w/ Novation guides)
- **Shift + Play** = start playback from current position and send MIDI Continue (vs Play from top).
- **Grid** button toggles the 8×2 pads between the last Sequencer subview and Template mode — fast way to flip between step-entry and clip/CC pads.
- Pads have **polyphonic aftertouch** (the keybed strip is only channel AT — the pads are the poly-pressure surface).

Community activity note: this is the single most active owner thread for the box, but it's front-loaded to 2018–2020 and goes quiet after firmware 1.4. Signal is good but not deep; no large bug-tracker culture like Elektron's own boxes.
