# Digitakt II Projects Sharing (community thread)

The central Elektronauts thread for swapping full **.dt2prj** project files (kits + patterns) for the Digitakt II. Useful because a DT2 *project* carries kits, samples, p-locks and patterns together — the fastest way to study someone else's sound design hands-on.

## Notable share: MINOR_KEYS_JAMS.dt2prj (by NKS)
- Posted 2024-10-23. Free, explicitly cleared for personal AND commercial use ("samples specifically made for this project, no copyright concerns").
- **8 pattern banks, one per key:**
  - Bank A = D min, B = E min, C = C# min, D = F min, E = A min, F = B min, G = F# min, H = D# min.
- **Structure inside each bank:** Pattern 1 = a demo "track jam" showing what the kit can do; Patterns 2–8 = the kit samples; Patterns 9–16 = empty (room to build).
- Download (mediafire, per thread): https://www.mediafire.com/file/qesm8y02w4ashiw/MINOR_KEYS_JAMS.dt2prj/file

## How to load a .dt2prj
1. Copy the project file onto the DT2 via **Elektron Transfer** (or place on the +Drive).
2. On the DT2: load the project from the project menu. It brings its own samples, kits, and patterns.

## DT2 vs OG
- **.dt2prj is a Digitakt II project format** — it does NOT load on the OG Digitakt. (OG uses its own project/syx format and is mono / single-LFO / single-filter.) Projects are the LEAST portable share type between the two boxes; sample packs are the most portable.

NOTE: This is a living community thread; more projects get posted over time. The MINOR_KEYS_JAMS share is the most concretely documented as of capture.
