# Amiva — ambient Digitakt workflow (Elektron artist spotlight)

Ambient producer **Amiva** (Alex) — ex-Berlin club scene, now rural Spain — uses the Digitakt as an ambient "canvas" rather than a drum machine. The cleanest minimalist recipe found; maps perfectly onto holding sampled fuzz-wall/banjo drones.

## The recipe (concrete)
- **No sequencer. One pattern.** He doesn't run steps — he **triggers each track by hand**.
- **AMP DECAY = INF on every track** → each triggered sample **loops/sustains endlessly**. The box becomes a tactile soundscape generator: hit a track, it drones; hit another, it layers.
- **Source material:** long samples — **field recordings, textures, piano motifs** (here: would be baritone/fuzz/banjo prints).
- **Route audio ONLY through the reverb/delay sends — skip the dry output** for a lush, all-wet atmosphere. (On DT2: turn a track's main level down / sends up, or use the send routing so only processed signal hits the mix.)
- Performance moves: **gentle pitch shifts, filter sweeps, real-time FX** tweaks to evolve the layers live with "emotional depth."

## Why it maps to this rig
- This is the "ambient sample bed" workflow (dossier §13d) reduced to its essence: INF decay + manual trig + all-wet. With DT2 stereo + 3 LFOs you extend it — but Amiva proves you can get there with almost nothing.

NOTE: "DT" in the spotlight — Amiva's method is OG-Digitakt-born but transfers fully to DT2 (and improves with stereo + Comb filters + extra LFOs). INF decay + send-only routing are the two load-bearing settings.
