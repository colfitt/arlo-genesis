# Where OT Projects/Sets/Templates actually get shared (the hubs)

Unlike a synth there is no big "Patchstorage for Octatrack." OT sharing is fragmented across forums, Blend, and a few paid creators. Honest map of where to look:

## Free / community
- **Elektronauts** — the main hub. Threads: "Octatrack Project Templates" (#3751), "Octatrack Templates" (#145666), "Definitive list — packs, chains & tools" (#32418), "Free OT tutorials list + live setups" (#61759). Mostly DIY-recipe discussion; a few attached project files (e.g. virtual_flannel's Crossfader Transition; an "OT Solo Template 1.2.2" referenced). Download attachments and copy the Set/Project folder to the CF card.
- **Gearspace** — "Official Octatrack Template Thread — Share Here!" (gearspace.com/board/...892415) — community-shared templates, same DIY tilt.
- **Blend.io** — blend.io/explore/octatrack — "Octatrack Projects, Templates and Lessons By Artists." Project/lesson sharing platform; worth browsing for actual project files.

## Paid (the polished, reliable templates)
- **EZBOT (Matthew Piecora)** — the de-facto pro template maker. Performance Mixer 3.x, Hyper FX, Standalone Template. ko-fi.com/ezbot + ezbot-shop.fourthwall.com + Patreon. (See artist-ezbot-templates-techno.md.) Best off-the-shelf "OT as live FX mixer" projects.
- **Modbang** — modbang.com sells EZBOT's Performance Mixer + sample-chain packs (e.g. "Micro Prophet 5 Sample Chains").

## Sample chains / packs (the OT's other shareable atom)
- **Official Elektron packs** (~€30): Octa Pak Vol.1–6 + genre titles; free "Drum Enthusiast." (Detail in samplechain-tools-roundup.md.)
- **Loopmasters** — sells in "Octatrack Set" / "Octatrack Programs" formats (project + chain bundles).

## How to load any of it
- A **Project/Set** = a folder (project.work/.strd, bank files, AUDIO pool) → copy to CF card → PROJECT/SET → LOAD. Templates usually ship as a Project you DUPLICATE per song.
- A **sample chain** = `chain.wav` + `chain.ot` → drop both in the Set's AUDIO folder → assign to a Flex/Static slot (slices come from the .ot).

## Honest note
Volume of genuinely downloadable, free OT *project* files is LOW (the format/CF workflow discourages casual sharing). The reliable polished projects are EZBOT's paid templates; the free value is mostly recipes + sample chains, not turnkey Sets.
