# Dataline — the deep live-resampling tutorial maker (improv OT practitioner)

Dataline is a Berlin-based improv electronic producer ("never the same track twice") who sells the most respected hands-on **OT MkII live-resampling tutorial videos** in the community. ## What the tutorials cover (HOW to resample live)
- "Octatrack MK2 — Live Resampling Tutorial Video": how the machines (with a Roland TB-03 + TR-09) are **connected, set up, and how the OT is configured for seamless live sampling**. Includes the full audio performance. ~£20.
- "Track & Teach: Octatrack Live Sampling & Remixing": configuring the OT for seamless live sampling + remixing, **mixing the inputs back in, and using the Cue Outputs as a Send-FX bus**, resampling the master track to capture all live nuances, and instantly switching between newly recorded material and the live performance for **beat-juggle-style** rhythms/fills.

## The resampling workflow it teaches (community-standard, distilled)
1. Track recorder buffer set up on a track; **REC trig** placed on the sequencer to capture either input pair (AB/CD) or the **internal master** (resampling).
2. A **Flex machine** plays the recorder buffer back.
3. **Resample the master track** to freeze a whole live state, then chop/rearrange it.
4. **Cue outputs as a send-FX** path (independent of mains) for auditioning/processing.
5. **Crossfader**: fast snaps (not just slow fades) to jump between live and resampled material; conditional/FILL trigs for rhythmic transitions.

## Rig relevance
This is the canonical "capture-and-destroy" loop the dossier describes — Dataline's material is the best paid resource for actually learning it hands-on. The Cue-outs-as-send-FX trick is directly useful with the Apollo x8 / Babyface (independent stem off the cue bus).
