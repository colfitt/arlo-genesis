# Octatrack artist roster — VERIFICATION roundup (who's confirmed vs. lore)

The dossier §7 lists many famous users. This file separates what's well-evidenced from what's repeated-without-proof, so attributions stay honest.

## WELL-EVIDENCED (own captured files where deep)
- **Alessandro Cortini (NIN)** — multiple on-record interviews (Reverb/Gearjunkies/Elektron). VERIFIED. → artist-alessandro-cortini.md
- **Panda Bear (Noah Lennox)** — Reverb interview + MusicRadar studio photos + Sydney Opera House 2018; two OTs, "DJ the stems." VERIFIED. → artist-panda-bear-dj-the-stems.md
- **Stimming** — Cercle (Phare de Cordouan) + DJ Tech Tools; dual-OT mixer/mothership rig. VERIFIED (deep routing partly from his own comments). → artist-stimming-dual-octatrack.md
- **Mumdance (Jack Adams)** — Elektron spotlight + MusicRadar + his own posts; one-shot live track-building w/ TR-909. VERIFIED. → artist-mumdance-live-improv.md
- **Surgeon (Anthony Child)** — CDM + RA "Art of Production" + SchneidersLaden; OT foundation + Faderfox. VERIFIED. → artist-surgeon-live-techno.md
- **Kangding Ray** — Gemtracks/equip listings cite OT DPS-1; performed Boiler Room Berlin CTM 2013 live electronics. Reasonably evidenced (gear listing + known dawless live electronics) — OT specifically plausible, exact live role not transcribed. MOSTLY VERIFIED.
- **Headless Horseman** — Boiler Room live performance referenced as OT user. Plausible, lightly evidenced.
