# FACTORY CONTENT — the only "preset bank" the OT ships with: Set/Project "PRESET MKII"

The OT has no patch presets in the synth sense, but it DOES ship one official factory Set+Project — this is the closest thing to a factory preset bank and is the canonical starting point.

## What it is (official spec, verbatim)
- Comes as an **Octatrack Set** — the Set folder is named **`PRESET MKII`**. Inside: an `AUDIO` folder holding all the sample folders (the audio pool).
- **1 Set. 1 Project. Samples 24-bit / 44.1 kHz. 16 patterns. 4 parts. 16 scenes per part. Content size 6.06 GB** (download ~5 GB, decompresses to ~6 GB).
- Demo patterns use samples assigned to **both Flex and Static** sample slots.
- Sample sources (5+ GB of loops & oneshots): **Origin Sound, Loopmasters, Sample Magic, Samples from Mars** (each is its own folder in the pool). One user note: the **Sample Magic and Origin Sound** folders are the strongest; Loopmasters content least useful.

## How it's organized (concrete, from the install sheet)
- **16 demo patterns live in Bank 1** — select with **[PTN] + [TRIG 1–16]**, audition with [PLAY].
- Each of the 4 parts of the 16 patterns has **16 scenes**; assign with **[SCENE A]/[SCENE B] + [TRIG 1–16]**, crossfader interpolates. **Scene 1 is empty by default** (so the fader's far-A end can be a clean/neutral state — useful template detail).
- Swap a sample on a machine: double-press [TRACK] to open the slot list → select + [YES]; or select a slot, press [RIGHT] to open the FILE BROWSER, pick from the audio pool, [YES] to load + [YES] again to assign.

## Install (official)
1. USB cable to computer; [PROJ] → SYSTEM → USB DISK MODE → [YES] (CF card mounts as drive).
2. Copy the unpacked **PRESET MKII** Set folder to the **root** of the CF card; unmount.
3. [PROJ] → SET CHANGE → [YES] → load Set **PRESET MKII** → then load Project **PRESET MKII** → [NO] to exit.

## Why it matters for this rig
This is the legitimate "factory patch list" — 16 patterns × 4 parts × 16 scenes of pre-built machine/FX/scene setups to reverse-engineer. Use it as a teardown reference (open a part, read its FX/scene assignments) rather than as final sounds; the rig's own material (fuzz wall, banjo, VG-800) replaces the stock samples via the slot-swap above. Note the **empty Scene 1** convention — worth copying into the user's own templates so the fader has a clean home position.
