# The Ultimate Octatrack FX Template / Performance Mixer — by EZBOT

The most fully-realized downloadable OT project template in circulation, by **EZBOT (Matthew Piecora)** — Seattle hardware artist/educator and Bitwig certified trainer (see artist-ezbot-templates-techno.md). Turns the Octatrack into a **DJ/performance FX mixer + dual looper** for processing external audio — exactly the Thru-machine-as-live-FX role this rig wants. Works on **MkI and MkII**. Now iterated as "Octatrack Performance Mixer 3.0.6."

## What it contains
- **64 effects** organized across **4 Parts**, themed: **Space / Build / Crush / Party**. Each Part = 16 scenes, so 4 Parts × 16 = 64 named FX states.
- Named scene-FX include: "Hallway Riser," "EZ Drummer," "Tape Stop," "Daft Punk Effect Alternate Version," etc. — i.e. each scene is a curated crossfader-ready FX preset.
- **2 independent loopers** (for FX captures / transitions).
- **2 compressors**, one per stereo input pair: **comp on inputs A/B, comp on inputs C/D**.
- **Sidechain pumping** available on every pattern (default groove = "4 on the floor").
- **Freeze-delay** effects pre-placed on **Track 1 and Track 5** (the two tracks that can host the input pairs).

## Concrete default values
- Tempo: **120 BPM**
- Pattern length: **64 steps (4 bars)**
- Recording starts at **step 1 of 64**
- All recorder captures at **unity gain**

## How to use it in this rig
This IS the "live-mangle-the-pedalboard" template, pre-built:
1. Patch pedalboard stereo print → OT inputs A/B (and a second source → C/D if wanted).
2. The template already has Thru/input handling + a compressor on A/B; pick a Part (Space for ambient drone, Crush for degraded walls).
3. Ride the crossfader between two scenes to morph the live signal through the 64 FX.
4. Use the two loopers to grab a fuzz wall and the freeze-delay (tracks 1/5) to sustain a fragment indefinitely.
