# Electro-Harmonix Effects Interface

**Summary** — Effects Interface "Hardware Plugin" (model **FXI**) — a USB-C, 24-bit, 2-in/2-out audio-interface bridge with three modes (Hardware Plugin / Pedalboard / Audio Interface). Runs plugins as pedals and re-amps via a DAW; can also reconcile the VG-800's line output with instrument-level pedals. ~125 mA draw; buffered bypass.
