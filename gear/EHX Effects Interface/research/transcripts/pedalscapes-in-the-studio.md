PedalScapes · EHX - Electro-Harmonix Effects Interface / Hardware Plugin - PedalScapes In The Studio · 2026-03-04

The most workflow-honest, real-use video so far — Matt (PedalScapes) runs the Effects Interface in Cubase, both as a re-amp/outboard insert AND as a plugins-as-pedals device, and is candid about the gotchas. (Auto-captions garble "Cubase"→"Qbase," "Nocturne"→"not turn," "BluGuru/Boudoir"→"Buddha"; cleaned where clear.)

UNBOXING / BUILD. From Bax Music; "fairly significantly chunky bit of kit." Faders feel "nice and resistant," soft-touch switch, stereo in and out, headphones, USB-C, and 9V power. EU power supply (with UK converter), USB-C cable, warranty, quick-start manual. Frames it: "it acts as a reamper so you can use your pedalboard in your DAW, but it also acts as a bring-your-effects-out-of-the-DAW onto your pedalboard. It works both ways."

SETUP WAS NOT PLUG-AND-PLAY. "I had some issues getting started. They were covered in the manual, but not in detail. I got in touch with EHX support and they came back to me the next day with a solution and it worked." Strong real-world caveat: budget setup time, the manual is thin on detail, but support resolved it quickly.

HARDWARE PLUGIN MODE (outboard in the DAW), Cubase. Uses Strymon Cloudburst, Halo, and Nocturne pedals; mainly the Halo and Nocturne. KEY OPERATIONAL POINTS:
- "In order to print it, you have to record it in REAL TIME." (No offline bounce — you must play it back and capture live.)
- REAL-TIME FADER ADJUSTMENT: touching the input/output fader on the hardware changes level live as the track plays.
- MONO/STEREO LIMITATION he hit: "I haven't found a way to do mono out / stereo in — it either seems to be stereo-to-stereo or mono-to-mono." So a mono source through stereo pedals to a stereo return isn't straightforward; you pick stereo↔stereo or mono↔mono. (He re-amped a part through a separate analog re-amper first because that pedal is stereo.)
- ONE INSTANCE AT A TIME: "you can only run one instance at a time because it's a physical hardware input and output." To move from the vocal track to the drums he must DISCONNECT on the vocal plugin (its light goes red), then go to the drums plugin and CONNECT there: Settings → Select Device → Audio (Stereo) → green light returns on the unit. Only one DAW plugin instance can hold the hardware connection.
- Uses the MIX to "drop the source in and out" for dub-style moves (100% wet when desired). Demonstrates Nocturne and Cloudburst on a vocal, then Halo + Nocturne (spring) on a drum bus, doing live dub mixing with tap tempo.

PEDALBOARD MODE (plugins-as-pedals), Cubase. Normal wet/dry/wet physical pedalboard: guitar → Mooer/Manis → Laney DI → compressor → record line-out → Strymon Halo → the Effects Interface → Cloudburst → simplifier into stereo → compressor → record. HOW IT'S WIRED IN THE DAW: "we've got TWO of these [plugin instances]. One is the input and one is the output. You select an input one, set it up as Pedalboard Input (stereo), then select another and set it up as Pedalboard Output (stereo)." Everything between them runs through the hardware.
- LATENCY/PHASE: even on "the lowest sample setting, the quickest setting," there's "a little bit of phase… a bit phasy," and a brief dropout when expanding the chain — "once I'm in this recording mode I'm asking too much of the sequencer." His fix: PUT THREE CLEAR/EMPTY PLUGIN SLOTS BETWEEN the input instance and the output instance, then load effects in between.
- Drives plugins-as-pedals: a "Transformer" ring-mod-style plugin (presets like Boomerang, pan-change), a multitap delay (filtered stereo percussion, "triplet dub verb… that's killer"), and an Octiva-type octaver — all now live on the physical board, into Cloudburst at the end. "This is opening up a whole new world."
- BYPASS NOTE: "there's a lot less latency with the pedal turned off, but it's still passing the signal through" — confirms buffered bypass still passes signal (and that the active loop is where latency lives).

VS THE RADIAL re-amper (he owns a Radial XTC stereo FX reamper). "It is a very different beast to the Radial XTC stereo FX reamper. They both have their place." The Radial is "more straightforward and instant" to set up and "all analog… no latency," but you can't use it as a plugin like this. The Effects Interface is "more quick and dirty once you've got it set up, but far more technical to set up — once it's set up it's dead easy: USB cable and done." Both are keepers; the EHX wins on DAW integration (use plugins as pedals, recallable, live mixing), the Radial wins on instant analog re-amping with zero latency. Verdict: "Bloody excellent… definitely going to feature in future scapes," and "it's making me better at Cubase."
