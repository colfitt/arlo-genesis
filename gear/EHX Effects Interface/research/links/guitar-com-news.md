EHX's own framing of the three modes, with the useful "vocals/synths/MIDI" detail.

THREE MODES:
- Hardware Plugin Mode — routes DAW audio through physical pedals and outboard gear. Per EHX, processes "vocals, synths and even MIDI instruments" via guitar pedals with simplified setup. (Confirms the re-amp use isn't guitar-only — synths and any DAW track qualify, which is exactly the Digitakt/OP-1/MPC-through-pedals use case.)
- Pedalboard Mode — reverses the signal flow: run instruments into a DAW, apply plugin processing, return the signal to amplifiers or pedal chains — "effectively converting software effects into stompbox-like tools for both studio and live applications."
- Audio Interface Mode — standard 2-in/2-out USB audio interface.

WORKFLOW FEATURES:
- Single USB-C connection "eliminates the need for separate converters or direct boxes."
- Real-time processing — "seamlessly integrates hardware and software effects" (real-time is inherent; offline render won't work because signal physically leaves to analog gear).
- Compact pedalboard-friendly footprint: stereo I/O with level-control sliders, footswitch, headphone output.
- Ships with power supply and USB-C cable.

EHX positioning quote: the device lets users "knock down the walls between physical and plugin effects."

Price: $359.
