The clearest concise statement of the three modes and the level-matching point.

THREE MODES (MusicRadar's framing):
- Hardware Plugin Mode — functions as an external effects loop connected via USB-C, routed THROUGH an associated DAW plugin in the insert chain. (You insert the EHX plugin on a track; the loop out/back becomes that insert.)
- Pedalboard Mode — incorporates software effects into a hardware chain: "audio enters the DAW through an INPUT plugin, gets processed with software effects, then returns to an amp or pedalboard." (Confirms the two-plugin-instance model: an input instance and an output instance, plugins in between.)
- Traditional Interface Mode — standard audio interface for recording/monitoring via its ins and outs.

LEVEL MATCHING (the key spec for this rig's VG-800↔board problem): inputs and outputs operate at "pedalboard-friendly levels," ELIMINATING THE NEED FOR ADDITIONAL CONVERTERS OR DI BOXES. This is the "no re-amp box needed" claim — the OUTPUT jacks are voiced to hand the board instrument level directly.

OTHER OPERATIONAL POINTS:
- Stereo / dual-mono: stereo inputs/outputs WITH DUAL-MONO capability for simultaneous dual routing (two independent mono paths at once).
- Dedicated headphone output with independent volume control.
- Level adjustment available via BOTH the hardware sliders AND the software UI (recallable in the session).

SPECS/PRICE: 2-in/2-out USB-C interface; $359; stompbox form with footswitch bypass.

EHX statement: "Effects Interface handles your signal and delivers it with care between your hardware and software via a simple USB-C connection."
