# Chroma Console firmware page (official)

Only ONE version is published on the page as the current release — no historical changelog is posted:

- **v1.04 (May 5, 2025)** — "Optimizes Preset Browser with faster preset load time, introduces Midi CCs for module bypass / engage, plus bug fixes and stability improvements."

Update method: firmware is delivered **over USB-C** via the Hologram firmware web tool (linked from the Support page). The unit ships with the USB-C cable and power supply needed to update.

## QC / honesty flags
- **No public version-by-version changelog.** Hologram lists only the *current* version on this page; v1.00–v1.03 changelogs are NOT published here. Any "old-vs-new feature differences" claim beyond "v1.04 added module-bypass MIDI CCs + faster preset loading" is **unverified** from the official page. (Community threads on Elektronauts/ModWiggler reference earlier behavior but do not constitute an official changelog.)
- The headline community-relevant addition in v1.04 is the **MIDI CCs for per-module bypass/engage** — this is the firmware feature that makes scene-based per-module switching from a MIDI brain (Digitakt/Morningstar/H90) practical.
