# AltWire (Derek Oswald) — Chroma Console review: one concrete chain-order recipe

This review is mostly aesthetic ("melting in the sun… floating underwater"), but it contains ONE concrete
module-order recipe worth keeping. It is a chain order, NOT per-knob values — capture accordingly.

## The one concrete chain-order recipe (verbatim)
> "Feed a beat through **Reels and Movement**, then once you're comfortable with the weirdness, take it through
> **Fuzz and Filter**."

Reading it as a Chroma module chain on a rhythmic/loop source:
- **DIFFUSION = REELS** (worn evolving tape echo) → **MOVEMENT = any** (vibrato/phaser/tremolo wobble) →
  **CHARACTER = FUZZ** → **TEXTURE = FILTER.**
- i.e. degrade-the-echo first, modulate it, then drive + filter the result. A "destroy a drum/banjo loop"
  progression — build the weirdness gradually rather than all four at once.

## Note on depth
No knob positions are given. This is a sequencing/exploration recipe, not a dial-in. Useful as a *chain template*
for the rig's loop-mangling (e.g. a Blooper banjo loop), but the AMOUNT/TILT/RATE/TIME values are left to taste.
The richer concrete material in this research wave is the wolfewithane "why buy" page (named chains + control-
position ranges) — see `wolfewithane-why-buy-chroma-console-chain-recipes.md`.
