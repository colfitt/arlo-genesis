# Darkside — Dave Harrington: Chroma Console + Microcosm in an improvised-ambient rig

Strong on-aesthetic, in-genre user, and a direct parallel to this rig's two-Hologram + Eventide
layout. Darkside (Nicolas Jaar + Dave Harrington) is improvised, textural, drone/psych-leaning.

## What's confirmed
- Harrington's main board ("three-section orchestra of stomps," ~26 pedals, one guitar) carries
  **Hologram Chroma Console AND Hologram Microcosm together**, alongside an **Eventide H90**,
  Eventide PitchFactor, Chase Bliss Habit / Thermae / Brothers AM, Walrus Monument/Slöer,
  Mu-Tron Micro-Tron IV, HexeFX reVOLVERs, etc. A second hip-level board adds a **2nd Microcosm**,
  EHX 45000 multitrack loopers, ZVEX Mastotron, EHX Cathedral.

> **Direct rig parallel:** Chroma + Microcosm + H90 on one improvising-ambient board mirrors this
> rig's Microcosm (Middle) + Chroma + H90 (End) chain. Harrington is real-world proof the
> Chroma/Microcosm/H90 trio is a coherent textural front end — granular looping (Microcosm) +
> tape/lo-fi morphing (Chroma) + algorithmic reverb/pitch (H90).

## Honest coverage note
- **No knob-level Chroma settings are documented.** The written Rig Rundown lists the gear and frames
  the board as a "three-section orchestra" but explicitly punts technique to the video
  ("tune in to the full Rundown to get the proper scoop"). The companion video
  (https://www.youtube.com/watch?v=DNl8oPs1sbs) would carry any spoken detail; not transcribed here.
- What's instructive is the **architecture, not a recipe**: stacked loopers + granular + tape-morph +
  reverb, played improvisationally — the same "build evolving stereo texture, then reverb over it"
  move the dossier prescribes for Chroma → H90.
