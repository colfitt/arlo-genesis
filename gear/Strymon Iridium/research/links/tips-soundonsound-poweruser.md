# Power-user tips & hidden behaviors (studio/SOS angle)

**MONO / STEREO / SUM rear input selector — what each does:**
- **MONO (TS):** standard guitar/instrument in.
- **STEREO (TRS):** a true stereo input — SOS notes this "should appeal to anyone working in the studio and synth/keyboard players." A stereo synth or stereo FX feed goes in over a TRS cable.
- **SUM:** collapses a stereo TRS source to mono before the engine.

**Stereo-IR depth trick (notable):** in Impulse Manager you can load a *different* IR into LEFT vs RIGHT. SOS: "it can have a huge effect on the tone, changing the same amp from warm and punchy to bright and cutting." Two mismatched cab IRs across L/R widen the stereo image and let you blend two voicings — directly useful for the rig's drone/wall aesthetic (e.g. a dark doom cab L + a brighter cab R for a wide bloom).

**Room:** combines a ~250 ms real-room IR with a synthetic-reverb tail for extended decay + modulation. Sizes small/medium/large via hold-ON + ROOM knob. SOS caution: "works best when used sparingly; it can sound a touch boxy if used to excess." For ambient pads keep ROOM modest and let downstream reverbs (H90 / Big Sky / Dark Star) do the long tails.

**Headphone out:** front 1/8" mini-jack for latency-free monitoring; shares main stereo voicing, LEVEL controls it. Good silent-practice + tracking-monitor path.

**Expression:** EXP defaults to volume (pre-amp); can be set to morph multiple parameters at once between two knob snapshots. MIDI needs the adapter cable; up to 300 presets.

**IR format:** Impulse Manager loads 24-bit/96 kHz only — convert other-rate packs first.
