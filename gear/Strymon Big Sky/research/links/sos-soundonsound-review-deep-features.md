# Big Sky — deep technical/feature notes (SOS review)

## Architecture standouts
- **All-analog dry path stays intact even when effect is engaged** + a **saveable-per-patch dry/wet mix** — SOS calls the programmable analog dry/wet "a world first in a reverb pedal." **50/50 mix = 3:00 position** on the Mix knob.
- **No menu diving by design.** Front knobs = Decay, Pre-Delay, Mix, Tone, Mod. The VALUE encoder menu holds **6 parameters common to all presets + 2–6 machine-specific parameters**, and you can assign **any two menu functions to the PARAM knobs per preset**.

## Machine character notes
- **Shimmer** — "so good it warrants the price of the unit on its own." Independent pitch-shifters tunable **octave-down → two-octaves-up**; "Eno-esque."
- **Cloud** — "ethereal, abstract reverb suitable for sound design."
- **Bloom / Swell** — variable fade-in to the reverb; **reversible for backwards pedal-steel effects**.
- **Magneto** — multi-head tape echo; **Pre-Delay = feedback**, adjustable head spacing.
- **Nonlinear** — backwards shapes **Swoosh / Reverse / Ramp** = "mind-bending results."
- **Reflections** — left/right + front/back placement of a source in a stereo reverb space; studio-focused, not live-oriented.

## Quirks called out
- **Bypass LED goes fully dark when bypassed** — you can't tell which preset is loaded once bypassed; you have to remember it. (SOS hoped Strymon would add a blinking-LED indicator.) A real live gotcha.
- **300 mA @ 9V**, incompatible with standard Boss-style supplies — needs a dedicated isolated 300 mA+ slot.
- **Cab Filter** switch = speaker emulation for direct-to-PA / interface; OFF when going into an amp/tape chain.
- (At review time MIDI-clock sync for Magneto tap "had yet to be implemented" — it shipped in later firmware; see firmware history file.)
