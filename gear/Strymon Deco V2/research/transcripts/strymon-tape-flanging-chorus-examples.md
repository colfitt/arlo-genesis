## Named patch recipes

**Thick Shake** — The lag deck is heavily wobbled with a very short lag time to create a phasey experience. The **Bounce** setting creates a thicker effect.
> Recipe: short Lag Time, high Wobble, Bounce.

**Flange Filtered** — A distorted signal is manually flanged for static filter effects by varying the Lag Time with the **Wobble turned down**.
> Recipe: distorted input, Wobble = 0, sweep Lag Time by hand = a movable/parkable comb-filter EQ. (This is the "static notch on a fuzz wall" technique.)

**Off-Center** — Turning Blend to **just the lag deck** creates a pitch-bent signal with heavy Wobble. Blending in the reference deck creates a wild tape flange.
> Recipe: Blend full CW (lag only) + heavy Wobble = pitch-bent vibrato; back off Blend toward 50/50 = wild flange.

**Off-Center (Bounce)** — Bounce mode creates a stereo panning effect as the lag deck wobbles around zero lag.
> Recipe: Bounce + Wobble at minimum Lag Time = auto-panning stereo movement.

**Vintage Doubles** — Maximum Wobble pairs with a longer Lag Time to create a pitchy double-tracked tape chorus.
> Recipe: long-ish Lag Time + max Wobble = pitchy vintage double-track chorus.

**Harmonics** — A wide double-tracked tape chorus is achieved with a moderate Lag setting and some Wobble with the **Bounce** setting. Turning up the **Saturation** adds rich harmonics.
> Recipe: moderate Lag + some Wobble + Bounce + Saturation up = wide rich chorus.

**Space Blues** — A short Lag Time and mild Wobble create a tape flange that brushes the through-zero point but doesn't go through. **Invert** mode creates a signature tonality.
> Recipe: short Lag (brushing but not crossing zero) + mild Wobble + Invert.

**Rocking Chair** — A doubling tape chorus using a moderate Lag Time, wide Wobble, and a **lower Blend** for a milder effect.
> Recipe: moderate Lag + wide Wobble + Blend reduced.

**Lagging in Stereo** — Wide Stereo Mode splits the reference and lag decks to L and R channels for an expansive stereo field. Tape Saturation adds mild overdriven harmonics.
> Recipe: Wide Stereo ON + a touch of Saturation.

**Subtle** — At minimum Lag Time, the lag deck passes through zero as it's slowed by adding some Wobble. (Variant: "Reducing the Blend makes the effect less intense.")
> Recipe: min Lag + a little Wobble (deck creeps through zero); pull Blend back to soften.

**Tape Flange** — The press-and-hold **Auto-Flange** mans the controls to transition seamlessly from any setting into a guided journey through zero time.
> Recipe: hold the Doubletracker footswitch for a one-shot studio-style through-zero sweep.

**Saturated Burst** — Tape Saturation adds fullness and overdriven harmonics; increasing the Wobble creates another level of richness.
> Recipe: Saturation up for body/drive + Wobble up for richness.
