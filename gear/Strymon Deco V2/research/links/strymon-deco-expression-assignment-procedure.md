Captured to nail down the EXACT mechanic for assigning an expression pedal to a single knob — the most-recommended live trick for the Deco (sweep Lag Time = walk flange→chorus→slapback→echo under one foot) needs this setup step, which the other captured sources reference but don't spell out.

## Assign EXP to control any one knob (the heel/toe range trick)
1. Unplug power to Deco.
2. Press and hold the **Tape Saturation (Saturation Bypass) footswitch** as you plug power back in.
3. While still holding the bypass switch, **turn the knob you want to control with the expression pedal.**
4. "**The maximum setting you turn to on the knob will set the maximum setting for the toe position on the expression pedal.**"

Behavior: the **knob's current panel position = heel** of the EXP travel; the **position you turn UP to during setup = toe** of the EXP travel. So the EXP sweeps the assigned parameter between those two endpoints — you choose both ends. (On V2 the EXP/MIDI jack must be set to Expression mode, which is the factory default per the Rev D manual.)

## Rig use
- **EXP → Lag Time:** set heel = flange region (~9:00, short lag) and toe = slapback/echo (~2-3:00). One rock of the foot walks the whole tape arc on a sustained banjo/baritone chord = the Hammock "perform the Deco" move without a second pair of hands.
- **EXP → Saturation:** set heel = clean glue, toe = pushed/compressed wall — a swell-into-drive under the foot (pairs with the Pete Celi "ride the drive" philosophy when you don't want to ride guitar volume).
- **EXP → Blend:** heel = dry-favored (subtle double), toe = 50/50 (max flange) or full-CW (lag-only vibrato) — fades the doubletracker intensity in/out live.
