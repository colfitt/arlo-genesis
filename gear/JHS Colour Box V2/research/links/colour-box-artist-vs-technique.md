# Artists: separating "named the Colour Box pedal" from "used the Neve technique"

The dossier flagged it could NOT verify a marquee artist naming the V2 specifically. This file resolves that with honest tiers.

## What JHS's marketing actually claims (read it carefully)
JHS's V2 page wording, verbatim: **"This unit can be heard on dozens of recordings from U2, Wilco, The National, Spoon, Collective Soul, Muse, the War on Drugs, and Phantogram amongst others."** (Other JHS/retailer copy extends the list with Beck, Mac DeMarco, St. Vincent, Foo Fighters.)
- This is a "heard on recordings" claim, NOT a per-artist endorsement claim, and it conflates the **Neve direct-in TECHNIQUE** (which all these artists' records use) with the **pedal**. Treat the bulk list as *technique*, not verified pedal users. JHS even says elsewhere "bands like the Beatles, Spoon, Wilco, and Steely Dan had used this *technique* for years" — confirming the slippage.

## VERIFIED — artists who actually NAMED / are shown using the pedal
These are real, sourced attributions that go beyond the marketing list:

1. **Mac DeMarco — Colour Box (V1).** In a **2019 Reverb.com** pedalboard interview/demo he explicitly praises it and says, on tap: **"If you wanna get nasty, you get the Colour Box."** He uses it to emulate a driven Neve preamp live. (Ground Guitar, Equipboard both document this; it's V1, not V2.) This is the strongest genuine "artist named the pedal" data point.

2. **Black Pumas — Colour Box V2 (×2!).** In Premier Guitar's **Black Pumas Rig Rundown**: **Eric Burton** (singer/guitarist) runs a **Colour Box V2** on his board (tech Bryan Wilkinson walks through it, ~52:14), and bassist **Brendan Bond** runs a **Colour Box V2** on his bass board (~44:58), describing using it humorously "as a simple overdrive." These are V2-specific, video-verified uses — they answer the dossier's open question: yes, touring artists are visibly using the V2.

## Honest framing for the Usage Guide
- The Colour Box recreates a **documented studio technique** (Beatles/Zeppelin/Motown/Steely Dan direct-in) — that history is solid.
- The big "U2, Wilco, The National…" list = **technique users**, not verified pedal owners; cite it as such.
- **Verified pedal-namers:** Mac DeMarco (V1, "get nasty"), Eric Burton + Brendan Bond of Black Pumas (V2, on-camera). Use these instead of the marketing roster when you want a real attribution.
