=== 1. THE TRIM↔FADER GAIN-STAGING GAME (the core discipline) ===
The strip is, in order: INPUT TRIM → CHANNEL FADER → MASTER VOLUME, with the 2-band shelving EQ in the path (per the Porta One architecture, the fader sits pre-EQ as a channel level). The builder's own framing of each control:
- Input Trim = "sets the preamplification of the input signal" — THIS is where the saturation/clipping lives.
- Channel Fader (the slider) = "controls the volume level of the second gain stage."
- Master Volume = "controls the final output level."
KEY DISCIPLINE: you can hit the SAME output loudness with very different grit by trading Trim against Fader/Master. High Trim + lower Fader/Master = lots of preamp clipping at a controlled level (gritty). Low Trim + high Fader/Master = clean DI warmth at the same loudness. Moving WHERE the clipping happens in the strip — not just "how loud" — is the whole game. Use Master to match the print level into the next stage (JHS 424 / Apollo) AFTER you've set the character with Trim+Fader.

=== 2. THE ONE WRITTEN SIGNATURE RECIPE (the Mk.gee dynamic move) ===
The canonical seller line, verbatim across listings + RockBoard: "With the input trim CRANKED and the channel fader turned UP it has a really great compressed feel and is super responsive to your pick attack, goes from CLEAN TO FILTH with just your attack."
→ Recipe: Input Trim = maxed, Channel Fader = high/up, EQ + Master to taste. The dynamics come from the PICKING HAND, not a knob — soft picking stays nearly clean, hard picking blooms into gooey grit. This is the only concrete settings recipe that exists in writing for the signature sound.
PLACEMENT SPLIT (builder, both directional, no numbers):
- END of chain, LOW Trim = "familiar DI tones" (clean console-ish print/glue). ← this rig's primary use.
- FRONT of chain, HIGH Trim = "a thick, trashy distortion" (amp-replacement). ← the Mk.gee-amp use (a real player, K_YEON_J, runs it "right after guitar line in" for exactly this).

=== 3. 9V vs 18V ===
- Builder spec: "9–18V for different levels of headroom." One owner called the 9–18V option "a huge plus."
- NO official tonal breakdown is published. Standard op-amp-headroom behavior (INFERRED, flagged): 18V = more headroom = cleaner/louder/tighter, breakup pushed higher up the Trim sweep, firmer low end → run it for clean-console-warmth print mode. 9V = lower headroom = earlier breakup, more compressed/"trashier" vintage character → run it for the gritty/Mk.gee mode. For a baritone/bass source, 18V keeps the bottom from flubbing when the preamp clips.

=== 4. THE B10k FADER TAPER (feel note, NOT a defect) ===
Current units ship with B10k (LINEAR) slide pots in place of A10k (log). Builder's exact words: "They are electrically the same, but the B taper will increase the volume faster, and actually allows finer control at the top." → Reframes the dossier's "feels more abrupt": the builder frames it as MORE fine control near the TOP of the fader's travel. Expect a fast volume rise low in the slot and finer adjustment near the top — set your "fader up" sweet spot in the upper third.

=== 5. THE MONO-PRINT QUESTION (the central practical gotcha) ===
The PORTA424 is MONO IN / MONO OUT (no XLR, no stereo). A real Tascam cassette channel IS mono, so collapsing to mono at the print is PERIOD-CORRECT, not a compromise. For a stereo-bus print stage you must deliberately sum L/R to mono into it (or feed one side / run it parallel) BEFORE this box — don't discover the collapse on stage. (No owner workaround for stereo exists because the pedal genuinely isn't stereo; the honest answer is "commit to mono here, or keep it out of the main stereo path and use it as a parallel color.")

=== 6. OTHER GOTCHAS ===
- NO published current draw (boutique, minimal docs) — meter it / give it an isolated slot; the op-amp strip likely sits ~20–50 mA (unverified).
- NO XLR/balanced out — that's the JHS 424's job; the PORTA424 hands off via 1/4" mono.
- MADE TO ORDER (~2-week lead per Reverb) — not an off-the-shelf grab; plan ahead if a unit ever needs replacing.
- The FADER is a slider, not a knob — set with hand/careful foot; builder insists "it doesn't sound as good with a knob instead of a fader" (the slider is deliberate, not just aesthetic).
