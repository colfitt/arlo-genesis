# Apollo x8 / all-UA synergy — the Del-Verb's in-the-box twins (concrete plug-in map)

The Del-Verb's quiet superpower for a UA owner: nearly every Del-Verb engine has a matching UAD plug-in you can run on the Apollo x8, so you can track a sound live through the pedal and re-create / extend it in the box. The modeled hardware lines up almost 1:1:

| Del-Verb engine | Modeled hardware | Matching UAD plug-in for the Apollo |
|---|---|---|
| **Hall 224** (reverb) | Lexicon 224 | **UAD Lexicon 224 Digital Reverb** — UA: "the exact algorithms and control processor code from the original hardware," nine classic algorithms, "famously lush reverb tail." Direct twin of the pedal's Hall. |
| **Plate 140** (reverb) | EMT 140, sourced from **The Plant, Sausalito CA** | **UAD EMT 140 Classic Plate Reverberator** — UA models "three uniquely different EMT 140s installed at The Plant Studios in Sausalito, California" — the SAME room the pedal's Plate 140 was sourced from. Tightest match in the box. |
| **Spring '65** (reverb) | '60s American (Fender-style) amp spring tank | UAD spring options: **AKG BX 20 Spring Reverb** (in the Apollo Hardware Essentials / Mix Tape bundle), or the spring built into the **Fender amp UADx plug-ins** / Galaxy Tape Echo's spring. No bit-identical "Fender '65 spring" plug-in, but the spring character is covered. |
| **Tape EP-III** (delay) | Maestro/Echoplex EP-style tape echo | **UAD EP-34 Tape Echo** (Echoplex EP-style, in UAD Ultimate/Complete bundles). Same tape-echo family. Also the UAFX **Orion Tape Echo** pedal uses the same Starlight EP-III algorithm if you want the hardware twin. |
| **Analog DMM** (delay) | EHX Deluxe Memory Man (BBD) | No exact UAD BBD plug-in; closest in-box is a third-party BBD or the **Galaxy Tape Echo** for warped analog echo flavor. |
| **Precision** (delay) | UA's own clean digital | UAD clean delays / DAW digital delay; trivially re-created in the box. |
| (bonus, dub) | Roland RE-201 Space Echo | **UAD Galaxy Tape Echo** — "the same sound that … shaped recordings by Pink Floyd and David Bowie, to seminal dub sides from King Tubby, Scientist and Lee 'Scratch' Perry." New: Tempo Sync, reverb/delay Pan, Tape Select. Great companion to the Del-Verb's Tape+Spring dub patch. |

## Workflow tips for the all-UA mobile / writing setup
1. **Track dry into the Apollo x8, monitor wet through the Del-Verb.** The pedal's analog dry-through + always-buffered output means you can run it as a real-time monitoring/printing ambience box while keeping a clean DI to re-amp/re-verb later in the box with the matching UAD plug-in. Print the dry, audition the same sonic family on playback with Lexicon 224 / EMT 140 — no tonal whiplash.
2. **Commit-or-recreate.** On a fly/writing rig: use the Del-Verb to commit a vibe instantly (no laptop). Back home on the Apollo, extend or automate it with the UAD twin (Hall 224 → Lexicon 224, Plate → EMT 140) — the decay/tone/pre-delay controls the pedal LACKS are all there in the plug-in. The plug-in is the Del-Verb's missing "deep edit" page.
3. **Kill-dry / 100% wet for parallel printing.** Turn Mix or Reverb fully CW (kill-dry) and feed the Del-Verb from an Apollo aux/send to print a 100%-wet stereo ambience stem alongside the dry — same trick you'd use with an outboard reverb on a console (see tips file).
4. **Stereo into the x8.** Del-Verb's two TS outs go to two Apollo line inputs for a true-stereo wet capture.

HONEST CAVEAT: the engines are the same MODELED hardware FAMILY and (for 224/EMT 140) the same source rooms, but UA states UAFX pedal algorithms are "recoded and tweaked specifically for guitar," so pedal and plug-in are voiced-cousins, not byte-identical. The synergy is real (coherent UA tone world, deep edit available in the box) but don't expect a perfect null between pedal and plug-in.
