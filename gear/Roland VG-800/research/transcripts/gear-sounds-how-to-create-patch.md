PATCH-BUILD WORKFLOW (the order he insists on):
1. INST FIRST. "Be sure you're genuinely happy with the instrument before you add any
   effects... once you select the instrument you're pretty much done." Press [INST] for the
   in-depth menu; set TYPE/variation. Each instrument exposes its own ALT TUNE options +
   per-string options (mostly per-string VOLUME/LEVEL).
2. Then the amp/channel section. EFFECTS screen shows the chain: Reverb slot, amp slots,
   noise suppressors right after the amps, the distortion (DS) on the channel.
3. CHANNEL = SINGLE vs DUAL. In SINGLE you pick Channel A (lower amp) or Channel B (top
   amp) via the [SELECT] knob — "two different amps with two different banks of effects, so
   you can have a dirty and a clean tone." (The DS distortion lives on one channel; switch
   channel to hear it engage/disengage.)
4. Turn each FX block on "for what it actually is" — phaser slot = use the phaser, etc.

CONCRETE PATCH #1 — "fake chorus via DUAL GUITAR" (no modulation pedal used):
- INST TYPE = DUAL GUITAR. Set a slightly DIFFERENT alt-tuning + per-string volume on
  guitar A vs guitar B. The detune between the two models = a chorus/doubling effect.
  "I get like a fake chorus effect, and with changing different tunings you're going to get
  all kinds of different combinations. This is to me the most useful function of the pedal —
  the alternate tunings." (Mechanism = the A:/B: STR DELAY + per-string PITCH/FINE offsets
  in the Parameter Guide.)

CONCRETE PATCH #2 — "epic GR-300 synth lead":
- INST = SYNTH, GR-300 voice ("sounds amazing and I love Pat Metheny"). Then on the channel
  he stacks DISTORTION + PHASER, then adds DELAY. Saved to an init memory slot 49-3.
