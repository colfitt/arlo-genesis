Most authoritative description of the VG-800's ON-BOARD FACTORY content and bank layout.

FACTORY BANK STRUCTURE (per mode):
- 20 banks of THREE factory presets each = 60 factory memory slots, all overwritable.
- A further 30 banks reserved for USER patches = 90 user slots.
- 50 banks total per mode (3 memories/bank). Guitar mode and Bass mode are separate
  banks (a startup routine reconfigures the unit for bass; menus change accordingly).
  This reconciles with Boss's "150 memories per mode" spec (the spec counts differently;
  SoS's "60 factory + 90 user" is the practical bank picture).

NAMED FACTORY PRESETS called out (real, copyable starting points):
- "24-String Guitar" — a doubled/jangly 12-string-style preset, "a really attractive
  'jangle' to it." Good template for the rig's alt-tuned/12-string-into-tape role.
- A "B-Bender" preset — "super-smooth," string-bender effect ACTIVATED BY HOLDING the
  CTL1 footswitch. (Shows the factory convention: CTL1 = the patch's performance toggle.)
- An Organ / Leslie preset — the modeled ORGAN "works best with the rotary speaker
  effects," and the rotary can be "set to ramp up and down in speed using a footswitch"
  (CTL1 by default). Template for the organ-drone role.

OTHER NOTED FACTORY STRENGTHS (good patch hunting ground): GR-300 synth lead ("brought
back many happy memories"), 12-string sims, distorted bass ("unbelievably fat"),
rotary-on-organ. Sitar model is the weak one.

EDITOR / SHARING: "free Boss Tone Studio software can make editing simpler... can also be
used for sound library organisation and sharing presets with the Boss user community."
(SoS doesn't name Tone Exchange vs Tone Central; that distinction is in the dedicated
links.)

Confidence: HIGH — primary written review, factory bank numbers + named presets quoted.
