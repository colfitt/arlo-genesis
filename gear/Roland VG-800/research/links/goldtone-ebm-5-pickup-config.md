# Gold Tone EBM-5 — pickup/body config (relevant to GK-5 mounting)

Grounding for the banjo end of the VG-800 rig — confirms the EBM-5 is a *solidbody electric* banjo, which is what makes GK-5 mounting physically plausible.

## Distilled facts
- **Solidbody electric 5-string banjo** — flat, guitar-like solid top rather than a tensioned drum head. This is the key enabler: a GK-5 needs a flat surface with strings passing over it at low, consistent clearance. A solidbody banjo gives you that; an acoustic open-back drum head would not.
- Stock pickups: **humbucker (bridge) + stacked humbucker (neck), 3-way selector, vol per pickup + master tone.** These feed the VG-800's GUITAR INPUT / NORMAL channel (the real magnetic blend), separate from the GK-5's divided signal.
- **5th string** is the short high drone string starting at the 5th fret with its own tuner peg on the side of the neck — the geometry wrinkle for any divided pickup: the GK element must still sit under it cleanly where it crosses the pickup zone, and it sounds a high G drone that the tracker must handle as a high, fast-decaying string.
- Upgrade pickups offered (Sliding Magnetic, Fishman Rare Earth) confirm the body routinely takes guitar-style magnetic pickups — supports DIY GK-5 mounting.

## Implication for calibration
- 5-string banjo → in GK SETTING treat as a guitar-mode source; the 5th drone string is the per-string SENS/DISTANCE problem child. Mount the GK-5 so all five strings (including the offset 5th) cross the element, set clearance with the included shims, then calibrate SENS per string with the 5th string getting individual attention.
