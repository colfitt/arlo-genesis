The GM-800 "Preset Scenes" liveset on Boss Tone Central — a PORTABLE reference for the
VG-800. The GM-800 shares the GK-5 / Serial-GK platform and the same synth-scene concept;
its scenes are organized by the same sound categories you'd author on the VG-800.

CONTENT (official description):
- "1100 unique GM-800 Preset Scenes" in one liveset.
- Organized by TAGS: Acoustic, Ambient, Electronic, Keys, Synth.
- Download = a .tsl liveset file → import in Boss Tone Studio → transfer to hardware.

WHY IT MATTERS FOR THE VG-800:
- The VG-800 is NOT on Boss Tone Central (its category 404s — confirmed in
  boss-tone-central-no-vg800.md), so this is the closest official curated bank on the shared
  platform. The Ambient / Synth / Keys tags map directly onto the rig's drone/pad needs.
- Individual scene NAMES are NOT exposed on the web page (it's a download portal, not a
  browsable DB) — the .tsl must be loaded in Tone Studio to read them. So this is a
  category-level reference, not a per-patch settings dump.
- The GM-800 is the synth-forward sibling; its GR-300/SOLO/CRYSTAL/ORGAN/FILTER-BASS scene
  values are the same engine families documented in parameter-guide-synth-engine-values.md.

Confidence: HIGH for existence + category structure (official portal); the per-scene values
require loading the .tsl. Listed as a portable cross-platform reference, flagged as such.
