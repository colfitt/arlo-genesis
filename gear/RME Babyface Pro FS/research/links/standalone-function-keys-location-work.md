Synthax Audio UK · "RME Babyface Pro Tutorial - Programmable Function Keys" · 2016-05-13

How the Babyface's hardware buttons and standalone mode turn it into a self-contained location/monitoring box — no computer required.

PROGRAMMABLE BUTTONS: A, B, and DIM. Three front-panel buttons are user-assignable via TotalMix FX. Assignable functions include:
- Phantom power (+48V) on/off — the key one for location mic work.
- Talkback — for engineer/talent comms while tracking.
- Mute / Solo, Fader Groups.
- Speaker B selection (A/B monitor switching), MONO fold-down.
- Mute FX, Effects ON/OFF.
- External input switching.
- Cue Phones mix → MAIN output.
- Recall a saved mixer snapshot or layout.

HOW TO ASSIGN: open TotalMix FX → Options → "ARC & Key Commands". Map your chosen function to button A, B, or DIM. The assignment is stored IN the unit, so it survives being unplugged from the computer — set it at the desk, then take the Babyface on location and the macro is still there.

FRONT-PANEL BASICS (standalone, no computer/iPad):
- Dedicated keys give instant access to all of the unit's basic functionality. SELECT steps through channels/links; the rotary encoder sets gain or output level depending on what's selected; DIM dims the monitor output; the SET/recall button stores and recalls a reference monitor level.
- The unit operates fully standalone — as a converter, a monitor controller, and (via its SteadyClock) a clock source — driving its stored TotalMix routing without any host connected.

WHY THIS MATTERS FOR THE RIG: bus-powered + button macros = a true grab-and-go box. Pre-store a "location" workspace at the studio (phantom on button A, talkback on B, a sensible monitor recall via SET), and the Babyface records the banjo or the pedalboard anywhere with full phantom/monitor control from the chassis alone — no laptop screen needed. For iPad control, toggle class-compliant mode from the hardware (hold DIM + SELECT while powering on); toggle back to PC mode by re-powering while holding DIM.
