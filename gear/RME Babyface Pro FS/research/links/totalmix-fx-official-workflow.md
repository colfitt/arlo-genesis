RME USA · official TotalMix FX feature/workflow page

RME's own description of TotalMix FX as a workflow tool — the authoritative feature map behind the "steep but powerful" reputation.

THE ARCHITECTURE (why the 3-row model exists):
- TotalMix FX is a DSP-based hardware mixer giving "fully independent routing and mixing of input and playback channels to all physical outputs."
- The genius/curse: you can create "as many independent submixes as output channels available" — RME calls it "unlimited routing and mixing." Each hardware output owns its own submix. That is exactly why you must SELECT an output first, then build its mix from the input/playback rows.

ROUTING MODES:
- Submix mode (default, easier): click a hardware output, then the input and playback faders shown are that output's mix.
- Free mode (advanced): route any source to any destination independently.

CONTROL ROOM SECTION (monitoring/cue):
- Up to 4 headphone mixes, plus Talkback signal when activated, speaker switching, cueing, and front-panel controls. This is what makes the Babyface a real monitor controller, not just a converter — "independent stereo submixes" feeding different phones/monitors.

EFFECTS (two kinds, handled differently):
- INSERT effects = EQ and Compressor, per channel (channel-by-channel; can be printed to the DAW via the "EQ + Dynamics for Record" setting).
- SEND effects = Reverb and Delay/Echo, on separate FX buses (aux-send/return style, monitor-only "comfort FX"). Result: "latency-free monitor mixes with EQ, Dynamics, Reverb and Delay."

LOOPBACK:
- Take the output of one software program and send it back as an INPUT for a different program; OR create a mix of input signals + playback signals (or both) and send it back as a stereo input for recording. (This is how you bounce a TotalMix-processed stem or capture a system-audio/mix recording.)

STATE MANAGEMENT (taming the complexity):
- Snapshots store the current mixer settings (faders, pan, mute/solo, EQ/dynamics) for instant recall — NOT layout.
- Groups: 4 storage places each for fader, mute, and solo groups.
- Workspaces: full configuration files — layout + mixes + window state — the master recall.
- Channel housekeeping: configurable mono/stereo channels, hide channels in the GUI / Mackie control / OSC, 2-row mode, zoom states, brightness control.

LEARNING CURVE NOTE: RME themselves ship a structured tutorial series (routing basics, snapshots, channel customization, headphone/monitor mixing, trim gains, channel groups, submix vs free modes) precisely because the power comes with a learning curve — the answer to "TotalMix is impenetrable" is to watch the beginner series and internalize the 3-row + per-output-submix model.
