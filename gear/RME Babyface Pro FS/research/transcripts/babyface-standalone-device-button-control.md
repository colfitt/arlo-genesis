RME USA - Synthax Distribution · RME Babyface PRO: full control with the device buttons. Monitoring control, stand alone operation. · 2025-01-11

Here's the Babyface Pro FS, and with this button layout it gives you the ability to use the unit WITHOUT actually needing the TotalMix FX window open. This is great for people who need mobility or to be on the go with the unit but may not have access to a laptop or tablet.

OUTPUT/MONITOR CONTROL. Operation is straightforward. To control the outputs for channels 1 and 2, select them with the OUTPUT button (already selected by default). To control phones volume, hit that button again — then use the rotary encoder to adjust the level. Same for the Optical output, and back to the main headphones. You can DIM the volume with the dim button (the LEDs show dim is engaged).

INPUT GAIN FROM THE FRONT PANEL. To select inputs — channels 1/2, 3/4, or Optical — use the input buttons. The difference with inputs: to adjust gain, select the input (e.g. channels 1/2), then hit SELECT — an LED appears at the bottom letting you increase/decrease gain with the encoder. Hit SELECT to step through channel 1, channel 2, or (one more press) BOTH channels linked for a stereo source. Same for channels 3/4. The Optical input has no gain, so there's no gain adjustment there.

THE SET BUTTON (volume recall). By default the SET button is your recall. Example: set the main output to a desired recall level, then hold SET for two seconds to store it. Later, raise or lower volume freely — then hold DIM and the volume jumps back to that recall state. (You can still use dim normally to dim.)

THE MIX BUTTON = on-board TotalMix. Think of the MIX (B) button as TotalMix on board the unit. To build a submix for an output (channels 1/2, headphones, or Optical), select the right INPUT channel, hit MIX, and the LED shows which channel you're on; raise the fader for that channel via the encoder. Each output keeps its OWN independent submix — switch from the 1/2 output's mix to the headphone output and the levels are completely different, exactly like TotalMix's per-output submix model. Same again for the Optical output.

CUSTOMIZABLE FUNCTION BUTTONS (the location-recording superpower). You can customize the button functionality on the unit. Example use cases for going on location: set up a PHANTOM-power button so you've got +48V for your mains when recording someone, or set up a TALKBACK button to communicate with the talent while they record. You can assign quick functionality to buttons A, B, or DIM. To set it up: open TotalMix → Options → "ARC & Key Commands" settings. A window lists all available options for the unit's buttons (or for a USB ARC if you have one). Example: assign button A = phantom power for channel 1, button B = phantom power for channel 2. Now everything else still works the same, but when ready to record, hit A and phantom engages (shown by the input LED); hit again to turn off. These customizations get preset and STORED INTO THE UNIT, so on location/remote you can recall them directly from the hardware.

PC MODE vs CLASS COMPLIANT (for iPad). You can toggle the unit between PC mode and class-compliant mode directly on the hardware, without TotalMix. Disconnect power, then press and hold DIM + SELECT while powering the unit back on — the status display tells you it's now in class-compliant mode (needed for iPad/iOS). To switch back, disconnect USB again, press and hold DIM, and it's back in PC mode.
