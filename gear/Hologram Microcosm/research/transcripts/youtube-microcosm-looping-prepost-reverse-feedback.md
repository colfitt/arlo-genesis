# Official looping deep-dive — pre/post-FX mental model + the "auto-feedback" gotcha

Hologram's own long-form looper tutorial. The conceptual framing here is the most useful part for actually USING the looper well.

## The pre-FX / post-FX analogy (verbatim-ish)
- **Effects BEFORE the loop (Pre-FX) = paintbrushes; the loop = a canvas.** "Each new addition to the loop can have a different tone, and by adjusting effect settings across multiple overdubs you can add a wide array of colors to your canvas." Use Pre-FX when you want each overdub layer to carry its own distinct effected tone.
- **Effects AFTER the loop (Post-FX) = an Instagram filter / a picture frame.** Effects "impart their particular character to the entire loop." And because many granular algorithms are constantly changing, "you can create situations where the loop keeps playing but never sounds the same way twice." Use Post-FX when you want the whole loop continuously re-granulated/evolving.
- The Microcosm is "uniquely suited" to demo this because you can **change the looper↔effects relationship immediately with the push of a button** (also CC 24).

## "Auto-feedback" — the looper's hidden decay behavior
- "Feedback in the Microcosm is a strange subject because it's not something the player can directly change." There is **no feedback/decay knob.**
- BUT "as you continue overdubbing new layers, you can reach points where older layers do start fading out" — i.e. **the oldest layers self-decay as you stack.** "Just jumping back and forth a couple of minutes will reveal some dramatic changes in the relative levels of older layers." Plan long ambient builds around this self-erasing behavior rather than fighting it.

## Rig takeaway
- For an **evolving granular bed**: route looper **Post-FX**, pick a constantly-changing granular engine (Haze/Tunnel/Warp) → the loop "never sounds the same twice."
- For **layered, distinctly-toned overdubs** (e.g. building harmony with Mosaic octaves): route **Pre-FX** so each pass can have its own engine/tone.
- Don't expect a feedback knob; let old layers self-fade, and use the Hold Sampler (separate buffer) for true infinite-sustain drones instead.
