## Navigation & knobs
- The big selector knob cycles all presets: Micro Loop, Glitch, Granules, Multi-Delay, then the User bank. Each effect (Mosaic, Sequence, Glide...) has four settings A/B/C/D. Rule of thumb: **C and D are usually the more extreme/nutty versions** of whatever effect you're on.
- **Reverse:** push DOWN on the selector knob — a little animation shows you're in reverse; push again to go forward.
- **Activity** = how "nutty"/how much activity. **Shape** = adjusts the waveform/attack of the effect (cycle the shapes for harsh-blocky vs. softer attack). **Filter** = fully right is no filter; turning left adds a low-pass, rounding off highs (good for going ambient). **Mix** = fully left is pure guitar, fully right is pure effect (kick it full-right to hear exactly what the pedal is doing). **Time** = does different things by context: in the looper it speeds up/slows down the loop; normally straight-up is 1:1, down toward quarter delays the response, up to 8x plays back at 8x speed. **Repeats** = how many times the effect repeats. **Space** = essentially a reverb; not a traditional reverb — for full ambient, dial Space up AND roll the Filter back so everything goes floaty. **Loop Level** = effect/loop loudness.
- **Secondary functions:** hold the Shift button and turn a knob (grey text); e.g. **fade time** = how long the effect keeps repeating after you switch the pedal off. LEDs go dim→bright to show what you're changing.
- Turning the pedal OFF shows which preset is currently selected (visual indication), so you're never surprised when you turn it back on.
- **Hold:** once the effect is doing its thing, hit Hold to lock it in (visual indication confirms).

## User presets (saving)
16 banks: four red, four yellow, four green, four blue. To save: dial in any preset and tweak it how you want → **push and hold the selector** (it starts blinking) → scroll to the User bank slot (e.g. yellow B) → **push and hold again to write** (shows a writing animation). Recall by scrolling to that slot — it lights up the original effect (e.g. Glide) it was saved from.

## Phrase Looper (step-by-step)
- Enter looper mode by pushing/holding the looper button.
- **Every layer you record carries the pedal effect on it** — layer and layer and it gets very busy.
- **Pre-FX option (the key trick):** lets you loop a DRY signal and then apply ONE set of effects to the entire loop. To enable: in looper mode, **push and hold the looper button → "Pre-FX" lights up → push the selector so Pre-FX lights → push the looper button again** to turn it on. (He notes this is confusing and cost him some frustrating attempts — to turn it back off, repeat and toggle the selector off.)
- Recording: hit record (turns **red** = recording) → play → hit stop (turns **green** = captured, looping) → hit again to record another layer (**yellow** = recording layer) → stop → repeat for more layers.
- **Erase:** push and hold the Hold button — it wipes the loop and is ready again.
- **Reverse:** reverses whatever you recorded; fun to reverse a loop then play over the top as your own backing track.

## Global Config
- Enter by pushing/holding the **Select + Loop buttons** together (the pedal goes colorful = global mode). Each knob selects a different config menu (reference the manual for which knob = which menu, since there's no readout).
- **Input mode (mono/stereo)** is on the **Activity knob:** one block = mono, two blocks = stereo. Turn the knob to the menu, set it, hit enter; push/hold Select+Loop again to save and exit. (He keeps his on stereo.)
- **Bypass mode** is on the **Space knob:** options are buffered (no trails), **buffered with trails** (second selection — effect continues for a set time after switch-off, length set by the fade-time secondary), and true bypass. He chose **buffered bypass + trails** because the pedal is so active that a hard cutoff sounds abrupt; trails cross-fade the tail into whatever you play next.
- Other global items: MIDI settings, etc. — only needed at first setup.
