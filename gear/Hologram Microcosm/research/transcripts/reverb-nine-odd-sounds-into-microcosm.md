# "9 Odd Sounds into the Hologram Microcosm" (transcript)

Source: https://www.youtube.com/watch?v=-WTCWLztwQY
YouTube · Reverb / "Nine Odd Sounds into a Guitar Pedal" (host: Martin) · 2020-2021
Note: presenter discloses Hologram sent the pedal to demo, "under no obligation."

Useful because each odd source is paired with a *named engine + mode + the controlling
setting*, which is rare. Concrete engine→mode mappings (verbatim where quoted):

| Source fed in | Engine / mode | Setting / behavior described |
|---|---|---|
| (intro tone) | **GLIDE A** | "glides between the audio at half and normal speed," Shape set to **triangle**, "very slow tempo division" |
| Ride cymbal scrape | **STRUM D** | "cascades chains of notes at various onsets" |
| Penny whistle (looped) | **STRUM B** | found the **speed knob and pitched it down an octave** |
| Kalimba | **INTERRUPT B** | "really glitchy algorithm with various pitch-shifted samples" |
| Springs on a spring-reverb tank | **WARP D** | "overlaps layers of samples with filter sweeps" |
| Brushed snare drum (looper) | **HAZE A** | "short diffuse effect created from stretching overlapping samples"; also engaged **Reverse mode** and changed **playback speed** |
| Analog delay feedback (Boss DM-2) | **BLOCKS D** | "rearranges the incoming audio with pitch-shifting and bit-crushing" |
| Melodica | **SEQ C** (approx.) | "overlaps layers of samples with filter sweeps" |
| Tapping on a guitar | **ARP B** | waveform **Shape set to Square**, running at **4× the tempo** |
| Toy xylophone glissandos | (final demo) | glissando textures |

On-aesthetic takeaways for drone/shoegaze:
- **Glide A + triangle Shape + very slow subdivision** = slow seasick pitch-drift bed — directly
  usable on baritone Jazzmaster drones.
- **Haze A + Reverse + slowed playback** = diffuse smeared wash from a sustained source, the
  Board-2 "smear" sound.
- **Strum B/D** chains transient onsets (ride scrape, plucks) into cascading pointillistic
  runs — the banjo-as-lead onset-chaining the dossier flags. Pitch the speed knob down an
  octave for a darker/doom voicing.
- Verdict quote: "the Microcosm is basically art in pedal form… going to be a classic."
