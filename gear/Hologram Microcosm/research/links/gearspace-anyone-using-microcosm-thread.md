# Gearspace owner thread — qualitative use + the Microcosm Editor pointer

A real-owner discussion (the modwiggler/gearspace forums 403'd in earlier project research; reachable now via
browser). No control-by-control patches, but several useful first-hand notes:

- **Sources it shines on (verbatim, Seamus TM):** "loving it on **vocals and fuzzed-out guitars**." (Confirms the
  rig's fuzz-wall-in thesis from another owner.)
- **arp synced to tempo on guitar** rated "a very good time" (electricsound) — backs the Arp-on-clock recipe.
- **itswintertime:** uses it for "**pads, swells, cool textures, intros/outros**"; praises that **tap tempo +
  octave up/down keep it "in context"** vs other granular/pitch units that "sound like a robot getting hit with a
  hammer." Notes "great on board **reverb and chorus** sounds, that **I have MIDI presets to utilize**" — i.e. he
  recalls Microcosm slots over MIDI PC (matches on-disk PC #45–60 map) and leans on the reverb/pitch-mod back end.
- **guyaguy:** mostly runs it "with **drive and distortion**"; says the busy granular stuff doesn't want other FX
  stacked — argues for keeping minimal chains *into* it (aligns with the rig's "minimal chains into the centerpiece"
  philosophy).
- **Microcosm vs CB MOOD / Blooper (owner consensus):** Microcosm = deeper but less immediate than MOOD; one user
  bought the **Blooper instead** for size + more adventurous modifiers, calling the Microcosm "more oriented toward
  a polished, washed-out 'ambient' sound." (Echoes the on-disk Board-2-vs-Board-3 division.)
- **NEW LEAD — post #19 (Dan Grain, 30 Jul 2024):** "There is a **Microcosm Editor** available now." → the
  SonicFreaks Max for Live editor (see sonicfreaks-microcosm-editor-m4l.md). This is the lead that overturns the
  "no off-device preset storage" framing.
- **Latency note (railkiller2376):** as a Cubase insert, the wet signal "needs to evolve" — up to ~8 s before the
  granular bed is fully present. Useful gotcha for DAW/Apollo printing: leave pre-roll before capturing a Microcosm
  bed.

No numeric knob positions anywhere in the thread — consistent with the pedal's no-shareable-preset reality.
