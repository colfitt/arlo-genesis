# Dark Star Stereo — named "scene"/starter-recipe settings (retailer page)

The Boost Guitar Pedals product listing for the Dark Star Stereo Reverb publishes four NAMED starter recipes with control-by-control directions — the only source found in this expansion wave that gives named patches with concrete per-knob directions (OBNE itself publishes no preset chart). Retailer-authored, directional (not knob-clock values), so treat as community/retailer-grade rather than factory.

Verbatim recipes pulled from the page:

- **Pad Factory** — "Pitch voices — 1 down, 1 up; long Decay; subtle Crush; high Mix; wide Spread."
  → i.e. Pitch 1 = down / Pitch 2 = up (opposed octaves), Decay high, Crush just off-noon (sample-rate side), Mix toward full wet, Spread wide (full up = normal stereo).

- **Ambient Rescue** — "Engage Aux for infinite trails; Lag low; Multiply mid; Mix high."
  → Aux = Hold (infinite sustain/freeze), Lag low (short predelay), Multiply ~noon, Mix high. The "drop a wash and freeze it" rescue/swell move.

- **Stereo Mayhem** — "Pitch1 +2; Pitch2 –2; Spread maxed."
  → Pitch 1 = +2 oct, Pitch 2 = -2 oct (full 4-octave split across L/R), Spread maxed. The widest, most chaotic dual-pitch cluster.

- **Classic Verb** — "Cut Crush; set Filter flat; mid Decay/Mix for smooth shoegaze results."
  → Crush at noon (clean, no degradation), Filter at noon (flat/native gentle high-roll), Decay ~noon, Mix ~noon. The "clean-ish shoegaze pad" baseline; page ties it to Slowdive / Cocteau Twins shoegaze.

Page also restates the control set (Filter / Pitch 1 / Pitch 2 / Crush / Lag = pre-delay / Multiply = feedback / Decay / Volume / Mix / Stereo Spread) and the Aux options (Infinite Sustain / Close LPF / Set Pitches +1 OCT) — all already in the dossier.

Notes / caveats:
- Recipes are directional ("down/up", "high", "mid", "low", "flat", "maxed"), not numeric clock positions — translate against the dossier control map (Filter bipolar w/ closeable LPF below noon; Pitch noon = unity; Crush below noon = bitcrush / above = OD; Spread full-up = normal).
- "Stereo Mayhem" (Pitch1 +2 / Pitch2 -2) is a NEW concrete recipe — the dossier's stacked-octave entry was ±1 oct; this pushes both to the ±2 extremes.
- "Pad Factory" (Pitch 1 DOWN / Pitch 2 UP) reverses the dossier's "Pitch 1 up / Pitch 2 down" stacked-octave; either polarity works, both are split-octave pads.
- No author/date on the retailer page; one source only — confidence = community/retailer-grade, not factory.
