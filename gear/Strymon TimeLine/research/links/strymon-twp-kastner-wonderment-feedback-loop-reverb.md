## "Wonderment" — dTAPE machine + external reverb in the feedback loop
- TYPE: dTAPE
- FEEDBACK-LOOP INSERT (rear EXP/loop): Kastner — "I plugged my old MIDIVERB II rack unit into the feedback loop of the Strymon TimeLine. The dTAPE machine played so organically with the reverse reverb trails."
  → an external REVERSE-REVERB sits inside the delay feedback path, so every tape repeat is re-reverbed and grows backwards.
- INFINITE HOLD: "Being able to hold the infinite repeats is a great bonus for creating walls of sound in live and studio settings." (hold the active A/B footswitch to freeze.)

Rig fit (high): this is the rig's feedback-loop-insert recipe with a real artist behind it — swap the MIDIVERB II for any of the rig's reverbs/Valhalla, or a fuzz/filter, into RIGHT IN/OUT (rear switch set to insert), run dTAPE, and HOLD the footswitch for a backwards-reverb tape wall. Direct support for the "runaway grit / re-process every repeat" + "infinite-hold wall" notes already in the guide. Costs stereo (the right channel becomes the send/return).
