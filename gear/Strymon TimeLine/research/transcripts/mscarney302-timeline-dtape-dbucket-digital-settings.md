Demo of three personal presets, saved in low banks. Default tempo 120 BPM (changeable via tap tempo). Common organizing trick across all three banks: **the A side is a quarter note, the B side is a dotted-eighth** of the same delay — so you can swap quarter ↔ dotted-eighth within a single bank.

## Bank 0 — dTape (digital tape)
Personal preset. FILTER / GRIT / SPEED / DEPTH / MIX / REPEATS all set to his taste (everything except TIME is baked into the preset). Machine params: **Tape Speed = Normal**, Low End set partway, **Tap = quarter on A / dotted-eighth on B**, **Boost = 0**. Demoed clean on a Strat neck pickup, then with a Timmy in the mix for dirt (bridge pickup), then the dotted-eighth side clean and with dirt.

## Bank 1 — dBucket (analog)
His filter/grit/speed/depth knobs set; same A = quarter, B = dotted-eighth layout. Machine params: **Range = Double**, Tap = quarter (A), **Boost = 0**. Demoed clean Strat neck pickup, then bridge with a bit of dirt and the dotted-eighth side, then with the Timmy added.

## Bank 2 — Digital
A = quarter, B = dotted-eighth again. Machine params: **Smear** set, **High Pass = 80 (Hz)**, **Repeat Dynamics = off**, Tap = quarter (A), **Boost = 0**. Demoed clean, then with the Timmy on the bridge pickup, then the dotted-eighth side clean and with dirt.
