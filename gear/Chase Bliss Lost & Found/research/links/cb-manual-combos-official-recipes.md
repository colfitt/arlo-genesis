## Swap-based recipe building (p.34–35)
> "Want to run the resonator through the chorus? Okay! Flick up the L SWAP dip switch, and both sides of the pedal will now contain the modes from the Right Channel. Use one channel to select the Sympathetic Resonator and one to select the Ensemble Expander."

Two starter patterns CB calls out for SWAP:
- **MEGA EFFECTS** — "Use the same mode twice with different settings to create a bigger sound, access unique rhythms, and fine-tune the response across the frequency spectrum. Start out in parallel (L+R), it's easier to hear what's happening." → **Try first: Pitch Repeater**.
- **FRESH VS. FROZEN** — "Have one instance of a mode processing your input and the other frozen to create a background for you to play along with." → **Try first: Grain Tumbler** (HOLD a footswitch to freeze one instance, play live over it on the other).

## Routing / BLEND mechanics that make recipes work (p.32–33)
- **Parallel `L+R`** — "Both channels are independent and run side-by-side. The BLEND knob sets the volume balance between the two. Parallel is useful for clean and predictable sounds that you can set just right. For example, a hybrid modulation that combines chorus and phaser."
- **Series `L>R` / `L<R`** — "One channel runs into the other. The BLEND knob sets the mix of the second effect in the series. Series is useful for enhancing one effect with another... For example, using some reverb to soften and expand the synth."
- **SPILL** (hidden option) — "By default the second effect does not apply to your input in series, but we can change that with SPILL. This will pour your input signal into the second effect as well, so that it doesn't stand out like a squeaky clean little weirdo."

## GLUE — finishing compressor/saturator (p.38–39)
> "GLUE is a compressor / saturator combo. As you turn it up it gradually applies more and more compression until it starts to overload... You will find it especially useful in the parallel (L+R) routing, making two distinct effects feel like one... GLUE is off by default because we didn't want to be presumptuous, but you will probably be happier if you use it."
- **GLUE-alone trick:** "You can use GLUE on its own as a bonus effect... simply turn on either channel and set its MODIFY knob at 12:00." (MODIFY at noon = no effect, so the channel passes only GLUE.)

## Dip-switch persistence (important for recipe sharing)
> "The dip switch settings are saved with your presets." — so a saved preset reproduces a full recipe: modes, routing, SWAP, ramping, EQ, GLUE, MIX. Share a recipe by sharing all of: mode L / mode R / MODIFY positions (A/B + character) / ROUTING / BLEND / MIX / which dips (SWAP, SPREAD, UNSYNC, TRAILS, BANK) / GLUE amount / any armed ramp/bounce.

## Rig-relevant reads (drone/doom/shoegaze, banjo+baritone via VG-800 → Microcosm)
- **SHOEGAZE SPECIAL** (Slow-verb > Ensemble Expander) is the on-aesthetic starting point; feeds the Microcosm a swirling reverberant bed.
- **ORCHESTRAL RESONANCE** and **SYNTH TOWER** turn sustained banjo/baritone chords into layered synth/pipe drones (pitch-tracking is cleanest on slow, sustained input).
- **LIGHT SLEEPER** (Orchestral Swell > Gen Lite) is the most "recorded-wrong" of the factory combos — digital swell smeared through the mini-GenLoss.
- **GRAIN DOUBLER** and **MYSTERY MOVEMENT** are the texture-injection picks to send into Microcosm's grain mill (non-reverb, so they don't stack reverb with Dark Star / Etherealizer downstream).
