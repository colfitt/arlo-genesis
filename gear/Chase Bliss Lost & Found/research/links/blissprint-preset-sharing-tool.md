# Blissprint — visual settings/preset-sharing tool for Chase Bliss pedals

Surfaced in a fresh 2026-06 search for L+F recipes. Blissprint is a **visual utility for designing
and archiving Chase Bliss settings as shareable images** ("A tool for creating and sharing custom
presets for your Chase Bliss pedals" — creator). It is NOT a sound editor and NOT a recipe library.

## What it actually is
- Lets you lay out a pedal's knobs/dips graphically and **save an image copy of the setup** to keep
  for reference or post online (the way owners share a "settings shot" of a CB pedal). Has a community
  discussion section for per-pedal preset talk.
- **Status: "In development"**; creator soliciting feedback via a Google Form.

## Honest relevance to Lost + Found (thin)
- **No dedicated Lost + Found support is documented** on the page as of this check — it lists CB models
  but L+F is not called out. So it does NOT host any concrete L+F patches/recipes right now.
- For L+F specifically the real software tool remains the **SonicFreaks Max-for-Live / standalone
  editor** (see `maxforlive-sonicfreaks-editor.md`), which actually reads/writes the pedal over MIDI.
- Captured only to keep the tooling map complete and to note that the **community recipe/preset-share
  culture for L+F is still essentially nonexistent** (consistent with `cb-lost-found-coverage-state.md`):
  a generic image-sharing tool exists, but no curated L+F patch dump. **No new owner settings obtained.**
