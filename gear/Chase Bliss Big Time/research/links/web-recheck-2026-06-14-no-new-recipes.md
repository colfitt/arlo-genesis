# Big Time community-recipe re-check — still no numeric recipes (2026-06-14)

Second-pass web sweep done while building the patch list (~2 weeks after the 2026-06-03
research). Goal: find any NEW documented artist/community dial-settings (exact fader positions,
named owner patches) for the Big Time.

## Result: nothing new with concrete values
- The dedicated Elektronauts Big Time thread (`/t/chase-bliss-big-time-analog-digital-delay/249961`)
  is **DEAD** — `curl` returns **HTTP 404 on BOTH the HTML URL and the Discourse `.json` endpoint**
  (a 404 on `.json` means the topic is deleted or unlisted/private, not merely a stale link). A
  Wayback CDX snapshot of `/249961.rss` (2026-04-30, status 200) confirms it *did* exist but was only
  an **announcement/reaction** thread (OP linked the product page + two demo videos; replies were
  "Very cool!"-grade) — it was never a recipe thread. **Stop citing /249961 as a live source.** The
  live discussion is the CB megathread (t/31096), which had no settings shares as of last capture.
- gearnews / geargods / thegearforum / GuitarPedalX / manuals.plus results all just **re-describe
  the controls** (COLOR = preamp gain → saturation + limiter intensity; STATE = Digital/Compressed/
  Saturated/#!&%; LONG + high CLUSTER + FEEDBACK + DIFFUSE = ambient) — i.e. they restate the
  manual, no numeric fader recipes, no artist-tied patches.
- No artist has been documented using a Big Time with published settings (pedal is weeks old).

## Bottom line for the patch list
The recipe canon is unchanged from the 2026-06-03 pass: **the 10 factory presets** + the
**manual's STATE×TEXTURE / COLOR-FEEDBACK gain rules** + the **captured demo transcripts**
(Secret Weapons, Superbooth/Snyder, official Walkthrough, Harp Lady, ambienttrash). All
designed (🟣) patches in the list are first-principles builds from that verified control
behavior, marked designed — not attributed to any artist.
