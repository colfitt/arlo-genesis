The reference for the X-Amp reamp loop, demoed on an Apollo Twin (identical logic on the x8 with more line outs to spare). Steps:

STEP 1 — Route the dry DI track out:
- Bring up the dry track's level "until you are getting as much signal out of the track as possible without clipping or 'going into the red.'" (Hot, clean DI = best reamp signal-to-noise.)
- Output selection: "On the Apollo Twin outputs 1 and 2 will usually feed your studio monitors, so we will select 'LINE OUT 3'." Avoid routing the dry-send out the main monitor outs. (On the x8 you have line outs 3–8 free; keep the send output OFF the monitor bus.)
- Cable: from the interface line output "to the input of your Reamp device of choice, using a balanced TRS or XLR cable," then from the reamp box to the amp/pedals using "an unbalanced TS instrument cable." (The Radial reamp box does the line→instrument level + impedance conversion; an X-Amp is the active version.)

STEP 2 — Set up the return/record track:
- Create a new mono audio track, output assigned to the MAIN stereo output — NOT Line Out 3 — to prevent a feedback loop.
- Input set to an available mic/line input (example uses "MIC INPUT 1").
- Mic the amp, XLR into the interface mic input. (For this rig the reamped return is a stereo pair printing to two fresh inputs.)

STEP 3 — Recording / avoiding feedback:
- When the amp is live in your listening space: "turn down your monitors and switch to headphones while recording your Reamp tracks." This is the key anti-feedback / anti-bleed move.

Key takeaways for the rig: send the dry DI out a SPARE line output kept off the monitor bus; balanced TRS/XLR into the X-Amp; X-Amp converts to instrument level into the front of the boards; the reamped (stereo) return prints to new inputs; monitor on headphones while the loop is hot. Push the dry send as hot as possible without clipping for the cleanest reamp.
