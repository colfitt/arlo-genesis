> Official, AC50-specific. The single best walkthrough of the four FX engines with exact button combos. Cleaned to prose.

Intro framing: the MPC Sample's effects are **four engines**, able to play **up to 7 effects at one time**, with **60 creative effect types**.

## 1. Pad FX
- Access: press the **Pad FX** button. The screen shows **16 effect types** triggered by **pressure on the pads** — push harder = more effect.
- Each effect is edited with the three knobs below the screen.
- **Lock it to the beat:** hold the pad + hit **Latch**.
- Up to **4 Pad FX at once.** Range: lo-fi, filters, flanging, "color."

## 2. Flex Beat
- Access: **Shift + Pad FX** → pads turn **purple**.
- **15 effect patterns:** stutters, tape stops, gates, beat repeats.
- **Mix (wet/dry) knob** sets how much.
- **Quantize button ON** = the new pattern triggers at the start of your beat (bar-synced). **Quantize OFF** = triggers instantly.
- You can change how the effect plays: **loop around** vs **one-shot** (triggered once on pad hit).
- **Turn Flex Beat off:** press **Pad 1** (it's the empty/null effect).

## 3. Knob FX
- A single plugin applied across the whole beat (or selected pads). **28 plugin types:** bus compression, limiter, side-chain pumping, tape + vinyl emulations.
- **Vintage Emulator** lives here: models of the **MPC60, MPC3000, and SP1200** (and SP1200Ring — see the sound-test transcript).
- Three control knobs set parameters per plugin. Examples named: **Vinyl** = tone / crackle / pitch. **Tape Emulator** = noise / tape drift. Also low-pass filter, reverbs, flanger.
- **Per-pad targeting:** select a pad → only those samples get the Knob FX. Use the **top button to select all pads.**

### Resample a Knob FX into a sample (free up the engine)
- Stop the beat → go to **Sample** → **Shift + Resample** → select a pad. The loop is resampled with the effect (e.g. vinyl emulator) **embedded**. Then turn off the live effect and you can load a *different* Knob FX (e.g. low-pass filter) on top. "Resampling really lets you get the most out of Knob FX by embedding them into the samples."

## 4. Color Compressor (master)
- Access: **Shift + Pad 5** — the new master "color compressor," adds retro punch and glues the beat.
- Compare bypassed vs on to hear the punch/attack.
- **Hold Shift + push the inputs** to drive it harder — snaps the kick and snares, more drive.
