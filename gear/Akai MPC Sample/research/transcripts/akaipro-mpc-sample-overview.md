> The authoritative model-specific tour of the AC50, presented by Akai's Andy Mac. Every button combo below is confirmed for the MPC Sample itself (not a bigger MPC). Cleaned to prose; timestamps stripped.

## Hardware / I/O recap (from the box)
Runs completely standalone, no computer required. 2 GB RAM + 8 GB internal storage. Built-in rechargeable battery (~5 hrs), dedicated speaker, built-in mic, color screen. Rear panel: 1/8" headphone out, external clock sync out, MIDI in/out, 1/4" audio in/out with gain control, USB-C (USB audio + MIDI, plus connecting smart devices for sampling or triggering iOS apps), microSD slot. Ships with **over 1,600 samples and over 100 pro kits** (lo-fi hip-hop through electronic).

## Pads, fader, banks
- 16 RGB velocity- + pressure-sensitive pads. **8 pad banks → 128 sample slots per project.**
- **Legacy fader:** by default adjusts the selected pad's level (hit a pad, move fader). **Shift + Fader** opens the fader-assign list: volume, pan, tuning, filters, attack, and whole-kit volume.
- Note Repeat: hold a pad and it follows the time division.

## Sequencer
- Record + overdub; add MPC swing and quantize **in real time**.
- Up to **128 sequences**, chainable via **Song Mode** (hit Song Mode → insert patterns → build a song structure → export as an audio file).
- **Step Edit:** insert/delete notes on the timeline; move notes on/off the grid by selecting the note and moving the fader back and forth.

## 16 Levels (for basslines / chromatic play)
- Select a pad, hit **16 Levels** → that sound is spread/pitched across all 16 pads (play a bassline).
- **Shift (cycles the 16-Levels Type):** Velocity across the pads (good for hi-hats) → Filter across the pads (opens/closes as you move up the grid) → (Tune).

## Four FX engines (real-time)
Tape emulations, reverbs, delays, tape dropouts, dialed with the three knobs.
1. **Flex Beat** — sequence-based / DJ-style master effects; a **Mix (wet/dry)** knob sets the amount.
2. **Pad FX** — apply effects by **pressure on the pads**; run up to **4 simultaneously** (lo-fi filters → retro color). Select an effect, then **hold the pad + hit Latch** to lock it on.
3. **Knob FX** — **28 plugin types** (bus compression, reverbs, tape delays, vinyl + cassette emulations). Hitting pads applies the effect **only to those pads** (per-pad targeting).
4. **Color Compressor** (master glue) — **Shift + Pad 5**. Start bypassed; **hold Shift to drive the input** (snaps the kick/snare).

## Resampling (bake FX in)
- With FX active on a loop (e.g. vinyl emulator), **Shift + Resample → select a pad** → the loop is resampled to that new pad **with the effects rendered in**. Frees up the FX engine to add a *different* effect on top (e.g. flanger).

## Sample Recall (the "I forgot to record" save)
- If you played a great melody in chop mode and didn't record it: **Shift + Sample Recall** drops it straight onto a pad. Then use **Trim** → set end point → truncate → tune → play a beat over it.

## Sample-edit page order (press the edit button repeatedly to cycle pages)
The three knobs below the screen edit each page; **hold Shift + turn the encoder to zoom** the waveform for precision.
1. **Trim** — start/end points (knobs); Shift+encoder zooms.
2. **Mix** — volume, pan.
3. **Amp envelope** — attack, decay (Shift makes samples snap).
4. **Tune** — pitch the sample.
5. **Warp** — real-time time-stretch **and** re-pitch; everything follows tempo when you change it.
6. **Play mode** — mute groups; mono / poly.
7. **Filter** — **7 filter types** (high-pass, low-pass, "classic", etc.) per sample, with resonance + cutoff.

## Chop modes (confirmed list)
Press **Chop** to slice a sample across the pads; press Chop again to return to the original.
- **Transients** (peaks)
- **Regions: 4 / 8 / 16** (auto across that many pads)
- **Manual ("lazy chop")** — strike the pads to enter slice points in real time
Edit each slice's start/end with the encoder; Shift+encoder zooms for precise chops.

## Sampling from a phone / iOS app over USB-C
- Connect a smart device by USB-C → run an iOS synth app, play it on the pads over your beat in real time, then **Sample Record** it.
- Set the sample length to **Sequence** (instead of Free) so it only captures the length of the sequence. Hit the pad → it waits → press Start.
- Built-in mic works the same way (Sample Record → hit pad → in Sequence mode it grabs exactly one sequence length).
