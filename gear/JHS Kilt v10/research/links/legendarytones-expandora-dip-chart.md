# Bixonic Expandora EXP-2000R — the canonical DIP-switch mode chart (maps directly to Kilt G1/G2)

The clearest concrete source for HOW the Expandora's two DIP switches map to the four modes —
which lets us map the Kilt's G1/G2 toggles AND Thom Yorke's live DIP setting precisely.

## DIP SWITCH CHART (quoted/tabulated)
| Mode | D1 | D2 | Description |
|---|---|---|---|
| **Crunch** | Lo | Lo | Least gain; "gritty edge" rather than fully crunchy. |
| **Overdrive** | **Hi** | Lo | More gain than Crunch; "brighter" with "more body"; "more of a hump in the midrange" closer to a Tube Screamer; good for a "Stones-like rhythm sound." |
| **Distortion** | Lo | **Hi** | Darkest mode; "harder distortion" resembling MXR Distortion+; "chunky metal rhythm sound," especially when "boosting an overdriven tube amp." |
| **Forbidden** | **Hi** | **Hi** | Extreme fuzz; "raspy, squealing, spitty"; "chaos." |

## INSTRUMENT ("Ins") SWITCH
- Third DIP selects **guitar or bass** operation. (On the Kilt this is mapped to the **Low-Cut flat/cut** toggle: flat = the fuller/bass-friendly low end, which JHS explicitly says is "much beefier than the original Expandora.")

## KEY MAPPING TO THE KILT 10 (cross-referenced, not from this source alone)
- Expandora **D1 = Hi → Overdrive** ≙ Kilt **G1 up**.
- Expandora **D2 = Hi → Distortion** ≙ Kilt **G2 up**.
- Both Hi = Forbidden ≙ Kilt **both up**. Both Lo = Crunch ≙ Kilt **both down**.
- ⚠️ Naming caveat: GPX & JHS both confirm Kilt **G1↓/G2↑ = "Distortion"** mode (matches Expandora D1 Lo/D2 Hi). This is the mode Thom Yorke's DIP setting selects.

## REVIEWER BASELINE SETTING (quoted)
- Most modes tested with **Gain and Level (Volume) set past noon, Tone at noon.**

## RELEVANCE
This is the missing concrete bridge to translate Thom Yorke's documented live Expandora DIP setting
("D1 Lo, D2 Hi, Ins B") onto the Kilt 10: **G1 down · G2 up (Distortion mode) · Low-Cut flat (bass/Ins-B analogue).**
