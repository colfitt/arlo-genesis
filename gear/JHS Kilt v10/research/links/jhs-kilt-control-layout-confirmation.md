# Kilt control-direction confirmation (Volume / Gain / Tone sweep directions; toggle behavior)

Plain confirmation of which way each control runs — used to keep clock-face patch values consistent.
(The V1 "boost side"/"Order" toggle described in some retailer copy is the discontinued V1 only;
the Kilt 10 / V2 has NO boost knob — three knobs + three toggles only. Flagged to avoid confusion.)

## CONTROL DIRECTIONS (quoted/paraphrased from JHS + retailer copy)
- **Volume** — master output; left = less, right = more. The 10's added headroom means "you no longer have to turn your volume up past noon."
- **Gain** — amount of drive/distortion; left = less, right = more. First 25% now usable (transparent OD); last 25% = gated fuzz.
- **Tone** — brightness; left = darker, right = brighter (a single high-frequency tilt, NOT a Muff-style bilateral control).
- **G1 up** = more overdrive. **G2 up** = more fuzz. **Both up** = heaviest gated fuzz.
- **Low-Cut** — down = flat EQ (fuller lows); up = "cut" mode (lows reduced to cut the mix).
- Clipping selection is by **resistance values in the clipping section** (Expandora method), not diode-swap selectors.

## HONEST FLAG
- Some older retailer pages (e.g. Russo Music) describe a "Boost" knob + "Order" toggle — that is the
  **2015 V1 only** (large enclosure, independent boost). The Kilt **10** = re-voiced V2: three knobs
  (Volume/Gain/Tone) + three toggles (G1/G2/Low-Cut) + Red Remote jack. No boost section.
