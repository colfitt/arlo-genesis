# Dream-pop / shoegaze chorus context (Cocteau Twins angle, MBV caveat)

Context for using the CE-2W in the shoegaze/dream-pop lane. No precise CE-2W RATE/DEPTH dials in these sources — captured for the *attribution honesty* and the genre-fit reasoning, not for fabricated numbers.

## Concrete-ish points (distilled / quoted)
- **CE-1-type chorus is a documented shoegaze/dream-pop staple**, and the CE-2W is used "in setups designed to create dream-pop vibes reminiscent of Cocteau Twins" (Sweetwater inSync shoegaze guide, distilled).
- **Robin Guthrie (Cocteau Twins)** is "renowned for using modulation effects like chorus, phasers, and flangers to create lush and ethereal soundscapes" — chorus is central to that ethereal dream-pop wash (distilled). NOTE: no specific Guthrie CE-2/CE-1 model confirmation or knob settings were found — do NOT attribute a numeric CE-2W patch to Guthrie. Treat Guthrie as genre context, not a cited settings source.
- **Honesty caveat — Kevin Shields / My Bloody Valentine did NOT use chorus.** Shields, quoted directly: "there's no chorusing or anything like that." MBV's sound = fuzz + reverse reverb + loud tube amps + glide-guitar whammy, not modulation. Do not build an MBV-attributed chorus patch.

## Relevance to the rig
- The dream-pop "ethereal lush wash" lane = CE-1 Chorus mode, slow rate, deep depth, into heavy reverb (Dark Star / Microcosm downstream). This supports a *designed* shoegaze/dream-pop patch (🟣) — genre-fit, not artist-attributed.
- Do not credit MBV with chorus; the on-taste shoegaze chorus reference is Cocteau-Twins-style dream pop (genre), not MBV.
