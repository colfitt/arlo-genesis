# BOSS CE-2W "secret" mode — Pedal Science (notes only)

No captions available; distilled from the channel's video description and the documented technique.

Author's description (quoted): "Simple dummy jack trick to hear wet signal only."

## The trick (what it demonstrates)
- Because the CE-2W's stereo behavior is a hard wet/dry split — **OUTPUT A = chorus (wet) only, OUTPUT B = dry only** once both outputs are engaged — you can get a **100% wet chorus in mono** by inserting a dummy/unused jack (a cable to nothing, or a TS plug) into **OUTPUT B**. This activates the split so OUTPUT A carries only the modulated (wet) signal, while OUTPUT B (the dry leg) goes nowhere.
- Result: a fully-wet chorus/vibrato voice out of a single amp, without the dry signal mixed back in — much more dramatic/"underwater," and the only way to hear the pure modulated path in mono.
- Confirms the dossier's §4 stereo-routing note from the player side: plugging OUTPUT B is what flips the pedal from "chorus+dry mixed" to "wet-only on A / dry-only on B."

## Relevance to the rig
- In this rig OUTPUT B already feeds Deco R, so the split is always active and OUTPUT A (→ Deco L) is already wet-only. This video is useful confirmation of that asymmetry, and a reminder that **the A leg into the Deco is the pure modulated signal, the B leg is pure dry** — exactly the divergent stereo pair the dossier describes.
