# Frusciante tone with Boss CE-1 (stereo) — David VDH (notes only)

No captions available; distilled from search/description. This is a **CE-1** (original) demo, not the CE-2W, but it is the canonical reference for the Frusciante CE-1-chorus sound that the CE-2W's **CE-1 Chorus mode** is built to reproduce — useful as a target tone.

## What it shows
- The lush, rich Frusciante chorus wash run in **stereo through two amps** (per search results, a Marshall JCM 800 and a Marshall SV20C) — the classic CE-1 dry-on-one-side / wet-on-the-other spread.
- Demonstrates that the magic of this sound is (a) the deep, slightly dark CE-1 chorus voicing and (b) the wet/dry stereo split across two amps — both of which the CE-2W replicates (CE-1 Chorus mode + OUTPUT A wet / OUTPUT B dry).

## Settings note
- No on-screen knob values captured (no captions). Take the CE-1 chorus target and translate to the CE-2W using Boss's own studio numbers for the same voice: **Mode = CE-1 Chorus, RATE ~10 o'clock, DEPTH ~1–2 o'clock** (see boss-studio-applications.md), which is the dossier's "CE-1 lush / Frusciante-style" setting.
