# Where to get OP-1 Field patches (annotated, 1 line each)

## Libraries
- **op1.fun** — https://op1.fun — THE hub: 12,500+ free user patches (synth/sampler/drum) + packs + Mac one-click loader + in-browser Drum Patch Builder + Web MIDI audition.
- **Teenage Engineering official sound packs** — https://teenage.engineering/downloads/op-1/sound-packs — free first-party packs; canonical drag-drop install steps.
- **op1.center** — https://op1.center — curated drag-and-drop sample/sound packs (Classic/Genre/Create/VIP), Field-compatible.
- **SoundGhost** — https://soundghost.net/product-tag/op-1-field/ — paid Field packs (e.g. "Float", 100 ambient presets).
- **Patchstorage** — has an OP-1 section but thin; op1.fun is the real hub.
- **op1essentials.com / soundpacks.com** — free drum-sampler kit collections (130+ kits).

## Creator packs (named, concrete)
- **Cuckoo** — https://op1.fun/users/cuckoomusic + Patreon — "OP-6" (40 patches sampled from a DSI OB-6), "Opines Collection" (15 DR Wave chiptune patches), NES/8-bit sets.
- **Red Means Recording (Jeremy Blake)** — https://redmeansrecording.gumroad.com — "RMR OP-1 Pack Vol.1" (24 kits + 14 pitched + 124 synth + 34 sampler presets; ~7 not Field-compatible) and "Dreams" (Field ambient/pad pack).
- **rephazer** — https://rephazer.gumroad.com — OP-1 Field packs ("Field Cassette", "Field Modulation").

## Tools (read/build patches, dump real parameter values)
- **op1.fun Mac app** — github.com/dustMason/op1.fun.app — menubar one-click loader.
- **padenot/libop1** — `op1-drum` builds drum kits; `op1-dump` prints the proprietary JSON of any patch.
- **schollz/teoperator** (op1z.com) — split long audio into 24-slice drum kits in-browser.
- **blattm/op1tools** — Linux: label/backup/count user-synth patches.

## Artist/technique references
- **Bon Iver / Justin Vernon** — Sound On Sound "Inside Track: 22, A Million"; tingen.org/bon-iver — radio/vocal sampling, play-one-shot-chromatically.
- **Andreas Roman (CDM)** — cdm.link OP-1 Field album writeup — one-note polyphonic sampling, Studio→Vintage tape re-print, Chase Bliss send/return, Tombola-on-FM-bells.
- **Hainbach** — Perfect Circuit Field demo — degrade-before-sampling, compose-for-half-speed.
