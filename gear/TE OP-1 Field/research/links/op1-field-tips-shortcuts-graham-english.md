# OP-1 Field — concrete shortcuts & techniques (for loading/saving patches + tape moves)

Distilled actionable shortcuts. Most relevant here: **how to save/rename your own patches** and the **content/disk-mode import** chord, plus tape performance moves for the degrade aesthetic.

## Saving / renaming patches (your own presets)
- **Save a snapshot to a preset slot:** hold **`[1]`–`[8]` for ~5 seconds** (in synth or drum mode). That writes the current patch to that numbered slot.
- **Rename a preset:** hold **`[Shift] + [1]`–`[8]`**, then use the **Blue and Ochre** knobs to scroll characters.
- **Jump to a saved preset:** press **`[1]`–`[8]`** (per the dossier; `[Shift]+[T1]` opens the full browser).

## Importing patches over USB (matches official + op1.fun)
- Enter **Disk/Content mode: `[Shift] + [Com]` → `[T4]`**, then drag **AIFF (44.1 kHz)** files into `synth/` or `drum/`. `[T4]` again ejects.
- **`[Lift]` / `[Drop]`** move a sample or patch between engines (e.g. lift a tape slice, drop into the Synth Sampler) — the on-box copy/paste.

## Sampling
- **Pick input + record:** `[Shift] + [Mic]` to choose input source (mic / line-in / FM / USB), then `[Rec]`.
- Drum mode loads samples **up to ~20 s** (`[Drum]` → `[T1]`); use **Blue/Ochre** to set slice boundaries, **Orange** for per-slice playmode, plus **Group Choke** so overlapping hits cut each other (hi-hat behavior).

## Tape performance (the degrade/scrub moves)
- Define a loop: **`[1]` = Loop In, `[2]` = Loop Out, `[3]` toggles** the loop.
- **`[Shift] + [Play]` = reverse** playback.
- Performance keys while playing: **`[4]` Brake** (tape stop), **`[5]` Reverse**, **`[6]` Chop** — tempo-synced stutter/scrub. **Double-tap `[Stop]`** resets the playhead.

> Honest flag: exact key labels (e.g. which numbered key is brake vs chop) vary by firmware and between OP-1 and OP-1 Field; treat these as the right *gestures* and confirm against the on-device prompts / the official Field user guide. The save = hold-slot-5-seconds and rename = shift+slot behaviors are stable across the platform.
