# apalazzolo — "Ten H90 Ambient Pads/Soundscapes" (.lst90) — UltraSwell-based pad List (NEW)

A 10-Program ambient-pad List ("The SINGULARITY") built around Eventide's factory **UltraSwell**
(UltraTap) program. Individual Program names aren't enumerated on the page (they're inside the file),
but the author's global setup notes ARE concrete and reusable. Provenance = community.

## Author's concrete global setup (applies across the List)
- **Mode: stereo insert mode** (built to sit with an external pedal in the insert loop).
- Inspired by Eventide's factory **UltraSwell** program (UltraTap swell engine).
- **Expression pedal globally linked to the P HotKnob** (one exp pedal morphs every Program).
- **Aux Switch #4 = always a +5 dB boost** (consistent across the List).
- **Aux Switches #5 & #6 = per-program**, described in each Program's developer notes.
- **P Out Gain HotKnob recommended at +12 dB** for headphone monitoring.
- Voiced/tested on middle + bass-toggle positions of a Les Paul.
- Playing technique baked in: **"PLAY DISTINCT RESTS BETWEEN NOTES to reset the UltraTap gate"** —
  note length loosely sets swell time.

## Why it matters for this rig
This is the apalazzolo UltraTap-swell pad well referenced obliquely in the existing pad-recipes file,
now captured directly. The global template (exp → P HotKnob, Aux4 = +5 dB boost, stereo-insert mode)
is a ready-made control scheme to clone for the End-board's pad Programs, and the "distinct rests"
gate technique is the key to playable UltraTap swells over a drone. Load the .lst90 and dissect the
UltraTap (Taps/Length/Slurm/Chop-Swell) values per Program.
