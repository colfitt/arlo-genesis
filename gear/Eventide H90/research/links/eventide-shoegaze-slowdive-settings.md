# Eventide shoegaze / Slowdive-style settings (Space → H90, transfers directly)

These are community-dialed settings from offset/shoegaze players for the **Eventide Space** reverb.
Every algorithm named here (ModEchoVerb, Shimmer, Reverse Reverb, Hall) is in the H90's reverb
library, so the parameter starting points transfer to the H90 directly. This is the single most
on-aesthetic technique source for the rig's shoegaze/drone wall goal.

NOTE: I could not load the OffsetGuitars thread directly (HTTP 403); these values are from the search
engine's extraction of that thread and corroborating shoegaze discussions. Treat the exact numbers as
*community starting points, lightly verified* — dial by ear.

## ModEchoVerb — "shoegaze for days" washy wall (the headline patch)
Mod set to **chorus**. Reported starting values:
- Mix **33**, Decay **12.75 (s)**, Size **55**, Delay **330 ms**
- Low EQ **−2**, High EQ **−5** (rolls top off for the dark/washed shoegaze haze)
- X knob = EchoFdbk **47**, Y knob = ModRate **63**
- FxMix = FlangeMix **67**, Contour = EchoTone **1**
ModEchoVerb (modulation + echo + reverb in one) is the classic Slowdive-style "everything blurred
together" voice. On the H90 this is one Preset; pair Preset B with a long Hall or Blackhole for depth.

## Shimmer — subtle "makes everything bigger" octave wash
- Pitch **+1 octave AND −1 octave** simultaneously
- Decay **~4–5 s**, **Mix ~25%** (kept low/subtle on purpose)
- *"Subtle and makes things feel bigger, perfect for single-note stuff and cleaner passages."*
For the rig's heavier drone wall, push Mix higher and use Pitch/Pitch+verb Freeze (per the dossier's
Shimmer notes) to lock a chord into an infinite cathedral and play over it.

## Reverse Reverb — backwards-swell shoegaze
- *"The reverse algorithm will give you shoegaze for days."*
- Space factory presets "Fluttering" and "Ghosts" are good reverse starting points (*"take a while to
  dial in properly"*). On the H90 use the **Reverse Reverb** algorithm; pairs with a volume-swell
  attack for the backwards/blooming entrance shoegaze loves.

## General washy-reverb principle (from the same players)
The key to a shoegaze wash is **long decay + modulation + rolled-off highs + moderate mix** so the
reverb sits *under and around* the note rather than on top of it. Use **Hall with heavy modulation +
Freeze/Infinite** as an alternative Slowdive-wash recipe (per the same community threads).

## How this maps to the rig
The End-board H90 catches the already-fuzzed/smeared signal off Boards 1–2. A **ModEchoVerb (Preset A)
→ Blackhole or Hall (Preset B), Series** Program with the values above is a strong "shoegaze wall"
starting Program; add a HotSwitch → Freeze to lock it. Then the PORTA424/JHS 424 tape stage degrades
the wash — which is exactly the intended "recorded-wrong, beautiful" result.
