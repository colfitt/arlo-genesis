# "Studio Box List" — 63-Program artist-flavored factory-style List (.lst90), by rustydutch

The single best free starting-point List for the H90. rustydutch's stated purpose: introductory
programs + practical recording tools to learn the box while building a useful toolkit. Many programs
are **artist/song-inspired** voicings — useful as named reference points and dissectable starting
patches. It's a `.lst90` (up to 99 Programs); import via H90 Control → Playlist → Import List.

NOTE: this is a community-authored list, not an official Eventide factory bank — names like
"Comfortably Gilmour" are the uploader's descriptive titles, not licensed artist presets. The exact
per-program algorithm assignments are inside the file (not all transcribed on the page). Treat the
list below as the authoritative program-name index for this download.

## Why it matters for this rig
The on-aesthetic standouts (marked above) — Swells and Clouds, Airport Music, Pads and Crystals,
Swirling Clouds, If Shimmer is Your Thing, Frippertronics, Airy Nimbus, Dark Echoes, War For Drugs —
are exactly the drone/ambient/shoegaze pad/shimmer voicings the End-board H90 is there to do. Import
the whole List into a User List, then dissect those programs to see the actual Blackhole/Shimmer/
Crystals/UltraTap parameter values the author used as reference points.

Same uploader also has a "Strange and Curious – List" (30 programs: delay/glitch/grain/pitch/reverb,
"glitchy, distorted, haunting, fairytale, nightmare") — even more on-aesthetic for the degraded/
"recorded-wrong" goal. See the patchstorage-ambient-programs capture file.
