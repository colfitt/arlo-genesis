# Named FACTORY H90 Programs (from two pro reviews) — the factory-bank index

Eventide doesn't publish a flat factory-Program name list online (you browse Factory 1 on the pedal /
in H90 Control). These two reviews name actual factory Programs by name + describe their algorithm
makeup. No reviewer gave numeric parameter values — these are factory-PROGRAM NAMES with their
algorithm content, useful as a "go find this preset on the pedal and dissect it" index. Provenance =
factory (Eventide stock Programs, as identified by reviewers).

## From Premier Guitar (factory Programs, with algorithm makeup)
- **"hey floyd"** — weedwacker + "spacetime" algorithm's "outer limits" preset; emulates Gilmour's
  Big Muff + Electric Mistress (*The Wall*).
- **"fuzzy old bits"** — vintage rack delay model + spring reverb; '80s dotted-eighth delay over
  '60s surf/psych ambience.
- **"gentle arps"** — harmonic modulation + modulated echo + reverb.
- **"Floating in Space"** — TWO Blackhole algorithms at different settings (dual-Blackhole stack).
- **"Dream Sequence"** — Eventide H910 emulation + spacetime algorithm.

## From Tape Op (factory Programs)
- **"Clean Ambient"** — reverb + delay combination (reviewer's favorite "way beyond their pay grade").
- **"Celestial Chimes"** — polyphonic (kalimba-into-bells character).
- **"Floating Space"** — Blackhole algorithms (likely same/related to PG's "Floating in Space").
- **"JazzyBZ"** — Hammond B3 / Leslie rotary background sound when nothing is played.
- **"Tight Drive"** — gain-gated drive (only responds above a gain threshold).
- **"Robocat"** — intense aggressive effect ("electric cattle prod").

## Why it matters for this rig
**"Floating in Space" / "Floating Space"** (dual-Blackhole) and **"Clean Ambient"** are the
factory drone/ambient starting points to load and dissect for real Blackhole parameter values.
**"Celestial Chimes"** (Polyphony/polyphonic) is the banjo-as-lead chordal-engine reference.
These are confirmed factory Programs — open them in H90 Control to read the actual values, since
the reviews give names + algorithm makeup but no knobs.
