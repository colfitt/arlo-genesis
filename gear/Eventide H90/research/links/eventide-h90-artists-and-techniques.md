# H90 / Eventide-lineage artists & techniques (honestly sourced)

Honest framing up front: the **H90 is new (2022)**, so its *own* artist mythology is still thin — most
documented "how an artist uses it" material is **H9 / Space / TimeFactor**, the H90's direct lineage.
Every algorithm those players name is in the H90, so technique transfers; I flag below which box each
artist actually used. Where a "user" is only a gear-database listing with no detail, I say so.

  Equipboard/rig tours). Use: polyphonic pitch + ambient textures over a clean amp platform. Detail

  aesthetic for drone, but a flagship H90 endorser.)

  `transcripts/ariel-posen-favorite-eventide-h9-sounds.md`.

## Transferable TECHNIQUES (the actually-useful part)
1. **The "two H9s" idea → one H90.** Pete Thorn famously ran **two H9s** so he could use *two Eventide
   effects at once* — one for modulation/pitch, one for ambient (reverb/delay), in stereo into a dual-
   amp setup. *"He liked having two H9's so he could use more than one Eventide effect at a time."*
   The H90's **two-Presets-per-Program** architecture is literally this in one box: put pitch/mod on
   Preset A and reverb/delay on Preset B. (Sources: Premier Guitar / Vintage King Pete Thorn demos.)
2. **Name presets by function for live recall** (Ariel Posen): name each Program/Preset after what it
   does so you switch fast on stage, and keep **failover copies** of your other pedals' core sounds.
3. **Shimmer subtle vs. wall** (shoegaze community): low Mix (~25%) octave-up+down Shimmer to "make
   things bigger"; crank Mix + Freeze for the drone cathedral. See `eventide-shoegaze-slowdive-settings.md`.
4. **ModEchoVerb / Reverse Reverb = "shoegaze for days"** (Slowdive-style wash). Concrete values in the
   shoegaze-settings capture file.
5. **Freeze/Infinite a chord, solo over it** — the universal Eventide ambient move; map it to a
   HotSwitch. See `h90-ambient-pad-settings-recipes.md` and the parallel agent's freeze-drone file.
6. **UltraTap-gate swells** for bowed/cello-like drones (apalazzolo's Patchstorage method): play
   distinct rests, wide vibrato, reverb after to smooth the release.
