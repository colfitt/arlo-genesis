# H90 Control — librarian & list-building workflow (the fast way)

H90 Control (Mac/PC, **USB only** — Bluetooth not implemented) is the editor/librarian: manage Programs/Presets/Lists, all settings (MIDI, tap tempo), firmware updates, and import/export User Lists.

## The hierarchy (so the librarian makes sense)
- **Preset** = one algorithm's saved sound (A or B slot).
- **Program** = a pair of two Presets — what you see on the pedal.
- **List (Playlist)** = up to 99 Programs, your performance order. Factory Lists are read-only; make **User Lists** to reorder.

## Fast preset-finding (the real time-savers)
- In the **Edit** tab, the **Preset Library** at the bottom lets you pick the slot (A or B); on the right, **filter by effect type and/or algorithm type** to narrow the list. Click a preset name to load it into the selected slot.
- Two filters that cut the list instantly:
  - **FAVORITE** — flag your go-to presets; this filter shows just that short list.
  - **USER PRESETS** — shows only presets YOU created (e.g. your ~20), instead of the whole factory library.
- Use these instead of scrolling hundreds of factory presets.

## Building / reordering Lists
- The **Program Library** is where you organize Programs into User Lists.
- **Create / delete / rename Lists**: icons at the bottom-left of the Edit screen (works like the H9).
- **Reorder**: click-and-drag a Program onto a Playlist slot to overwrite the Program in that location. You can also **swap Preset A and B** within a Program.
- **Import/Export User Lists** (and import H9 lists — see the H9-import file; not while in Dual Mode, v1.7.6+).

## Workflow gotchas (from bugs file)
- Windows app crashes (save often); typed numeric values sometimes don't stick; the fine-adjust "Control" key doesn't slow range edits. Patchstorage imports have triggered crashes.
- "The great thing about the H90 is you don't need a control app to use it" (Gearspace) — but the app is far faster for list-building and bulk organization.

## Rig relevance
For a one-player live rig the recommended discipline (per reviewers): **don't browse factory presets on stage.** Build a small **User List** of named Programs (e.g. "Blackhole drone," "Shimmer wall," "Delay+Verb," "Banjo harm"), flag them FAVORITE, order them for the set, and recall by footswitch/PC. Use USER PRESETS / FAVORITE filters to keep the working set tiny. Store the List export with the rig backup.
