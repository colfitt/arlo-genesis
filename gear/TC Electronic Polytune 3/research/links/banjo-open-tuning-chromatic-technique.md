SoundHalo · "How to Tune a Banjo" guide

Concrete technique for tuning a 5-string banjo with an electronic tuner — the technique that applies to running the EBM-5 through the Polytune in chromatic/needle (NOT polyphonic) mode.

The open-G layout (gDGBD):
- Notes G, B, D arranged 5th-to-1st as g–D–G–B–D; all strings open ring a G-major chord.

The re-entrant 5th string (why poly mode can't map a banjo):
- The short 5th string is "a high G, one octave above the 3rd string."
- It "starts at the 5th fret, not the headstock" and "has its own tuning peg mounted partway up the neck."
- This re-entrant high-g sits ABOVE the strings around it in pitch, breaking the low-to-high string order a polyphonic tuner expects from a 6-string guitar. A poly display has nowhere to put it. → tune the banjo one string at a time.

Why chromatic mode:
- Set the tuner to "chromatic mode (displays all 12 notes, not just guitar or bass)." A chromatic tuner just names whatever pitch it hears, so it handles open-G, the high-g 5th string, and any alt/open tuning without trying to force strings into a guitar layout.

Method — string by string, ascending:
- "Pluck it and watch the tuner display," then "turn the tuning peg slowly until the tuner reads [target note]."
- Creep UP to pitch: "if you overshoot and go sharp, loosen the string below the target note and come back up. Tuning up to pitch seats the string more securely around the peg and helps it stay in tune longer." (Same "approach from below" rule Chris Berrow gives for guitar in the transcripts.)

Rig application: on the Polytune 3, use Guitar/Chromatic-Needle (or Strobe for the drone) and tune the banjo single-string, never strum-all. Banjo's fast attack/decay also reads more stably in strobe. And because the EBM-5's GK signal runs into the VG-800 ahead of the tuner, tune off a CLEAN VG-800 patch so the pitch detector sees a clean fundamental.
