JustinGuitar Community · forum thread "Polytune 3 and built in buffer - where to place it in my pedal chain?"

The most useful real-world thread on the Polytune 3 buffer + fuzz question. Concrete field reports, not theory.

Placement consensus — tuner goes FIRST:
- roger_holland: "The only pedal which should ALWAYS go before the wah pedal is a tuner pedal. This is because tuner pedals require the cleanest signal for the most accurate results." (Tune off the cleanest possible signal; a buffer/wah/dirt ahead of the tuner muddies pitch detection.)

The buffer is switchable — you don't have to use it:
- AndyTake2: "The TC3 can be switched to True Bypass instead of a buffer if you wish."

The key buffer-in-front-of-fuzz field test (user Kamkor, A/B'd it directly):
- "I tried it placed before a fuzz face type pedals with buffer disabled and enabled:
  - buffer disabled: I can clean up the fuzz nicely by reducing the volume on the guitar.
  - buffer enabled: The tone is harsher. The clean-up of the fuzz is not working well."
- His recommendation: "when using TC3 mini with a fuzz face type pedal, I'd either disable the buffer or put it after the fuzz face type pedal."

Takeaway: even for a SILICON fuzz-face-type pedal, the buffer can flatten the guitar-volume cleanup interaction and harshen the tone — so true-bypass is the safer default in front of any Fuzz-Face-style dirt. (Note: this is the cleanup/feel interaction, distinct from the germanium-fuzz oscillation problem; a Big-Muff-derivative is less sensitive than a Fuzz Face to either.) The practical rule that emerges: if you run fuzz, either leave the buffer OFF or place the tuner after the fuzz.

Cross-reference for this rig: matches the dossier's "leave it True Bypass" call. CB Clean immediately downstream already buffers the board, and keeping the tuner true-bypass preserves the guitar-volume cleanup into the silicon 108 / Big-Muff-style Hizumitas.

Related threads (corroborating, accessed via search summaries — buffer preference is setup-dependent):
- One user switched a standalone BonaFide for a Polytune 3 Mini buffer and "couldn't tell a difference."
- Another preferred the Polytune 3 ON buffered, reporting "a big difference in the fuzz sound" the other way — so it cuts both ways depending on the specific fuzz. Plug in and A/B is the universal advice.
