TC Electronic · Polytune 3 Quick Start Guide (official, retailer mirror)

The authoritative operating-workflow reference, straight from TC's quick-start diagram. Confirms the controls and the DIP buffer table verbatim.

Controls (panel callouts):
- USB — "Connects to your computer for firmware updates." (firmware only, not audio)
- Display Mode button — "Choose between Needle or Strobe tuning modes for guitar (G) or bass (B)." Cycles G/Needle, G/Strobe, B/Needle, B/Strobe.
- Display (the LED screen) — "Pick one string for single-string tuning, or strum all of the strings for polyphonic tuning." This is the MonoPoly auto-detection: pluck one string = single-note needle/strobe; strum all = polyphonic readout. No mode button to press — it just switches based on what you play.
- Tuning Mode button — "Select standard tuning or one of several dropped tuning or capo modes." (Standard, all-flat Eb/D/Db/C/B, capo 1st-7th.)
- Footswitch — "Turns the tuner on and off. Press and hold the footswitch to toggle between standard and drop-D tuning." Engaging the tuner mutes the output for silent tuning.
- Ambilight Sensor — "Detects the strength of the surrounding light and automatically adjusts the display brightness." (Bright on stage, dim in a dark room.)
- Power Input — "Connect a 9 V > 100 mA power supply." (Confirms the >100 mA draw — budget a real isolated slot.)
- Power Output — "Supply power to other daisy-chained pedals."

Buffer DIP switches (official table, verbatim) — "Switching between True Bypass and Buffered Bypass. Unscrew the back plate of the pedal and look for the two small DIP-switches in the upper left corner."

| Switch configuration | Mode |
| Both switches set to the left | True Bypass |
| Top switch set to the right, lower switch set to the left | Buffered Bypass |
| Both switches set to the right | Buffered Bypass, Display Always On |

Workflow notes that follow from this:
- Everyday: G/Needle, pluck string by string (fastest, most reliable). Strum-all poly for a quick whole-guitar glance between songs.
- Strobe (G/Strobe or B/Strobe): the precision mode for intonation and for getting a drone/sustained string dead-on — slows as you approach pitch, stops when centered. Drift looks like rotating segments.
- Drop-D: hold the footswitch to toggle. Alt flat tunings + capo: step with the Tuning Mode button; settings persist across power cycles.
- "Buffered Bypass, Display Always On" keeps the readout live while you play (signal passes) — a glanceable tuner without muting; the configuration the Equipboard reviewer runs.

Spec-confusion note: TC's full bundled manual still carries the older "±0.1 cent" strobe figure inherited from earlier Polytune manuals, but the Polytune 3's marketed and reviewer-confirmed strobe accuracy is ±0.02 cent (chromatic 0.5 cent). Both are extremely accurate; 0.02 is the current correct figure for the 3.
