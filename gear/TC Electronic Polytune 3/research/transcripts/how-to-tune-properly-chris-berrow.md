How To Play Guitar with Chris Berrow · How to use a guitar tuner PROPERLY (PolyTune 3 | 4K Full Guitar Lesson | Chris Berrow) · 2025-12-26

A technique-focused lesson on getting strings genuinely in tune with the PolyTune 3 — not a feature tour. Chris bought the PolyTune 3 because Noel Gallagher (Oasis) used it, but says the tricks here work with any tuner, particularly this one.

Placement: put the tuner as the very first thing the guitar plugs into. He uses a gold-tip input cable just so he always knows which lead is the very start of the chain; the instrument goes through the tuner, then to a wireless pedal, then to the board.

Core technique — the things most beginners get wrong:

- Use a pick to tune, not fingers, for a consistent attack.
- Start at the bottom (low E) and work up.
- MUTE the other strings while tuning a single string. If you don't, sympathetic vibrations from the other strings (he demonstrates the high E ringing in sympathy) make the tuner read the target string as sharper than it actually is. Muting reveals the string is actually a touch sharper than the unmuted reading suggested. He uses his thumb to damp the lower strings as he goes.
- Pluck dynamics matter: the HARDER you pluck, the sharper the tuner reads — without you having touched the tuning peg at all. Hit it hard and the tuner thinks it's way up high. So pluck once, consistently, not aggressively.

The "tune up from below" method (the heart of the lesson):

- Deliberately drop the string BELOW the target note, keep the other strings muted, pluck once, and wind the tuning peg UP to the note.
- Coming up to the note from below holds tune better than overshooting and dropping down to it.
- When you let go of the peg / your fretting-hand pressure, the pitch universally drops slightly. So nudge it a fraction sharp to compensate for that settle, then let go — and it lands bang on.
- Repeat string by string (E, A, D, etc.).

Let it ring: actually let the note ring for a long time while reading. If the pitch is DROPPING OFF as it decays, you've got a slightly flat string that you might otherwise call "in tune" — the decay reveals it.

After tuning all strings, go BACK and recheck the bottom strings. Adjusting tension on the upper strings can pull the lower ones out, especially if the guitar started badly out of tune or you've moved between tunings (e.g. Eb up to E). Check the bottom two or three strings again to confirm they've stayed put.

Quick version: drop below, come up to the note, let go with the fretting hand, see if it stays — good enough. Closes by inviting other PolyTune 2 / quirk tips from viewers.
