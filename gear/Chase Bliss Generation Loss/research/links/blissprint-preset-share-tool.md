# Blissprint — the Chase Bliss preset-SHARE tool (and why it confirms "no Gen Loss recipe culture")

**This is the one genuinely new find of the expand pass.** Blissprint is a free
interactive **visual pedal-parameter editor** built specifically for Chase Bliss
pedals (by community dev "Synlyfe"). You pick your pedal from a built-in list,
match the on-screen graphic to your physical knobs / latches / DIP switches, name
the preset, and **save it as an IMAGE you can archive or share online.** It is the
de-facto "where Chase Bliss presets would live if a share culture existed" — the
prior research flagged the CB Discord + Elektronauts as the only (gated) venues
and **missed Blissprint entirely**; this corrects that.

- App + web build: https://synlyfe.itch.io/blissprint (Windows build + browser version, v1.4+).
- Community board: https://synlyfe.itch.io/blissprint/community → "Share your Presets".
- Generation Loss MkII is a supported pedal in the editor (full knob + DIP layout).

## THE KEY RESULT (honest, load-bearing)
There **IS** a dedicated **"Generation Loss MKII — Share your Presets"** thread
(https://itch.io/t/3725219/generation-loss-mkii, started by Synlyfe 2024-05-04).
As of this fetch it has **96 views and ZERO shared presets** — the opening post
("Share your presets for Generation Loss MKII below!") has **no replies, no
images, no settings.** So even on the *purpose-built Chase-Bliss preset-share
platform*, the Gen Loss MkII recipe shelf is **empty.** This independently
confirms the prior pass's "no community recipe-share culture; demo-video-driven"
conclusion — now verified at the one site designed to hold exactly those recipes.

(For contrast, other Blissprint pedals DO get shared prints — e.g. an "Onward"
thread and a Billy-Strings-tied "Wombtone" preset exist for other CB units —
so the emptiness is Gen-Loss-specific, not a dead platform.)

## So: no NEW concrete Gen Loss patch settings came from here.
Blissprint is captured as the correct **venue** to (a) monitor for future shared
prints and (b) self-author + archive your own Gen Loss patches as shareable
images. The actual copyable patches for this pedal remain the manual's starting
points + CB's dip-technique recipes already on disk.
