# Ricky Tinez — Clean as a live limiter + pseudo-sidechain on a drum/synth bus

Ricky Tinez (electronic/house producer) runs Clean **end-of-chain on a drum machine / live mix**, not on guitar. Most instructive non-guitar use captured. His core philosophy: he plays live with a **limiter, not a compressor** — "I just love how hard it squashes."

## Recipe — "live limiter + pseudo-sidechain" (his main live preset), step by step (paraphrased from his walkthrough):
1. **DRY all the way down** (no parallel blend for the limiter sound; though he notes parallel "is actually kind of nice" if you want it).
2. Set WET for output volume; set **SENSITIVITY** so the left LED "is moving" — "if our sensitivity is too low, not much is happening."
3. **DYNAMICS into limiting** (noon→3:00 zone): "I just start limiting."
4. **Fast ATTACK + fast RELEASE:** "I like a fast attack 'cause I want the sound to really squash down, but I want a fast release as well."
5. The trick: "I like my kick to be a bit too hot to push the entire signal down, **acting as a pseudo side-chain effect**." He prefers this to the real SIDECHAIN input — "I like the sounds fighting against one another." (Limiter ducking the whole bus on kick hits = the classic dance-music pump, without using the sidechain jack.)
6. **ATTACK to taste** to let kick transients pop through while the limiter still grabs the tail.
7. **Watch the low end:** "the low end really overwhelms compressor circuits" — when adding a loud bass, cut some lows (use the EQ knob CW / envelope balance) so the bass doesn't trigger excess ducking.
8. Save to a preset: hold the right footswitch ~3 s, then add left.

## Dynamics map he confirms by ear
"From basically 7 o'clock to noon it's compression, from noon to about 3 is limiting, and then from 3 to time-to-clock-out is that sag stuff." Even in higher compression it stays effective.

## Physics / Sag as performance texture
With **PHYSICS all the way right** (twitchy), bringing in a clap "causes these sounds to kind of start breaking up... falling apart." Adding Sag on top: "see, that's a little too much" — flavor, not default.

## EQ as a live build-up tool
EQ knob noon = manual mode flat; sweep for "DJ-style high-shelf / low-pass" filter moves during a build-up/drop. Wishlist: he wanted a way to fully mute/kill the signal with it.

## Takeaways for this rig
- Clean is genuinely strong as a **drum-machine/synth-bus limiter** — directly applicable to the VG-800 modeled output and any Elektron/MPC bus.
- "Pseudo-sidechain" (hot kick + fast limiting) is a cheap, musical pump effect without patching the 1/8" sidechain.
- Cut lows before the limiter when bass is hot — same Envelope Balance logic the dossier recommends for the baritone Jazzmaster.
