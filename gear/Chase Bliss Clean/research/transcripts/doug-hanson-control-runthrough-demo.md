# Control Run-through — Doug Hanson (concise, accurate quick reference)

A tight, accurate control narration (then sound-only demo). Overlaps the official video manual but is a fast, no-fluff reference for what every control and dip does. Distilled verbatim-faithfully.

- **Dynamics** = ratio; increases toward noon, then enters limiting, eventually **Sag** (emulates an overloaded tube, makes your highest peaks lower than your quiet playing).
- **Sensitivity** = threshold (the level that triggers the compressor).
- **Attack** = how quickly the compressor turns on, 0.5 ms to 300 ms.
- **Release toggle** = three release times incl. a user-adjustable setting; 50 ms to 1.5 s.
- **EQ — three modes (Mode toggle):**
  - Middle = fixed EQ; cuts highs left of noon, cuts lows above noon.
  - Right = EQ that modulates when you play; EQ knob sets the modulated center point, Attack sets the speed.
  - Left = EQ that shifts when you play (like an envelope filter): when the signal passes threshold the EQ moves toward full frequency; below threshold it shifts back to the EQ-knob setting. Attack and Release set the sweep speed.
- **Physics toggle** = adds instability/fluctuations (dropouts, level variation). Middle = stable, left = subtle movement, right = unstable.
- **Wet / Dry (blue knobs)** = parallel-compression blend, mix wet and dry as you wish.
- **AUX (Clean's swell) switch** = swell effect, two modes: Dynamic (auto-swell) and Manual (trigger with the AUX footswitch). Holding the **bypass** footswitch maxes out the Sag effect.
- **Dip switches:** DUSTY = turns Clean into an overdrive · SWELL AUX = manually trigger the swell · MOTION = modulates the amount of compression (Dynamics = depth, Attack = rate) · NOISE GATE = filters noise amplified by compression · SIDECHAIN = external signal triggers compression/EQ/swells · LATCH = footswitches momentary → latching · SPREAD = adds stereo movement · MISO = splits a mono signal to stereo output · (green dip bank) = ramp Clean's parameters.
- **Hidden options (hold both footswitches):** Swell In / Swell Out timing · Gate Release / Gate Threshold · User Release (the Release toggle's middle value) · Envelope Balance (prevent low frequencies from triggering the compressor) · Envelope Type (three types) · Spread Routing (assign SPREAD to EQ, volume effects, or both).
- Stereo in/out; expression, CV, and MIDI control; two onboard presets.
