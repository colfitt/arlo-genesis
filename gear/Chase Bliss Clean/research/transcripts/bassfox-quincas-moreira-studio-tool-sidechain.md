# Way More Than a Compressor — Bass Fox / Quincas Moreira (studio tool, sidechain ducking, presets)

Frames Clean as a complete studio production tool (stereo I/O + sidechain), not just a guitar/bass comp. First half: builds an entire song ("Chasing Your Bliss") using Clean on every track. Second half: bass-focused feature tour. Distilled below.

## Per-track settings used to build the song
- **Bass** — preset 2: "simple compression." Removing a little high end with the EQ, mid attack, mid release, fairly high compression. (Sensitivity isn't at the panel position — it's stored in the preset, "somewhere in the middle.") Demonstrates that the panel knobs lie when a preset is recalled; the saved values rule.
- **Keys (pad, Yamaha CK61)** — run through Clean in stereo with **SIDECHAIN engaged, keyed by the kick drum**, so the pad ducks on every kick hit. Sends a stereo pad from the interface into Clean, sends the kick into the 1/8" sidechain input, records Clean's stereo output back to a separate track. (This is the headline production trick: classic sidechain pumping, done in hardware, in stereo.)
- **Guitar (Telecaster, stereo)** — MISO on (mono in / stereo out), plus SPREAD and MOTION on, for the tremolo-EQ effect; Physics added on top so you get *double* tremolo (the compression amplitude moving AND the EQ moving, both in stereo).
- **Drums** — stereo in/out as a "physical plug-in." Played with the tilt EQ (brighter/darker), Dynamics all the way up into Sag (inverts dynamics — soft parts loud, loud parts drowned out), slower attack = punchy (lets transients through), fast attack = squished/sustained. Then tried DUSTY for distorted-drum character; used the left (Shifty) EQ mode as an envelope EQ.

## Feature tour notes (bass-centric but general)
- **Compression in three zones:** regular VCA compression up to ~middle of the Dynamics knob; past middle it's a squashy **limiter** (play loud, holds the same level — and as a loud note decays it gets darker/boomier, acting like a dynamic-EQ swell); the very end is **Sag**, which *inverts* dynamics (loud → soft, soft → loud), the thing tube amps do when overdriven, now on a clean signal.
- **Sensitivity = threshold; Dynamics = ratio/strength.** Watch the activity LED: doesn't light at all when not acting; brighter = more compression. Low sensitivity = high threshold.
- **Attack** continuous: fast = sustain-y, slow = punchy/percussive (lets the initial transient pass).
- **EQ (tilt)** middle = fixed tilt (spongy lows CW / bright filter CCW — note this presenter's CW/CCW is reversed vs. the official manual wording; trust the manual: CCW removes highs, CW removes lows). Right = tremolo controlled by Attack speed. Left = dynamic/envelope EQ (loud = bright, decays darker — like an envelope filter responding to Sensitivity/Dynamics).
- **Physics** adds spring-like glitchy volume variation; more extreme as Dynamics goes up.
- **Noise gate** demonstrated on a noisy vintage Jazz Bass — dead silent when not touching the strings.
- **Dusty** converts the outboard limiter into a drive; "a really nice overdrive for bass." Works with the EQ/envelope-filter at the same time.

## Live/preset strategy (useful, general)
Because it's analog signal path but **digitally controlled, the presets save the dip-switch settings too** — anything on the panel saves to the left or right preset; the middle preset position is "live" (whatever the panel shows). The presenter's standard layout: **one preset = basic compressor, one preset = Dusty (dirt), and use the middle/live position for the creative stuff** — toggle between the three during a performance. Also controllable via expression/CV (modular) and MIDI (AUX jack).

## Bottom line
Calls it possibly "the only pedal a bass player might need" — compression, balanced-ish stereo outputs, tilt EQ for room conditioning, noise gate — plus the creative sag/Dusty/stereo-tremolo/sidechain features. Says after two months he still hasn't found everything it does.
