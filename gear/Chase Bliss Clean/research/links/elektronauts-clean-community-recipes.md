# Clean — community recipes & use notes (Elektronauts thread)

Synth-focused community (relevant: this rig's VG-800 modeled source + the owner's Elektron boxes). Thin on exact clock-face numbers — like most Clean coverage, users describe dip/mode recipes rather than full knob maps. Best concrete community shares below.

## Recipes / preset strategies shared
- **User "Python" — two saved presets he'd keep the pedal for:**
  - **Preset A = DUSTY mode** (DUSTY dip ON), with **DRY blended up** to keep "nuance" and avoid the volume dropouts you get from a wet-only Dusty signal; PHYSICS on the **subtle** (left) setting. He said he'd *"keep the pedal on my board if dusty mode would have been the only available mode."*
  - **Preset B = pseudo-tremolo via the MOTION dip** (MOTION dip ON) — saved to the second onboard preset slot for instant recall. (Recall: MOTION = DYNAMICS sets depth, ATTACK sets rate.)
- **Noise-gate caveat (Python):** when using the NOISE GATE dip, *"completely turn down the threshold or it will swallow up lightly finger-picked notes."* (i.e. SENSITIVITY = gate threshold; keep it low for quiet/fingerstyle passages.) Practical for single-coil hum without killing soft playing.

## Use-cases relevant to this rig
- **End-of-chain processor for an Octatrack** (Python, via another user) — confirms Clean's true-stereo end-of-chain "glue" use on a hardware sampler bus, the alternate deployment noted in the dossier.
- **Guitar FX chain example (Python):** `Clean → Generation Loss MkII → HX One → Lossy → Iridium` — Clean placed FIRST, as the front-end dynamics/level stage feeding dirt+texture (same logic as this rig's Clean-first slot).
- **Loudness without noise (user "aaanicholls" / Andrew Nicholls):** *"with the wet and dry knobs this pedal can get things really loud but with no noise"* — supports using WET/DRY as a quiet front-end boost/level-setter.
