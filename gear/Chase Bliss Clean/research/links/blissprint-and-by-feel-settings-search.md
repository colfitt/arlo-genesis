# Clean — fresh settings search (2026-06-14): by-feel confirmed + Blissprint preset tool

Light NEW WebSearch/WebFetch pass for documented artist/community DIAL settings on the Chase
Bliss Clean, to feed the patch list. **Outcome: no new concrete clock-face recipes exist on the
open web beyond what the repo already captured.** This confirms (again) the dossier finding: Clean
has no signature-artist mythology (Nov-2024 release) and the entire community/press corpus dials it
**"more by feel than by exact ratios"** — they describe dip/mode recipes, not full knob maps.

## What the fresh pass returned (all already on disk)
- The only repeated *number* anywhere is the maker/specialist start point: **Dynamics ~10:00 ≈ 4:1**
  (compressorpedalreviews) and **Wet/Dry unity ≈ noon** (video manual). Everything else is "watch
  the left LED / set Sensitivity per instrument."
- BuyOrBorrow, Guitar World, sixstringsensei, Pedal of the Day = promotional/feature-level, **zero
  concrete dial positions**. sixstringsensei explicitly refuses to give knob numbers ("philosophy
  over patches").
- Elektronauts thread (already captured) remains the only place with named owner recipes
  (Python's Dusty+Dry / Motion-preset; the noise-gate-threshold gotcha).

## NEW useful find — Blissprint (preset capture/share tool, NOT a preset pack)
- **synlyfe.itch.io/blissprint** — a 3rd-party app: *"A tool for creating and sharing custom
  presets for your Chase Bliss pedals. Pick your pedal from the list and match the parameters
  on-screen to your current physical settings."* It supports Clean.
- It ships **no factory/named Clean presets** — it's a visual logbook for *recording* your own dip
  + knob states so you can recall/share them. Relevant to this rig because Clean only has 2 onboard
  preset slots + middle "live"; Blissprint is how you'd document the 10–14 patches below beyond
  those 3 physical slots (e.g. a "banjo tamer" sheet vs. a "doom feeder" sheet).
- (There is also `alpfv.gumroad.com/l/onwardsettings` — that's an *Onward* sampler settings pack,
  NOT Clean; noted so it isn't mistaken for a Clean resource.)
