## Consensus tips / character
- "Natural feeling and easy to dial in"; "works well with both guitar and bass"; "really sweetens everything." Compression dialed **"more by feel than by exact ratios."**
- Mode + Physics switches enable "gated velcro fuzz" character **without a fuzz**, plus sagging/sputtering lofi textures. (SixStringSensei: "you wouldn't use these for 70s funk" — i.e. it's a character/vibe comp, not a transparent funk-clean comp.)
- The DRY knob being explicitly labeled "DRY" (not "Mix") is praised as clear for parallel-comp dialing.
- EQ is a single dial (no separate low/mid/high) — a limitation vs. multi-band comps like the Jackson Audio Bloom V2.

## The value debate (the main forum argument)
- **Pro:** "a lot of functionality crammed into a little box, nothing else like it"; noise gate + external sidechain make the price "quite the value"; "the MIDI control can do things none of the other popular compressor pedals can do."
- **Con:** several bass players **returned theirs** — "cool, but overkill for bass," wouldn't use all the features. Some guitarists "don't see how they'd need this kind of capability in a guitar signal chain" — feel these complex comps are "better suited for studio environments."
- **Bottom line (recurring phrasing):** "not so much about whether you need it, more about whether you will enjoy it."

## Bass-specific gotcha (recurring)
Use **Envelope Balance** (hidden, under EQ) to keep the low B / boomy lows from over-triggering the compressor; avoid **Sag** on bass (it favors higher frequencies).
