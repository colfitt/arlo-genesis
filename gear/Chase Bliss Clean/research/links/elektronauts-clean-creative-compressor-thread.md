Best community thread found for real-world owner workflow on the synth/ambient side. High-signal poster is "Python" (Nov 7, 2024). Distilled tips:

## Power-user tips (quoted / attributed)
- **Python (11/7/24):** **Dusty mode** is "the nicest sounding overdrive pedal I've ever had" (for guitar); "I would keep the pedal on my board if dusty mode would have been the only available mode." Notes Dusty is **conspicuously absent from most promo guitar demos.**
- **Python:** runs Clean **first in chain** → GenLoss II → HX One → Lossy → Iridium. Uses it as a **flexible panner + saturation source.**
- **Python:** **Physics on the subtle (left) setting** gives gentler volume modulations — "more nuanced than the heavy-handed volume dropouts on GenLossII and Lossy." Saves **Motion (pseudo-tremolo)** to a preset; calls EQ Modulated mode "nice and vibey."
- **aaanicholls / Andrew Nicholls (11/8/24):** end-of-chain it makes "really loud clean tones that can smack the other pedals in new ways"; **Wet/Dry boost "with no noise."**
- **MichaalHell / Grååskala (11/8 → confirmed keeper 11/30):** "End of chain, flexible panner, saturation, EQ" — "a lot of use cases."

## Pitfalls flagged by users
- **Noise gate threshold:** Python warns you must turn the gate threshold **fully down** or "it will also swallow up lightly finger-picked notes."
- **Sag (hold right footswitch maxes the ratio into Sag):** Python questioned its utility and **wished hold engaged Dusty instead.**

## Placement takeaway
Multiple users land on **either end of the chain** depending on goal: first = panner/saturator/level-setter that "smacks" downstream pedals; last = stereo glue/mix-bus. No one reported it being noisy or causing tone-suck.

## Note
Thread is synth/ambient-leaning (Hydra, Digitakt, etc.), so it under-covers dip-switch minutiae but is strong on creative/stereo placement and Dusty advocacy. No firmware issues raised.
