Useful mainly for the **complete control + hidden-options map** (GPX transcribes every knob's secondary function). Confirms a couple of hidden options not detailed elsewhere:

## Primary knobs → [hidden-option secondary function]
- Dynamics **[Gate Release]**
- Sensitivity (Ramp) **[Gate Threshold]**
- Wet **[Swell In]**
- Attack **[User Release]**
- EQ **[Envelope Balance]**
- Dry **[Swell Out]**
- Release (toggle)

## Toggle hidden functions (confirmed)
- **Physics toggle → Spread Routing: EQ / Both / Volume.**
- **Release toggle → Envelope Type: Analog / Combo / Adaptive.**
- **EQ Direction: Standard <> Inverted** (an EQ-mode hidden option — listed here, not prominent in the video manual).
- EQ modes: Shifty / Manual / Modulated.
- Swell: Dynamic Swell / Manual Swell (via Swell-Aux dip).

## Dips (full list)
Dusty, Swell Aux, Motion, Noise Gate, Sidechain, Latch, Spread, MISO.

## Other
- Positioned as a "full Stereo Dual Stage VCA Compressor/Limiter" placeable **"at either end of your chain."**
- Sidechain port for "smart-triggering the compression," esp. drums/synths.
- Price: $399 USD / €469 EUR.
