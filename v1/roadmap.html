04

Principles

local-first · free

Onboard/free models are the runtime target. Claude is an opt-in seeder, never a dependency.

markdown is canonical

The RAG index is a rebuildable cache. Folders are the database; lose the index, lose nothing.

inventory-honest

Recommend only gear & software you own — unless you explicitly ask "what should I buy." GAS off by default.

cited or it didn't happen

Every instructional claim links to a real file. "Falling back to generic" is stated out loud.

patchbay → MCP / CLI

Capabilities surface as MCP so any model or shell can call them. Not Claude-locked.

the moat stays yours

A 3-day-old odysseus is a swappable face. Your data & capabilities never depend on it surviving.
