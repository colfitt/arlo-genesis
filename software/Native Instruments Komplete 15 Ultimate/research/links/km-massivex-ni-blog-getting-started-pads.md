NI's "5 patches from init" starter (taught alongside Rishabh Rajan videos). Two patch skeletons are directly relevant to pads/drones. Distilled.

## Minimal pad ("quick and dirty")
- ONE wavetable oscillator + the **Wanderlust** reverb algorithm + an **LFO modulating Wavetable Position**.
- That single LFO→Position move + a lush reverb is enough for a usable evolving pad. Start here, then stack.

## Fuller pad
- BOTH wavetable oscillators + an **extra Insert oscillator** + a noise generator + the **Blue Monark filter** + multiple modulation sources.

## The "seven oscillators" trick
- Despite looking like a 2-oscillator synth, Massive X can run up to **seven oscillators** by routing audio-rate LFOs (and oscillators placed in Insert FX slots) into the signal path. Useful for thick, detuned walls. (This is the same insert-oscillator move Rishabh uses for his third oscillator.)

## Takeaway
The endorsed pad starting point is dead simple — one oscillator, LFO on Wavetable Position, Wanderlust reverb — then layer the second wavetable + an insert oscillator + noise + Monark filter for the full evolving texture.
