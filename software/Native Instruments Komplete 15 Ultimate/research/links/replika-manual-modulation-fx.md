# Replika XT — the 7 effects + the feedback-loop routing (official manual)

These are the "7 additional effects." The killer feature is the **Feedback Loop toggle.**

## ★ Feedback Loop placement (the shimmer/cascade mechanism)
**Freq Shifter, Filter, and Pitch Shifter** each have a Feedback Loop toggle:
- **Loop symbol ON** → effect sits *inside* the delay feedback loop → *"each delay repetition is
  progressively processed by the effect, resulting in a cascading sound effect."*
- **Arrow OFF** → effect processes only the delay output (once).
- Manual's own example: **Pitch Shifter in the feedback loop → each repetition shifts higher →
  shimmer textures.** This is the in-the-box shimmer-delay (no separate shimmer reverb needed).

## The 7 effects
- **Phaser** — harmonically unrelated peaks/troughs sweeping; Depth, Feedback, Rate (Hz/Note), Width.
- **Flanger** — jet-swoosh comb; Depth, Feedback, Rate (Hz/Note), Width.
- **Chorus** — detuned-copy thickening (shoegaze widen); Mix, Depth, Rate (Hz/Note), Width.
- **Freq Shifter** — shifts freqs up/down by Hz (NOT musical) → inharmonic/metallic; 5 modes (Up/
  Down/UpDown/DownUp/**RingMod**); Mix, Shift, Mode, **Feedback Loop**. In-loop = drifting inharmonic wall.
- **Filter** — LP/HP/BP/Peak/Notch, Cutoff, Resonance, Depth, Rate, **Feedback Loop**. In-loop =
  each repeat darker/more resonant = analog/dub self-filtering, or resonant howl near self-osc.
- **Pitch Shifter** — musical-interval transpose, preserves harmony; per-channel L/R semitones;
  algos **Smooth / Tight / Drums**. ★ In feedback loop = **shimmer** (e.g. +12 → octave-climbing cloud).
- **Micro Pitcher** — tiny L/R detune for width/ensemble; Mix + Style. (Subtle stereo widener.)

## Rig takeaways
- **Shimmer wall on banjo/synth:** Diffusion or Tape style + Pitch Shifter +12 (Smooth) **in the
  feedback loop**, high feedback → rising octave cloud.
- **Inharmonic doom drift:** Freq Shifter (small Hz) in feedback loop → seasick detuning each pass.
- **Self-filtering dub:** Filter LP in feedback loop, resonance up, Hi Cut down → repeats melt darker.
