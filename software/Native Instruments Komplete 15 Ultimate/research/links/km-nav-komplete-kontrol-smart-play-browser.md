# Komplete Kontrol — Smart Play + tag browser

## Smart Play (scales / chords / arp)
- **Scales:** map the keyboard to **100+ scales and modes**. Press SCALE to activate (default C Major). SHIFT+SCALE for Edit mode → knobs 1–3 pick root key, scale bank, scale type. Out-of-scale notes snap in.
- **Chords:** SHIFT+SCALE → knob 5 = Harmonizer or Chord Set → knobs 6–8 explore chord types. One key triggers a whole chord, staying in the selected scale. **Only works on polyphonic instruments (pads/keys).**
- **Arpeggiator:** press ARP. When SCALE is on, the arp follows scale notes. SHIFT+ARP → rate, octave range, etc.

These are **MIDI-layer features in the Komplete Kontrol plugin/standalone** — they work regardless of which keyboard sends the notes (the controller just triggers; the scale/chord/arp logic lives in the KK software). So a **Novation 61SL MkIII playing into KK** still gets Smart Play.

## Tag-based browser (the real reason to run KK)
- **Filters** search by **NKS tags** + Library tiles across ALL Komplete Kontrol content (Instruments, Loops, One-Shots, Effects) **plus your own user content**.
- Filter by tags, brand, product, bank, text search, user presets, Favorites.
- KK auto-sorts every sound by tags — find an "analog" bass, a "gloomy" soundscape, a specific kick by **describing it**. This is the unified cross-bundle preset browser that turns 150+ instruments into one searchable library.

> Implication for the rig: even if the 61SL MkIII never gets NI's hardware light-guide, running instruments **inside the Komplete Kontrol plugin** still buys you (a) the unified tag browser across the whole bundle and (b) Smart Play scales/chords/arp on any synth — both are software features.
