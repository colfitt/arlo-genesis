blog.native-instruments.com · NI Blog · "Building tracks with generative ideas: how iorie uses Reaktor"

Ambient producer **iorie**'s real generative workflow with Reaktor — the "let it run, capture, lightly edit" method that suits the whole rig.

**Tools:** Reaktor factory plugins + **Reaktor Blocks** patches. Highlights the **Flip Gen** block as ideal for generative sound design, and recommends starting from prebuilt patches before building your own.

**The core workflow (verbatim):**
> "I just let it run and record the output in Ableton until I hear something cool. Then I often only quantize it and EQ it a little bit."

Three stages: **generate → capture promising moments → minimal processing.**

**His generative cheat-code (verbatim):**
> "The generative cheat code is the 'Metaphysical Function A&B.'"
Variation: **manually change the sample selection** (rather than hitting randomize) for better results.

**Processing discipline (verbatim):**
> "Many generative sounds contain subharmonic frequencies, so remember to low-cut those sounds."
He avoids over-processing — generative output is "complex enough on their own."

Rig note: this is the exact loop for the user's setup — run SpaceDrone / CRUDE / a Blocks patch, record into Logic (or Ableton), pick the good 30 seconds, low-cut the mud, light EQ, then drop it under banjo and into Valhalla/EchoBoy. Metaphysical Function A&B is a factory ensemble — confirmed reach-for.
