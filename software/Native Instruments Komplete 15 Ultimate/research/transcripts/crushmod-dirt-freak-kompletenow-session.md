# DIRT + FREAK in a real session (distilled)

In-track demo of both, with preset counts and practical-mix advice.

## Preset counts (useful)
- **Dirt = 74 presets.** **Freak = 79 presets.** Both browse via Komplete Kontrol / the plugin browser. (Freak preset named **"Old School Maker"** used on vocals — pull Tuning down, change modulation type → degraded/fermented vocal character.)

## DIRT in practice
- Put on an **808 / sub** to make it cut through: flip presets, then tweak Mix (effect amount), Routing (A↔B), Drive on A and B.
- **Reverse the routing (B→A) genuinely changes the tone** — "you get different tones just by changing the direction of the input"; turn up Drive before it hits the second stage.
- **Amount:** saturation first half, **wavefolding second half** — "folds the waveform back into itself → cleaner distortion with less clipping."
- Modes I (subtle) / II (default, balanced) / III (extreme, crushed dark).
- As an effect on a **lead synth:** Drive all the way up on both sides, then **automate the Mix knob up from the start of the song** for a build/intro. When the distortion sounded **thin, turning up Bias** (asymmetry → even harmonics) gave "a warmer, less hollow sound."
- On **drums:** adds punch — but "a little goes a long way; you don't want to distort every element or things start to sound muddy."

## FREAK in practice
- **Oscillator mode = pure sine modulation;** the rate knob sets oscillator rate (fine adjust available by clicking the value).
- **Radio mode** emulates AM-radio demodulation circuits; **Tuning** = "dialing in a specific radio station."
- **Sidechain mode** connects external sources; **Contour** = how much sidechain affects the signal.
- **Harmonics** = amount of harmonic overtones from the circuit.
- **Type morph** between amplitude → ring → frequency shifting demonstrated as the central sound.

## Rig relevance
- Dirt's **automate-Mix-from-0 intro build** translates directly to a doom track's slow degrade-in.
- Bias = the go-to fix for thin/hollow wavefold distortion.
- Freak's big preset banks (79) are worth browsing for inharmonic/sci-fi starting points before dialing by hand.
- "A little goes a long way" — Dirt restraint advice applies to layered drone walls (distort one layer, not all).
