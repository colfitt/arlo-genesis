Making ambient pads in Massive X (the techniques transfer from older synths — they're the same).

Drop the volume to start (it can get loud). Pick basic wave shapes: oscillator 1 = Sine-Triangle, oscillator 2 = another basic shape. Pads need long attack and release — set those. Now bring the volume back up because it won't sound as harsh after filtering.

Filtering: use the blue filter (Monarch), keep it light, nice and low to start. Add a tiny bit of resonance.

Movement: pads need movement. Use a Performer. Bring the bell/level down, want unison. Throw a Performer on the filter frequency — make it go down a little, with a bit of resonance, and make it unidirectional (unipolar). Throw in a second Performer/modulator for amp and frequency on a slow rate.

Voicing: add voices, a tiny bit of stereo width. (Still not fully happy with the voicing but it works.)

Noise oscillators: blend them in subtly — it's tricky to get just a tiny bit. Try the second noise oscillator without filtering. Add ring modulation (set around -12). A touch of distortion.

The magic: Valhalla reverb ("this is where the magic is at — I love this plugin"). Back the reverb off when it's too much.

Post-processing / layering: group the patch, then DUPLICATE it. On the second copy, take the Performer off the filter frequency and select two DIFFERENT wavetables. The effect/FX wavetables work pretty well. Solo each layer to check. Increase the mix, add a little decay, want it a bit late/delayed — that's cool.

You don't have to stop there — add another layer. For this one, take off the noise. Try Soft Sync. Add a bit more movement, faster, and try something different. Use an LFO (unidirectional), slow rate, small amount. If using it on a track, touch the lows because there's too much low end.

That's just ONE patch built from scratch. Don't blindly copy — understand what was done and make your own things by applying the technique (layering multiple slightly-different copies + reverb).
