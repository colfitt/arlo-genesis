## Technique 5 — glitch/stutter a vocal loop ("the effects ARE the texture")
Source = a mono vocal loop ("cool vocal loop, just needed some flavor"); routed into Guitar Rig with input on Left only (mono in).
- Built by browsing: **effect type = modulation → character = Mashup**, which surfaced a preset full of **Traktor-derived components** — "I've been using Traktor components forever, I love the slicers and the Gators." (Confirms GR's rack can hold DJ/Traktor-style rhythmic components.)
- **Favorites by color**: right-click a preset to tag a color, e.g. "guitar presets orange, synth presets purple."

### The chain, built one module at a time
1. **Beat Slicer #1** — driven by a **step sequencer**: when a fader is up it slices, down it doesn't. 16 steps at **eighth-note** tempo. Choose a **style** (rhythmic pattern set) and a pattern within it.
2. **Beat Slicer #2** — same 16 steps but **sixteenth-note** tempo (faster) and "kind of backwards" — off, then on for the last two steps of every four. Stacking the two = a mix of slow + fast slices.
3. **Gator** — 16th-note gate cutting the volume in/out by a set shape = stutter/glitch movement.
4. **Delay** — quarter-note.
5. **Reverb.**
Delay + reverb sit **at the END of the chain** — all the beat-slicing/gating happens *before* them, so the time effects smear the already-chopped signal. Result = "a lot of movement, a lot of glitches, different stutter effects." Closing line: "that's five ways to use Guitar Rig on sounds that aren't guitars."

## Takeaway for a degraded rig
GR is a full modular FX rack that can host sequencer-driven rhythmic gates/slicers (Traktor-style) plus delay/reverb — so a sustained source (drone, banjo wall, synth pad) can be rhythmically chopped/gated and then washed, entirely inside one plugin instance. The chain-order principle (motion/glitch BEFORE time effects) is explicit.
