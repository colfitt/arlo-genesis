# Komplete 15 Ultimate — Navigator

The "what's in it, how you browse it, and which piece for what" map for a drone/doom/
shoegaze rig. Deep flagship guides beside this: **`Massive-X-UsageGuide.md`**,
**`Reaktor-6-UsageGuide.md`**, **`Guitar-Rig-7-UsageGuide.md`**. **Kontakt 8** (the other

## 1. What's in K15 Ultimate + what's new (verified, Sept 2024)

Three tiers (counts vary by source → ranges): `research/links/km-nav-k15-whats-new-synthanatomy.md`, `research/links/km-nav-tier-differences-buying-guide.md`
- **Ultimate** ≈ 150+ instruments/FX · 80+ expansions · ~100k sounds.
- **Shared flagship engines (all tiers):** Kontakt 8, Massive X, Reaktor 6, Guitar Rig 7
  Pro, iZotope Ozone 11 Standard.
- **New in K15:** Kontakt 8 (Tools/Leap/Conflux — see the Kontakt guide), **Guitar Rig 7
  Pro** (loop machine + new ICM amps), iZotope **Trash** + **Ozone 11** + VocalSynth 2, and
  new instruments incl. **Bazzazian Tapes**, Glaze 2, Saffron.
- **What Ultimate adds over Standard** = mostly *more high-end sampled/orchestral content +
  mixing tools*, **not new synth engines.** The drone-relevant Ultimate-only wins (all run
  in Kontakt): **Ashlight** (dark granular), **Schema Dark/Light**, **Thrill**. `research/links/km-nav-k15-review-musictech.md`

## 2. Komplete Kontrol front-end + the 61SL MkIII reality

**Komplete Kontrol** is the unified browser/host (plugin or standalone) — the one thing the
bundle needs: a **single tag-based search across all 150+ instruments** (filter by NKS tags/
brand/text/favorites — find a "gloomy soundscape" by describing it) + **Smart Play** (100+
scales, one-finger chords, scale-following arp). `research/links/km-nav-komplete-kontrol-smart-play-browser.md`

> **★ The 61SL MkIII reality:** run instruments *inside the KK plugin* and the SL is a
> strong NKS controller — **unified browsing + 8 knobs mapped to NKS parameter pages on its
> 5 screens** (load Template 1 MIDI). **What it does NOT get: the NI on-key light guide** (NI
> never licensed it to third parties). **Smart Play still works** (it's software/keyboard-
> agnostic). For a sustained-chord drone player the lost auto-light-guide is minor. `research/links/km-nav-61sl-mkiii-nks-reality.md`

## 3. Bite + the key NI effects for degrade/ambient

> **⚠️ INSTALL-STATUS CORRECTION (2026-06-11, verified on disk):** of these effects **only Bite is
> actually installed** (CONTENTS.md / Components). **RAUM, Replika XT, Dirt, Freak, Phasis, Choral,
> Flair are NOT on disk** — they're NI Effects-Series tools referenced here but absent; install via
> **Native Access** (RAUM is **free**; Bite's presence implies the Crush Pack license exists; verify
> the Mod Pack license). Each now has a deep-dive guide written ready for when installed.

- **Bite** (★ INSTALLED) — anti-aliased sample-rate + bit reduction: Freq 0.1→44.1 kHz, Bits,
  Crunch, **Jitter** (clock instability → "broken tape"), Pre/Post filters. **★ Automate Freq
  44.1 kHz → 300 Hz under a held drone = a "tape-disintegrating" arc.** → `research/Bite-UsageGuide.md`
- **RAUM** (not installed; free via Native Access) — the doom/shoegaze reverb: infinite drones via
  **Grounded/Airy + Freeze** *or* **Cosmic + Predelay Feedback 100%** (⚠️ **Freeze is NOT on Cosmic**
  — corrected); predelay+feedback = resonator/comb/looper. → `research/RAUM-UsageGuide.md`
- **Replika XT** (not installed) — 5-mode delay (Tape flutter / Analogue dub / **Diffusion**
  reverb-wash); self-oscillate for walls; FX-in-feedback-loop = shimmer/detune. → `research/Replika-XT-UsageGuide.md`
- **Crush Pack** = Bite + **Dirt** (wavefolding) + **Freak** (ring-mod/freq-shift → inharmonic;
  *the jewel*). **Mod Pack** = Phasis / **Choral** (shoegaze widener) / **Flair** (tuned-resonator
  flanger). (Dirt/Freak/Mod Pack **not installed**.) → `research/Crush-Mod-Packs-UsageGuide.md`

## 4. The reach-for map (whole bundle, this aesthetic)

| Need | Reach for | Notes |
|------|-----------|-------|
| **Evolving wavetable pad/drone** | **Massive X** *(own guide)* | Performers + feedback + noise→PM-AUX |
| **Generative drone / granular / modular** | **Reaktor 6** *(own guide)* | SpaceDrone, Metaphysical Function, Blocks, Travelizer |
| **Dark sampled/granular textures** | **Kontakt 8** + Ashlight/Straylight/Pharlight/Thrill/Schema | *(Kontakt guide §8)* |
| **Hybrid wavetable+sample texture** | **Conflux** (in Kontakt 8) | new in K8 |
| **Amp/cab + degrade + all-in-one FX rack** | **Guitar Rig 7 Pro** *(own guide)* | walls, lo-fi, on non-guitar |
| **Infinite reverb / freeze drone** | **RAUM** (Grounded/Airy + Freeze, *or* Cosmic + Feedback 100%) | the core ambient move — *not installed, free via Native Access* |
| **Dub/wash/lo-fi delay** | **Replika XT** | self-oscillate for walls |
| **Bitcrush / tape-disintegrate** | **Bite** (Freq-sweep automation) | installed |
| **Ring-mod / wavefold grit** | **Freak / Dirt** (Crush Pack) | doom detune |
| **Shoegaze widen/shimmer** | **Choral / Flair Voices** (Mod Pack) | |
| **Front-end / browse / play** | **Komplete Kontrol** + 61SL MkIII | tag search + Smart Play |

**Signal-chain idiom:** synth/sampled source → Mod Pack (Choral/Flair) → Bite/Freak
(degrade) → Replika XT (Diffusion) → **RAUM (Cosmic + Freeze)**. Modulate → delay → infinite.
