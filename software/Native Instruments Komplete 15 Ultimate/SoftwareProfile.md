# Native Instruments Komplete 15 Ultimate

**Summary** — NI's largest production bundle: Kontakt, Massive X, Reaktor, Guitar Rig, libraries.

> Kontakt 8 has its own folder for deep tracking. Sample libraries live on external drive — see CONTENTS.md.

## Why I have it
The big NI synth/FX/library bench. For the drone aesthetic the load-bearing pieces are
**Massive X** (evolving wavetable walls), **Reaktor 6** (generative drones + granular +
Blocks modular), **Guitar Rig 7 Pro** (amp/cab + modular FX rack), **Kontakt 8** (engine +
the Ashlight/Straylight/Thrill granular libs — *its own deep wave*), and the **NI effects**
(RAUM, Replika XT, Bite, Crush/Mod Packs). Research = a navigator + 3 flagship guides (deep
4-facet dive). → `research/Komplete-Navigator.md`

## Signature uses
- **Evolving pad/drone:** **Massive X** — stack 3 unsynced modulators on cutoff + Position;
  feedback + noise→PM-AUX for grit. → `research/Massive-X-UsageGuide.md`
- **Generative/granular drone:** **Reaktor 6** — SpaceDrone / Metaphysical Function (let it
  run, record, low-cut); **Travelizer** to granular-cloud the banjo. → `research/Reaktor-6-UsageGuide.md`
- **Walls + lo-fi + non-guitar FX rack:** **Guitar Rig 7 Pro** — distortion-LAST wall trick;
  Color + Tape Wobble + Noise Machine degrade; Split/Crossover/M-S routing on a send. →
  `research/Guitar-Rig-7-UsageGuide.md`
- **Core FX (deep-dived 2026-06-11):** **Bite** (★ installed — automate Freq 44.1k→300 Hz =
  tape-disintegrate; `research/Bite-UsageGuide.md`); **RAUM** (infinite drone: Grounded/Airy +
  Freeze *or* Cosmic + Feedback 100% — ⚠️ **Freeze is NOT on Cosmic**; `research/RAUM-UsageGuide.md`);
  **Replika XT** (Diffusion delay-wash; `research/Replika-XT-UsageGuide.md`); **Crush/Mod Packs**
  (Dirt wavefold / Freak ring-mod / Choral / Flair / Phasis; `research/Crush-Mod-Packs-UsageGuide.md`).
  **⚠️ Of these only Bite is installed on disk — RAUM/Replika XT/Dirt/Freak/Phasis/Choral/Flair are
  NOT installed (Native Access; RAUM is free).**
- **Browse/play:** **Komplete Kontrol** tag search + Smart Play.

## Notes
- **Komplete Kontrol + 61SL MkIII:** gets NKS browse + 8 knobs on its 5 screens (load
  Template 1), and Smart Play works — but **NOT the on-key light guide** (NI never licensed it
  to third-party keyboards). Minor for a drone player.
- **Kontakt 8 is tracked separately** (`Software/Native Instruments Kontakt 8/`, deep wave);
  the granular libraries (Ashlight etc.) are covered there.
- Ultimate over Standard = mostly *more content + mixing tools*, not new synth engines.
- **★ Install-status catch (2026-06-11):** CONTENTS.md shows only 6 NI plugins installed
  (Bite, Guitar Rig 7, Komplete Kontrol, Kontakt 8, Massive X, Reaktor 6). The navigator had
  presented **RAUM, Replika XT, Crush Pack (Dirt/Freak), Mod Pack (Phasis/Choral/Flair)** as
  active rig tools, but they are **not on disk** — one Native Access install away (RAUM free).
- Deep usage research → navigator + **3 instrument guides + 4 effect guides** (first pass:
