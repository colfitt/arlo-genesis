# Leo Abrahams — Ableton as a parallel guitar processor (NOT a looper)

THE most rig-relevant artist source: a guitarist running a pedal-front signal into
Ableton and treating it with parallel Audio Effect Racks controlled by a MIDI foot
controller. Direct analog to a three-pedalboard + baritone/banjo rig feeding Live.

## Signal flow (sourced quotes)
- "My setup is to have my guitar going into a Gamechanger Audio Plus Pedal with the
  dry signal going into one input and the plus pedal signal into another."
  → guitar splits: DRY into interface input 1, the Plus Pedal's granular-freeze
  sustain into input 2. (Map to YOUR rig: dry baritone in one input, the wet/
  sustained pedalboard-end tail — e.g. Big Sky / Microcosm / MOOD freeze — into a
  second input.)
- Each input is treated heavily with its own **parallel Audio Effect Rack** in Live.

## The racks
- "a long list of combinations of plugins that I've found to be inspiring and useful."
- "the fact that you could plug things into other things, have signals running in
  parallel and the routing was so flexible meant that it felt made for that."
- LFOs assigned to macros so the chains "react in slightly unexpected ways to
  whatever input I put in." (NOTE: Live's LFO device is Max-for-Live → NOT in Lite.
  Lite substitute = the built-in LFOs inside Auto Filter / Phaser-Flanger / Chorus-
  Ensemble, or clip envelopes/automation, to get the same moving-target behavior.)

## MIDI foot control
- "I use a MIDI foot controller to control a lot of parameters in audio racks and
  that feels like a very friendly, open environment."
  → Macro-map a footswitch/expression pedal to rack Macros. Fully Lite-loadable
  (Audio Effect Racks + MIDI map are in Lite). The Novation 61SL MkIII or any MIDI
  foot pedal can drive macros.

## Looping stance (important counterpoint)
- "Not like a looper where you hit repeat and it acts as an accompaniment, but more
  as an aid to exploration." He consciously avoided looping for the record.
- Relevant because **Lite has no Looper device anyway** — Abrahams shows a serious
  ambient-guitar workflow that needs NO looper: parallel reactive FX racks + foot
  control + record each layer separately.

## Lite verdict
The CORE method is Lite-safe: 2 inputs → 2 parallel Audio Effect Racks (built from
Reverb/Delay/Auto Filter/Erosion/Saturator/Phaser-Flanger) → macros on a MIDI foot
controller → record passes. Only the M4L LFO device must be swapped for device-
internal LFOs/automation.

## Certainty
HIGH for the workflow shape and quotes (direct Ableton interview, 2024). The
Lite-adaptation notes (LFO swap, input mapping to the user's pedals) are my
reconstruction, flagged as such.
