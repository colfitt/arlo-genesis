# 13 artists on using Ableton — the Lite-relevant ones

Filtered to performance/Session-view + Audio-Effect-Rack technique that loads in Lite.

## Kayla Painter — Follow Actions + blank scenes (LITE-SAFE, performance template)
- Live improv with audio/visuals; Session View with "real emphasis on the live feel"
  via **Follow Actions**.
- Uses **blank scenes with edited launch tempos** to "seamlessly blend one idea (or
  track) into another." Locked-tempo safety net under improvisation.
- → Direct Lite performance template: scenes as song sections, follow actions to
  auto-advance/randomize clips for generative ambient motion, blank scenes as
  transition/breath points. All stock, all in Lite (Session view + follow actions +
  scenes are core).

## The "Audio Effect Rack as your own plugin" school (LITE-SAFE)
- Jauz: saves effect racks/synth presets/loops to the **User Library** as a reusable
  "utility belt."
- KSHMR: audio effect racks to "essentially make your own plugin"; grab anything from
  the arrangement into the browser to store.
- Zanias: "Build and save complex effect chains and just see what happens"; Session
  View "for triggering new ideas."
- Laughing Ears / S.C.D.D. / Sky Deep: Session View + nested groups + effect racks for
  flexible layering; Sky Deep makes a custom default session template that opens on launch.
- → All fully Lite-loadable (Audio Effect Racks, macros, User Library, default
  template are in Lite). This is the foundation for building your own ambient rack
  library and a custom drone default set.

## Workflow hygiene that fits Lite's ceiling
- Prequel Tapes: "Always bounce everything to audio. Never leave plugins plus effects
  running." → identical to the Lite resample-to-free-tracks discipline.

## NOT Lite (flagged)

- Greg Fox / KSHMR: Comping — Standard/Suite (not in Lite).
