# Ableton stock Delay — ambient/dub recipes (all Lite-safe)

The stock **Delay** device is in Lite. These techniques use only Lite devices
(Delay, EQ Three, Auto Filter) — directly applicable to dub/ambient walls.

## Core Delay settings
- **Feedback fully down** = single slap echo (no repeats).
- **Link + Sync ON** = tempo-synced repeats in 16ths.
- **Link OFF + Offset** = nudge L/R times "small percentages in opposite
  directions" for width/spatial character (manual ping-pong feel).
- **Dry/Wet** balances source vs echoes.

## Filtering the repeats (the dub move)
1. **EQ Three before/after Delay**: kill L and H bands, sweep the M band's
   FreqLow/FreqHi → classic darkening dub-delay (telephone repeats).
2. **Auto Filter** for modulated filtering (LP/HP/BP/notch + LFO + envelope
   follower) → repeats that move/breathe.
3. Per-channel filtered/feedback delay lines for L / center / R for stereo spread.

## Routing
- Put delay (and reverb) on **Return tracks**; "Pre" send mode lets you control
  echo level independent of the dry automation. Lite has 2 returns — enough for a
  delay return + a reverb return.
- Chain: delay return → reverb return (serial), or both parallel.

## Lite verdict
100% Lite-loadable. This is the backbone for the dub/ambient delay sound without
needing the Suite-only Echo device. Pair with high feedback (just below
self-oscillation) + Auto Filter LP sweep for the "degraded wall" tail.

## Certainty
HIGH — straightforward stock-device techniques, all devices confirmed in Lite.
