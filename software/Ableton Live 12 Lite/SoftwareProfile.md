# Ableton Live 12 Lite

**Summary** — Session-view performance/clip-launching DAW (the **Lite** edition).
Secondary to Logic; also the **VST fallback** since Logic is AU-only.

## Why I have it
Performance + sketch surface and VST host. Work within the Lite device floor,
commit textures by **resampling**, perform via Session clips, then hand stems to
Logic for the studio mix. The hardware carries the sound; Live provides
clip-launch + rack-dense texture tracks.

## Signature uses
- **Warp → Texture** turns any pedalboard/OP-1/banjo recording into a drone;
  **Drift** + **Reverb Freeze** for evolving pads/walls (all stock-Lite).
- **Loop-without-Looper** Session template (Looper isn't in Lite); clip/scene
  performance with **Follow Actions**.
- **Racks = 1 track** to beat the 8-track cap; **resample (not Freeze)** to
  commit clean ambient tails.
- **Clock sync** to the Elektrons/MPC/OP-1 (Live as master, 64-sample buffer,
  dedicated MIDI port, no USB hub).
- VST fallback for the owned plugin set.

## Notes
- **Lite device floor:** 8 tracks · 16 scenes · 2 returns · 5 instruments (Drift,
  Simpler, Impulse, Drum/Instrument Rack) · ~15 audio effects. **No Max for Live,
  no Looper/Echo/Grain Delay/Shifter/Hybrid Reverb/Spectral.** (Lite ≠ Intro.)
- **Trap:** opening a Suite/M4L set in Lite → "Device is not available"
  placeholders **and Save/Export disabled**; bounce those tracks to audio to
  re-enable saving.
- **Freeze warps in Beats mode** (degrades long tails) → resample instead.
- **Upgrade verdict:** stay on Lite; Standard (~$189, ~20% off via Novation
  ownership) only if the track cap bites; Suite only for the M4L ecosystem.
- Deep usage research → `research/Live-12-Lite-UsageGuide.md` (6 transcripts +
  26 links).
