# Nudist Audio Nudistort

**Summary** — Single-band **chaotic / nonlinear / time-varying** distortion + an
"abnormal" delay + modulation, by solo dev **Caleb Reske** (~$32–40; VST3/AU/AAX,
Apple-Silicon native, very low CPU). NOT multiband, NOT a compressor, NOT Devious.

## Why I have it
The in-the-box **generative-chaos** mangler — it does things no other dirt in the
rig can: ring-mod, a **self-oscillating abnormal delay**, telecom grit, and
note-keyed harmonic resampling. The "broken in a way you've never heard" texture/
drone end, complementary to Decapitator (analog color) and the hardware fuzzes
(performed wall).

## Signature uses
- **Drone / self-oscillation:** Delay **high Feedback + high Abnormality** =
  "beautiful disorder"; add **Species = the track's tonic** for hypnotic harmonic
  lock. Banjo/baritone sustain → endless evolving drone.
- **"Recorded-wrong" grit:** `Mu-law` (telecom companding) + `Buzz` + dark `Tone`.
- **On reverb/delay sends** — mangle the wash, not the dry source (the shoegaze move).
- **Per-string** via the **GK-5 → VG-800 hexaphonic** rig (one instance per string).

## Notes
- **Nonlinear by design:** `Drive` is **non-monotonic** (higher ≠ more distorted —
  audition by sweeping); **Damping** is a tone/voicing control, not just a transient
  tamer; **Mix Lock** holds wet/dry while you audition Types/Modes/presets.
- **Delay runaway** (high FB + Abnormality) spirals fast — a feature; automate/gate.
- **No multiband / no oversampling toggle** — HP into it / LP the wet on bass; it
  can alias (part of the character).
- **★ Data-hygiene flag:** AI search summaries confabulate a fake "Drive/**Bias**/
  Tilt" 3-knob layout with bogus percentages — **there is no "Bias" knob.** Ignore.
