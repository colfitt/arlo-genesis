soundonsound.com · "Arturia Solina V" review · undated

The string-machine bed (ARP/Eminent Solina String Ensemble emulation) — the lush
shoegaze ensemble-strings layer.

**The ENSEMBLE effect (the whole point):** the original used modulated BBD (bucket-
brigade) analog delay lines = a massive built-in chorus. Solina V faithfully models both
the V1 and V2 ensemble. **Caveat from the reviewer:** the lower-frequency modulator runs
"around 1.3 Hz — too quickly and too prominent"; the original patent specified **≤1 Hz**
for the proper luscious effect. Practical fix → if the default ensemble sounds too
fluttery, the V2 ensemble / lighter chorus settings sit better for a slow wall. Plus side:
the software removes the hardware's noise halo (though for THIS rig, a little noise back
in via SketchCassette is desirable).

**Voices:** violin, viola, cello, **horn, trumpet** — plus a new **Humana** voice
(approximates the Vox Humana preset: doubled + slightly chorused, moves even without the
ensemble effect). Sections: **Upper** (paraphonic) + **Lower** (monophonic bass).

**Paraphonic vs polyphonic:** the original is paraphonic (all notes share one contour).
Solina V adds a **POLY switch** for true polyphony (each note its own contour) — the
reviewer found this transformative for pad use.

**Added synth/FX section (the V version's bonus):**
- **Upper Resonator** — three formant filters (60–300 Hz / 300 Hz–1.5 kHz / 1.5–7.5 kHz)
  with resonance + gain → vowel-y, vocal string textures.
- **Bass section** — 24 dB/oct low-pass with its own AR envelope + arpeggiator.
- **Effects** — dual-stage phaser, 3 chorus complexities, analog/digital delays, 24 reverbs.

**How to get the lush bed (reviewer's recipe):** apply a wash of the **onboard effects to
individual voices, often WITHOUT the ensemble**; use the Master section to **darken** the
initial tone; add expression via **velocity/aftertouch**. The **dual phaser** creates
"huge, whooshing panoramas" by processing stereo sides differently; the **chorus** has
stereo rate + width knobs for width.

**Rig takeaway:** Solina V2 = the always-on string-ensemble bed under banjo/baritone. Stack
violin+cello (or add Humana for movement), turn POLY on for sustained chords, darken the
master, then run into EchoBoy + Valhalla. Use the phaser for shoegaze whoosh; add
SketchCassette to put the noise/degrade back.
