The how-to for the standout evolving-pad tool in the bundle: Prophet-VS vector synthesis.

**The four oscillators (A–D):** each loaded with a distinct digital waveform from the
library. The **joystick** crossfades between all four in real time — move it while
holding a note and the timbre morphs.

**Making the pad evolve ON ITS OWN (the key trick) — the Mixer Envelope / vector field:**
1. **Activate the envelope** — enable the power button in the mixer-envelope section.
2. **Plot waypoints** — click/drag the center square to position points across the
   vector field; each point is a blend of the four oscillators (i.e. a position in the
   A/B/C/D square).
3. **Set transition timing** — adjust the **Rates** on the right to program how long the
   sound takes to travel between points.

Result: the synth auto-travels between waveform blends — timbral evolution with no hand
on the joystick. This is the closest thing in the bundle to a true "evolving pad"
generator: a slow multi-point vector path = a pad that never sits still.

**Extra motion:** the filter and amp envelopes offer **loop points**, so you can loop a
segment for continuous cyclical movement on top of the vector path.

**Rig takeaway:** Prophet-VS V is the pick when you want a pad that *evolves by itself*
under a static drone. Program a slow 4–6-point vector loop (long Rates), add an env loop
point on the filter, and you have a self-morphing bed. Pair with the analog filter for
warmth, then send to Valhalla for the wall.
