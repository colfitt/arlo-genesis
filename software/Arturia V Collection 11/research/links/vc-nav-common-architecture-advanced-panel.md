The SHARED "Advanced" panel architecture that hides behind the faithful front
panel on Arturia's modern V instruments (the design philosophy: "a faithful
front panel and a deeper 'lab' behind it"). Common building blocks:

ADVANCED PANEL:
- Multi-arp: up to 4 stacked arpeggiator layers.
- 3 modulator slots, each can be: ADSR Envelope, Function generator, Random
  generator, Voice Mod, or Mod Sequencer. Plus keyboard + macro modulators.
- Drag-and-drop modulation routing to most parameters (the modern Arturia
  "drag the source onto the destination" mod matrix).
- 4-slot FX RACK with 17+ effects (the same FX concept exposed in Analog Lab's
  "pedals" view).

UNISON / VOICE MODE:
- Unison stacks up to six (triple-VCO) voices with adjustable detune + stereo
  spread → up to 18 oscillators on a single key. (Huge for walls/supersaw drones.)

MPE / MICROTUNING:
- Full MPE: aftertouch, release velocity, keyboard tracking — assignable to any
  parameter via the Advanced panel.
- MTS-ESP microtuning support.

UI:
- Resizable interface across the line.

NOTE on the "lab behind the panel": the same skeleton (multi-arp + 3 modulators
+ function generators + 4-slot FX + unison + MPE) recurs across the newest V
instruments (Synthx V, MiniBrute V, Jup-8000 V, Augmented, SEM V3, MiniFreak V),
so once you learn one Advanced panel you know them all. Older/legacy panels
(e.g. Mini V4, CS-80 V4, Prophet-5 V) carry their own period-appropriate
modulation but still share the FX rack + resizable UI + preset browser.
