WHICH V INSTRUMENT FOR AMBIENT/DRONE/EVOLVING TEXTURE — and the core technique.

CS-80 V (→ flagship guide separate):
- Atmospheric pads via slow modulation on the hi-pass filter + evolving tremolo
  pads. The original CS-80's ring-mod, dual-layer + brass patch with long release
  = the Vangelis Blade Runner immersive/droning sound. Ribbon + poly-aftertouch =
  the expressive wall.

MELLOTRON V (→ flagship guide separate):
- Shoegaze strings/choir/flute beds; use the tape Flutter + Mellotron mechanics
  for the authentic degraded wobble directly on the sample sets.

SYNTHI V (→ flagship guide separate):
- "Instantly want to make drones and sound effects." The pin-matrix patchbay +
  added modulations (beyond the hardware) = drone/noise/FX engine. Joystick +
  the matrix for evolving chaos.

BUCHLA EASEL V (→ flagship guide separate):
- Generative drones: the Random Voltage generator (4 unique outputs) + Sequential
  Voltage Source → Krell-style self-evolving patches; route random voltage to the
  Period rate / complex-oscillator for organic, non-repeating evolution.

EMULATOR II V + CMI V (→ flagship guide separate):
- The lo-fi sampling core: 8-bit DPCM crunch + warm analogue VCF (Emulator),
  gritty early-digital texture (Fairlight). Import samples, smear with the filter.

THE UNIVERSAL "EVOLVING" TECHNIQUE (applies to ALL of them):
- Use multiple ASYNCHRONOUS, very slow, UNSYNCED modulation sources (free-running
  LFOs + looping envelopes / function generators at unrelated rates). The
  "conflicting cycles" almost never line up → the pad never audibly repeats.
- Long release times + chorus + delay + reverb are what sell "ambient."
- In the V instruments: park 2-3 of the Advanced-panel modulator slots on
  Function generators / Random at different (non-synced) speeds onto filter
  cutoff, oscillator pitch/detune, and FX amount.
