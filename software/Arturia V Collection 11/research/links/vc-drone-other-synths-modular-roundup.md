== Modular V3 (Moog Modular emulation) ==
https://www.arturia.com/products/software-instruments/modular-v/overview
- The most "anything goes" patchcord box in the bundle: VCOs, filters, VCAs, env, plus
  Bode frequency shifter + sequencers, fully self-patchable. Best for FULLY self-generating
  drone webs: patch slow LFO/S&H → filter cutoff + osc fine-tune, loop a sequencer into
  pitch, feed noise through a resonant filter. Free patches at presetpatch.com (search
  "Arturia Modular V").

== ARP 2600 V (semi-modular; pointblank tutorial, Dan Herbert) ==
https://www.youtube.com/watch?v=QfsiDs4QO6o  (captured: vc-drone-arp2600-pointblank-modular-basics-p1.md)
- "Great synth to learn modular principles on — not too complex but loads of potential."
  3 VCOs (VCO1/2/3) + filter + main output with a simple spring REVERB built in; 2 envelopes
  (ADSR + AR), a DAW-syncable LFO, and the VCOs can run in LF mode as extra modulators.
- Noise generator + voltage processors + Sample & Hold + the 1601 sequencer. For drone:
  detune the 3 VCOs a few cents apart, run them into the filter near self-oscillation, then
  modulate cutoff with S&H(noise) for slow random shimmer; ride the onboard reverb.
- It's pre-normalled so a drone needs almost no cabling — fastest "semi-modular" drone of the set.

== KORG MS-20 V (gritty filter + ESP) ==
https://www.arturia.com/products/software-instruments/korg-ms-20-v/overview
- The aggressive one. Meticulous model of the MS-20's "ultra-fat oscillators, rule-breaking
  External Signal Processor (ESP), and screaming filters," every non-linear nuance kept.
- KILLER FEATURE for our rig: the ESP processes EXTERNAL AUDIO via sidechain — run the banjo
  (or a pedalboard reamp, or a Digitakt loop) INTO the MS-20 V's filters + crunchy output,
  build "wild feedback routing," and use the incoming sound to drive an envelope follower.
  This is the in-the-box version of screaming the board through a gritty analog filter.
- The two filters (HP + LP, each Cutoff+Peak) self-oscillate; push Peak to oscillation for a
  pure drone tone, patch the ESP envelope-follower back to cutoff for a self-playing wall.
- Free standalone "Filter MS-20" effect exists too (insert on any track for the same crunch).

== MiniFreak V (digital hybrid; Arturia mod-matrix tutorial, Matt Pike) ==
https://www.youtube.com/watch?v=Da5mtDCI0jQ  (captured: vc-drone-minifreak-arturia-mod-matrix.md)
- Best for EVOLVING DIGITAL textures. 2 LFOs/voice with custom wave shapes, ADSR + a CYCLING
  ENVELOPE (loops like an LFO with Rise/Fall/Hold) for continuous evolution, velocity/AT/key-
  follow, mod wheel + 2 macro sliders, and a 4-lane modulation SEQUENCER — all into a generous
  mod matrix (9 assignable destinations).
- Drone recipe: set Cycling Env to loop slowly → route to filter + an FX param; second LFO
  (slow, random/S&H shape) → oscillator type/wavetable position; spread two presets across
  octaves and hold a low cluster. Digital oscillator engines (wavetable/FM/etc.) give it a
  colder, more "broken digital" character than the analog models — good for shoegaze grit.
