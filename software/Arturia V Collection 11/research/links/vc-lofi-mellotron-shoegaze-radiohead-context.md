Artist/aesthetic context for the Mellotron string/choir sound in shoegaze + Radiohead.

## The Mellotron string sound (why it's the shoegaze/prog staple)
- The famous **"3 violins" M400 string tape** is THE recognizable Mellotron
  sound: lush, slightly seasick, with audible tape wobble + the ~8-second
  per-note tape limit and worn-tape dropouts.
- Loved by **King Crimson** ("Epitaph"), **Genesis** ("Watcher of the Skies"),
  **The Beatles** ("Strawberry Fields" flute), **Led Zeppelin**, **David Bowie**.

## Radiohead
- Used the **M400** (the most common model — the one Mellotron V is based on).
- Mellotron strings on **"I Promise"** and **"Polyethylene (Parts 1&2)"** (cut
  from OK Computer, released on OKNOTOK 1997 2017). Reverb notes these can be
  recreated with an **M400 string patch** "likely sampling the same tapes."
- Implication for this rig: Mellotron V's authentic M400 string tape IS that
  patch — start from the **3-violins/strings** rack.

## My Bloody Valentine / shoegaze
- MBV's *Loveless* uses **Mellotron-like strings** ("Touched") and Mellotron-ish
  keyboard melodies ("What You Want"). The shoegaze move = take the woozy tape
  strings/choir and bury them in pitch-bend, reverb, and saturation so they sit
  as a **wall** under the guitars rather than a clean orchestral line.

## Practical takeaway for Mellotron V
- Strings/choir + heavy **Flutter** + **Saturation** + long reverb = the
  "recorded-wrong" shoegaze string wall.
- Layer two tapes panned L/R (per the Big Stereo Choir recipe) for width.
