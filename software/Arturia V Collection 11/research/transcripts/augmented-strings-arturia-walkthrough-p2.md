Official Arturia walkthrough of the Augmented STRINGS Advanced panel. Distilled. (Architecture is identical for Augmented VOICES — same engine, different sample content.)

== Two pages ==
- **Main page**: the big central **Morph** control + **7 macro knobs** to drastically reshape the current sound in a few clicks.
- **Advanced panel** (behind it): deeper sound design, 5 tabs total — Layers, Modulation, Arpeggiator, Effects, Macro.

== Layers tab — the engine architecture ==
- **2 Layers (A and B)**, each with **2 sound generators** = **4 generators total**, all playable simultaneously. Each generator has its own VCA, envelope, and filter.
- Three ways to arrange each layer: **2 synth engines**, **2 sample engines**, or **1 synth + 1 sample** (3 combinations).
- **Sample engines** = "rich, natural, authentic recordings": violins, violas, cellos, double basses, full ensembles. (VOICES: choir + solo voices instead.)
- **Synth section** = **wavetable, granular, subtractive, harmonic** synthesis. Browse articulations: legato, staccato, pizzicato, tremolo (STRINGS).

== Modulation page ==
- Built-in modulators: **LFOs, Functions, Random generators, keyboard behavior** — each with many parameters.
- **Bottom bar** lists every modulator + macro. To route: click a modulator/macro block in the bar, drag onto the destination parameter. Same drag-to-assign for adding params to Morph or any macro.

== Morph + macros (the evolving-wall heart) ==
- **Morph** can control **up to 8 parameters** from anywhere in the instrument (on top of its default layer-blend job).
- Sound macros **Color / Time / Motion** can each carry **6 additional parameters**:
  - **Color** → engine + filter parameters.
  - **Time** → specifically VCA + filter envelope params (attack / release times).
  - **Motion** → modulation params (LFO / function **rates**) — i.e. dial in how fast everything wobbles with one knob.
  - **FX macros** → 4 additional effects params each.

== Arpeggiator page ==
- Enable/disable steps; per-step velocity + gate; add chords; playback direction; time division; swing; octave switching. "Great for designing rhythmic patches and sequences."

== Effects page ==
- **Layer effects**: each layer holds **2 effects** → 4 at once, **14 effects** available (reverbs, delays, sound-sculpting, modulation).
- **Global effects** (output of both layers): **3 delay types + 2 reverb types**, always present, "each as spacious as the next."

== Macro page (last tab) ==
- Shows the current mapped parameters for each macro (Color/Morph/Time/Motion + Reverb/Delay/FX); set per-destination **modulation amount** and **modulation curve**. "Always be in control of what's being modulated on the front page."
