- Hi I'm Glen Darcy, vice-president of product management for Arturia. This is part 2 in our Buchla 
music easel video series. In this video I'll be covering the main modulation and control sources. 
The main control source on the music easel is the keyboard. We have a standard keyboard output that 
you can route from here. There's also portamento control. The keyboard also has an arpegiater. You 
can have ascending patterns, and then random patterns. You can change the speed and set whether 
it's sync or free speed. The preset voltage source consists of these four knobs and these four 
pads. They can be routed to the keyboard to be added of with a keyboard, via this "add to main" 
button. You can take the direct output of them from this jack here. Using the "add to main" I can 
set them as octave buttons. Or I can set preset voltages. This will allow me to set different 
transpositions. So if I hold one chord on the keyboard, I can then set these to different voltage 
settings, and then transpose. On the far right is the master clock. This can be used to clock the 
sequencer. Next to the this is a square wave pulse output that comes from the keyboard itself. On 
the upper left front panel, is a 5 step sequencer. You can choose the number of steps from 3-5 
using this switch. You can also choose the trigger select source. In this case I have the clock, 
down here, control that. I can also have the keyboard do it. So every time I play a note, it will 
advance the sequencer. Or I can use the pulsar. The pulse sequence switches will set what rhythm is 
output. Modules like the envelop generator and pulsar can be set to sequencer as their triggering 
source. While you can hear that the triggers are skipping, if you extend out the decay time you'll 
hear that the pitches are still going playing. The sequencer voltage levels are where you can set 
the output for each step. In this case I've taken the sequencer output, which are these blue jacks 
here, and I've just routed it into the pitch of the complex oscillator. By varying these I can 
change the pitch on each step. In a similar way I can also just use this timbre control instead of 
pitch control. Or I could route it to both. The envelop generator can operate in three modes. It 
can be an attack sustain hold release, with a sustain setting. It can be an attack hold release 
with a transient, where it will raise up on the attack, it will sustain for a given amount of time 
based on the slider, and then release. It can also self trigger itself and be a looping envelop. 
This allows it to actually work as an LFO. The envelop generator can be triggered by the keyboard, 
it can be triggered by the pulsar, and it can be triggered by the sequencer. The pulsar has the 
same kind of trigger options. It can be controlled by the keyboard, the pulsar, itself, or the 
sequencer. The pulsar is a simple clock pulse with a release time. It can be used just as a sustain 
and release envelop if you want. In this case I'm going to patch the output, into the low pass 
gate. And you can hear the period with which it's going to decay out. Between the multiple trigger 
options in the modulation sources, and the routing available in the dual low pass gate, the music 
keys all hides a lot of complexity in a few controls. The patching system is fairly simple once you 
understand it. The outputs are all color coordinated to match the modulation source. Blue is 
sequencer, white is your random voltage control, orange is the envelop generator, yellow is pulsar, 
and lavender is after touch. The black connections are inputs to your modulation destinations. All 
of the input destinations have an arrow that lead up to a slider. This slider gives you the amount 
of modulation for that destination. Just patched any source to any destination by clicking, 
dragging and releasing. In this example I'm patching the sequencer to the pitch of the complex 
oscillator. I could then just patch the envelop generator into the pitch. Or lets do the pulsar 
instead. In this case I'll do the random voltage. This covers the features on the front panel. You
