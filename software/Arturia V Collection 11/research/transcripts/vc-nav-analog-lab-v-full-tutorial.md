Walkthrough of Analog Lab V (a.k.a. Analog Lab 5 / Analog Lab Intro) — the
unified front-end/browser for the V Collection. Distilled workflow:

BROWSE BY TYPE (fast path to a sound)
- Default screen = "Browse sounds by type" tiles: Leads, Bass, Pad, Keys,
  Strings, Organ, etc. Click a type, pick a preset, audition. (Demo: Leads →
  "Saw Pulse Lead".)
- "See more" on any tile jumps to the full EXPLORE list of every preset.

EXPLORE / FILTERING
- In Explore you filter by category. Click "Instruments" to filter by which
  soft-synth the preset comes from (e.g. show only Jupiter-8000 presets) — this
  is how you audition one engine's character without owning the standalone.
- "Styles" → Characteristics section: tag-filter by sonic trait, e.g. select
  "Reverb" to find drenched/ambient presets (demoed the "Ambient Spectrum" pad).
- Explore by GENRE too (e.g. "Disco", and there is an "Ambient" genre tag).
- Sound Banks = purchasable + free expansion banks; Sound Designers = browse by
  who made the preset. The Store tab auditions paid banks; there are FREE banks
  worth grabbing (presenter installed "MiniBrute Funk" + a "Lily" signature pack
  — auditionable and downloadable for free).

MAIN VIEW (the playable screen)
- Good for randomly auditioning presets; arrows cycle through presets in place.
- Can also search by category from here.

TAGS / LIKES / PLAYLISTS (organising)
- Heart icon = "Like" a preset → shows up under the Liked category.
- You can RENAME the tag categories (e.g. rename one to "Pads") and right-click
  any preset → add it to that category.
- Right-click a preset → Add to Playlist → New Playlist (named e.g. "Cool
  Sounds"). Playlists group presets for live recall / Stage view.

PRESET CONTROLS = THE 4 MACROS
- Each preset exposes a small set of macro knobs (Brightness, Chorus Mix, Delay
  Volume, Release time shown). Turning Brightness opens the tone; Chorus Mix
  thickens; raising Delay Volume + Release time pushes a key into a long ambient
  wash — the fastest way to make a stock preset more "drone-ready."

PRESET FX RACK (Edit Preset)
- "Edit Preset" reveals a row of FX "pedals" (Reverb, Delay, Wah, Phaser, EQ…).
- Each slot has a fader/mix; you can pull the reverb fader down, push delay up,
  raise release for long tails. Slots are swappable (swapped slot 1 to a Wah,
  slot 2 to a Phaser) and bypassable. Treble/Mid/Bass EQ sit top-right.
- This is the same shared FX concept exposed across the V instruments.

SPLITS / MULTIS
- "Add Instrument" loads a second sound; drag the split bars at the top to set
  the keyboard zones (left = bass, right = lead). Adjust the split point live.

SAVE
- Waffle menu → "Save Preset As" → name + author + type + description → Save.
- Recall from Sound Banks → User category, or by name in the Explore search bar.
