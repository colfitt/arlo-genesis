Hi I’m Matt Pike, and in this video I will guide you through the modulation options and mod matrix 
of the MiniFreak, Arturia’s unconventional hybrid synthesiser. We’re going to look at how this 
carefully designed selection of modulation tools, intuitive mod matrix, and performance macros help 
us exploit all the power this little synth has to offer. The MiniFreak features 2 LFOs with custom 
wave shapes, an ADSR Envelope, a Cycling Envelope, velocity, aftertouch, and keyboard follow with 
various modes, an assignable mod wheel with a dedicated vibrato option and 2 assignable macro 
sliders, so our modulation options are really extensive That's not even factoring in the 4 lane 
modulation sequencer which we will cover in a separate video. Let’s start with the Envelope. This is 
the only modulation source in the MiniFreak which is hardwired to control elements of the synth. It 
will always control the VCA volume and can be routed to the filter via the Envelope knob in the 
filter section. The Envelope is the same one found in the PolyBrute and can go from extremely tight 
and snappy to long and smooth. We have options to change the curves in the envelope settings menu, 
which can be accessed by the shortcut of holding Sound Edit then pressing the Envelope Mode switch. 
This same shortcut process can be used to access the settings menu for all the modulation sources. 
In the Envelope settings menu we also have options to add velocity modulation of the Envelope to 
its various sources, which is a great way to get extra dynamic into a patch quickly and without 
using up space in the mod matrix. Before we move on to the other modulation sources in the 
MiniFreak, let's look at where we will actually assign and make use of them. The mod matrix will 
hopefully look familiar to any MicroFreak users and has been expanded to include all the new mod 
sources and to have 3 pages of assignable mod destinations. Mod destinations are found along the 
top of the matrix, and mod sources are down the side. We navigate around the matrix horizontally 
with the Matrix knob and vertically by holding shift and turning the Matrix knob. When we find a 
modulation we want to assign, we have two options. Either click the Matrix knob and then adjust the 
amount, then click to exit back to navigation. Or we can push and turn to make quick edits without 
leaving the navigation mode. To remove an assignment you just have to click and hold the Matrix 
knob for a few seconds until it blinks. Pitch 1+2, Wave 1, Timbre 1 and Cutoff are preassigned for 
speedy patching but if you want to assign a mod destination just hold the Assign button on the 
matrix, then wiggle the control you want to modulate. For instance, here we'll go for Shape 1 You 
can assign almost all of the rotary controls on the front panel, and also if you hold the assign 
button and turn the Preset / Edit knob some extra controls such as the LFO amplitudes or the VCA 
with are not available on the front panel. You can also assign a patch point on the matrix as mod 
destination, say for instance I wanted to have velocity control the cycling envelope to timbre in 
this FM patch. If I hold the Assign button and then navigate to the patch point using the Matrix 
knob we can see it displayed as the destination on the screen. Now if I assign some velocity to it, 
we get some nice dynamic FM. To navigate through different pages of the assignable mod destinations 
just hold shift and press the assign buttons, you can see the assignments for that page show up on 
the screen. Now we have seen how to assign the modulators, let's take a look at the rest of them. 
The Cycling Envelope is somewhere between an LFO and an Envelope, and may seem familiar to any 
modular users out there. It has three modes we can choose between Envelope Run which behaves like a 
mono LFO and doesn’t retrigger and Loop which behaves like a looping envelope or polyphonic LFO and 
retriggers Rise and Fall control the shape of the Envelope or LFO We can change the shape of these 
by holding shift while turning the knobs Note that in Envelope mode we have the same 2 options as 
we got on the main ADSR Envelope, but in Run or Loop we get a continuous curve so we can create 
really snappy LFOs with ease. Hold/Sustain is a sustain level when in Envelope mode or a hold time 
when in Run or Loop mode We can also change when this hold takes place with the Stage Order options 
in Cycling Envelope settings, allowing us to create gaps between snappier LFO shapes In these 
settings, we can also tempo sync the stages and choose the retrigger source including having it 
retriggered by the LFOs which can tie modulations together really nicely. For instance here LFO 1 
is triggering the Cycling envelope, locking their movements together Moving on to the LFOs, we have 
two available, they share the same features and we choose which LFO we are controlling with the 
Select switch. The Rate control can be unsynced or synced by clicking the knob And the rate is 
displayed via the LEDs above LFO section which really speeds up the process of knowing which is 
which while tweaking a sound. If we hold shift while turning the rate knob we can also choose the 
retrigger options for the LFO including turning it into a one-shot Or having it retrigger from the 
other LFO or Cycling Envelope, which is great for creating swung or complex modulation We can 
choose the shape of the LFO with the Wave knob and as you can see we have a bunch of options beyond 
the standard wave shapes including this final option: Shaper. This is where we can use custom 
waveshapes or sequences for each of the LFOs. We have some preset options we can quickly load in 
here to try it out by holding shift and turning the Wave knob You’ll probably find you need to 
increase the rate of the LFO to use these custom shapes as they are made up of multiple stages as 
you’ll see in a moment. To create our own custom shape we can enter Shaper Edit mode by holding the 
shift button and then pressing the LFO Select button. We use the sequencer steps and touch strips 
to edit the stages of our shaper, which can be up to 16 stages long with each stage lasting 1 LFO 
cycle hence us having to speed up the LFO before. If we press and hold the step we want to edit, we 
can set its level with the left slider and its curve with the right slider. We can also choose the 
shape of a step using the sequence length buttons to the right of the step buttons. The graphics 
underneath are the shape options. You can turn a step off by tapping it and also tie steps together 
to create longer shapes by holding the first step and then tapping the step you want to tie up to. 
From a Shaper, you can really quickly get a pattern by wiggling the level touch strip and just 
painting along the step buttons We can set the length of the shaper pattern by holding the Last 
Step button on the right-hand side of the sequencer length buttons and pressing the step we want 
the pattern to finish on. So now we’ve seen how it all works, let’s make something... If you like a 
Shaper pattern you can save it to one of the 8 User Shape slots to recall later, to start this 
process just hold save and press the LFO button. When we’re finished editing we can close the Shaper 
Edit mode by pressing the LFO Select button again. So that's the LFOs, now let’s take a look at some 
special features of the final few modulation sources in the Matrix. The first of these is 
Velo/Aftertouch which is a single mod source for both Velocity and Aftertouch, You can also choose 
to only use one of them in the Keyboard settings menu. For instance, if you have assigned velocity 
via Envelope settings. Then we have mod wheel assignments and keyboard. Keyboard can have a few 
modes which can be selected from the settings menu, including a random mode which generates a 
random value on every key press. This is a great way of quickly getting either subtle or wild 
variation into a patch, like this patch for instance which uses it to change the Engine Type of the 
oscillator on each note. Beyond the mod matrix and its modulation sources, we also have some extra 
modulation and performance options in the touch strip section of the synth. We choose the touch 
strip mode with these three buttons The first mode has them act as a Pitch bend & Mod Wheel. As 
well as assigning the mod wheel in the matrix, we can also turn it into a vibrato control to make 
use of the dedicated Vibrato LFO by holding shift and pressing the Bend/Wheel button. We can adjust 
the Vibrato LFO by holding the Bend/Wheel button and using the left touch strip to change speed and 
right touch strip to change its depth. We can also assign these as mod destinations from the 
additional mod destinations list in the mod matrix, hold assign and use the preset/edit knob as 
discussed earlier. This means if we assign the Vibrato amplitude in the matrix we can use it even 
when it is turned off and not controlled by the mod wheel. For instance, here I’ll turn it off and 
then control it with our other LFO. The second touch strip mode turns them into two Macro controls. 
These Macros can each have 4 assignments which we edit by holding shift and pressing the Macro 
button. We choose which macro to edit by touching its respective touch strip then if we turn a knob 
on the UI, it not only assigns it but also sets its macro offset amount We can navigate the list of 
assignments using the Preset/Edit knob If we click in an empty assignment slot, we can add the same 
additional mod destinations we get in the mod matrix If we click on an existing assignment, we can 
edit its value and we can click and hold to delete an assignment. One sneaky little trick we can do 
with these macros is to assign the Macro itself as a mod destination in the mod matrix. So we are 
able to modulate up to 4 parameters with a single matrix assignment. The final touch strip mode 
controls the sequencer so we’ll cover that in the next video. So that's modulation in the MiniFreak.
