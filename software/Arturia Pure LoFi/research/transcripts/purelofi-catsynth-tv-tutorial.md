CatSynth TV · "Arturia Pure LoFi - embrace imperfection! A detailed tutorial & review" · 2025-05-13 · 34:40

> Arturia Pure LoFi instrument (V Collection 11). Deepest control-by-control video walkthrough; the presenter is already using it in compositions.

**Architecture.** Look and feel familiar to anyone who's used Arturia **Pigments** or the Augmented series. Two oscillators / sound engines, each set to one of three oscillator types; a dedicated noise oscillator; a multi-type filter. The Advanced pane exposes effects + extra modulation. "Under the hood it uses the Pigments engine, but the interface exposes a subset of features optimized for the lo-fi ethos."

**The LoFi Processor (the heart, at the top).** Several **coloration modes** identified by retro floppy-disk icons. The big **LoFi knob** sets the overall amount of lo-fi processing. Six controls fine-tune the processing — "fancy lo-fi macro controls" that, under the hood, modify engine parameters (distortion types, tape models, etc.). Demo: starts clean, turns LoFi up → more overdriven. Walks the modes: **Velvet Frost** (cooler, tape sound), **Golden Age** (warmer analog tone + saturation), **Fuzzy Line, Cathodic Tube** (tube distortion + buzz), **Dim Memories** (degraded tape). The six fine controls (visible when the mouse is in the upper area, otherwise hidden under the Arturia logo): **Wobble** = wow & flutter of old tape recorders (a lot of the audible character comes from here); **Wear** = condition of older/worn tape; **Drive** = nonlinear analog processing ("warmth"/distortion); **Speaker** = vintage speaker cabinets.

**Main engines.** Each slot → one of three types.
- **Realistic Instrument**: standard multisampler layering multiple samples for conventional instrument sounds (e.g. a nylon guitar he keeps coming back to; classic Casio CZ sounds).
- **Hardware mode (per engine)** emulates vintage samplers, "the digital counterpart to the analog character of the lo-fi section": Clean → **Fairlight CMI** (aliasing + bit crushing) → **EMU Emulator II** → **Casio SK1** (loses a lot of frequency content, just like the real SK-1) → **EMU SP1200** → **Akai S900** → **Akai MPC60**, plus Arturia Custom **Deteriorate / Damage / Crush** (increasing bit-crush + downsampling).
- **LoFi Oscillator**: waveforms via analog modeling, wavetables, or harmonic generator. **Timbre** controls the harmonic profile. Examples: B3 organ wave, basic morphing sine/triangle/square, Yamaha CS-80 saw/square, "Bode Metal," "grains," Korg MS-20 feedback. Hardware modes work on the oscillator too (running it through SK1 loses frequency content).
- **Creative Sampler**: up to five samples, edited and played back various ways. Press **edit** for details; load different samples into slots A/B/C/D; play each in turn or switch to **round-robin** (each key triggers the next sample, then loops). Set loop points + loop mode per slot.

**Noise oscillator.** Two modes: continuous **texture** and **transient**. Texture demo = "forest rain" sample; large palette: FM digital aliasing noise, old J37 tape-machine feedback, fake vinyl crackles. Mixed with the engines + LoFi processor → rich worn lo-fi timbres. Transient mode = little noise bursts at the note start (e.g. a "metal hit" adding a sharper attack transient).

**Filter.** Three types. **Clean** = essentially the Pigments multi-filter (12 dB lowpass with gentle resonance shown; also highpass, 24 dB bandpass). **Analog** = modeled on the Oberheim SEM filter (LP/BP/HP). **LoFi** = the downsampling filter introduced in the latest Pigments — resamples the input at a lower rate for a "pleasant grungy quality"; modulate its frequency with the filter envelope. Dedicated **filter ADSR** + a separate **VCA/amp envelope** (staccato → very long stretched notes).

**Advanced page.** Larger amp/filter envelope visuals, four FX slots in series, and modulation sources (same as the JUP-8000 V / Pigments family). Demo: a **Function** generator assigned to the LoFi Oscillator's **Timbre**; a **Voice Modulator** modulating noise level per note; **Aftertouch** controlling the LoFi-Processor level. **Effects** include the new **Trance Gate** (change number of steps — e.g. 13 — remove steps for rhythmic patterns; gate shapes triangle/ramp; morph + speed; many presets), **Super Unison** (detuned copies, voice count + detune), **Reverb** (Cathedral preset, room size + decay), **Phaser** (Resonator preset). Effects order is free — reverb need not be last.
