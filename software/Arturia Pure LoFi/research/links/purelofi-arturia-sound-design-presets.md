Arturia (official) · "Pure LoFi — Sound Design" page · 2025

> Named factory presets with the sound designers' own descriptions of HOW they were built. The most concrete "copyable recipe" source for the drone/ambient/pad world, since each says which engines, LoFi params, macros and FX were used.

## Signal-flow confirmation
The page repeatedly describes the two-part lo-fi: **"modelling of old digital units on the single sound sources, and the tape emulation on the master output"** (= per-engine Hardware mode + the master LoFi Processor). Four macros recur by name: **Timbre, Brightness, Movement** (and per-preset custom macros) — these are the "play with one knob" controls baked into presets.

## Named presets + technique (ambient/pad-relevant first)

- **"Somber Mood"** — Jörg Hüttner. A **harp-based pad**, "long and reverb-filled." Built on the **Tape Processor modes + the lo-fi parameters, specifically Wear, Wobble, and Vintage.** A **Movement Macro** adds a rhythmic element to the sustained pad. (The most drone-relevant factory recipe: harp/plucked source → tape processor heavy on Wear/Wobble/Vintage → long reverb → Movement macro for motion.)

- **"Snow Ghosts"** — Vincenzo Bellanova. A **simple ambient pad** using **the fourth tape processor (= Dim Memories)** to add **density and warmth** to washed-out sounds. (Minimal source + a single coloration mode does the heavy lifting.)

- **"It Blossoms"** — Marco Iodice. A **lost piano** with "slow pitched delays and great reverberations." The **Timbre Macro manages the amount of sound deterioration**; the **Brightness Macro filters the sound, drastically changing the overall flavor.**

- **"Little Pleasures"** — Marco Iodice. A **sine wave + a noise texture** run through the tape processing. **Timbre Macro + Aftertouch + Mod wheel** dynamically alter the degradation quality — warmth reminiscent of a vintage electric piano. (Direct template for a degraded drone: simple sine + texture into the LoFi Processor, expression-controlled degrade.)

- **"Reversed Breeze"** — Laura Katic. A **soft, flowing pad with gentle drift, processed through effects.**

- **"Time Drift"** — Chad Beckwith. **Creative Sampler mode**, loading different electric/taped-piano samples across slots with **random triggering**; **vibrato and filter modulated by LFO/Function speeds derived from vinyl-record rotation rates** (≈0.55 Hz / 33⅓ rpm flavor), paired with a **slow filter sweep + ambient reverb + delay.** (A masterclass recipe for "warped record" motion.)

- **"Muted Memories"** — Zane Twice. A **soft piano** using **programmed Functions** to evolve from deeply muted to defined tones; macros add perspective and a sense of depth.

- **"Grand Reversal"** — Ale Mastroianni. **Heavily degraded piano** from **layered reversed piano samples**, controllable via the **mod wheel**; combines the per-source digital modelling with the master tape emulation.

- **"Corrupted Sub"** — Gustavo Bravetti. A **bass** using dynamic modulation to shift from soft to extreme lo-fi while keeping a powerful, evolving low end.

## Takeaway for the rig
The factory designers' shared playbook = **simple source (piano / harp / sine / reversed sample / noise texture) → LoFi Processor heavy on Wear + Wobble + Vintage → long ambient reverb + delay/tape echo → a macro (Timbre/Brightness/Movement) or expression (mod wheel/aftertouch) riding the degrade amount.** That is exactly the degraded-tape drone target — and it's all inside one instrument.
