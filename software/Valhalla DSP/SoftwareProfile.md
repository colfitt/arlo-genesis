# Valhalla DSP

**Summary** — Reverb/delay plugins. **Installed: VintageVerb + ValhallaRoom ONLY**
(Shimmer/Delay/Plate/ÜberMod/**Supermassive**/FreqEcho/SpaceModulator NOT installed —

`Supermassive-UsageGuide.md` (free ★) + `Shimmer-UsageGuide.md` + `Valhalla-Delay-UsageGuide.md`.

## Why I have it
The in-the-box ambient/drone reverbs — recallable, automatable, near-zero CPU, and
the studio-side complement to the hardware reverbs (Big Sky, H90, Microcosm).
**VintageVerb** = colored/vintage/character; **Room** = transparent/realistic
(next wave).

## Signature uses
- **VintageVerb** for walls of sound: **Chaotic** modes + **1970s color** = degraded
  tape drones; long Decay + big Size + slow **Attack** swell + **Mod** for evolving
  tails. 100% wet on a send; saturate the return (Decapitator/RC-20). The **Tycho
  two-insert + delay-after-reverb** wall. → `research/VintageVerb-UsageGuide.md`
- **Room** for **depth + lush walls without color**: it's a **dual-engine** reverb
  (separate Early + Late, crossfaded by **Depth**) — the transparent space *under*
  VintageVerb. Drone walls = a dark/slow mode (**LV-426 / Dark Space / Dark Room**) +
  Depth 100 + long Decay + big Late Size + **Early Send ~1.0** (slow swell) + Late Mod +
  **Bass Mult < 1** so lows don't mud out. Depth bed under VVV = **Depth 0% (Early only),
  Early Size < 40 ms.** 100% wet on a send; saturate the return. → `research/Room-UsageGuide.md`

## Notes
- **VintageVerb has NO Freeze and a 70 s decay ceiling** — for held infinite drones
  use Big Sky's Freeze or the free Supermassive. No post-reverb EQ on the tail (do it
  on the return). Mono behavior varies by mode (Sanctuary = mono-in/stereo-out).
  1970s/Dirty modes add real downsampling/quantization artifacts (that's the
  character).
- **★ Free adds worth installing (NOT on disk yet): Supermassive + FreqEcho + SpaceModulator.**
  **Supermassive** fills VVV's gaps (genuine infinite drone: pick a mode, **Feedback 100% + Warp
  + Density**, Mod 0% to hold pitch — the freeze VVV/Room can't do) → `research/Supermassive-UsageGuide.md`.
  **FreqEcho** = free Bode freq-shifter + analog echo (broken/clangorous drone delays). Paid
  **Shimmer**/**Delay** mostly skippable (Supermassive fakes shimmer; EchoBoy owns delay; H90 does
  real shimmer) — guides + buy-order in `research/Valhalla-Navigator.md`.
- **Room ≠ VintageVerb:** Room = transparent/realistic/anonymous **dual-engine** (Early
  + Late + Depth), 12 modes, decay to **100 s**, 3-band decay multipliers for surgical
  tails. VVV = colored/vintage/era-degradeable. Use Room for depth+placement, VVV for
  color; or stack (Room places, VVV colors). **Room has NO Bloom mode** (that's
  Shimmer), **no Halls** (that's VVV), **no Freeze, no real Shimmer** — bloom job = Bright
  Room / LV-426; Freeze = Big Sky / Supermassive; pitched shimmer = Shimmer / H90.
  **Dark Room is deliberately lo-fi** (11 kHz ceiling, noisy interpolation). **Plates are
  Room's weak point.** **Space** (v2.0.0) self-feeds the Early engine into a drone.
