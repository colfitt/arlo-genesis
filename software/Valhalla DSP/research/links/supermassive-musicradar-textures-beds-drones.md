# MusicRadar — "The perfect tool for textures, beds and drone effects"

Free-plugin spotlight tutorial on Supermassive for ambient beds/drones.

## Drone/texture recipe (insert on a pad/string)
- Load a sustained **pad or string**, bypass other processing first.
- Insert Supermassive; select **Pleiades** mode (their pick for texture/drone from
  sustained sources).
- **Warp ~50%** — *"introduces more delay mixing and left/right blending."*
- **Feedback** — push up to extend decay (author rides it live while auditioning).
- **Density** — *"control how smeared the sound is."*
- **Mix** — blend to taste; more wet = more prominent wash.

## Live-performance trick
*"Adjusting Warp and Delay controls in real time introduces temporary pitch changes
to the effect"* — ride Warp/Delay live to build evolving, pitch-bending texture
movement (great for drone evolution / risers).

## Insert vs send
For a **spot effect** on one dedicated sound (e.g. a "drop boom"), load it as an
**insert** after the channel's other processing. (For a shared ambient wall use a
100%-wet send instead — standard send-reverb practice.)
