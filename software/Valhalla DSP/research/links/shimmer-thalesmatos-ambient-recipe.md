## Concrete ambient settings (reported)
- **Feedback**: as you add it, "it gets more and more chaotic, but for ambient styles
  it's typically kept UNDER 50%." → so the doom/drone move is the opposite of the Eno
  octave-climb: keep Feedback modest (<50%) for a controllable evolving pad, push it high
  only when you want the runaway climb.
- **Diffusion**: comes "at almost 90% by default" (immediate reflections); **lowering to
  ~60% creates a reverse-reverb effect** (slow fade-in swell). ~0.5 = "fairly high echo
  density."
- **Mix / send**: "**Start with Mix ~20–30% on a send.** Go **100% wet only for pads or
  transitions.**" → so: send at 20–30% for an instrument that should stay legible; full
  wet only when the Shimmer IS the pad.
- **Pitch / Shift**: octave **up OR down** (±12) for the classic shimmer; "the pitch shift
  only affects sound when Feedback > 0."
- **Size + Diffusion** set the space: larger = slower, softer bloom.

## The evolving-pad idea (reported)
Pitch-shifted feedback in the tail makes each repeat bloom upward (or downward) in pitch →
"massive, evolving pads and drones that feel like they're breathing." For a darker/haunted
wall, use −12 instead of +12.
