## STYLE = delay-line routing
- **Single** — one delay time for L+R, same mod waveform; SPREAD adds slight L/R offset
  for a 3D image.
- **Dual** — independent DELAY L / DELAY R, each feeds back on itself, no cross-feedback
  = two parallel delays.
- **Ratio** — LEFT set by DELAY knob; RIGHT = RATIO × left. Feedback combines through a
  unitary matrix. **RATIO 61.8% + higher feedback = a "cool pseudo-reverb" (golden-ratio
  reverb).** ★
- **PingPong** — L+R summed into the left delay, then feedback alternates channels;
  supports two different delay times; WIDTH −100% flips which channel sounds first.
- **Quad** — four taps (A/B/C/D), each toggleable, SPACING sets tap intervals. REPEAT/
  SWELL button sets feedback: REPEAT = recirculate the last tap (multi-head); **SWELL =
  feed the SUM of all taps back = blooming, building wall** (RE-201/RE-301 multi-head
  swell). ★ the self-building ambient move.

## Key controls (with values)
- **DELAY L/R + Sync** — ms or tempo (notes / dotted / triplet).
- **FEEDBACK** — self-oscillation "usually begins around 100%"; **goes to 200%** for
  howling self-oscillation.
- **WIDTH** — −100…+100%; **0% mixes L+R** → mono + strong flanging.
- **DIFF Amount** — 0% = off; ~68% = balanced; **91% = reverb from the diffusion network**
  (delay → reverb wash). **DIFF Size** — small = blurs attack but still a delay; large =
  transforms into a reverberant sound. (Spans up to ~20 s of delay space without sounding
  sparse.)
- **MOD Wow/Flutter** (Tape) · **MOD Rate/Depth** (HiFi/BBD/Digital).
- **PITCH Shift/Detune** (semitones/cents) · **FREQ Shift/Detune** (Ghost, Hz).
- **COLOR Drive** (input saturation dB) · **COLOR Age** (mode-specific artifacts).
- **ERA** — variations of the active mode (filters, saturation, diffusion behaviour).
- **EQ High / EQ Low** — high-cut / low-cut **in the feedback path** (repeats darken/thin
  as they recirculate = the tape-into-the-distance behaviour; how you keep a wash clear).
- **MIX** — 0 dry / 100 wet / 50 equal. (MIX LOCK by clicking "MIX" to keep the wet level
  while preset-browsing — useful on a 100%-wet aux.)
