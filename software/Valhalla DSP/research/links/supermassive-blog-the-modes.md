# Valhalla Supermassive — The Modes (authoritative, original 8)

The official descriptions of the launch modes. Each is a different FDN topology; the
listed character = attack / decay / echo-density behaviour. (Later modes added by
update — see supermassive-version-history-modes.md.)

- **Gemini** — *the most "conventional"* mode. **Very quick attack, natural exponential
  decay.** Lower DENSITY = sparse/pointillistic; higher = smooth reverb. → smooth reverbs.
- **Hydra** — **fast attack, very lush + long exponential decay**, *very* sensitive to
  DENSITY (0% = simple echoes → high = complex reverb/echo structure). → lush complex
  reverbs.
- **Centaurus** — **slow fade-in/out**; attack **tied to FEEDBACK** (higher feedback =
  faster attack), shorter decay with sustain, medium density. → slow swelling reverbs.
- **Sagittarius** — **VERY long attack** (depends on FEEDBACK), slow decay; *"slow to
  get going, slow to go away."* Builds sonic mass/inertia. → echoes that fade in slowly /
  massive reverbs.
- **Great Annihilator** — slow attack with a fair amount of predelay, **super-long
  decay**; like Centaurus but MORE MASSIVE. → super-long decays.
- **Andromeda** — **very slow attack + predelay**, **decay into the THOUSANDS of
  seconds**; the supermassive variant of Sagittarius. → the longest, most smeared drone.
- **Lyra** — **quick attack, long decay, VERY low echo density**; lives more in the
  "echo" domain than "reverb." → spacey echoes that gradually diffuse.
- **Capricorn** — **quick attack, long decay, medium density**; a larger Lyra → between
  echo and reverb. → pointillistic reverbs + lush chorus. (Spring-like at tight delays.)

★ For this rig: **Andromeda / Great Annihilator / Sagittarius** = the long-smear drone
family; **Gemini / Hydra / Lyra / Capricorn** = the clean-sustaining infinite modes (see
infinite post); **Centaurus / Sagittarius** = slow-swell pads.
