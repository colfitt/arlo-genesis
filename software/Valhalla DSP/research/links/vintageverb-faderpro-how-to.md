Clean parameter-by-parameter how-to. Best for the fundamentals (predelay, send vs insert, EQ vs
damping).

## Predelay for clarity
- Generous predelay = hear the dry source distinctly before the reverb enters (keeps walls from
  swallowing the attack). Low predelay = immediate blend. Crank predelay to keep a drone's pitch
  readable under a long tail.

## Send vs insert (the Mix dial)
- Mix blends Dry↔Wet: 0% = dry, 100% = wet. "100% wet is the choice if you're using VintageVerb
  on an Auxiliary or Bus." → On a send/aux, set Mix 100% and control level with the send.

## EQ vs Damping (two different jobs)
- **Damping** shapes reverb TONE over time — HF limit + High Shelf contour (how the tail darkens
  as it decays).
- **EQ (HiCut/LowCut)** controls the VOLUME of the reverb shape at low/high — a static filter on
  the wet, distinct from damping. Use LowCut to keep low drones from turning to mud; HiCut +
  damping to pull the tail back into the fog.

## Decay & Size for ambience
- Decay Time max = 70 s. Size/Attack design the virtual-space proportions. Drums = Ambience mode
  + short decay; ambient walls = large Size + long Decay.

## Color modes
- 1970s = noisier, less hi-fi. 1980s = rich modulation, brighter, pristine. NOW = cleanest/most
  transparent/digital.

## Suggested workflow
Pick Mode first (Plate=vocals, Concert Hall=cinematic, Ambience=drums) → set Decay Time → set
Diffusion (early vs late reflections) to taste for the source.
