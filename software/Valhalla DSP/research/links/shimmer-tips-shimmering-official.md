The canonical Eno/Lanois "shimmer" recipe, from the designer.

## The classic shimmer settings
- **Reverb Mode**: Medium Stereo or Big Stereo — "the smoothest shimmer sounds."
- **Pitch / Shift**: **+12 semitones** (octave up).
- **Feedback**: **0.5 or greater** — below that the pitch climb is too faint; 0.5+ makes
  each repeat audibly bloom an octave higher.
- **Diffusion**: **~0.9** for a reverberant shimmer wash. **Below 0.5** instead gives a
  pitch-shifted *echo* effect (discrete climbing repeats, not a smooth wall).
- **Color Mode**: **Dark** — "produces a natural roll-off of high frequencies, which
  eliminates almost all of the aliasing noise in the feedback path." Essential for a
  clean octave-up climb (the aliasing is worst on the highest, most-shifted content).
- **Mod Depth**: keep LOW initially — the pitch shifting is itself a source of modulation.

## Pitch-mode choice for shimmer
- **Single / Dual** = more noise/grain in the decay → better for orchestral/dense textures.
- **SingleReverse / DualReverse** = much smoother → better for organ-like, glassy shimmer.

## Workflow order
1. Set Feedback for the pitch-shift intensity (how hard it climbs).
2. Set Size for the decay character.
3. Fine-tune Diffusion for how reverberant vs echo-like it is.
