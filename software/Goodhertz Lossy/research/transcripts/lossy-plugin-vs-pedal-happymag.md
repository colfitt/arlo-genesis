Comparison of the Goodhertz Lossy plugin against the Chase Bliss x Goodhertz Lossy
hardware pedal. Useful here mainly as a clean recital of the plugin's control set
and the plugin-vs-pedal division of labor (the owner has the hardware-degrade
axis covered by Generation Loss / Big Time, so this frames where the software fits).

## The plugin's controls (as listed)
The Lossy plugin is described as a tool to "replicate vintage digital playback
devices with modern controls." Its feature set:

- **Reverb with precise spatial control** — a built-in verb (pre/post position).
- **A flexible filter section** for digital frequency shaping (low/high cut + slope).
- **Loss modes**: Standard, Inverse, Phase Jitter, Packet Repeat, Packet Loss,
  and hybrid combinations (Standard + Packet Loss / Standard + Packet Repeat).
  These introduce "digital imperfections and compression effects allowing for
  unique sonic textures and character."
- **Advanced controls**: Master Mix, Verb Decay, Auto Gain, Gate Threshold,
  Stereo Mode, and Weighting options — "further sonic control and creativity."

## Plugin vs pedal (the takeaway)
- **Plugin** = "digital precision and detailed sound design," intuitive GUI
  tailored for studio/DAW workflows and electronic-music production. Full preset
  recall, deep advanced page. The studio/back-end tool.
- **Pedal** = tactile analog control, low-fi modes, freeze function for evolving
  ambient textures, a built-in limiter, all-wet/slow modes — "expressive
  real-time adjustments" for live performance and pedalboards. The played-in,
  hands-on, real-time tool.
- Both share the core Loss modes (Standard, Inverse, Phase Jitter) and the
  packet controls (packet loss / packet repeat). The pedal adds a freeze and
  limiter; the plugin adds full advanced controls + recall.
- Verdict: choose the plugin for "digital precision and studio integration,"
  the pedal for "tactile analog control and live-performance versatility."
