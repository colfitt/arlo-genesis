Practical, settings-level guidance on the **Choir Vocal Multiplier** from hands-on reviewers — the "how it actually sounds, what to dial" layer the official guide doesn't give. Triangulated across the musicngear musician review + the SoS AVOX review:

**Size, by ear:**
- **4 and 8 voices = most realistic** (SoS: "the smaller four- and eight-voice choir sizes sounded the most realistic"). 8–16 with moderate variation = convincing pro double-tracks.
- **16 / 32 = bigger but more "smeared chorus" risk.** musicngear got "a huge ensemble sound without utterly smearing intelligibility" at **32 voices + subtle EQ + reverb**. So the wall is achievable at 32, but you must clean it up.
- **Build big choirs from MULTIPLE INSTANCES, not one maxed track** — both reviews say layering 2–4 instances across separate parts beats cranking one instance to 32. (Matches the official "multiple instances = realistic large ensemble.")

**Variation controls:** "each knob has an audible effect, very little menu diving." **Moderate variation = realistic doubled-takes; pushed to extremes = synthetic/chorus-y.** So for the rig: *moderate* for believable thickening, *extreme* deliberately for the synthetic detuned wall.

**Stereo Spread:** spreading voices "added a three-dimensional sheen that sat well in the mix." Width is where the value is.

**Realism depends on reverb:** SoS — "slightly artificial in isolation, but **far more realistic when you add concert-hall reverb**." Reverb after Choir isn't optional polish; it's what sells the ensemble. (Straight into Valhalla VintageVerb for the rig.)

**Instruments / non-vocal:** confirmed — SoS: "these effects **also work well on monophonic instruments**"; musicngear: "works on monophonic instruments too, but its voicing is optimized for the human voice so results vary by source." Sanctioned for banjo/baritone/synth drones, expect voice-flavored results.

**Gotchas (real):**
- **CPU scales with voice count** — 32 is the heaviest; reasonable to run multiple smaller instances. Print/freeze in Logic if stacking.
- **SoS flag:** in the older AVOX version, "**only the left channel is processed if you use the plug-in in stereo mode**" — i.e. it multiplies a mono source; feed it a true mono signal (or mono→stereo) and don't expect it to handle an already-stereo source per-channel. Consistent with the manual's mono-source / mono-to-stereo requirement for Stereo Spread.
- It does **not** create harmony or change pitch — it only multiplies the pitch that's there. For intervals/chords use Harmony Engine; for a unison cloud on a held note, Choir.

Rig recipe distilled: **held drone/banjo/baritone/synth note → Choir (mono→stereo) → Size 16–32, moderate-to-high Pitch/Timing/Vibrato Variation, Stereo Spread ~70–100 → Valhalla VintageVerb (long, concert-hall)**. For a stacked choral *chord*, run 2–3 instances on different held pitches (or after Harmony Engine's voices) rather than one maxed instance.
