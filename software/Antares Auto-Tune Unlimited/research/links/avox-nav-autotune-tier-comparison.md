# Auto-Tune tier ladder — what each edition adds (so you know which to open)

Auto-Tune Unlimited installs FIVE pitch engines (Pro, Artist, EFX+, Access, plus Slice which is
a separate sampler — Hybrid is the Avid-DSP/live version and is NOT in the AU/VST3 CONTENTS list).
They're cumulative tiers of the SAME tuning engine; the question is only how many knobs you get.

## The ladder (each adds to the one below)
- **Access** (entry) — automatic pitch correction ONLY: Key, Scale, **Retune Speed**, **Humanize**.
  No Graph, no Classic Mode, no MIDI, no formant. **Lightest CPU** of the family — its whole reason
  to exist is "tune in a session that's already maxed out."
- **EFX+** — adds adjustable retune + transpose + **throat/formant modeling** + the **Auto-EFX**
  and **Auto-Motion** modules (Auto-Motion = a melodic-pattern/arpeggio generator that *forces* the
  pitch through a programmed melody — a built-in creative-effect toy). Positioned as "effect-first."
  No Graph mode, no Classic Mode in the older positioning (see note).
- **Artist** — adds **Classic Mode** (the Auto-Tune 5 "the sound" voicing), **Flex-Tune** (note
  protection), **vibrato editing**, and **real-time MIDI control**. Streamlined pro Auto view, no Graph.
- **Pro (Pro X)** — adds **Graph mode** (graphical pitch + time editing, per-note hand drawing),
  back-compat with older Auto-Tune, and bundles the **Auto-Key** plugin. The flagship.

## Reach-for in THIS rig (drone/doom/banjo, Auto-Tune-as-EFFECT)
- **Pro X = the one you open.** Graph mode is the single most creative feature here: draw a mono
  banjo/synth line to *wrong* pitch targets for a designed inhuman melody (not possible in any
  lower tier). Retune Speed 0 + Classic Mode for the hard robotic snap.
- **EFX+ = the fun second choice.** **Auto-Motion** auto-generates a stepped melodic pattern from a
  held/sustained note — feed it a drone and it spits out a rhythmic robotic arpeggio for free. The
  fastest path to "effect now" without touching Graph. Throat control is built in.
- **Artist / Access = skip in this rig.** Artist is just Pro-minus-Graph (Graph is the reason to
  use Pro creatively). Access is the CPU-saver auto-tuner — only reach for it if a heavy
  Choir/Harmony-Engine session is already redlining and you just need quick correction on a part.
- **Hybrid** isn't installed (Avid DSP / live-console build); irrelevant to a Logic/Ableton rig.
