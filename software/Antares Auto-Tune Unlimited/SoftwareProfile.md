# Antares Auto-Tune Unlimited

**Summary** — Subscription bundle of **22 installed plugins** (8 AVOX + Harmony Engine 4x +
Mic Mod + Auto-Key + 6 Auto-Tune editions + Slice + SoundSoap + Vocodist + Vocal Comp/EQ/
De-Esser/Reverb). Deep-researched (2nd pass): a full roster `research/Antares-AVOX-Navigator.md`
+ flagship deep-dives for **Harmony Engine, Choir, Throat, Mutator**. (Plan/memory's "41
components" was wrong — verified 22 against CONTENTS.md; the marketing "Sybil de-esser" is
NOT installed, only the standalone Vocal De-Esser.)

## Why I have it
Not a vocal-forward studio — so this bundle earns its keep as a **sound-design / texture**
toolkit, not a tuning utility. The pitch engine becomes an *effect* (the robotic snap) on
voice or a monophonic banjo/synth line, and the **AVOX** suite + **Harmony Engine** + **Vocodist**
turn a single held note into alien textures, detuned choir walls, played choral beds, and
vocoded machine-choirs — fuel for the drone/doom/shoegaze "recorded-wrong, sustained wall"
aesthetic. The vocal-mix utilities (Comp/EQ/De-Esser/Reverb) are redundant with the rig's
existing tools. Subscription = always-latest, everything unlocked.

## Signature uses
- **Hard-tune-as-effect** — Retune Speed 0, Flex-Tune/Humanize/Vibrato 0, Classic Mode ON, on a
  clean **mono** banjo/synth/voice line (sparse scale = more dramatic stepping).
- **MUTATOR** for alien/monster/inhuman texture (24 ring-mod mutations + throat reshape + tempo-
  synced Alienize reverse-chop) — under a drone or as a foreground creature.
- **CHOIR** (Size 32 + variation + spread) = one held note → shimmering detuned choir wall;
  layer instances for chords.
- **Harmony Engine 4x** in MIDI Omni (played from the 61SL MkIII) + per-voice Choir multiplier +
  Freeze = a held, breathing drone-choir from one instrument input (works on instruments, not
  just voice). The **Prismizer** (slow Glide + MIDI-played voices) = the documented Bon Iver /
  Frank Ocean smeared-choir wall — the rig's one real genre-adjacent artist technique.
- **Auto-Tune Slice** — the bundle's hidden gem: it **sidesteps the mono-input limit**, slicing
  & tuning ANY phrase (banjo, vocal, found-sound) into a playable MIDI chop-instrument with 14
  built-in FX. Pairs with the MPC / 61SL workflow. (**EFX+ Auto-Motion** = generative robotic
  arpeggio from a held note.)
- **Vocodist** with an external **carrier** = drone/pad/VG-800/S08 chord vocoded by a played
  banjo/voice → robotic machine-choir texture.
- **THROAT** for beyond-anatomy / designed formants; **WARM** as a tube/tape saturation stage;
  **Mic Mod** as character-EQ coloration.

## Notes
- **Mono only:** Auto-Tune / Mutator / Throat track a single pitch — banjo *strums*/double-stops
  won't track; comp a clean single-note take. For chords, *generate* harmony with Choir / Harmony
  Engine instead of tuning a chord.
- The audible part of hard-tune lives in **note transitions** — scoop/glide and don't hold
  sustains for a stronger effect; push input off-pitch for bigger snaps.
- **CPU:** pitch-tracking + ring-mod + 32-voice/choir-multiplied settings are heavier than static
  plugins — print/freeze in Logic.
- **Subscription:** if it lapses, the plugins stop authorizing. All AU + VST3 (Logic AU-fine).
- **Honesty:** ~all Antares presets/tutorials assume a pop/rap lead vocal; most creative recipes
  here are by *capability + technique*. The **one** documented genre-adjacent credit is the
  **Prismizer** (Harmony Engine — Bon Iver / Frank Ocean / Francis & the Lights); no verified
  drone/doom/shoegaze credit exists for Choir/Throat/Mutator. Antares' own pages are JS-walled →
  several specs triangulated from PDFs/reviews (labeled in the captured files).
