# Motions — Lo-fi Ambient for Ableton Live

The flagship ambient/texture pack — the most rig-relevant of the Reverb Machine
sample packs.

## Contents
- **50 lo-fi ambient patches** built around **128 recorded samples** sourced from
  cassette loops, granular synth devices, and guitar pedals.
- Delivered as **Ableton instrument racks** with custom effect chains baked in.
- Each preset has **2–5 Macro Variations** (accessible via the Instrument Rack's
  Macro Variation list) — so 50 patches give well over 100 usable starting points.
- **3 bespoke effect racks** included separately and reusable on any track:
  - **Tape emulator** rack
  - **Pitch-shifting quad delay** rack
  - **Shimmer reverb** rack
  These "add an ethereal character... whilst being fully flexible to adjust to your
  own taste."

## Requirements / Lite caveat (IMPORTANT)
- **Ableton Live 11 or later** required.
- **Compatible with ALL editions: Lite, Intro, and Standard.** Confirmed by the
  product page. So Motions loads on the user's **Live 12 Lite** — the racks are built
  from stock devices that exist in Lite (Sampler/Simpler + EQ/Delay/Reverb-class
  effects), not Suite-only Max-for-Live devices.
- Will NOT function on older Ableton versions (pre-11).

## Price
$30 USD.

## Designed for
"Producers and musicians working in ambient music, soundtracks, or looking to add
unique textures." Emphasis on **dynamic movement and evolving textures** — keyboard-
playable, layered soundscapes. The whole point is "motion": the patches modulate /
evolve while held, which is exactly the sustained-wall / drone use-case.

## Rig note
Because the samples are cassette-loop / granular / pedal recordings, the raw WAVs
(if extractable from the pack) could be re-mapped into Kontakt to build custom drone
instruments — but as shipped this is an Ableton-rack product. In Logic, the cleanest
path is to run it in Live 12 Lite and resample/bounce textures to WAV, then bring
those into Logic / Kontakt.
