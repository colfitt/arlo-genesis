# Reverb Machine Remake Projects — paid learning templates

Paid **Ableton Live project files** that recreate famous songs. Each includes **frozen
tracks** (playable without owning the plugins) + **multitrack WAV stems + MIDI** (any DAW).
$15–$40 each. "For learning and educational purposes only" — not royalty-free.

## Most rig-relevant
- **Vangelis – Tears in Rain** ($25): CS-80 brass, VP-330 strings, Rhodes melodies. Uses
  Arturia CS-80 V4 / Prophet-5 V / Stage-73 V2 + Valhalla VintageVerb (all owned).
- **Aphex Twin – Selected Ambient Works 85–92** ($35): lo-fi ambient reconstruction —
  DX7 pads, cassette saturation, vintage drum machines.
- **Rival Consoles** ($35): organic Prophet-inspired evolving-modulation textures.

## Why it matters for this rig
- The **frozen-track + stems + MIDI** format means even in **Logic** (no Ableton project
  import) the user gets usable WAV stems and MIDI to study/reuse.
- These templates are arguably the deepest learning resource Reverb Machine sells — you see
  the exact arrangement, routing and plugin chains (many of which the user owns) rather than
  just presets.
- Caveat: native project = Ableton Live 11/12; the user's **Live 12 Lite** opens Live
  projects but Lite's track/scene caps and missing Suite devices may prevent a full project
  from loading intact — the stems+MIDI path is the safe one in Logic.
