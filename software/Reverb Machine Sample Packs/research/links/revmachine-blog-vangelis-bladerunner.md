# How Vangelis Created "Tears in Rain" — CS-80 cinematic pads/brass

Free synth presets + a paid Ableton project/MIDI/stems link at the article. The remake
was built with **Arturia CS-80 V4, Prophet-5 V, Stage-73 V2, and Valhalla VintageVerb**
— ALL owned in this rig.

## The three layers
1. **Fender Rhodes (→ Arturia Stage-73 V2)** — main melody, onboard graphic EQ + chorus/
   ensemble.
2. **Roland VP-330** — lush string-pad / vocoder-string foundation, automated tone control.
3. **Yamaha CS-80 (→ Arturia CS-80 V4)** — brassy swells using the CS-80's expressive
   controls.

## Key techniques (copyable)
- **Velocity + polyphonic aftertouch** on the CS-80 layer for note-to-note brightness/
  intensity → "almost orchestral." Map aftertouch to filter cutoff / brightness.
- **Ribbon controller** for dramatic descending pitch slides at the end → emulate with the
  pitch-bend wheel / MIDI pitch-bend automation.
- **Reverb via send/return** (Lexicon 224), not per-track inserts — one big cavernous send
  for cohesive depth. Recreate with **Valhalla VintageVerb on a send**, Space-Echo-style
  delays.
- Prioritise **expressive real-time performance over static patches** — play the swells in.

## Rig application
This is a turnkey recipe with owned tools: CS-80 V4 brass + Stage-73 Rhodes + VintageVerb
on a send. The Novation 61SL MkIII can send aftertouch/CC for the brightness modulation.
For drone/doom, hold the CS-80 brass swell and let the VintageVerb send tail bloom; bow the
banjo or guitar underneath as the moving "vocal" layer.
