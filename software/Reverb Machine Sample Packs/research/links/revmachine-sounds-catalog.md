# Reverb Machine — full sound-pack catalog

Reverb Machine sells two broad kinds of products:
1. **Synth preset packs** — banks of presets FOR a specific 3rd-party synth (mostly
   Arturia V Collection / Analog Lab, TAL U-NO-LX, Cherry Audio, Arturia Pigments,
   Arturia MiniBrute V). These are `.fxp`/Analog Lab banks, NOT samples — they only
   work if you own that exact synth.
2. **Ableton-native sample/rack packs** — `.adg` instrument & drum racks built from
   recorded WAV samples (cassette loops, hardware drum machines, vintage keyboards).
   These are the ones relevant to a sampler/texture workflow.

## Ableton-native sample + rack packs (the rig-relevant ones)
- **Motions** — $30 — 50 lo-fi ambient patches from 128 recorded samples (cassette
  loops, granular synth, guitar pedals) + 3 custom effect racks (tape emulator,
  pitch-shifting quad delay, shimmer reverb). Ableton Live 11+. **Works in Lite/Intro/
  Standard.** The most drone/ambient-aligned pack.
- **DrumVerse R8** — $15–30 — complete Roland R-8 drum-machine sample set + Ableton
  Live racks emulating its functionality.
- **Drumtraks** — $20 — 195 sampled sounds from a 1984 SCI Drumtraks; kits for Ableton
  Live, FL Studio, Logic Pro X (so WAV-based, multi-DAW).
- **Vice** — $10–20 — 326 LinnDrum samples + Ableton Live Pack.
- **VP-30** — $30 — sampled Roland VP-330 strings/choir/vocoder → Kontakt, Decent
  Sampler, AND Ableton Live instrument packs. (Multi-sampler — Kontakt-ready.)
- **PS-48** — $30 — Yamaha PSS-480 FM "Phoenix-style" samples → Kontakt, Decent
  Sampler, Ableton Live. (Kontakt-ready.)
- **RM-20²** — $30 — sampled Yamaha PS-20 organ/keys (Beach House-style organs), Ableton.
- **Omni²** — $20 — Suzuki Omnichord recreation, Ableton instrument racks.
- **Monolith** — $20 — Korg Mono/Poly samples, 80 presets + MIDI device routing, Ableton.
- **Horizons / Wavetable Presets** — Ableton-native (Wavetable Presets = 120 presets
  for stock Ableton Wavetable, works in any edition that has Wavetable).

## Key takeaway for this rig
Most Reverb Machine "packs" are **synth presets**, not samples — they need the matching
synth. Only a handful are true sample/WAV/Ableton-rack packs (Motions, the drum packs,
VP-30, PS-48). VP-30 and PS-48 ship Kontakt/Decent Sampler versions = directly loadable
in Logic via Kontakt. Motions is Ableton-rack-based but explicitly Lite-compatible.
