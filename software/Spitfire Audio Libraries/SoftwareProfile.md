# Spitfire Audio Libraries

**Summary** — Orchestral / cinematic / evolving-texture sample libraries. **Installed
plugins:** LABS, BBC SO, Fractured Strings, Ólafur Arnalds Cells, Originals Rare
Flutes, Resonate. **On the (offline) drive:** Symphony Orchestra, Loegria, Orchestral
Swarm, Symphonic Strings Evolutions.

## Why I have it
The **evolving-texture / string-wall** source set — layered under banjo/baritone into
Valhalla/EchoBoy/Decapitator. Three evolving-wall poles owned: **SSEvo** (symphonic,
scripted Evos), **Orchestral Swarm** (micro-detune shimmer), **Ólafur Cells**
(intimate generative chamber strings) — plus **LABS** (free textures) and the
sound-design pair **Fractured Strings** (refracting string texture) + **Resonate**
(resonance-after-strike / metallic drones).

## Signature uses
- **Generative bed (Cells):** Played mode, soft velocity → record a held progression →
  **offline-bounce** to a wall → banjo over it. → `research/Olafur-Arnalds-Cells-UsageGuide.md`
- **Free drone walls (LABS):** one patch per layer, long attack/release, EQ off the
  top, 100% wet through Supermassive; **Cinemagnetic** (Amulets) is the bullseye. →
  `research/LABS-UsageGuide.md`
- **Designed texture:** Fractured Strings ("seasoning, not the dish") + Resonate
  (metallic resonant washes) for the non-realistic colour.
- **Real-orchestra ghost (BBC SO):** Super Sul Tasto / Long Flautando / Con Sordino /
  Harmonics, mod-wheel (Dynamics) low for the breathy pp layer, in-plugin reverb OFF →
  **Close mic only → Valhalla Room→VintageVerb → EchoBoy/Decapitator** (feed reverb the DRY
  mic; baked hall mics ring metallically). Bounce→PaulStretch for an evolving wall under banjo.
  The Jóhannsson move. → `research/BBC-Symphony-Orchestra-UsageGuide.md`
  **⚠️ Edition unconfirmed:** Discover (free) = Longs only (no sul tasto etc.) → use LABS Frozen
  Strings; **Core = the sweet spot** (all articulations); Pro adds mics/leaders. Confirm on disk.

## Notes
- **Three hosts:** dedicated plugins (LABS/BBC SO/Fractured/Resonate/Cells/Rare
  Flutes — native AU) vs **free Kontakt Player** (Loegria/Swarm/SSEvo/Symphony — load
  inside **Kontakt 8**) vs full Kontakt. → `research/Spitfire-Navigator.md`
- **★ Drive:** content is on the offline `PlaySomeGodDamnMusic` drive (Mac OS Extended
  Journaled) — **mount it BEFORE launching Logic** or libraries show "not found"
  (per-product **cog → Locate** to relink). The AU plugins themselves always load.
- **LABS was rebranded "Splice INSTRUMENT" (late 2025)** — installed copy keeps
  working into 2026; no subscription needed for the drone use case.
- **Symphony Orchestra (SSO)** — the big drive-resident Kontakt orchestra — now deep-dived:
  `research/Symphony-Orchestra-UsageGuide.md` (17 sources). The **WET Lyndhurst-hall** counterpart
  to BBC SO's dry Maida Vale: the hall IS the sound (even the close mic is roomy — you can't dry
  it), so degrade the *character* (EchoBoy/Decapitator), don't add reverb. Reach-for: "want the
  gorgeous hall baked in" → SSO; "want to build the space" → BBC SO; "alive evolving texture" →
  Swarm/SSEvo/Cells/Fractured. Runs in **Kontakt 8** (not a native AU); ~345–368 GB on the drive.
