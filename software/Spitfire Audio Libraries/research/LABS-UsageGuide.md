# LABS — Usage Guide

Spitfire **LABS** is the free, sample-based **texture instrument** — minimal
interface, real instruments often "recorded wrong," running in **Spitfire's own
player (not Kontakt)**. For this rig it's a deep well of free, hidden-gem
evolving/lo-fi/textural patches to **layer under the banjo/baritone** and push into
Valhalla/EchoBoy/Decapitator. It's a *source* with one expressive dial — the "warp"
happens in your post chain.

> **★ Rebrand note:** LABS was folded into **"Splice INSTRUMENT"** (late 2025) —
> `labs.spitfireaudio.com` now redirects to splice.com. **Your installed copy keeps
> working into 2026**, all installed packs keep loading, and projects play back
> unchanged. Only *new* free content moved (monthly "Free Drops," 500+ presets). No

## 1. What LABS is + the plugin controls

Identical minimal interface across patches: `research/links/labs-plugin-controls.md`, `research/links/labs-technique-tips.md`
- **Expression slider (CC11)** — ride for swells on held notes.
- **Dynamics slider (CC1 / mod wheel)** — crossfades velocity layers (fuller/
  brighter); on some patches reassigned to **cutoff/brightness**. The primary
  "make it move" control.
- **The rotary dial** — defaults to reverb, but per-instrument it's a different
  "warp" param. **★ Hidden trick: click the centre of the dial** to expose baked-in
  **Attack / Release / Sustain / Decay / Variation** — the key to **long-attack /
  long-release drone behaviour.**

## 3. Technique — building walls from LABS

The gold-standard rule across every good tutorial: **one LABS patch per layer, never
one patch doing everything.** `research/transcripts/labs-drones-pads.md`, `research/transcripts/labs-track-only.md`
- **The drone-wall build** (DaveMarks): 3 LABS instruments split by keyrange (pad L /
  dirt-bass mid / pad R), each with **long attack + long release** for breathing,
  aggressive **EQ off the top** to carve space, one pad run **100% wet through a
  plate** (Valhalla Supermassive). **The "orbit" trick:** two copies of the same
  patch with different settings drift in/out of phase for free modulation.
  **Keyrange-split** so the dirty sub stays out of the highs.
- **The arrangement method** (Larsen): stack ~10 instances and grow layers in by
  **automating EQ (highs out → highs in)** rather than volume fades; add a
  movement/LFO modulator for motion.
- **Rig integration:** LABS gives the source + one subtle dial; the real warp is your
  **post chain** — into Valhalla/Supermassive + EchoBoy for the wall,
  Decapitator/SketchCassette for grit, under the banjo/baritone lead. Ride mod-wheel +
  expression for life.

## 5. Common pitfalls / gotchas

- **The Splice rebrand** changes only where *new* free content lives — installed
  packs and the standalone plugin keep working; **don't pay $12.99/mo** for the
  drone/ambient use case (the free catalogue already covers it). `research/links/labs-splice-instrument-rebrand.md`
- **It's a source, not an effect** — the dial is subtle; do the heavy lifting in the
  post chain.
- **Community usage depth is thin** — the YouTube tutorials carry the weight (no
  strong Reddit/Gearspace drone-technique thread surfaced).
