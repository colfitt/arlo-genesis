# Spitfire Fractured Strings — SOS review (4/5)

**What it is.** A chamber ensemble of **eight string players** plus patches featuring **violin and cello soloists**. Made with Hans Zimmer's Bleeding Fingers Music collective; featured in the BBC's *Frozen Planet II*. Recorded at AIR Lyndhurst Hall.

**Format.** Runs in **Spitfire's own dedicated plug-in** (not Kontakt) — minimalist UI, frozen-landscape background. **12 mic positions + 3 macro mix options.**

**Signature feature — the "fractures."** This is NOT a normal sustain library. It's built on intervallic, evolving performances:
- **42 violin section "fractures"** with controllable **speed** parameters (the textures glitter/refract and move).
- **54 violin + 51 cello techniques**: Statements (short phrases returning to the start note), natural-harmonic patterns, **Rotations** (tempo-synced phrase performances), **seven trill variations ("Dispersals")**, pizzicato strums, and nine arpeggio performances for solo violin/cello.

**Scale Mode.** Conform performance patches to a scale or mode — velocity-controlled interval variation *without* changing fingering. Pick any of the seven diatonic modes or build a custom scale by clicking the notes you want.

**Evo Grid.** Present (as in other recent Spitfire releases) — map articulations/sounds across the keyboard, with a randomise for happy accidents.

**Sonic character (reviewer).** "Glittering and refracting textures"; dynamic layers move "from warm pianissimos to icy borderline ponticello-like fortissimos." Built to **duck and hide inside an arrangement, poking out for flavour** — seasoning, not the main dish.

**Price.** £249 / $299.
