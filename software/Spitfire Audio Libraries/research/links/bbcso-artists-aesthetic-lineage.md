## The honest verdict

BBC SO is overwhelmingly a **media-composer / realism** tool — that's its documented
user base. The citable thread for THIS aesthetic is the **neoclassical-ambient lineage**

processing IS the recognised method, and BBC SO is one of the libs reached for.

## The citable aesthetic anchor — Jóhann Jóhannsson (scene, not a BBC-SO credit)
- VI-C "Jóhannsson sound" + "tape saturation for neoclassical/ambient (Jóhannsson,
  Arnalds, Sakamoto-inspired)" threads are the on-target community knowledge:
  - String beds = "mostly slow **legatos and sustains**, pad-like," using delicate
    **flautando + sul tasto**, **crossfading articulations** for a "woozy" feel.
  - Texture = "delicate layers of **tape hiss, static, blurred field recordings**"
    (cf. *Orphée*) over the strings.
  - Processing = **tape emulations** (Studer A800, J37, Satin, Reels) for saturation,
    wow/flutter, hi/lo rolloff — the "recorded-wrong" sound.
- This is the documented blueprint the rig already follows: BBC SO sul-tasto/flautando
  sustains → EchoBoy/Decapitator (tape/saturation) → Valhalla (space). **Medium
  certainty as a scene method; NOT a claim that Jóhannsson used BBC SO.**
