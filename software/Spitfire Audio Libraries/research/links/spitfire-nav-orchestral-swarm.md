# Spitfire Swarm / Orchestral Swarm — the "living detune" texture engine

**The "Swarm" concept.** Inspired by **Pointillism** (neo-impressionist art): "discrete short notes blend in the listener's ear to create the impression of longer continuous tones." In practice a section of players each performs slightly differently around a note — organic micro-detune and constant internal movement → **living, breathing sustained textures** (no static-sample stillness).

**Format.** Runs in the **free Kontakt Player** (Swarms reissue: 7.5.2+; **Orchestral Swarm: Kontakt / Kontakt Player 5.6.8+**). Recorded to **tape** at AIR Lyndhurst Hall.

**Orchestral Swarm specifically** (the one the user owns on the drive): a string-section "swarm" — the orchestra playing as a shimmering, slowly-shifting cloud rather than clean unison. Four mic positions (close, tree, ambient, outriggers). Articulations include tremolos, glissandi, rhythmic patterns, sustained "swarm" tones, with dynamic + speed control.

**The Swarms bundle** (separate, Kontakt Player) gathers Harp Swarm (9 harps), Marimba Swarm, Mandolin Swarm (18 mandolins/charangos/ukes) — "haunting tremolos, epic glissandi, intricate rhythmic patterns, soaring melodic lines." (Not owned, but same concept — flagged for the reach-for map.)

**For drone/doom/shoegaze.** Orchestral Swarm = the **moving, alive string-wall** — where SSEvo *evolves through scripted Evos*, Swarm *shimmers via human micro-variation*. Great under banjo as a bed that never sits still; layers beautifully into Valhalla/EchoBoy.
