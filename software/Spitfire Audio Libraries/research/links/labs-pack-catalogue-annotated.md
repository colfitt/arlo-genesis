# LABS pack catalogue — annotated for drone / ambient / shoegaze / lo-fi

There are ~60+ free LABS libraries (66 cited in mid-2025). Below = the ones that matter for sustained walls, textures, lo-fi keys, and degraded/"recorded-wrong" beds — plus the on-aesthetic premium/LABS+ standouts. The free packs are tagged the user almost certainly already has many of these installed on the external drive.

## TIER 1 — grab these first (textural / evolving / drone)
- **Cinemagnetic** (LABS+ / premium) — THE one. Sampled from Amulets' tape-loop/guitar/synth drone rig: evolving, glitching, dissolving drones, "ambient music with grit and grain." See labs-cinemagnetic-amulets.md. Bullseye for drone-doom-shoegaze.
- **Textural Pads** (FREE) — 8 granular pad patches (Cyclosa, Deneb, Osny, Petrichor, Suddadi…). Built from bowed/scratched bass guitar, kalimba, cello, vocals + synths, mangled with bespoke granular FX into "otherworldly textures." Inspired by sci-fi soundtracks. Quirky + usable. See labs-textural-pads.md.
- **Organic Textures** (FREE) — warped/mangled orchestral recordings twisted through analogue gear: "eerie drones, dystopian ambiences, shimmering pads." Pure ambient-dreamscape / wall fodder.
- **Synth Pads** (FREE) — vintage synths fed through reverb, guitar pedals and outboard gear → "celestial sonic universe." Classic evolving pad bed.
- **Frozen Strings** (FREE) — glacial, Icelandic/neo-classical string tones (Albion-Tundra flavour). Sustained, fragile, cold — ideal under a baritone.
- **Cinematic Frozen Strings** (premium) — 8 London string players doing experimental techniques at AIR; deeper Tundra-style frozen strings.
- **Scary Strings** (FREE) — dissonant/unsettling sustained strings; doom-leaning texture.
- **Amplified Cello Quartet** (FREE) — four cellos run through FX pedals → distorted electric-string texture; "experimental, dark atmospheres." Great as a gritty pad layer.

## TIER 2 — strings / choir / brass beds (lush, cinematic)
- **Strings (LABS Strings)** (FREE) — 14 London players; warm, emotional, leans to smooth cinematic pads. Use sustained, avoid fast playing.
- **Choir** (FREE) — Eric Whitacre Singers; 3 evolving choral textures. "Smooth, evolving layers" — instant mood. HP-filter + light comp to sit it in.
- **Trumpet Fields** (FREE) — soft atmospheric brass: "swirling and droning brass textures" + granular elements. Memory-like ambient brass.
- **Vocal Pad / London Atmos** (FREE) — sustained vocal/room atmospheres for glue.

## TIER 3 — lo-fi keys & guitars (the "recorded-wrong" character)
- **Soft Piano** (FREE) — the famous felted piano (felt between hammers and strings, recorded 2008 at Air-Edel). Soft, intimate, low-dynamic. The default lo-fi/ambient piano bed.
- **Tape Piano / Glass Piano / Modular Piano / Piano Felt / Electric Piano** (FREE) — variations of the lo-fi/degraded/granular piano theme; Tape Piano = built-in wow/flutter character.
- **Wurli** (FREE) — sampled Wurlitzer (eng. Henry Cornish). Sharp tone + vibrato "cutting like a knife," but the **"Super Soft" presets** give a soulful, mellow, lo-fi tone — that's the one for ambient.
- **Mrs Mills Piano** — the famous detuned/honky-tonk Abbey Road upright (Beatles fame). NOTE: now lives in Spitfire **Originals**, not the free LABS tier (it was a LABS highlight historically). Lovely "wrong"/period lo-fi piano if owned.
- **Peel Guitar** (FREE) — Fender Telecaster through a 15W tweed amp with light distortion, vibrato, tremolo. "Warm, vintage British-indie" tone — soft, textured, dreamy. Perfect shoegaze/lo-fi guitar layer; pan multiple takes L/R for width.
- **Electric Guitar / Ambient Guitar / Moon Guitar / Guitar Harmonics** (FREE) — pedal-treated/ambient guitar textures and harmonics for shimmer layers.

## TIER 4 — plucked / folk / experimental (banjo-adjacent + oddities)
- **Charango** (FREE) — Andean small guitar; bright, earthy, warm; "really fun strumming engine," 3 expressive sounds. Banjo-adjacent plucked texture — sits well next to the user's banjo work. EQ out lows.
- **Mandolin / Dulcimer / Autoharp / Monochord** (FREE) — folk/plucked textures; Autoharp = "soft, dreamy," great layered with strings; Monochord = unique theatre-production instrument. All pair with the banjo aesthetic.
- **Drums / Drum Kits / Percussion / Tape Percussion / Hand Bells** (FREE) — dynamic session-drummer kit; Tape Percussion adds degraded character; Hand Bells for nostalgic/ghostly bell layers.

## Cross-pack technique tips (gathered)
- **Layer, don't solo:** stack different LABS instruments (e.g. Autoharp + Strings, Soft Piano + Frozen Strings) for warmth/fullness — single patches are thin by design.
- **Reverb knob = instant space:** crank the built-in dial to put bells/strings "in a big echoing church"; then add the real reverb (Valhalla) after.
- **EQ-cut lows** on plucked/mid instruments (Charango, bells) to avoid mud in dense walls; EQ-cut HIGHS to push a layer back / make it "muted" and lo-fi.
- **Automate mod wheel (dynamics) + expression** on long notes for organic movement — the static-sample fix.
- **Best-for-ambient community picks:** Autoharp (87/100) and Peel Guitar (82/100) for dreamy texture; Synth Strings + Choir for "instant mood" cinematic pads.
