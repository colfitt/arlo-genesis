# KVR: which Spitfire Evolutions library — user comparisons

Cross-library opinions on the Evolutions/textural range (useful for the reach-for map):

- **Ólafur Arnalds Chamber Evolutions (OACE)** — "my favourite of the Evo libraries, with Woodwind and LCO Textures a close second" (Mr Arkadin). Best first Evo purchase. Loads as a full chamber orchestra with **intelligent voice allocation** — low notes trigger cellos, with natural crossover between sections (a "feathering" technique where not all players sound at once → seamless crossfades). [Cells is the user's owned Ólafur library; same world.]

- **Symphonic (Strings) Evolutions** — Spitfire's **earlier** attempt at the concept; "not as varied or useful somehow" than the later chamber ones. Still the symphony-SIZED option (bigger, darker wall) where OACE is intimate/chamber.

- **Woodwind Evolutions** — "very underrated"; great for pad-type/evolving textures. (Runs in the **dedicated Spitfire plugin**, not Kontakt — unlike SSEvo.)

- **LCO Textures / Fragile / Angular Strings** — "probably the least traditional"; **Fragile String Evolutions** = "aleatoric/horror type stuff," "softer, more haunting"; the dedicated-plugin version adds **2× and 8× stretching modes**. Strong experimental sound-design potential.

**Takeaway for drone/doom/shoegaze:** the chamber Evos (OACE/Cells) = intimate, haunting; **SSEvo** = the big symphonic evolving wall; the Fragile/LCO end = horror/aleatoric texture. The user owns the two poles already — **Cells** (intimate) and **SSEvo** (symphonic).
