Making Music · review: "Spitfire Audio's Olafur Arnalds Chamber Evolutions"

The best independent breakdown of the Evo Grid + the two innovations (general Evolutions technique — NOT a confirmed-owned product; installed library is Cells).

## The Evo Grid
- A **VCS3-style plug-board**: vertical axis = key ranges (each covering a **perfect fourth**); horizontal axis = numbered Evolutions. Click a junction to insert/remove a **peg** to apply different Evos across the keyboard.

## What Evos are
- Extended samples (up to **one minute**) that transform as they play: crescendos that fade, tremolo, harmonic evolution, flutter effects. Categorized **subtle / thrills / episodic / dissonant**, color-coded.

## Key features
- **Randomization** — the Dice button generates random peg combinations (trillions possible); randomize by column or by Evo type. (The Basses Grid lacks this.)
- **Waves** — pre-recorded crescendo-decrescendos; normale, tremolo, vibrato; four lengths; via interface or key-switch.
- **Feathering** — only half the ensemble plays at any moment, preserving individual-player detail while keeping full impact without density.
- **Mic positions** — Close / Stereo / Tree / Ambient; mix independently; route to separate outputs for surround.

## Musical application + verdict
- Excels at "textured, expressive, Icelandic chill, intimate string lines"; NOT for "ostinato lines or fast, flashy passages." A few held keys create evolving soundscapes that sound genuinely performed.
- Verdict: "Top quality and first-rate" production/programming, articulations "not readily found in other sample libraries," "absolutely perfect" for atmospheric string composition.
