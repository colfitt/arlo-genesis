A "4-bus reverb fan-out" template for turning one sustained note/synth into an evolving drone wall. The principle: multiple complementary reverbs + de-synchronized tremolo panning = overlapping waves of sound that never lock in phase.

## The technique (stock Logic)
1. **Pick a source** with a good timbre (tutorial uses "SYN Golden" from the Collage preset pack). Route it to **4 separate send buses** using **Pre-Fader** (so the bus level is independent of the track fader).
2. **Reverb per bus** — different Space Designer IR on each:
   - Bus 1: a long-decay IR like **"Big Cave"**
   - Buses 2-3: different IRs for complementary textures
   - Bus 4: **reverse** the reverb for rhythmic interest
   - Use each reverb's onboard EQ to give each layer a distinct frequency character (so they stack without mud).
3. **Tremolo on every bus** — set to **slow L↔R panning** (not volume fade). Give each bus a **different rate & depth** so they never sync → "overlapping waves" of motion across the stereo field.
4. **Optional**: layer chorus / phaser / delay on individual buses for more complexity.

**Key principle quoted**: the skill is choosing *complementary* reverb presets that enhance rather than muddy each other.

## Rig fit
This is the stock-Logic analogue of running one banjo/baritone drone into several of the time-pedals at once (Big Sky / TimeLine / Microcosm) — here done as parallel aux reverbs with de-synced tremolo. Pairs naturally with printing a sustained source from the OP-1/Octatrack and fanning it across four reverb buses.
