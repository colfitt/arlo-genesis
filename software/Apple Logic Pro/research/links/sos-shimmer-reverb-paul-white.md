A second, slightly different stock-Logic shimmer build — useful because it gives concrete filter frequencies and an alternative interval set. Built on a pre-fade bus send into a compressor → pitch-shift → tape delay → reverb → chorus chain.

## Optional track inserts (on the source track itself)
- **Enveloper**: Threshold to your recording level; both Time controls at max (**200 ms**); left Gain fader at **-100%** (removes attack for a slow swell).
- **SilverVerb**: for early spatial dimension; use the low-cut slider to stop bass buildup; default modulation is fine.

## Shimmer chain on the aux (fed via PRE-FADE bus send)
**Opto Compressor (stock Logic)**
- ~**6 dB gain reduction** on loud notes (evens the feed into the pitch shifter).

**Pitch Shifter (stock Logic)**
- **+12 semitones**, plus **3 cents** of detune for thickness
- Mix: **60% wet**
- Timing/Pitch-Tracking mode set for single notes
- Alternative intervals worth trying: **+7, -5, or -12 semitones**

**Tape Delay (stock Logic)**
- High-pass: **580 Hz**
- Low-pass: **1.6 kHz**
- Subtle LFO + flutter modulation on
- Set feedback **last**, by ear

**SilverVerb (the wash)**
- Size and Reflectivity at **max**
- Low-cut: ~**900 Hz**
- High-cut: above **3-4 kHz**
- Mix: **100% wet / 0% dry**
- Density/Time: **60-70%**

**Ensemble**
- **"Vintage Dimension"** preset, depth reduced by ear → subtle chorus on the tail (the reverb-modulation-as-Lexicon-224 trick Eno/Vangelis used).

**Channel EQ (final)**
- Extreme low-cut
- ~**220 Hz** boosted for warmth

## Honesty note
Paul White / SOS is a reliable, well-regarded technical source; settings are explicitly given. The Eno/Vangelis/Lexicon-224 reference is a stylistic comparison the author draws, not a claim those artists used Logic.
