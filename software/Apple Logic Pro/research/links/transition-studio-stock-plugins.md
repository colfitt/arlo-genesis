Maps the stock-Logic plugins most useful for textural/ambient/lo-fi work and the specific creative moves for each. Good "what reaches-for what" reference for the UsageGuide §3.

## Plugins & moves
**Alchemy** (hybrid: additive / granular / spectral / FM / sample)
- Drag field recordings or vocal chops into the **granular** engine → ambient pads, evolving drones, glitchy textures, rhythmic plucks. The granular engine is the headline ambient tool.

**Step FX** (step-sequenced multi-FX)
- Automate **"Pattern Select"** across the arrangement for evolving variety.
- Put it on a **return/aux** channel for parallel glitch textures (don't commit the whole signal).

**Vintage Console EQ** (Neve/API-modeled)
- Use **Neve-style mode** to sculpt a synth bus — high-end sheen or mid-push.
- Push input gain into saturation, compensate on output → console-style drive. Place **pre-reverb** on melodic elements.

**Phat FX** (distortion / comp / filter / mod multi-fx)
- **"Big Bottom"** preset for basslines; adjust the Multiband Exciter.
- Stack multiple instances, each doing one job.

**Space Designer** (convolution)
- Load **percussive samples as IRs** → metallic / industrial reverbs.
- **"Reverse Cello Hall"** or cathedral IRs for ambient breakdowns.

## Suggested processing chain
**Step FX → Phat FX → ChromaGlow → Space Designer** for stacked high-impact texture processing.
(ChromaGlow = the newer stock saturation/tube plugin — relevant to the "degraded/recorded-wrong" aesthetic.)
