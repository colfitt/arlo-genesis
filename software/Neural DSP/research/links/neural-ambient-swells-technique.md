Technique notes for using ANY amp/FX (incl. Archetype JM X / Nolly X) for ambient walls:
- **Volume swells**: strike the chord with volume off, fade it in (volume pedal, guitar knob, or DAW automation) → removes the percussive attack → "breathing" post-rock bloom. Timing so the note blooms with the bar. In-the-box: automate the JM X / Nolly X **Output** or a pre-volume, or use a volume-pedal plugin before the amp.
- **Chain order for walls**: (modulation: chorus/tremolo) → **delay** (adds the wash/texture) → **reverb LAST** (wetten, room) — slow-decay Hall/Cathedral/Shimmer for the big spacey size.
- **Stacked delays + long reverb + swells** = cascading orchestral layers from a simple progression. Sparse voicings keep the wall defined.
- Maps directly onto the plugins: Gravity Tank harmonic-trem (JM X) or pre-Delay (Nolly) for movement → Dream Delay / Delay 2 (ping-pong, long feedback) → Studio Verb Hall / Nolly reverb cranked decay last.
