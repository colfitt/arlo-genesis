guitar.com / MusicRadar / Guitar World (aggregated) · What the Neural DSP "X" platform adds · 2025

The **"X"** suffix = Neural DSP's new plugin platform / big free update to existing Archetypes.
Shared across all X titles:
- **Resizable UI** (click-drag window resize); refreshed visuals/navigation.
- New **global meta-features**: **Transpose**, **Doubler** (one-pass double-tracking), **Live Tuner** (always-visible bottom-left), **Metronome** (standalone).
- **Quad Cortex compatibility** coming via a future CorOS update (run the same plugins on the hardware).
- Per-title feature bumps. **Nolly X specifically**: doubled cab tonal territory — **4 speaker options per cabinet** (each cab now offers its original voicing PLUS an alternate, e.g. Cabinet 3 = Zilla + Vintage 30; open-back vs closed-back greenbacks), the **"Nolly's Mod" EQ** dual-band set, and a **Room Send on every cab**. **Cory Wong X** added a Room Reverb. **Tim Henson X** reworked the Multivoicer (4 pitch voices) + Harmonic/Melodic Minor scales.
- Existing owners upgrade free.
