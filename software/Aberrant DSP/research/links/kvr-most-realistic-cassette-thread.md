Comparison thread — where SketchCassette II sits vs rivals, in users' own words.

## On SketchCassette II
- **Lotuz2019**: realistic option; you can get convincing results WITHOUT leaning on artifacts. Workflow tip: "Just select a tape setting you like and turn up the Input. I prefer the Chrome settings." (= Type II Chrome profiles for the most authentic character to this experienced user.)
- **Lotuz2019**: "You should really give it a try yourself. It's not a blatant lofi plugin at all." (i.e. it can be subtle/realistic, not only cartoon lo-fi.)

## Rivals named in the thread
- **Cupwise Nebula libraries** (plexuss): "the most realistic" — but workflow friction ("I don't like working with Nebula 4"). Not a daily driver.
- **XLN RC-20** (Rivanni): suggested as an alternative (no deep commentary). [Rig note: user owns RC-20 — see stacking recipes.]
- **Wavesfactory Cassette** (plexuss): "not bad" but less authentic than Nebula.
- **ChowTape** (TIMT): "can get some of the way there for the saturation" when under-biased (free/cheap, reel-to-reel not cassette).
- **Flywheel / Fuse** (midi_transmission): good transients but unconvincing as cassette — loses bass when driven, lacks warmth.

## Distillation
- SketchCassette II is positioned as the most CONTROLLABLE/usable realistic cassette plugin (Nebula more "accurate" but unworkable). For the rig: SketchCassette = the in-the-box cassette workhorse; RC-20 = the more cartoonish multi-module lo-fi suite for when you want overt destruction.
