## Tapetown Studio practical guide (source-by-source starting points)
- **Vocals**: Type II tape + moderate warble = "wobbly, 90s bedroom pop vibe."
- **Drums**: high dropout + hiss on a loop = "gritty, tape-degraded texture" for lo-fi hip-hop.
- **Synths/pads**: an "Old Tape"-style preset = warm cassette pads.
- **Clean guitar**: crank the Dropout control = "broken-tape effect."
- **Ambient layers**: put the NOISE/hiss setting on an EMPTY track = subtle ambiance bed. (= a deliberate hiss-only texture track; pairs with the Theatre of Noise "ghost-tape layer" hack.)
- CPU: "light on CPU for home studios."
- Honest limit: "sticks to cassette-style effects, so it's not great for clean mixes." A few users wish for more presets; "Worn Mixtape" cited as a good starting preset.

## CPU + gotchas (aggregated from KVR product page / reviews)
- CPU: uses MORE than a basic EQ/saturation module, but LOW compared to other tape-FX plugins. Fine to run several instances; freeze/bounce if you stack many with heavy modulation.
- **Preset gotcha**: there has been an issue with the preset system in SOME DAWs (noted in the manual); Aberrant moved toward a DAW-independent preset system. If presets don't show/save in your host, that's the known cause. (Save to the plugin's own User bank rather than relying on the host's preset menu.)
