# Decapitator vs other saturators — the "why Decapitator"

## The three-way comparison everyone makes
- **Decapitator** — "the most versatile all-around choice." 5 hardware-modeled
  preamp/console/tube flavors, simple Drive + Mix + Tone + filters. Simplicity +
  musical analog character. The default reach-for when you want *vibe* fast.
- **FabFilter Saturn 2** — multiband, 28 distortion styles, per-band dynamics, deep
  modulation engine (LFOs/env-followers/XY). "Unmatched precision for advanced
  users." Use when you need *surgical/multiband* control or modulation — overkill
  when you just want color.
- **XLN RC-20 Retro Color** — a *lo-fi character bundle*, not a pure saturator:
  Distort + Noise + Wobble + Magnitude + Digital + Space modules. "The definitive
  creative texture tool." Use for full vintage/degraded treatment (wobble + noise
  + bitcrush), not for clean harmonic warmth.

## The takeaway for this rig
- **Decapitator = clean, controllable analog *harmonic* color** (the "make it
  warmer/edgier" tone-shaper). Reach for it for musical saturation that stays
  intentional.
- **RC-20 / SketchCassette II = degradation & motion** (wobble, dropouts, noise,
  bitcrush) — the "recorded-wrong" character. These overlap *very little* with
  Decapitator; they're complementary stages, not alternatives.
- **Saturn 2** (if owned) = the precision/multiband option; rarely needed when the
  goal is vibe.
- **vs hardware Colour Box V2 / Generation Loss:** Decapitator is recallable,
  automatable, and free to instance — but the hardware has committed, unmodelable
  signature character (Colour Box = transformer/EQ console-preamp grind;
  Generation Loss = real tape-degradation modulation+filtering). General community
  framing: software saturation = surgical/recallable/per-stem inside the box;
  hardware drive = the front-end commitment you print. Use Decapitator on the
  studio/back end, the pedals on the way in.
