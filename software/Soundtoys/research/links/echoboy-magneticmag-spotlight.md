## Character
- "Warm, clean, smooth echo" for front-of-mix vocals; also "massive lo-fi crushing echo sounds" for
  guitars/drums/anything that needs mojo. 30+ styles blending classic pedal models + modern digital.

## The saturation-only trick (most concrete version found)
- Switch from "note" to **"time," set Time = 0**, push **Saturation to ~50%**, then adjust Input/Output
  for extra character. Now audition the **styles as pure saturation/coloration** (no delay).
- **Caveat (important for the rig):** Width, Diffusion, and Modulation/Wobble still add delay to the path
  regardless of the Time setting — so at Time 0 they can cause **phasing when mixed with the dry signal.**
  Keep those at zero if you want truly clean coloration, or run 100% wet on a send.

## Other named techniques
- **CE-1 Chorus** style = "one of the richest, warmest choruses" — stunning.
- **Decay shaping:** per-band EQ Decay controls darken or brighten the feedback tail (the tape-loss move).
- **"Bouncing ball" effect** (inspired by Aphex Twin's technique): Rhythm Echo mode, controlled via the
  **Warp** and **Shape** knobs.
