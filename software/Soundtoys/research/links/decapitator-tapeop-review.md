# Decapitator — Tape Op review (the "why Decapitator" + Mix-knob praise)

- **Primary real-world use:** Weiss uses it to "**sand down**" bright top-end on
  pop/rock vocals while **preserving transients via the Mix knob** — on "nearly
  every one of my mix projects." (i.e. it's a *de-harsher/warmer*, not just a
  dirt box.)
- **Five models** = Neve preamp, Ampex tape preamp, Chandler/EMI input channel,
  and two Thermionic Culture Vulture variations. "All a bit different, but all
  very useful and realistic-sounding."
- **Mix knob** — "so simple, yet so effective" — parallel processing *inside* the
  plugin, no DAW routing. He wishes every plugin had it. This is the single
  most-praised feature.
- **Auto switch** (default on) — tethers Drive to Output so gain stays consistent
  while you experiment.
- **Punish** — +20 dB onto the Drive.
- **Character vs hardware:** unlike a SansAmp (one signature sound), Decapitator is
  "**a Swiss Army Knife capable of many harmonic flavors**" — thickening drums,
  warming vocals, broad mix duty. The flip side: it's *less* of a one-trick
  signature than a dedicated hardware box. (Relevant to the "software vs hardware
  saturation" decision: Decapitator = versatile/recallable; the hardware
  Colour Box / Generation Loss = committed signature character.)
