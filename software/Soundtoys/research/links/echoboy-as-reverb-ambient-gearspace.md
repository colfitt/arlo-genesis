The drone/ambient-relevant takeaways on bending EchoBoy into reverb territory:

## Using EchoBoy as a reverb / ambient wash
- EchoBoy "has nice reverb possibilities." The key trick is the **Prime Numbers** switch: it restricts/
  de-aligns the repeat times so feedback "thins out in a nice proportion between short and longer
  repeats" — i.e. it stops the wash from turning into one resonant ringing pitch.
- **"EchoBoy's Galaxy"** factory preset = a reverb-style preset built on the prime-number approach **with
  a lot of feedback** — a good starting point for a self-feeding ambient cloud.
- Combine with the **Diffused / Splattered / Verbed / Ambient** styles (per manual) + Style-Editor
  **Diffusion** (Loop position, so each repeat smears more) for a plate/hall-like tail.

## Honest limitation
- Users note it's **hard to get a *pure* reverb tail** from EchoBoy — the discrete repeats tend to show
  through (it's a diffused multitap, not a true reverb algorithm). So: great for a *grainy, rhythmic,
  reverb-ish wash with character*, less so when you want a smooth featureless tail. For the latter, feed
  EchoBoy into a real reverb (Valhalla) rather than asking EchoBoy to *be* the reverb.

## Drone-chain note (from the linked MOD WIGGLER "Reverb and drone" discussion)
- Drone/ambient players favour **delay → chorus → echo → reverb** chains of heavily-effected sound; use
  Mid/Side processing on the reverb to keep width/depth without the center becoming a mushy wash. EchoBoy
  sits naturally as the "echo + chorus + character" stage *before* the final reverb.
