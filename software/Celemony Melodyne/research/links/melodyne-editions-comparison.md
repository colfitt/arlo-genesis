The four Melodyne 5 editions, what each unlocks (the dividing line for THIS rig is **Editor**, because that's where polyphonic DNA — needed to edit banjo/guitar **chords** note-by-note — appears):

## Essential (entry)
- **Main tool only**: pitch, timing, duration, note separations. Basic macros (pitch / timing / dynamics).
- All the **monophonic + percussive + complex** algorithms (so it can transpose/time-stretch a whole mix), but **no DNA polyphonic** note editing and **no** dedicated formant/drift/vibrato/amplitude sub-tools.
- Often bundled free with interfaces/DAWs.

## Assistant (full vocal toolkit, still mono)
- The **entire tool set**: separate **pitch-center, pitch-modulation (vibrato), pitch-drift**, plus **formant, sibilant, transient, amplitude, fade, timing** tools and their **Inspectors** (numeric entry).
- **Audio-to-MIDI export** (Save as MIDI).
- Still **no** polyphonic per-note editing.

## Editor (instruments & samples — the meaningful tier here)
- Everything in Assistant **plus DNA Direct Note Access** — edit **individual notes inside chords** on **piano and guitar** (and banjo, and library samples).
- **Tempo mapping / tempo detection**, **chord-track adaptation** (adapt a sample's notes to the song's chords), **scale creation/editing**.

## Studio (pro multitrack)
- All Editor features plus **multi-track simultaneous editing**, the **Sound Editor** (overtone-level sound design — re-shape the harmonic spectrum of a note), **track-transcending macros**, and **quantize-to-reference-track**.

## DNA in one line
DNA Direct Note Access = the tech that lets you reach **inside polyphonic material** (a strummed chord, a piano voicing) and grab one note. Essential/Assistant can still transpose/stretch/retune a whole **monophonic** part or a whole mix as one object — they just can't isolate a single note from a chord.

## Rig takeaway
- A single **banjo/baritone lead line** (one note at a time) → **Assistant** is enough (Melodic algorithm).
- **Strummed banjo chords / fingerpicking / re-voicing a chord / building a sampler instrument from a chordal recording** → need **Editor** (DNA).
- **Overtone-level sound design** (re-sculpt the spectrum, true resynthesis textures) → **Studio**'s Sound Editor.
