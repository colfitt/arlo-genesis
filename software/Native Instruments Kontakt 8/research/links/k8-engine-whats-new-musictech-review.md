musictech.com · Native Instruments Kontakt 8 review · 2024

Verified "what's new in Kontakt 8" (released Sept 2024) from a hands-on review.

## Conflux (new hybrid instrument)
- Pairs sampling with an upgraded **wavetable** synthesis engine that now also
  does **FM, phase and ring modulation**.
- "Excels at evolving textures, big swamping bass swells, and cinematic sound
  design." (Directly on-aesthetic for drone/ambient.)
- Bundled with extensive modulation options and onboard effects.

## Leap (new performance sampler)
- "Sampler within a sampler" for performance. **White keys** trigger one-shot
  samples and **tempo-synced loops**; **black keys** trigger performance FX —
  pitch/time shifts, stutter, granulation, and other modifications, applied live
  without breaking flow.
- Ships with **12 Leap expansion packs** in the full version.
- LIMITATION: **stereo output only** — no per-loop multi-out, so you can't
  record Leap stems separately. Reviewer's main gripe.
- Performance effects apply to the whole Leap track, not per-sample.

## Tools (new MIDI effects, instrument-agnostic)
- **Chords**: fire stacked harmonies from a single root note — **130 harmonic
  progressions**. Each white key triggers a chord; adapts to selected key/scale.
- **Phrases**: same idea for melodic passages — **181 melody presets**, editable
  timing/velocity/playback.
- Run on ANY sample instrument loaded in Kontakt; features randomization,
  inversion, humanization, MIDI export.

## UI / browser
- Expanded **browser** with dedicated tabs: **Instruments, Combined, Loops,
  One-Shots, Tools, Leap**.
- New **Combined** preset format saves multi-instrument setups with embedded
  Tools, macros and layers.
- Classic View toggle preserves the Kontakt-7-style interface.

## Pricing / version
- Full: £269 / $299 / €299. Upgrade: £89 / $99 / €99.
- Tools and Leap are included in BOTH the free Kontakt 8 Player and full Kontakt 8.
