Kontakt 8 Factory Library + included sounds

## Factory Library redesign (verified)
- Redesigned Factory Library, **HiDPI** interface, **seven new collections** of
  instruments (band, orchestral, electronic), fresh sounds + added FX/functionality.
- **Orchestral Tools collaboration** noticeably upgraded the orchestral patches —
  "now sounding professional" (multiple reviews).
- Full Kontakt 8 ships with **900+ instruments** in the Factory Library.

## Named included instruments worth knowing
- **Noire** — detailed/intimate felt-piano instrument (ambient-friendly).
- **Stradivari Violin** — solo violin.
- **Hybrid Keys** — hybrid/processed keys (textural).
- **Conflux** — the new hybrid wavetable+sample synth; **~200 atmospheric
  presets** (musicradar), strong for evolving textures/cinematic.
- **Leap** + 12 expansions (80s New Wave, Afrobeats, Latin Trap, acoustic drums,
  drum breaks, etc.).

## Performance / CPU (honest)
- Loading times improved vs Kontakt 7, but Kontakt "still demands a decent amount
  of computing power," especially with multiple instances (magneticmag).
- Standard CPU/RAM levers still apply: **Purge** (drop unused samples from RAM),
  **DFD vs sampler/RAM** source modes, **Batch Resave**, and freezing instances
  in the host. Logic AU host: run Kontakt as a multi-out AU and freeze when not
  actively tweaking.
