# What's new in Kontakt 8 + the new-UI / Quick-Load gotchas

## New for power users
- **New side panel browser:** search and **audition/demo presets from any loaded instrument without leaving Performance View** (Kontakt 7 made you switch to Library View — this removes a navigation step). Six-tab library: Instrument / Combined / Tools / Leap / Loops / One-shots.
- **Conflux** — hybrid wavetable+sample synth engine (FM, phase-mod, ring-mod, virtual-analog osc; dual LFOs, two envelopes, deep mod routing). Reviewers note it's aimed at **media/experimental soundscapes** — strong fit for drone/ambient, less for pop/EDM.
- **Leap** — "sampler within a sampler": 16 sample slots mapped to white keys, performance FX (stutter, pitch-shift, speed) on the black keys; drag-and-drop your own samples to build a kit. Good for live texture mangling.
- **Chords & Phrases** MIDI tools — single-key → full chord / melodic phrase; ship in BOTH full and free Player; work with **any** Kontakt instrument; 130 progression sets / 181 melodic ideas.
- **Engine/perf:** upgraded playback engine, faster loading, more instances, better stability (per NI; reviewers didn't benchmark).
- **8.1+ added effects:** new amp/stomp FX (Super Fast 100, Chainsaw, Kolor, Skreamer Deluxe), more delay/reverb send types; later: Moog-style passband filters (Monark) + dual Sallen-Key (MS-20) filters — useful for in-Kontakt drone shaping.

## The gotchas (real complaints)
- **Quick Load is unavailable in the new Default View** — it only works in **Classic View**. Long-time users who organized everything via Quick Load consider this the biggest regression. Workaround: switch to **Classic View** to get Quick Load back (loses the new browser niceties — "best of both worlds" only with extra legwork).
- **MIDI-channel changes** are now hidden behind a right-click and not visible at a glance (annoying for multitimbral setups).
- **Can no longer hide instrument UIs** when stacking libraries → you scroll through all the big Performance-View graphics to reach one control. Called a "frustrating, time-consuming oversight."
- **Tools presets audition on the default "Piano Uno"** instrument, not your loaded sample — frustrating back-and-forth when picking Chords/Phrases presets.
- **No "export full MIDI" from Chords/Phrases** (you record the trigger notes, not the generated notes); Phrases can't be manually edited; **no MIDI Learn on Conflux macros** (DAW-level mapping still works).
- **Version lock-in:** opening a library in **Kontakt 8 (incl. K8 Player) can make it no longer load in Kontakt 7/older** — be deliberate about which version you open shared/legacy content in.

## Verdict (reviewers)
Upgrade pricing is modest; the creative tools (Conflux/Leap/Chords/Phrases) are the draw, the browser redesign is divisive. Classic View remains, so nothing is forced.
