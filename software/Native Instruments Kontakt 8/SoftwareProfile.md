# Native Instruments Kontakt 8

**Summary** — Industry-standard **sampler engine** (Kontakt 8, Sept 2024). Both a
**builder** (turn your own recordings into playable instruments) and the **host** for
the Komplete granular libraries + all Spitfire-in-Kontakt content.

## Why I have it
The rig's superpower for the degraded aesthetic: **sample your own sound — pedalboard
tails, banjo/baritone, OP-1/Digitakt/MPC/Octatrack/VG-800 one-shots, field
recordings — and loop it into an evolving drone.** Plus it runs the big libraries you
own (Komplete's Ashlight/Straylight/Pharlight granular trilogy, Spitfire LABS/Ólafur
Arnalds) and now ships **Conflux** (hybrid wavetable+sample) for evolving textures.

## Signature uses
- **Build instruments from your gear:** sequence-and-record hardware (External-MIDI +
  pre-programmed notes, hands-free) or Logic **Auto Sampler → EXS → Kontakt**; map +
  group + save (Patch+Samples).
- **Drone looping:** record 4× a cycle → loop the middle two bars; **vary loop length
  per zone** so a held chord boils. Don't loop plucked banjo → **round robins**.
- **Bowed→wall (Clinton Shorter):** pan takes L/R, stretch one note across the keys,
  modulate **pan+volume not pitch.**
- **Evolving pad:** Group-Insert filter + slow LFO→cutoff + 2nd LFO→pan + random→pitch;
  or **granular** (grain ~600 ms, sub-0.1 Hz LFOs on grain params).
- **Own convolution IRs:** drag a pedalboard tail/banjo scrape into the Convolution FX.

## Notes
- **★ Build essentials:** auto-map needs filenames `name_group_note` + **sharps only**;
  turn OFF "Edit All Groups"; **Kontakt-set loops can't be exported** (set loops in the
  DAW if porting); **only Source/Amp/Group-Insert FX are internally modulatable** (not
  Instrument-Insert/Send/Main — use Group Insert or host automation).
- **Management:** **Batch Re-Save** every library after install (and when the drive
  path changes); **Purge → Update Sample Pool** after playing to shrink RAM; the **"Add
  Library" button is gone** → cogwheel **Import Content**.
- **Logic AU:** use the **Multi-Output (16×Stereo)** variant for per-part routing;
  **Freeze is disabled** for multi-out → **Bounce In Place**; **Quick Load only works
  in Classic View.**
