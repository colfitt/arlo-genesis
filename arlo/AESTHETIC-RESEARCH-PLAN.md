# ARLO Aesthetic-Specific Research Plan

A targeted source map for teaching ARLO the production language behind a hybrid aesthetic:

> Emotionally direct but experimental music blending Bon Iver–style vocal layering and processed folk intimacy, LCD Soundsystem–style dance-punk repetition and live electronic groove, Charli XCX–style maximalist/hyperpop sound design, mk.gee–style DI/reamped guitar weirdness, and Billy Strings / Punch Brothers / Noam Pikelny–style acoustic virtuosity and modal drive.

The first 38-file corpus seeded the *general* production vocabulary. This plan adds the *aesthetic-specific* vocabulary — the moves, gear, and conventions that make a Bon Iver vocal sound like Bon Iver, an LCD groove sound like LCD, and a Punch Brothers arrangement feel modal-yet-bluegrass.

## Sourcing philosophy

| Tier | Source type | Why it's useful |
|---|---|---|
| **S** | Inside Track / Tape Notes / AudioTechnology long-form production breakdowns | Real session details — track counts, plugin chains, mic choices, structural decisions — written by working journalists with studio access |
| **A** | Premier Guitar Rig Rundowns; manufacturer official docs; Berklee Press books | Authoritative on a specific gear/technique, format-friendly, citable |
| **B** | Reputable production-education sites (MusicRadar, Reverb Machine, Native Instruments blog, Pro Audio Files) | Concrete how-to content, generally well-cited |
| **C** | Wikipedia (orientation only); artist profile interviews in NPR / Pitchfork / Dazed / NYT | Frames context; rarely deep on technique |
| **NOT** | Generic listicles, "5 ways to..." blog spam, plugin-marketing content | Filler; doesn't teach underlying mechanic |

The 9 categories below each list sources at these tiers, with per-source evaluation.
