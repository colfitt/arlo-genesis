# Chris Messina on the Messina + "715 - CRΣΣKS"

Distilled concrete info:

- Messina: **"It's not a thing"** — "a laptop running software, then that software run through a
  physical piece of hardware, that is then doing another thing." "Many things working together and
  none of them are ours, but the product is." They used gear **"the way they're not normally
  intended."**
- Operationally: **"Justin is singing a song, and he's playing a keyboard that can create harmonies
  simultaneously."** The innovation = **real-time** harmonic generation (normally you record first
  then harmonize).
- **"715 - CRΣΣKS"** = essentially **just Justin + the Messina, one take** (a few extra takes done,
  each **hitting the H8000 differently / cranking the output to hit it harder** for more distortion).
  No instruments besides the voice-through-Messina — the purest example of the halo as the whole song.
- Prismizer described as a similar "cloning and disseminating" effect (let Frank Ocean have "two
  versions" of his voice on *Blonde*).

Rig takeaway: "715" proves the vocal-halo can stand alone as an entire arrangement — the rig
analogue is a QI Etherealizer freeze-pad/Dark Star dual-pitch wall played as the whole bed, voice
(banjo) on top. The "hit it harder for more distortion" detail = lean into Crush/saturation when
emulating, don't keep it clean.
