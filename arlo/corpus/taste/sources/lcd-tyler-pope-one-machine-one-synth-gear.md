# LCD Soundsystem — gear & "one machine + one synth" method

**The constraint:** James Murphy challenged Tyler Pope to make an EP "**only using one drum machine and a synth**." Pope used a **Roland CR-78** drum machine + an **ARP Odyssey**. His ongoing rule: "**I try to stick to using one drum machine or combining sounds with a 909.**"

**Drum machines used:** Roland **TR-808** (heavily), **TR-909**, **CR-78**; Behringer RD-808 clone for retracking.

**Synths across LCD / DFA:** Roland **SH-101** (live), Yamaha **CS-60** (prominent on several tracks; "Get Innocuous" kick/snare were built FROM the CS-60 and sampled), **ARP Omni II**, **Oberheim SEM**, **Moog Voyager / reissue Minimoog** ("on nearly everything"), **Moogerfooger** pedals, Simmons SDS-V e-drums, Gretsch acoustic kit.

**Philosophy:** "I just prefer the **imperfection of using analogue gear** — the whole signal chain is different." Thermionic Culture Vulture for character. Started on Pro Tools, moved to **Logic** with DFA. Soft-synth sketching = Arturia (Juno-60/MS-20/Minimoog), NI Battery for drums.

**The DFA method (from "Get Innocuous"):** build the beat out of one synth's sounds — kicks made with the synth, snares synthesized then sampled — then loop and layer. Cowbell + four-on-the-floor + analog drum machine is the backbone.

**Rig translation:** the "one drum machine + one synth" discipline = build a whole groove on the Digitakt or MPC from a tight palette; sample-and-resample a synth voice for the kit (CS-60 method → VG-800/OP-1 sampled into the MPC). Keep the analog imperfection (run prints through the degrade boards).
