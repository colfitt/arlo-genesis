# Daft Punk "Around the World" — synth recreation (concrete patch values)

**Bass synth:** start with a **sawtooth**; lower the **filter cutoff to ~2.7**; crank **resonance high** (high but not screeching); add a small amount of **envelope→filter modulation (ENV ~2)**; ADSR = **attack 1, decay 6, no sustain** → the "subtle pluck" that defines the bouncing bass.

**Lead synth:** **square wave**, **filter ~3**, **resonance ~6**, **env-mod ~halfway**, **attack ~2** → the funky wah-wah lead.

**Filter-house core takeaway:** the resonance + envelope-driven filter is the whole sound; the part itself is a short repeating bass figure. The "open/close over a build" effect is **manual filter-cutoff automation** on the looped part, not a different note pattern.

**Rig translation:** VG-800 SYNTH FILTER BASS (saw, low cutoff, high reso, snappy filter decay) = the closest hardware acid-bass; OP-1 synth or Digitakt sample with an LFO/manual cutoff sweep does the open/close.
