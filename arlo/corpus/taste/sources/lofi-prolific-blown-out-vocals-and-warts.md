## Distilled — the SIGNATURE SOUNDS of this cluster, sourced

This file pins the five signature sounds named in the cluster brief to documented method:

1. **Bedroom/4-track cassette degrade** → Toledo: laptop + Audacity/GarageBand + bundled PC mic;
   Modest Mouse: 2-week fast tracking, Calvin Johnson "let the tape roll." Both = budget capture,
   not designed degrade. (RIG: PORTA424 + Generation Loss + JHS 424 = the cassette/4-track print.)
2. **Clipped / overloaded preamps** → Toledo recorded "relatively quietly and boosted it in the
   mix" → the boost reveals clipping; cheap mic/laptop input overloaded. (RIG: Colour Box V2
   broken-console preamp; drives pushed by a low-level source.)
3. **Room-mic'd amps bleeding** → Modest Mouse's Moon-Studios room sound; Toledo's Supro recorded
   in a room then boosted. The room/bleed is *left in*. (RIG: Deco V2 tape glue + a touch of the
   degrade chain to fake "captured in a room"; this rig prints dry-forward, so room is approximated.)
4. **Vocals pushed into distortion** → Toledo: "singing or screaming into the laptop mic" — the
   distortion is a cheap mic overloaded by a screamed source. (RIG: the user's voice isn't in the
   target gear; the *character* — an overloaded cheap-mic clip — is portable to a STRING source via
   Colour Box broken-preamp + 424 gated grit. Flag: this is a character analogy, not a vocal patch.)
5. **Jangly brittle clean + odd tunings (Modest Mouse)** → bright single-coil Tele/Wicks into a
   clean Twin, double-tracked, sparing tremolo/chorus, microtonal bends/slide. (RIG: bright
   edge-of-breakup drive + banjo-as-lead + Deco doubletracker for the double-track chorus.)

## The load-bearing finding
Neither artist used "lo-fi gear" as an aesthetic *device* the way the rig does — they used **cheap
gear and fast capture out of necessity**, and the warts became identity. So **every rig patch in
this cluster is 🟣 designed-to-emulate the RESULT** (the blown-out, brittle, recorded-wrong sound),
never a found artist setting. The honest move: chase the sound, name the record, flag the method gap.
