# Even-Vibe — Univibe Swirl (stereo)

## Concept
Stereo Uni-Vibe-style chorus/vibrato swirl for that watery, throbbing Hendrix/Gilmour movement — lush and three-dimensional. The Even-Vibe algorithm lives in the H90 modulation set; set a slow-to-medium rate for the classic throb, Depth to taste, and choose chorus mode (mixing dry and wet) versus vibrato mode (full wet). It sits great in series before a delay or Blackhole for swirling ambient washes.

## How to play it
1. Load the Even-Vibe algorithm on Preset A.
2. Set a slow throb rate and medium depth for a ballad swirl.
3. Choose Chorus mode (dry+wet) for swirl, or Vibrato mode (full wet) for pure pitch wobble, to taste.
4. Map Rate to expression/HotKnob to speed it up for solos.
5. Optional: run in series before Blackhole/delay so the modulation rides the repeats.

## Notes
- The Even-Vibe algorithm exists in the H90 modulation set; exact knob values are not published, so dial the controls above from the recipe by ear.
- Sits great in series before Blackhole/delay for swirling washes.
