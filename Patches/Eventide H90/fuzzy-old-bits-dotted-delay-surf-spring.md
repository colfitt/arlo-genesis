# Fuzzy Old Bits — Dotted-1/8 Rack Delay + Surf Spring (factory)

## Concept
A factory H90 Program — "fuzzy old bits" — that stacks a vintage rack Delay model against a '60s surf/psych Spring reverb. The delay carries an '80s flavor; set it to a dotted-eighth subdivision with modest feedback and the repeats lock into The Edge's signature 16th-note weave. The Spring sits underneath for springy, surfy ambience, with light tremolo adding period-correct movement. Algorithm makeup is published (Premier Guitar review); the knob values are not, so dial from the recipe or just load the program in H90 Control.

## How to play it
1. Load "fuzzy old bits" (or build it: vintage rack Delay + Spring reverb).
2. Set the delay to a dotted-eighth subdivision with modest feedback.
3. Add Spring reverb with light tremolo for surf character.
4. Play steady eighth notes against the dotted delay so the repeats weave into a fast 16th-note feel.

## Notes
- Confirmed factory Program (Premier Guitar's H90 review) — algorithm makeup is published, but no knob values are. Dial the controls above from the recipe by ear, or read the loaded program in H90 Control. The Spring controls (Tension, Number of Springs, Tremolo Intensity/Rate, Large/Small tank) reflect the Spring algorithm's parameter set, not transcribed settings.
- The Edge's dotted-1/8 = 75% of a beat; eighth notes played against it sound like fast 16ths. Slave the delay to MIDI clock for tight tempo-sync.
