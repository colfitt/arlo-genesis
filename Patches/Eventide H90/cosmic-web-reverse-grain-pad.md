# Cosmic Web Reverse-Grain Pad

## Concept
A reverse-pitched granular pad flowing into reverb — the single most rig-relevant new granular-firmware algorithm for the wall-builder. Cosmic Web blends reverse-pitched grains with classic Eventide reverb; the grains "stretch and bloom" into deep delay and a pad-like modulated space.

## How to play it
1. Engage Cosmic Web on Preset A.
2. Run Series into Blackhole or Plate on Preset B.
3. Print the result to the PORTA424 / JHS 424 tape stage downstream.

## Notes
- Algorithm-character description, **no per-param numbers** (granular firmware ~v1.12.x).
- Requires recent firmware (post-v1.1.4, free via H90 Control).
- Provenance = Eventide blog / Sound On Sound -> factory algorithm. The standout granular for this rig.
