# Biblically Accurate — VocalShiftMIDI + Wormhole (Vines artist preset)

## Concept
Vines artist preset: deeply modulated reverb with otherworldly vocal textures and long gliding tails that feel unsettling yet beautiful. Wormhole's tilting, pitch-modulated space carries the MIDI-driven VocalShiftMIDI harmonies, so the vocal voices smear into the reverb rather than sitting on top of it.

## How to play it
1. Load VocalShiftMIDI + Wormhole (VocalShiftMIDI on A, Wormhole on B).
2. Set Wormhole long with active Stability/Warp modulation for the gliding feel.
3. Drive the harmonies via MIDI.
4. Use the Warp footswitch (momentary) for pitch glides.

## Notes
- Long gliding reverb tails are described as "unsettling yet beautiful."
- Factory Wormhole presets for reference: Deep Spaced, Wormhole, Spaceplate, Snake Portal.
- Distinct from the existing wormhole-hyperspace-wall / wormhole-serpent-warp-latched patches — this is the vocal-harmony pairing.
- HONESTY: the VocalShiftMIDI + Wormhole pairing is published (Eventide blog), but raw knob values are NOT published. Treat every control value above as a dialable recipe, not a sourced setting.
