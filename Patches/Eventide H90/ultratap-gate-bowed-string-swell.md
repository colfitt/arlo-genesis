# UltraTap-Gate Bowed-String Swell (apalazzolo technique)

## Concept
Turn the baritone Jazzmaster or banjo into a bowed / cello-like drone voice in a small hall. UltraTap in GATE mode (factory "UltraSwell" is a good start) gates the attack into a swell; a reverb on B smooths the release. The technique is in the playing: play distinct rests, use wide vibrato, "play the effect."

## How to play it
1. Engage UltraTap in GATE mode on Preset A (start from factory "UltraSwell").
2. Add a reverb on B to smooth the release.
3. **Play distinct RESTS between notes** to reset the UltraTap gate.
4. Let note length loosely set the swell time; use **wide vibrato**.
5. Delay algorithms also mimic pads with a healthy amount of SLUR / SLURM.
