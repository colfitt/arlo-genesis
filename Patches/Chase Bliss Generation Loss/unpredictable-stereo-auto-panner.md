# Unpredictable Stereo Auto-Panner (clean skitter)

## Concept
A clean-sounding mono source that randomly skitters L↔R — width/movement without obvious degradation. Failure-driven SPREAD splits the mono→stereo image into a random pan while staying mostly clean (manual pg.31).

## How to play it
1. Set **SNAG BYP dip = on**, **SPREAD dip = on**, **MISO dip = on**.
2. Turn all main knobs **DOWN** (Wow/Flutter/Saturate/Failure low) so it stays clean.
3. Set Dry to taste and listen on headphones to hear the pan.

## Notes
- Not signature-fit — it's a clean width utility — but useful.
- Best on headphones to hear the L↔R skitter.
