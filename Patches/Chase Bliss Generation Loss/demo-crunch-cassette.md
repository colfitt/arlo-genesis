# Demo-Crunch Cassette (Car Seat / Modest Mouse lean)

## Concept
The "the demo *is* the song" 4-track-cassette degrade on guitar/banjo — bedroom-tape warts left audible, sitting ahead of the PORTA424 print for a double-cassette feel. DROP BYP keeps it a *consistent* bedroom warble (full drops would make parts non-repeatable).

## How to play it
1. Pick **Model 7** (Dictatron-EX) or **10** (MS-Walker).
2. Set **DROP BYP dip = on** so Failure gives occasional crinkle without full volume drops.
3. Set Wow/Flutter ~9:00, Saturate ~11:00 for a touch of crunch, Failure ~9:00, Dry **Small**.
4. Keep hidden Hiss low + HUM BYP on so it doesn't stack a second noise floor into the downstream cassette print.
