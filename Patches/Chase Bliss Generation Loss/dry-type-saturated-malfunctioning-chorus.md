# DRY TYPE — Saturated/Malfunctioning Chorus

## Concept
A degraded chorus where the *dry* path also gets processed (everything except Wow/Flutter). The DRY TYPE dip removes the normal wet/dry chorusing — the dry now gets the same effects — so it reads as saturated/broken doubling rather than a clean wet/dry chorus.

## How to play it
1. Set **DRY TYPE dip = on**.
2. Choose the Dry toggle: **Small** (community default for melodic material — tape-dominant, keeps body), **Unity** (more dramatic), or **None** (full wet).
3. Dial Wow/Saturate/Model to taste.

## Notes
- **Small** is the Elektronauts-cited default for melodic parts.
- DRY TYPE removes the wet/dry separation, so it's a thicker, mangled doubling instead of a sparkly chorus.
- Store as an onboard preset.
