# Tape Overdrive / EQ-Only Color Box

## Concept
Using Gen Loss purely as a tape-saturation drive / 12-flavor EQ color — no wobble. Modeled magnetic hysteresis is warm, not harsh, and also boosts level slightly. Good for de-perfecting a VG-800 modeled amp.

## How to play it
1. Turn **Saturate up**; keep Wow/Flutter/Failure off.
2. Use **Model** to pick the EQ color (or OFF for none).
3. Set Dry to taste.

## Notes
- Saturate and Model are **interactive** — each model exaggerates different distortion bands.
- Subtle through most of the sweep; gnarly/mis-biased at the extreme (manual pg.27).
- Store as the drive scene.
