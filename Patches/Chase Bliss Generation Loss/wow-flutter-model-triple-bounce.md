# Wow + Flutter + Model Triple-Bounce (drunken-tape swell)

## Concept
A pronounced drunken-tape swell — wow + flutter + model morphing in one moving patch, saved so a song scene recalls the motion. The clearest "the whole pedal breathes" patch. Demoed by Nick Rees on the actual MkII.

## How to play it
1. Enable the **WOW + FLUTTER + MODEL** ramp-target dips.
2. Enable the **BOUNCE** dip and set **SWEEP = B** (travel min→current).
3. Set each of those knobs to its extreme destination — that's the far end of the ramp.
4. The **Volume/mix = SPEED**: start fast, slow it for a more pronounced swell.
5. **Save to a preset** — the moving patch is the preset.

## Notes
- 🔵 community — demoed by Nick Rees on the actual MkII (not a manual page).
- Trick: Aux = Filter + tap aux pauses the Model ramp.
