# Laptop-Bounce Cassette Demo (Dictatron crunch)

## Concept
CSH laptop/bedroom budget-DAW degrade ahead of the cassette print — bedroom-tape warts left audible, voiced explicitly for the lo-fi-prolific-volume taste wave. Essentially the existing "Demo-Crunch Cassette" (#11) anchored to the Car Seat Headrest result. DROP BYP keeps it a *consistent* bedroom warble (no full drops → repeatable). Pairs with PORTA-LF1 for the double-cassette / laptop-bounce feel.

## How to play it
1. Pick **Model 7** (Dictatron-EX) or **10** (MS-Walker).
2. Set **DROP BYP dip = on** so Failure gives occasional crinkle without full drops.
3. Wow/Flutter ~9:00, Saturate ~11:00, Failure ~9:00, Dry **Small**.
4. Keep hidden Hiss low + HUM BYP on so it doesn't stack a second noise floor into the PORTA424 print.
