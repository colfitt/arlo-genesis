# Failure Surgery — isolate the texture you want

## Concept
Keep only the part of the malfunction you want — drop the drops, the snags, or the hum independently while keeping the motion. This is how you make Failure *repeatable enough* to gig with, and the essential surgery for the doom-bed patches.

## How to play it
1. **DROP BYP dip = on** → no volume cut-outs.
2. **SNAG BYP dip = on** → no pitch-warps (keeps only crinkles/pops).
3. **HUM BYP dip = on** → no mechanical noise.
4. Mix the remaining elements to taste; set the **Crinkle/Pop Level** with the hidden Failure knob (CW loud / CCW soft).

## Notes
- Example: SNAG BYP + DROP off leaves a gentle crinkle texture only.
- Applies to whatever base you save.
