# Reduktion / HexCells / Broken FM (texture sources)

## Concept
Three texture-source packs: Reduktion (110 stereo, most abstract/textural), HexCells (203 stereo + 86 ready DT2 presets, granular percussion), and Broken FM (~1 GB stereo DX7/Digitone FM one-shots + 400 glitch loops + 16 pre-built DT2 patterns) — the standout for rich detunable drone/metallic source. Glitch/texture beds plus ready DT2 presets.

## How to play it
1. Load via Elektron Transfer → +Drive.
2. Reduktion (Keinseier) — 110 stereo, most abstract/textural.
3. HexCells (Dissonant Witchcraft) — 203 stereo + 86 ready DT2 presets, granular percussion (load samples first for correct folder structure).
4. Broken FM (Yves Big City) — ~1 GB stereo DX7/Digitone FM one-shots + 400 glitch loops + 16 pre-built DT2 patterns; standout for rich detunable drone/metallic source.

## Notes
- HexCells: load samples first so the preset folder structure resolves correctly.
- Packs ship baked-in values (not text-published) — load and reverse-engineer.
