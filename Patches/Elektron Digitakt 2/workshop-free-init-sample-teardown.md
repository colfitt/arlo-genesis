# WORKSHOP (free init-sample teardown) + MINOR_KEYS_JAMS.dt2prj

## Concept
The best free way to reverse-engineer DT2 synthesis from a known sample: WORKSHOP is 68 presets all built from the 16 INIT samples (heavy modulation + velocity + keytracking), including 26 synth-pad-lead-DRONES. MINOR_KEYS_JAMS.dt2prj is a free 8-bank project (one minor key per bank) that brings its own samples/kits/patterns so you can study someone else's kits and p-locks hands-on.

## How to play it
1. Load via Elektron Transfer → +Drive.
2. WORKSHOP — 68 presets all built from the 16 INIT samples (pay-what-you-want; 16 bass / 21 drum / 26 synth-pad-lead-DRONES / 5 FX), heavy modulation + velocity + keytracking; best free way to reverse-engineer DT2 synthesis from a known sample.
3. MINOR_KEYS_JAMS.dt2prj — free 8-bank project (one minor key per bank), Pattern 1 = demo jam, 2–8 = kit samples; brings its own samples/kits/patterns.

## Notes
- ⚠️ MINOR_KEYS_JAMS is hosted on an unverified 3rd-party file-host — scan before loading; `.dt2prj` is DT2-only.
- Packs ship baked-in values (not text-published) — load and reverse-engineer.
