# DigiSphere II — atmospheric (project carries sends)

## Concept
Lush atmospheric ambient beds with reverb/delay already dialed. The real deliverable is the project file: its Bank-A patterns carry the author's preferred REVERB + DELAY send settings, because the DT2's default sends "lack depth and size." 121 DT2-only sounds across the full stereo + 3-LFO + comb engine.

## How to play it
1. Load via Elektron Transfer → +Drive.
2. 121 sounds, DT2-only (`.dt2pst`).
3. Ships a project file whose Bank-A patterns carry the author's preferred REVERB + DELAY send settings (the defaults "lack depth and size," so the project's pre-set sends are the deliverable).
4. Full stereo + 3-LFO + comb engine.
5. Bug: a few preset names get an `X` prefix through OS1.10A.

## Notes
- Bank A patterns carry the preferred sends.
- These ship values baked into `.dt2pst` objects (not text-published) — "load and reverse-engineer."
