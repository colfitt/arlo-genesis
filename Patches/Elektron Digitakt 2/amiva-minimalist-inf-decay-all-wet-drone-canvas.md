# Amiva Minimalist INF-Decay All-Wet Drone Canvas

## Concept
The lowest-setup drone canvas: no sequencer, one pattern, `AMP DECAY = INF` on every track so each triggered sample loops/sustains endlessly. You trigger tracks by hand and route everything only through the reverb/delay sends for a fully wet atmosphere, then perform with gentle pitch shifts, filter sweeps and real-time FX. An OG-Digitakt-born method that transfers fully and improves on the DT2 (stereo / Comb / 3 LFOs).

## How to play it
1. No sequencer, one pattern.
2. Set `AMP DECAY = INF` on every track (each triggered sample loops/sustains endlessly).
3. Trigger each track by hand.
4. Source = long samples (baritone / fuzz / banjo prints, field recordings).
5. Route audio only through the reverb/delay SENDS — turn each track's main level down, sends up — for an all-wet atmosphere.
6. Perform with gentle pitch shifts, filter sweeps, real-time FX.

## Notes
- The two load-bearing settings (`AMP DECAY = INF` + send-only routing) are explicit.
- OG-Digitakt-born method; transfers fully and improves on the DT2 (stereo / Comb / 3 LFOs).
- Use one pattern; one kit of long held samples.
