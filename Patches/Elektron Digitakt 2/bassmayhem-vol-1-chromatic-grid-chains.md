# BASSMAYHEM Vol.1 — chromatic Grid chains

## Concept
Chromatic sub-bass plus Grid-machine sample-chain leads for the doom register. Ships `.dt2pst` presets and a `.dt2prj` demo, with chromatic sample chains on the Grid machine. The load-bearing gotcha: set chromatic presets' root to C1 or you get the wrong octave. Low end is kept mono to avoid phase issues on the sub.

## How to play it
1. Load via Elektron Transfer → +Drive.
2. `.dt2pst` presets + a `.dt2prj` demo. Chromatic sample chains on the Grid machine (`32` = 32-note chain, `1S` = one-shot). Chains 17–25 MB each.
3. Gotcha: set chromatic presets' root to C1 or you get the wrong octave.
4. Stereo upper register, low end kept MONO (no phase issues on sub).
5. Includes a `RANDO` pattern built with OS1.15 "Create Random Locks."

## Notes
- These ship values baked into `.dt2pst` / `.dt2prj` objects (not text-published) — "load and reverse-engineer"; only documented gotchas are concrete.
