# Cloud signature wash (high diffusion)

## Concept
The signature huge Strymon ambient Cloud wash. Cloud is deliberately sparse (its menu is just Low End / Diffusion / Boost), so high diffusion is what turns it into a smooth pad. The rig's drone-anchor machine; works on guitar, vocals, drums, and Rhodes/EP.

## How to play it
1. Dial a lot of Diffusion for the smooth-pad character.
2. Set Tone darker or lighter to taste.
3. Use it as the drone anchor under leads.

## Notes
- Relative values only — no clock positions stated.
- PARAM1 = Low End, PARAM2 = Diffusion; menu = Low End / Diffusion / Boost.
