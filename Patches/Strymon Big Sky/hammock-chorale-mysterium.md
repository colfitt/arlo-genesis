# Hammock Chorale (Mysterium go-to) — machine only

## Concept
Hammock's go-to for sustained ambient/cinematic vocal-pad walls (on Jazzmaster + baritone, the rig's instruments) is the CHORALE machine. Only the machine is confirmed — no knob values are documented, so none are given here rather than inventing them. For actual settings, use the "Wide & Deep Ambient Swell" patch (chorale-wide-and-deep-ambient-swell).

## How to play it
- Two-person live knob manipulation as performance: one plays, one rides DECAY/MIX/TONE or unplugs cables.
- Stereo doubling: stereo wash + mono-right tripped-out + mono-left dry-ish.
- Lean into MOD/detune ("more warble").

## Notes
- Real artist attribution but only the machine is documented — knob values would be invented, so none are given.
- Hammock's Strymon chain = Big Sky + TimeLine + Deco + Flint.
- Hammock's current top reverb has since rotated away from Big Sky.
