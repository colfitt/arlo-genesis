# "Ever After" — orchestral shimmer (controlled)

## Concept
The best fully-specified TWP: a resplendent orchestral Shimmer with voice 1 up two octaves and voice 2 down one. INPUT mode applies the shifts to the input only, so they don't self-regenerate — the result stays controlled rather than runaway. Demoed on an Oberheim Matrix-6 pad, which maps directly to VG-800 modeled pads into the Big Sky.

## How to play it
1. Feed a synth/VG-800 pad in.
2. Hold a chord; the +2/-1 octave shimmer wraps it in orchestral ambience.
3. Keep Mode = INPUT to stop the shifts piling up — that's what keeps it controlled.

## Notes
- Shift1/Shift2/Amount/Mode/Decay are exact and cited. MIX/TONE/PRE-DELAY/MOD are dial-art — the .syx is authoritative.
- INPUT vs REGEN is the lever: INPUT = controlled, REGEN = self-regenerating/bigger.
