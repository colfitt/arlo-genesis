# Swell "Dry Mode" banjo-attack eraser (technique)

## Concept
The banjo-pluck eraser for banjo-as-lead pads. In Dry mode (SWLDRY), Swell swells the dry signal itself into the reverb, hiding the picking attack entirely — versus Wet mode, which swells only the reverb behind the dry. A short Rise Time gives a fast auto-volume-swell, and the reverb is set for maximum ambience.

## How to play it
1. Set Mode = Dry (SWLDRY) so the dry signal itself swells in.
2. Use a short Rise Time for a fast auto-volume-swell.
3. Pick the banjo — the percussive attack is erased into an instant pad.
4. Pair under it with a frozen Shimmer fifth (shimmer-self-build-freeze-a-fifth) as a drone bed.
