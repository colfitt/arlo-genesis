# "Star Cloud" — zero-dry ambient pad

## Concept
An ethereal wall of sound played zero-dry (Mix full wet) so the source — guitar, banjo, or VG-800 — becomes an organic ambient pad. MOD sits at 2:00, described as max modulation depth at the slowest speed possible, for shimmer without obvious wobble. This zero-dry-pad move IS the rig's "sustained walls / banjo as pad" technique.

## How to play it
1. Run zero-dry (MIX full wet) so only the wet pad is heard.
2. Use the expression pedal on TONE for hands-free filter sweeps.
3. Live fade-out: roll TONE all the way down, then DECAY to minimum for a fast smooth tail kill.

## Notes
- Only MOD = 2:00 is a stated knob value; zero-dry and EXP-on-Tone are stated techniques; the rest is dial-art.
