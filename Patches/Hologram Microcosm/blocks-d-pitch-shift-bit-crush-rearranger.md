# Blocks D Pitch-Shift + Bit-Crush Rearranger

## Concept
Degraded/bit-crush rearrangement of incoming audio; or, under Hold, a pad/drone. Blocks D "rearranges incoming audio with pitch-shifting and bit-crushing," with Activity setting the amount of variation / sample manipulation.

## How to play it
1. Select engine BLOCKS (Glitch) variation D.
2. Use Activity to set the amount of variation / sample manipulation.
3. For a pad/drone, run Blocks under Hold ("typically produces a pad or drone").

## Notes
- REAL demo (Blocks D from a Boss DM-2 analog-delay feedback source).
- Blocks B = rearrange + pitch-shift only. Blocks under Hold = drone/pad per Delicious Audio.
