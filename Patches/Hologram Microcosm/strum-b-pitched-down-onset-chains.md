# Strum B Pitched-Down Onset Chains

## Concept
Banjo-as-lead — chain plucks into phasing/cascading pointillistic runs, pitched dark for doom voicing. Strum B overlaps many copies of the most-recent note onset into phasing/cascading chains. Find the Speed knob and pitch it down an octave for a darker voicing. Feed transient/plucked input (banjo, ride scrape).

## How to play it
1. Select engine STRUM (Granules) variation B.
2. Find the Speed knob and pitch it down an octave for a darker voicing.
3. Feed transient/plucked input (banjo, ride scrape).

## Notes
- REAL demo (Strum B + "pitched the speed knob down an octave").
- Strum D cascades chains and adds double-speed grains for more density.
