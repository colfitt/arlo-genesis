# Pattern A Classic Linear Multidelay

## Concept
A conventional tempo-locked multidelay — the essential "normal delay" recipe and the right tool for Jazz Bass dub taps. Pattern A is a classic linear delay (Activity = number of active taps), best against a source with empty space. Sync Time to MIDI clock (or tap) so the taps lock.

## How to play it
1. Select engine PATTERN (Multidelay) variation A.
2. Set Activity for the number of active taps (low for dub bass).
3. Sync Time to MIDI clock (or tap) so the taps lock.
4. For dub bass: low Activity, Filter rolled back to avoid mud.

## Notes
- Documented engine behavior. Not signature — the most conventional, least drone-leaning engine.
- B = more syncopated; C/D = further rhythmic patterns (D "pretty subtle").
