# Interrupt B Glitchy Pitch-Shift Bursts

## Concept
Degraded glitch stabs interrupting the dry signal — recorded-wrong texture. Interrupt B is a "really glitchy algorithm with various pitch-shifted samples" interrupting the dry signal; Repeats sets how often the glitches fire. At Mix 100% the dry signal is muted entirely (dry returns as Mix drops below 100%).

## How to play it
1. Select engine INTERRUPT (Glitch) variation B.
2. Use Repeats to set how often the glitches fire.
3. Set Mix to 100% to mute the dry entirely, or drop below 100% to let dry back in.
4. For the bit-crush alt: switch to Interrupt D and add Reverse (CC 47) — "bit crush reversed sounds great."

## Notes
- REAL demo (Interrupt B from a kalimba source).
