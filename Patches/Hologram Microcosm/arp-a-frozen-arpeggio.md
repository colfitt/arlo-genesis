# Arp A Frozen Arpeggio

## Concept
A multi-note freeze: chain distinct plucks into an arpeggiating pattern that sustains past natural decay. Unlike Tunnel B (single sample), Arp A keeps multiple notes present and loops them as an arpeggio. The banjo-as-lead onset-chain the dossier calls out.

## How to play it
1. Select engine ARP (Glitch) variation A.
2. Set Activity all the way down to repeat only the most recent sample, or turn Activity UP to feed more buffered samples into the pattern (Activity = number of steps).
3. Play distinct notes for clear arpeggios.
4. Engage HOLD: it loops an arpeggiating pattern of the recently-sampled notes.
