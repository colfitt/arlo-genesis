# Looped Banjo Layers — clock-face starting point

## Concept
Octave-stacked looped banjo layers built inside the Microcosm (granular bed-building) — a dial-it-now clock-face starting point. Engine Mosaic A (octave-up harmonies) with the looper Post-FX and Quantize on. The dossier explicitly defers structured song loops to the CB Blooper — the Microcosm looper is a granular-capture bonus.

## How to play it
1. Select engine Mosaic A (octave-up harmonies).
2. Set looper POST-FX, Quantize on.
3. Dial: Activity 11:00 · Mix 12:00 · Loop Level 12:00.
4. Record a banjo phrase (left footswitch), close the loop, overdub octave-stacked layers.
