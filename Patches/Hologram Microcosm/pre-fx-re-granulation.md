# Pre-FX Re-Granulation

## Concept
Record a banjo roll, then re-process the recorded loop through the live engines (Tunnel/Haze) — bed-building without a second pedal. With looper routing set to Pre-FX, the recorded loop is re-granulated in real time as you twist the engines. Pre-FX = "paintbrushes" (each overdub keeps its own tone); Post-FX default = "Instagram filter" (the whole loop re-granulated, "never sounds the same twice").

## How to play it
1. Record a phrase into the Phrase Looper.
2. Set looper ROUTING = PRE-FX: looper mode → hold looper button → "Pre-FX" lights → push selector → push looper again (or CC 24 ≥ 64).
3. Twist the live engines so the recorded loop is re-granulated in real time.
