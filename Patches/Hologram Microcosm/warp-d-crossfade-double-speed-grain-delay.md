# Warp D Crossfade Double-Speed-Grain Delay

## Concept
A spacious textural multidelay that smears taps with grains — the dossier's "ambient smear wash" alt. Warp D crossfades delay taps with double-speed grains ("overlaps layers of samples with filter sweeps"). Activity sets the number of active taps; Whirrings calls "Warp D. Beautiful. Four repeats sounds great," so set Activity around 4.

## How to play it
1. Select engine WARP (Multidelay) variation D.
2. Set Activity to about 4 active taps ("four repeats sounds great").
3. For the alt smear voice, switch to Warp B (resonant bandpass per tap).

## Notes
- REAL demo (Warp D from a spring-tank source) plus Whirrings' "4 repeats" setting.
- Warp B = resonant bandpass per tap (the §13c alt smear voice).
