# Arp B Square-Shape 4× Rhythmic Arp from Plucks

## Concept
Turn banjo rolls into sequenced cascading leads locked to the board's clock. Arp B (pitch-shift) with a square Shape and Time at 4× the tempo turns distinct plucked notes into a fast rhythmic arpeggio. Sync Time to the Digitakt clock to lock the arp to the board.

## How to play it
1. Select engine ARP (Glitch) variation B (pitch-shift).
2. Set Shape to square.
3. Set Time to 4× the tempo.
4. Feed distinct plucked notes (tapping demonstrated in the demo).
5. Use Activity to set the number of steps.
6. Sync Time to the Digitakt clock to lock the arp to the board.

- REAL demo (Arp B + Shape=square + 4× tempo all stated).
