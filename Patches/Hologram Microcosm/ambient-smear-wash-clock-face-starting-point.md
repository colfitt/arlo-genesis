# Ambient Smear Wash — clock-face starting point

## Concept
A glacial, diffuse wash feeding the QI Etherealizer + Dark Star smear stage — the Board-2 smear sound into the diffuser stage. Engine Haze B or Warp B (resonant bandpass) with a full clock-face chart; let the downstream pair catch the output and diffuse it.

## How to play it
1. Select engine Haze B (or Warp B resonant bandpass).
2. Dial: Activity 1:00 · Shape 1:00 (slow envelope) · Filter 11:00–12:00 · Space 3:00 (mostly wet) · Mix 1:00.
3. Let QI Etherealizer + Dark Star V3 catch the output.
4. Pull Dark Star back so it diffuses rather than double-granulates (watch granular-into-granular mud).
