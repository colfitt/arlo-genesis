# Seq D Bit-Crushed Sub-Octave Layer

## Concept
A degraded, sub-weight rhythmic texture — "gone a bit Aphex Twin / Window Licker." Seq D produces interlocking rhythmic samples with bit-crushing; Activity adds rhythmic layers. Turn fully clockwise to add a bit-crushed sub-octave (octave-down) layer. On-aesthetic for the degraded/recorded-wrong lean.

## How to play it
1. Select engine SEQ (Micro Loop) variation D.
2. Use Activity to add rhythmic layers.
3. Turn fully clockwise to add the bit-crushed sub-octave (octave-down) layer.
