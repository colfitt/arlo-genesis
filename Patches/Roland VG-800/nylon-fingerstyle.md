# Nylon String — the standout acoustic model

## Concept

The VG-800's NYLON acoustic model — the one the Gear Sounds presenter calls "the best acoustic guitar simulator in the world as of now," and that Marcus Curtis singles out as a real test the box passes ("how well did they nail nylon — not bad"). A warm classical/nylon fingerstyle voice: a genuinely strong, pristine clean source for the tape-degrade stage. `INST TYPE = ACOUSTIC` → cycle the acoustic sub-type to `NYLON`. The acoustic types cycle `MA28 / TRP0 / GB48 / GLD / NYLON` (per Gear Sounds).

## How to play it

1. Press `[INST]`, set `INST TYPE = ACOUSTIC`, then cycle the acoustic sub-type to `NYLON`.
2. Set `BODY / RESONANCE` moderate; fingerpick for the classical voice.
3. Blend `NORMAL MIX` low-to-moderate to keep real finger/pluck noise.
4. Pristine → Generation Loss → JHS 424 / PORTA424 to ruin, or leave dry for an intimate bed.

## Notes

- The nylon model is the consistently-praised acoustic voice (Gear Sounds calls it the best acoustic sim he's heard; Marcus Curtis: "how well did they nail nylon — not bad").
- Acoustic sub-types named by Gear Sounds: `MA28` (Martin), `TRP0`, `GB48`, `GLD`, `NYLON`. Exact per-model EQ/body values are **not published** — `BODY`/`RESONANCE`/`NORMAL MIX` here are a dial-by-ear recipe, not sourced values.
- Provenance: named INST model (no published knob values) — dialable recipe.
