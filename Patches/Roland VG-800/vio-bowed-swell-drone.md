# VIO Bowed Swell Drone

## Concept

An attackless violin-swell drone from the banjo or baritone — it removes the pick attack entirely for a Dark Star V3 / QI Etherealizer smear. `INST TYPE = VIO GUITAR` with `LEAD EMPHASIS` up gives the bowed/synthy character ("guitar and a synth filter together" — Gear Sounds). This is the strongest single aesthetic fit for drone/doom in the rig.

## How to play it

1. Set `INST TYPE = VIO GUITAR`, push `LEAD EMPHASIS` up for the bowed/synthy voice.
2. Add `SLOW GEAR` or `SOUND HOLD` in an FX1–3 slot for the volume-swell envelope.
3. `NORMAL MIX = OFF` to kill the real pick attack.
4. Optional `ALT TUNE` to set the drone pitch.
5. Onboard REV/DLY off; double the swell with the instrument volume knob.

- VIO is a real INST type (manual); the swell-drone build is designed for the rig.
