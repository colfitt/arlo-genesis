# Fretless Guitar — gliding funk lead

## Concept

The VG-800's FRETLESS electric-guitar model — "Boss does pretty good fretless, great for funk" (Marcus Curtis). A smooth, glissando-capable lead/comp voice with no fret definition, on-point for sliding funk lines and the rig's "strange-source" aesthetic. The fretless model is listed in the E.GUITAR lineup (Parameter Guide); Juca Nery also tours a "Fretless guitar" voice.

## How to play it

1. Recall the factory fretless-guitar memory (or set `INST = E.GUITAR`, `INST MODEL = FRETLESS`).
2. Play slides/glissandi to exploit the absence of fret definition.
3. Keep `NORMAL MIX` low so the real fretted instrument doesn't fight the model.
4. For funk, add a touch of envelope/auto-wah in an FX slot; for texture, run it clean into a smear reverb.
