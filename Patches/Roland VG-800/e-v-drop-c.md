# E V Drop C (alt-tuning, no retune)

## Concept

Drop-tuning weight without physically retuning — the doom-baritone-weight enabler. `ALT TUNE TYPE = DROP C` (Drop C/B/Bb/A are all available via the DROP family; D-MODAL / Open D / Nashville are also on tap). A real named factory preset demonstrating the DROP family.

## How to play it

1. Set `INST TYPE = E.GUITAR`, `ALT TUNE SW = ON`, `ALT TUNE TYPE = DROP C`.
2. Assign the `ALT TUNE SW` toggle to CTL-1 = back to standard tuning on a footswitch.
3. Pick a dark AMP.

## Notes

- Real named factory preset (Juca Nery) demonstrating the DROP family; the `DROP D–A` mechanism is manual-verified.
- CTL-1 = standard-tuning toggle. Copy to User Memory to tweak.
