# Around-the-World Acid Filter-Bass (baritone)

## Concept

Baritone JM (or banjo) as the Daft Punk "Around the World" bouncing sawtooth acid-pluck bass — a short repeating figure whose filter opens over the build (Daft Punk "Around the World": sawtooth acid-pluck bass, low cutoff, high reso, snappy decay). A 🟣 designed-to-emulate build.

## How to play it

1. Recall GK SETTING BARI JM. Switch to BASS mode (a separate 150-memory set).
2. Set `INST TYPE = SYNTH`, `SYNTH TYPE = FILTER BASS`.
3. `FILTER CUTOFF ~50` (low), `FILTER RESO ~60` (high — acid signature), `TOUCH SENS ~40`.
4. `FILTER DECAY ~25` (snappy = the pluck / no sustain), `COLOR 70–90` (weight).
5. Optional `ALT TUNE = -12 STEP` for a sub-octave.
6. Play a short repeating figure; ride `FILTER CUTOFF` open over the build (or have the 61SL/Elektron do it).

## Notes

- BASS mode = separate 150-memory set. FILTER BASS param names manual-verified; values designed (verify by ear).
