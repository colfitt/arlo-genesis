# CRYSTAL Shimmer Pad (bell drone)

## Concept

A glassy bell-attack pad/drone for the Dark Star V3 / Etherealizer smear — the "Crystal Echo" ("Cristo Echo") factory voice, drone-fied. The CRYSTAL synth's short shimmer attack with modulated tune/depth makes the undulating bell entry; high `SUSTAIN` holds it as a long drone (the fast-decay-banjo fix for shimmer drones).

## How to play it

1. Set `INST TYPE = SYNTH`, `SYNTH TYPE = CRYSTAL`.
2. `ATTACK LENGTH ~30` for a short shimmer attack; `MOD TUNE ~50`, `MOD DEPTH ~40` for the undulating attack.
3. `ATTACK LEVEL ~70`, `BODY LEVEL ~80`.
4. `SUSTAIN = 90–100` for a long bell drone.
5. `NORMAL MIX = OFF`. Onboard REV/DLY off — let the End Board reverbs do it.
