# Sufjan Fingerpicked Banjo Lead

## Concept

The cluster's lead voice — Sufjan's Deering Goodtime played like a 4-string guitar, fingerpicked rolling arpeggios (Sufjan Stevens — banjo-as-lead, Seven Swans / Carrie & Lowell). The 5th-string exclusion is Sufjan's "4-string guitar" approach, already the rig's banjo strategy. A 🟣 designed-to-emulate build.

## How to play it

1. Recall EBM5 BANJO.
2. Set `INST = E.GUITAR` or `ACOUSTIC`, 5th string excluded (`STRING MUTE 5` / `STRING LEVEL 5 = 0`) = "banjo as a 4-string guitar."
3. Keep the real 5th drone via `NORMAL MIX ~20–40%`.
4. Bright/clean, no dirt; keep `NORMAL MIX` moderate so the real pluck/finger noise stays.
5. Onboard REV/DLY off. Fingerpick rolling arpeggios.
