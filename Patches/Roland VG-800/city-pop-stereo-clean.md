# City Pop Clean — sparkly stereo doubled clean

## Concept

The factory "City Pop" preset Marcus Curtis and Juca Nery both flag — a bright, glassy, stereo clean voiced for 80s J-pop sparkle. `INST = E.GUITAR` (a clean single-coil-style, stereo-doubled model) with the onboard chorus + delay engaged as the factory chain. A pristine, chorus-y clean source useful as the rig's "recorded-clean" bed before the tape stage, or doubled across the stereo split.

## How to play it

1. Recall the factory "City Pop" memory (or build: `INST = E.GUITAR` clean, add the onboard chorus + delay).
2. Pick a glassy single-coil-style model; keep it bright and present.
3. Blend `NORMAL MIX` low so the model's stereo sparkle dominates.
4. For a pristine tape source, leave onboard REV/DLY minimal and route clean → Generation Loss → JHS 424 / PORTA424.
