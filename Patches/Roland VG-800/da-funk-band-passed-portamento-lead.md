# Da Funk Band-Passed Portamento Lead (banjo-as-lead)

## Concept

Banjo via GK-5 as a Daft Punk "Da Funk" lead — a distorted, band-passed, glided mono line that reads like an overdriven guitar (Daft Punk "Da Funk": distorted glided band-passed lead, melody in perfect fourths). A 🟣 designed-to-emulate build; the banjo-as-lead carries the line. Into the rig distortion (MXR 108 → Hizumitas) for the overdriven-guitar read.

## How to play it

1. Recall GK SETTING EBM5 BANJO first.
2. Set `INST TYPE = SYNTH`, `SYNTH TYPE = SOLO` (or `GR-300` for more grit).
3. PITCH-glide/portamento ON (the Da Funk slide — moves in 4ths).
4. `FILTER CUTOFF` mid + `TOUCH SENS` up = pick-triggered filter wah (the band-passed, overdriven-guitar voice).
5. `RESONANCE` moderate, `NORMAL MIX` low.
6. Into the rig distortion (MXR 108 → Hizumitas) for the overdriven-guitar read.
