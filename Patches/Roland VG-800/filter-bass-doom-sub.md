# FILTER BASS Doom Sub

## Concept

Baritone JM driving a fat synth-bass sub-octave for low-end weight/crush — "distorted bass = unbelievably fat" (MusicRadar). The FILTER BASS synth with a mid cutoff, moderate resonance, and snappy filter decay gives a pick-responsive filter close; `COLOR` is the weight knob. Into a dirty bass AMP (Ampeg/Darkglass model) for the crush.

## How to play it

1. Switch to BASS mode (a separate 150-memory set). Set `INST TYPE = SYNTH`, `SYNTH TYPE = FILTER BASS`.
2. `FILTER CUTOFF ~50`, `FILTER RESO ~50`, `TOUCH SENS ~40` for a pick-responsive filter.
3. `FILTER DECAY ~30` for a snappy filter close (needs `TOUCH SENS` not too low).
4. `COLOR = 70–90` — low-range strength is the weight knob.
5. Optional `ALT TUNE = -12 STEP` for a sub-octave.
6. Run into a dirty bass AMP (Ampeg/Darkglass model) for the crush.

## Notes

- BASS mode is a separate 150-memory set. Param names/ranges are manual-verified; the numbers are designed.
- "Razor Bass" is a related factory name (Juca Nery).
