# Open G Five-String (banjo-style, 5th muted)

## Concept

A banjo-emulation open-G tuning with the low string muted — a 5-string open-G feel that's directly on-point for the EBM-5 banjo-as-lead premise and the 5th-string-mute strategy. Juca describes "a different distortion flavor, everything set in the tuning." A real factory preset, unusually on-point for the rig.

## How to play it

1. Set `INST TYPE = E.GUITAR` (or ACOUSTIC), `ALT TUNE TYPE = OPEN G` with the low string MUTED for the 5-string open-G feel.
2. Keep `NORMAL MIX` low.
3. Pairs with the EBM5 BANJO GK profile's 5th-string exclusion.

## Notes

- Real factory preset — a string-muted open-G "5-string banjo-style" factory tuning. Marcus Curtis also demos an "OPEN G distorted patch."
