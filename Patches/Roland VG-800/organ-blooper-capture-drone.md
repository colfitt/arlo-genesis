# ORGAN → Blooper Capture Drone

## Concept

A continuous-waveform organ source built specifically as CB Blooper / Microcosm capture-and-layer food — an infinite drone wall. Same as "Organ Drone + Rotary" (ORGAN, `SUSTAIN = 100`, heavy 16/8/4 drawbars) but with no rotary — the waveform stays dead-steady for clean granular/looper capture. A steady drone is better looper/granular food than a modulating one.

## How to play it

1. Set `INST TYPE = SYNTH`, `SYNTH TYPE = ORGAN`; heavy `FEET 16/8/4` (e.g. 90/100/70).
2. `SUSTAIN = 100` for the continuous waveform.
3. No rotary — keep the waveform dead-steady for clean granular/looper capture.
4. `NORMAL MIX` minimal.
5. Add `SLOW GEAR`/`SOUND HOLD` if you want attackless entry.
6. Route into Blooper (layer) or Microcosm (granular). Onboard REV/DLY off.
