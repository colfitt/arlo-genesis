# Open-Tuned Acoustic Bed

## Concept

An open-D/G droney acoustic banjo/baritone bed, kept clean — a pristine source for the tape-degrade stage (indie-folk banjo-bed territory). Clean source → tape destruction is the degraded-tape signature: bypass dirt → Generation Loss → PORTA424.

## How to play it

1. Set `INST TYPE = ACOUSTIC` (J-45 / D-28).
2. `ALT TUNE TYPE = OPEN D` or `OPEN G`.
3. `BODY`/`RESONANCE` moderate.
4. Minimal/no dirt, `NORMAL MIX` low.
5. Keep `ACOUSTIC RESONANCE` OFF if you want per-string PAN downstream (it collapses stereo silently).
6. Route bypass-dirt → Generation Loss → PORTA424.

- Real model names; values designed (verify by ear).
