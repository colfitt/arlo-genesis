# Organ Drone + Rotary

## Concept

An organ pad-drone generated clean inside the VG-800 — ideal granular/looper food for Microcosm / Blooper / Etherealizer. The factory Organ/Leslie preset, drone-fied: heavy 16/8/4 drawbars with `SUSTAIN = 100` give a continuous waveform, and a `ROTARY` FX latched slow/fast on CTL-1 supplies the Leslie ramp. MusicRadar called the Leslie/organ "convincing." Big `SUSTAIN` defeats the banjo's fast decay.

## How to play it

1. Set `INST TYPE = SYNTH`, `SYNTH TYPE = ORGAN`.
2. Drawbars (each 0–100): `FEET 16 = 90`, `FEET 8 = 100`, `FEET 4 = 70`, `FEET 5+1/3 = 40`, rest 0–30 to taste.
3. `SUSTAIN = 100` for a continuous drone.
4. Add `ROTARY` in an FX slot with `RATE SLOW` and `RATE FAST` set; latch SLOW/FAST on CTL-1 (the rotary ramps up and down in speed via the footswitch).
5. Onboard REV/DLY off — let the End Board chew the continuous waveform.

## Notes

- Drawbar feet labels + SUSTAIN are manual-verified; the ROTARY-ramp-on-CTL-1 is the factory Organ convention (SoS + Juca).
- See the related "ORGAN → Blooper Capture Drone" for a rotary-free, dead-steady looper variant.
