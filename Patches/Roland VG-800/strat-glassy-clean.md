# Glassy Strat Clean — praised single-coil model

## Concept

The VG-800's Stratocaster model — reviewers say the Strat (and Tele) models "seemed better than ever." `INST = E.GUITAR → STRAT` set to a single-coil pickup position, run into a clean amp model with light spring reverb, gives classic glassy Strat sparkle from the banjo or baritone source. Pick the neck position for warmth or the bridge for snap, add a light compressor for even picking, and you have a present clean bed — distinct from the Tele-neck Stereo Telecaster Clean and the stereo-doubled City Pop Clean. A good foundation patch to layer synth/harmony on top of.

## How to play it

1. Select the Stratocaster model (`INST = E.GUITAR → STRAT`).
2. Choose a single-coil pickup position (neck for warmth, bridge for snap).
3. Run into a clean amp model with spring reverb.
4. Add a light compressor for sparkle and even picking.
5. Play funk chinks or clean arpeggios for the glassy Strat voice; layer synth/harmony on top.

## Notes

- Review-cited: MusicRadar / forum users say the Strat and Tele models "seemed better than ever." Strat is in the repo's E.GUITAR model list (DeepResearch §2).
- Honesty: single-coil models read quieter/cleaner than the real thing because the hex pickup rejects hum (Sound on Sound). Exact knob values are **not published** — dial by ear (recipe only).
- Distinct from Stereo Telecaster Clean and City Pop Clean (those are Tele-neck and stereo-doubled cleans).
