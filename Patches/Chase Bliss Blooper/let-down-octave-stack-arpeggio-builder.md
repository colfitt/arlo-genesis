# "Let Down" Octave-Stack Arpeggio Builder

## Concept

The literal "Let Down" move — Jonny doubled his arpeggio an octave higher, then Ed added a third layer. Blooper builds that stack from ONE recorded arpeggio: print it, overdub an octave-up copy, then a third pass — the doubled/tripled Spector wall from a single line. Designed to emulate (Radiohead used overdubs, not a Blooper).

## How to play it

1. In ADD mode, record the clean arpeggio (banjo/guitar, bright) — keep STABILITY off so the stack stays clean and chimey, matching Big Time TB1's HiFi voicing.
2. Set MOD A to Pitcher (ALT bank A2) at +1 octave (CW; length unchanged = the ADD-safe shift) — requires the BANK A dip.
3. Punch in an overdub pass → now you have the original plus an octave up.
4. Optional third pass at a 5th or octave again (the "Ed Jazzmaster third layer").
5. REPEATS max so all layers ring as the cascade.

## Notes

- This is the real doubled-octave layer the King-of-Gear / PoeticWax breakdown describes — pair it with Big Time TB1 running the multi-tap cascade over the stack for the full jangle wall, into a long hall (St Catherine's ballroom).
- Keep it clean and bright here — the jangle is the hook, so resist the degrade tie-breaker on this one patch.
