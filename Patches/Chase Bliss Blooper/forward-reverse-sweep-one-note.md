# Forward/Reverse Sweep — one note swept up then back

## Concept

Tap a long tempo, turn REPEATS up, and set the quantized pitch modifier (Stepped Speed) so a single recorded note plays forward and then backwards — constantly sweeping up to it and back. Running it in ADD mode with STABILITY up makes the sweep progressively less stable over time, so it slowly decays into a wobbling drone rather than holding a clean cycle.

## How to play it

1. Tap in a long tempo to set a long loop, and turn REPEATS up.
2. Play a single note into the loop.
3. Set the quantized pitch modifier (Stepped Speed) so the loop sweeps the note forward then backward, constantly.
4. In ADD with STABILITY up, the sweep gets less stable over time — let it decay into a wobbling drone.

## Notes

- Andy's words: "plays it forward and then backwards… constantly sweeping up to it and then back" and "less stable over time because we're in Additive mode and stability's up."
- A long tap tempo gives the sweep room to breathe.
