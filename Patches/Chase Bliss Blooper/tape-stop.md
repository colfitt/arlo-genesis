# Tape Stop — rubbery Smooth Speed halt

## Concept

With Smooth Speed set close to real-time speed, engage the modifier and immediately bring the knob to noon to create a rubbery effect like a tape machine slowing to a halt. The single most iconic Smooth Speed move and the literal opposite end of the Glacier recipe.

## How to play it

1. Set MOD A to Smooth Speed (slot A1, default bank).
2. Position the MOD A knob close to real-time speed (clockwise of noon).
3. With the loop playing, engage the arcade button and immediately sweep the knob down to noon.
4. The loop rubberizes and halts like tape slowing to a stop.
5. To keep it in the loop, do this in ADD (or one-shot the pass); note Smooth Speed is time-based so the record/play heads desync while it runs.

## Notes

- Smooth Speed is time-based — overdubbing under it records a different part of the loop than you hear (two-headed anomaly).
- Counter-clockwise of noon on Smooth Speed engages reverse playback (manual); expect to wander out of tune as the speed drops.
- Pair with the Starter trick (Stopper alt-bank) for a stop-then-start gesture, or use Glacier for the near-stopped drone instead of a full halt.
- Exact knob clock positions beyond "close to real-time then noon" are not published — dial from recipe.
