# Blippertronics / Frippertronics fade-looping

## Concept

An endless evolving ambient base — a slowly evolving sustained-loop meditation that breaks up repetition. With REPEATS fairly high, all layers subtly fade so the loop always shifts and moves; once you've built a nice ambient base, max REPEATS to lock it. The "old classic" Frippertronics voice, hands-free.

## How to play it

1. In NORM, build a delay-style base — tap the RECORD footswitch like tap-tempo into the overdubbing state, with a nice long delay time.
2. Set REPEATS fairly HIGH so all layers subtly fade — it always shifts and moves.
3. Improvise over it (bad notes fade away).
4. Once you've built a nice ambient base, turn REPEATS all the way up (max) to LOCK it — nothing fades.
