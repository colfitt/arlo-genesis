# Filter-fades-on-repeat

## Concept

A delay where each repeat gets progressively darker. With Blooper set as a delay and recording, a slightly-engaged low-pass Filter prints the effect each pass, so every repeat darkens further. A staple ambient move into the H90's bigger spaces.

## How to play it

1. In ADD mode, set Blooper as a delay — REPEATS rolled back, leave it recording.
2. Engage MOD B Filter (B6) very slightly toward the low-pass side (CCW).
3. As it records, it prints the effect each pass, so every repeat darkens.
4. Turn the Filter knob further CCW for faster/heavier filtering per pass.

## Notes

- Knob positions are relative (from Andy's livestream), not clock-face.
- A staple ambient move into the H90's bigger spaces.
