# Kid-A "Loops Mistreated" Aging Wall

## Concept

The Kid A method verbatim — "loops mistreated, stretched, pulled out of phase… everything breathes wrong on purpose." Build a loop, then let it rot and drift like a tape machine failing while you keep playing clean over it. This is essentially the Accumulator patch aimed squarely at the Kid A aesthetic — the most on-lens of the Radiohead Blooper patches.

## How to play it

1. In ADD mode, record a clean loop (banjo/VG/Rhodes-like phrase) with STABILITY off.
2. Turn STABILITY up — mild to moderate "analog feel" — so each pass ages; older layers wobble, lose highs, and drift out of phase ("breathes wrong").
3. Keep overdubbing; pull REPEATS back (~11–1 o'clock) so old layers fade → continuous time-shift.
4. Add Dropper (A2) mild for granular crumble (Aphex/Autechre rhythmic artifacts left uncorrected).
5. Use the BLIP Stability slider to bias toward flutter, not white-noise hiss.

## Notes

- Fresh notes appear clean while the bed rots — exactly Ed cutting up the Rhodes in real time over a mistreated loop.
