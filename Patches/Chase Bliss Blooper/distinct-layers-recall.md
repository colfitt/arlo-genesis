# Distinct Layers — re-tap to commit independently recallable layers

## Concept

Andy Othling's core layering technique: instead of circling and stacking on one layer, deliberately re-tap the LEFT footswitch to commit each idea as a separate layer (bed / stacking / chords / melody / effects pass). This is how you get independently-recallable layers and navigate them with the LAYERS knob — the foundation of Loop Capsules.

## How to play it

1. Record layer 1 (a bed), then re-tap the LEFT footswitch into playback to commit it.
2. Re-tap to record layer 2 (stacking), commit; layer 3 (chords underneath); layer 4 (melody on top); a final layer with effects applied (switch to ADD for that effects-printed pass).
3. Don't just keep circling one layer — each record→playback transition is what creates a distinct, recallable layer.
4. Navigate between committed layers with the LAYERS knob (you can only peel from the top of the stack).

## Notes

- Andy: "I could have kept stacking on one layer... but I wanted to get all those layers distinct."
- You cannot pull out one layer in isolation — the LAYERS knob is undo/redo: navigating back + pressing record erases everything above (use it to fork).
- Limit: 8 discrete undo/redo points (1–7 independent, 8–16 lumped).
- No published knob values — this is a footswitch/workflow technique. Treat REPEATS (and any other knob) as a dialable recipe set to taste.
