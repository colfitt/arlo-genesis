# Skip Protection

## Concept

An occasional disturbance, like a bumped Discman — a subtle "recorded-wrong" character. Set Scrambler to the lowest possible random setting and the loop only glitches now and then.

## How to play it

1. Set MOD B to Scrambler (B5) at the lowest possible random setting — knob just CCW of noon (CCW = random).
2. The loop occasionally disturbs, like a bumped Discman.

## Notes

- Andy uses the same mild Scrambler over a falling/octave-down delay so it "just kind of falls apart."
