# Clumped Chords — Stepped Speed remixes a loop into chords

## Concept

Adjusting Stepped Speed during Additive recording reconfigures the loop into what sounds like chords — extending, shrinking, and repeating sections along the way. A single melodic loop becomes a clustered, chordal remix of itself, staying in key because the steps land on octaves and fifths.

## How to play it

1. Record a loop with some note/chord movement (a single note or short phrase also works).
2. Set MOD B to Stepped Speed (B4) and enter ADD mode.
3. While overdubbing, step the speed up and down — the loop reconfigures into chord-like clumps as sections extend, shrink and repeat.
4. Because Stepped Speed is quantized to octaves and fifths, the result stays in key.

## Notes

- ADD mode is essential — the speed moves get baked into the loop.
- Stepped Speed is time-based, so the fixed-loop-length rule applies — slowing past the container loses content, speeding repeats it within the container.
- Different from Chord-from-one-root (Chromatic Speed punch-ins) — this is continuous octave/fifth speed remixing rather than chromatic interval punch-ins.
- Cheat-sheet tip: shift a fifth up or down and play alongside the original key (see Harmonious Fifth).
- Honesty flag: exact knob clock positions are not published — the other knobs dial from this recipe.
