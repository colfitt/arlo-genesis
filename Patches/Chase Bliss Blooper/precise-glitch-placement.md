# Precise Glitch Placement — record a glitch into one exact spot

## Concept

Because the loop length is fixed and the record head is respected, you can plant any additive material at a specific moment in the loop: start overdubbing exactly when a target note plays, create the glitch, and when the loop comes back around the glitch lives permanently in that exact spot. This is KNOBs' precise-placement technique — the two-headed model keeps the record head steady, so it lays audio down wherever it currently is.

## How to play it

1. Enter ADD mode with a loop already playing.
2. Wait for the exact note/moment you want to replace; start overdubbing (LEFT footswitch) right then.
3. Make the glitch — a quick Trimmer/Scrambler burst on MOD A/B, or a played sound.
4. Stop overdubbing. When the loop comes around, the glitch is permanently fixed at that spot.

## Notes

- Relies on the steady record head + fixed measure (the two-headed "anomaly" model) — the record head lays down audio wherever it currently is, so a precisely-timed punch lands precisely.
- Related to the existing Punch-in patch, but this one is specifically about replacing one exact note/moment rather than general punch-ins.
- No published knob values — this is a placement technique. The MOD A/B settings are a dialable recipe: pick a modifier, engage it briefly, then stop. Dial the rest to taste.
