# True Glitch

## Concept
Natural-sounding glitches that pop in and out instead of repeating steadily — the sparse, "recorded-wrong" character in one toggle. Per the manual: "The middle ERROR TYPE makes it easy to create more natural sounding glitches that pop in and out instead of repeating steadily. Adjust ERROR to control how broken and sporadic the glitches are."

## How to play it
1. Set the channel to **GLITCH**.
2. Set **TYPE = middle = CONDITION**.
3. Turn **ERROR up** to control how broken and sporadic the glitches get.
