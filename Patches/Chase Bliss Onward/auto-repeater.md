# Auto-Repeater

## Concept
A simple, classic repeater that samples and infinitely loops each note you play — repeats that disappear cleanly after a set time. The easy GLITCH-channel starting point. Per the manual: "Turn SUSTAIN down and the repeats will cleanly and instantly disappear after a chosen amount of time."

## How to play it
1. Set the channel to **GLITCH**.
2. Turn **SUSTAIN down** so the repeats fade and disappear cleanly after a set time.
3. Set **FADE toward fast/right** per the icon.

## Notes
- Manual specifies only SUSTAIN-down + FADE; SIZE/ERROR are open.
- The "Classic Repeater Effect" Swain reads aloud is the same idea at **SUSTAIN max** (infinite loop) instead of down.
