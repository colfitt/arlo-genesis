# Dancing Drone

## Concept
A frozen chord that turns generative — momentary pitch shifts make a simple freeze feel like a self-composing piece. The shoegaze wall that never sits still. Per the manual: "Having TYPE set to Playback will introduce momentary pitch shifts that can make a simple freeze feel like a generative composition."

## How to play it
1. **Hold/lock the FREEZE footswitch** for an infinite pad.
2. Dial **ERROR up**.
3. Set **TYPE = PLAYBACK** — this introduces the momentary pitch shifts that make the freeze feel generative.

## Notes
- Correction vs the old dossier, which merged this into Living Pad — the manual lists it **separately** in Error Ideas, keyed on **TYPE = Playback**.
- Use over Living Pad when you want pitch/direction drift specifically, not just timing variety.
