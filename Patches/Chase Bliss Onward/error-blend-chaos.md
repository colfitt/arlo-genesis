# Error Blend Chaos (the "secret")

## Concept
The secret unlisted mode CB revealed — blend all three error types at once over a chosen base TYPE for maximally chaotic, organic instability. This is the engine behind Uh Oh exposed as a continuous control rather than a named recipe. The ERROR knob still sets overall error *chance*; ERROR BLEND sets the *likelihood an error is one of the unselected types*.

## How to play it
1. Set a base **TYPE = TIMING**.
2. Enter the hidden option: **hold both footswitches + turn the ERROR knob** to set **ERROR BLEND** (mixes in the two unselected error types).
3. Crank it right — random Condition + Playback errors come in too.
