# Swain Freeze-Base / Glitch-Top

## Concept
One voice as a sustained harmonic base, the other as an active rhythmic textural top — played together, "like bringing in 20 more musicians." The most on-aesthetic documented technique. FREEZE is the sustained base voice; GLITCH is the active rhythmic textural top that chimes in. Maps straight onto the rig: banjo drives the Glitch top, baritone/chord locked as the Freeze base.

## How to play it
1. Set **FREEZE = the sustained base voice** (lock/hold, or MANUAL-capture a held tone).
2. Set **GLITCH = the active rhythmic textural top** that chimes in.
3. Set the **MANUAL dip ON** for hands-on capture/stop control.
4. **"Bounce the SIZE"** — assign the Ramping › Bounce LFO to SIZE, with the SWEEP toggle, for evolving glitch subdivisions (manual p.42).
5. Place the pedal at the **end of the chain**.
