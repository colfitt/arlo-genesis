# Sprawling Ambient (Swain demo)

## Concept
The same "turn any sound into a sprawling ambient composition" target as Instant Eno — but with the only two **hard numbers** anyone has captured on video. Courtney Swain, reading off the unit: "size to 3:00, mix all the way, baby — this setting will turn any sounds that you play into a sprawling ambient composition." Effectively Instant Eno with two on-screen-confirmed values.

## How to play it
1. Set **SIZE = 3:00** (3 o'clock).
2. Set **MIX = all the way up** (full wet).
3. Play and let it sprawl.

## Notes
- These are the only two on-screen-confirmed (hard) values for this sound — the rest of Instant Eno is interpolated.
- delaydude independently lands on MIX-full = "float away completely," so the all-wet target is double-sourced (see mix-full-float-away-wall).
