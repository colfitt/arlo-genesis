# Getting-Started Dip Setup

## Concept
A safe starting dip configuration if dynamic re-triggering feels unpredictable, then add width/pumping as desired. Because dips save into presets, settle this before you store anything.

## How to play it
1. Start with **ALL dips OFF except LATCH.**
2. Then add as desired:
   - **SPREAD** — stereo width/movement tied to Error.
   - **MISO** — mono→stereo.
   - **½ SPEED** — longer/lo-fi glitch.
   - **DUCK** and/or **SIDECHAIN** — pumping.
3. For the hot GK-5/VG-800 line source, set hidden **SENSITIVITY (on SIZE) toward LESS.**

## Notes
- Community/CB-consensus rule, aligned with the manual; all dip definitions are manual-verified.
- Because dips save into presets, settle this before you store anything.
