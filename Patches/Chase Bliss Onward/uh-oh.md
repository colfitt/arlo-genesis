# Uh Oh

## Concept
The maximal "recorded-wrong" patch — every error type equally likely, maximum variation and surprise. The rig's ERROR-forward target taken to the limit — the "Beautiful Errors" (ALPFV) aesthetic engine maxed out. Per the manual: "Anything can go wrong in this setting. Each error is equally likely to occur… Adjust the ERROR knob to have more or less errors overall."

## How to play it
1. Turn **ERROR up**.
2. Engage the **ERROR BLEND hidden option ON** — hold both footswitches, the hidden option under the ERROR knob (runs Timing + Condition + Playback simultaneously, p.17).
3. Adjust the ERROR knob for more or fewer errors overall.

## Notes
- The rig's ERROR-forward target taken to the limit — the "Beautiful Errors" (ALPFV) aesthetic engine maxed out.
- ERROR BLEND is the load-bearing hidden setting.
