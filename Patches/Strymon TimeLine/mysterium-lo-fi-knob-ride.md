# Mysterium Lo-Fi knob-ride

## Concept
THE rig-relevant gold standard. Hammock's main machine on *Mysterium* was Lo-Fi, with one player riding the Lo-Fi knobs — and unplugging cables mid-decay — to mangle the tail. The technique *is* the patch: there are no fixed knob values, it's a live performance of degradation placed at the very end of the chain, post-reverb (after the BigSky / H90). Wash-on-wash.

## How to play it
1. Run the TimeLine last in the rig, after the reverb.
2. Sweep Sample / Bits / Vinyl by hand as the tail decays.
3. Solo adaptation: assign all three to EXP via EP SET and ride the pedal.
4. For the mangle move, interrupt cables mid-decay to corrupt the tail.

## Notes
- No fixed knob values are documented — the technique is the patch.
