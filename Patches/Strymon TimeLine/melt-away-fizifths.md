# Melt Away / Fizifths

## Concept
A +5th short Ice voicing — the upper note a perfect fifth above the root, set very smeared and diffuse. This is the factory preset Chords of Orion adopts as the drone-bed foundation and the base layer of the silent-loop drone. Melt Away (01A) is the short version; Fizifths (06A) is the longer Oct+5th variant with a Short slice.

## How to play it
1. Load 01A (Melt Away, ~290 ms) or 06A (Fizifths, 900 ms, Oct+5th).
2. Push SMEAR high so the fifth diffuses into a bed rather than discrete repeats.
3. Set HP ~80–160 Hz so a bright banjo doesn't ice-pick.
4. Use as the base layer for the silent-loop Ice drone method.

## Notes
- BLEND and SMEAR are bar-graphs on the factory card; SMEAR "high" is described, exact value not numerically captured.
- Fizifths uses Oct+5th interval and a Short slice for a wider, longer voicing.
