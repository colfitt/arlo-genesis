# Megaphone — Lo-Fi as filter

## Concept
The Lo-Fi machine repurposed as a degraded tone-mangler/filter rather than a rhythmic delay: delay time pulled to near-zero so a megaphone/telephone gadget filter does all the work, with the expression pedal sweeping feedback for a performable runaway. Pure "recorded-wrong" color.

## How to play it
1. Import the free `.syx` via Nixie 2 (no `.syx` on disk; transcribed setting).
2. Pull delay time to near-zero so the gadget filter is the effect.
3. Sweep the expression pedal to push feedback into a live runaway.

## Notes
- The specific megaphone/telephone gadget filter # is image/.syx-only and not captured.
- Sits right next to the Mysterium Lo-Fi knob-ride — color, not a rhythmic delay.
