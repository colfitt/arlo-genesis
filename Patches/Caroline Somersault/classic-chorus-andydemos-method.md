# Classic Chorus (AndyDemos method)

## Concept
The reference recipe for guitar, built around the key OFFSET tip. The counterintuitive lesson lives here: OFFSET fully CCW is the *tight* end. AndyDemos sets "DEPTH about halfway and the OFFSET fully counter-clockwise for the fastest delay time."

## How to play it
1. MIX ~11:00 (a dry+wet blend so it stays a chorus, not vibrato).
2. OFFSET full-CCW (7:00) for the fastest delay / least warp.
3. DEPTH ~12:00 (halfway), SPEED ~12:00, triangle WAVE, Bright TONE to start.
4. Raise OFFSET alone to add warp + apparent depth — then flip TONE to Dark to "smooth the whole thing out."

## Notes
- Raising OFFSET alone adds warp + apparent depth; flipping TONE to Dark smooths it once OFFSET is pushed up.
- This is the patch that teaches the OFFSET-as-tightness rule.
