# True North

## Concept
The home base for any source — guitar, banjo, baritone, or VG-800. Everything points straight up at noon for a smooth 80s-leaning chorus with a slight time-lag fullness. This is the start point every demo agrees on; tweak from here for tighter/looser/shimmer/water.

## How to play it
1. Set all four knobs to 12:00, WAVE to triangle, TONE to Bright.
2. Play and listen — it's a classic dry+wet chorus that keeps a clear pitch reference.
3. Adjust from here: raise OFFSET for warp, drop SPEED for slow drift, flip TONE Dark to darken the wet.
4. HAVOC stays unused on this one.

## Notes
- The factory chart row literally reads "True north — they've told me that if everything points straight up, it will all be OK."
- Herndon at NAMM compared it favorably to a Boss CE-2.
- MIX at noon = dry+wet blend = chorus (keeps a pitch reference). Going full-CW (100% wet) would turn it into vibrato.
