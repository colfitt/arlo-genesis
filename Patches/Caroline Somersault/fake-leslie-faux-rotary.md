# Fake Leslie (faux-rotary)

## Concept
A faux-rotary voice for guitar and VG-800 organ-pad patches. The chart pairs SQUARE with a slow base SPEED + high DEPTH + mostly-wet MIX; the rotary "spin" is the HAVOC gesture — instant max rate, coast back on release. Factory row: "Sometimes a fake Leslie sound is as cool as the real thing, and it's almost always lighter."

## How to play it
1. MIX ~1:30 (mostly wet), OFFSET ~9:00 (low), DEPTH ~1:30 (high).
2. Set the base SPEED slow ~9:00 so the HAVOC ramp reads as a Leslie accelerating.
3. WAVE square, TONE Bright.
4. Stab HAVOC for the speed-up "spin," release to coast back to the slow base.

## Notes
- Dial the base SPEED slow ~9:00 so the HAVOC ramp reads as a Leslie accelerating.
- HAVOC is momentary: instant max rate, coasts back on release (no ramp on the way up).
