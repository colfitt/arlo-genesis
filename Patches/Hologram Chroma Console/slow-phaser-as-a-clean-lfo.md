# Slow PHASER as a Clean LFO

## Concept
Smooth slow auto-movement on a sustained/drone source — "a little bit like an LFO," the cleanest movement on a synth or pad. MOVEMENT PHASER with RATE all the way down (slow) and AMOUNT all the way up adds super-slow, smooth movement.

## How to play it
1. MOVEMENT **PHASER**.
2. Set **RATE all the way DOWN** (slow).
3. Set **AMOUNT all the way UP**.

## Notes
- KVNDRA's favorite movement on an arp.
- wolfewithane and KVNDRA both confirm RATE down + AMOUNT up.
