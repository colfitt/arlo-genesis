# Best Chorus

## Concept
The most musical chorus on the pedal — a random-waveform + stereo-width chorus built from the VIBRATO effect. Turning the secondary DRIFT up morphs the sine toward a random waveform AND adds stereo width; backing MIX off slightly blends the dry with the widened wet.

## How to play it
1. MOVEMENT **VIBRATO**.
2. Turn secondary **DRIFT up** (morphs sine toward random AND adds stereo width).
3. Back **MIX** off slightly so dry blends with the widened wet.
4. Set RATE/AMOUNT to taste.

## Notes
- Untagged — a conventional clean chorus (left untagged on purpose so the bank stays complete).
- Factory trick: back MIX off for chorus; leave MIX up for vibrato. Sine lives at DRIFT/RATE ~7 o'clock, randomness increases from there.
- Wittek's own "Chorus" preset is instead built on the **PITCH** module ("so much versatility") for a more unusual voice — try that variant too.
