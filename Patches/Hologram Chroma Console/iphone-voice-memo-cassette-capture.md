# iPhone Voice-Memo Cassette Capture

## Concept
The taste-profile cluster's true "recorded-wrong" patch — but the INTIMATE/lo-fi kind, not doom. It maps the Carrie & Lowell iPhone Voice Memos capture (A/C hiss left in, recorded in a hotel room) onto the Chroma: a CASSETTE degrade, a gentle SWEETEN lift so the close source still reads present, and a subtle pitch-warbled VIBRATO that suggests a "wrong machine."

## How to play it
1. TEXTURE **CASSETTE**, AMOUNT ~**10–11 o'clock** (wow/flutter/hiss = the iPhone-in-a-hotel-room degrade).
2. CHARACTER **SWEETEN** both ~**9–10** (gentle preamp lift so the close source still reads present).
3. MOVEMENT **VIBRATO**, RATE ~**9**, AMOUNT ~**8**, secondary **DRIFT ~9** (subtle pitch warble = "wrong machine").
4. DIFFUSION bypassed, or **SPACE** tiny (halo, not hall).
5. Set **MIX ~65–70%**. Run Buffered+Trails, both outs connected.

## Notes
- 🟣⭐ The intimate recorded-wrong degrade, NOT doom. Maps to Chroma sheet #2/#4. Suggested slot A (tape/lo-fi).
- Don't double Gen Loss tape — pick this OR Gen Loss as "the tape."
