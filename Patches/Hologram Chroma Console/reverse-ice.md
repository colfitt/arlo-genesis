# Reverse Ice

## Concept
A pulsing, rhythmic reverse texture with no dry attack audible — "a pulsing rhythmic texture … side-chain sort of thing." DIFFUSION REVERSE at 100% wet, with the reverse repeats pitched up an octave via the top TIME/pitch knob.

## How to play it
1. DIFFUSION **REVERSE**.
2. Set **MIX = 100% wet** so there's no dry attack audible.
3. Pitch the reverse repeats **UP an octave** (top TIME/pitch knob = +1 octave).

## Notes
- The only stated numeric is MIX = 100% wet (Wittek said it aloud); the rest is recipe-only (PDF not on disk).
- REVERSE's own DRIFT is inert — animate with a GESTURE on TIME instead.
