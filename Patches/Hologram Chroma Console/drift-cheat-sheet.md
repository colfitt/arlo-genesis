# DRIFT Cheat-Sheet

## Concept
A reference recipe (not a single patch) for what the secondary DRIFT does on each effect, so you reach for the right module. Clean sine lives around 7 o'clock; randomness increases as you turn up.

## How to play it
- On **VIBRATO** → morphs sine → random waveform + adds stereo width (back MIX off = best chorus).
- On **REELS** → saturates/ages the delay line "as though the tape has aged or degraded," adds stereo flutter (official demo ~50).
- On **SPACE** → pitch-modulates the reverb tail (detuning verb).
- On **REVERSE** → DRIFT is inert; gesture TIME instead.

## Notes
- This is a reference applied across the bank, not a single dial-in patch.
