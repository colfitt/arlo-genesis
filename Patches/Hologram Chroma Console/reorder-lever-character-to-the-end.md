# Reorder Lever — Character to the END

## Concept
A mechanism recipe (not a single patch) — treat module reorder as a primary sound-design lever. Moving CHARACTER (drive) to the END produces very different results than the default placement, and a few specific orderings unlock distinct sounds.

## How to play it
1. In **FX Setup (A+D)**, tap modules in a new order.
2. Moving **CHARACTER (drive) to the END** "produces very different results than default placement."
3. Reorder is a primary lever:
   - FILTER-before-SPACE filters the source, not the tail.
   - PHASER-after-REELS modulates the delay trails.
   - SPACE-before-SWELL = a free **gated reverb** (the swell envelope chops the reverb tail).

## Notes
- This is a technique recipe applied across the bank, not a single dial-in patch.
