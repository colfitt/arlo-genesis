# Glue Only (Comp/Saturator Utility)

## Concept
The genuine utility setting the rig's old shorthand gestured at — run the hidden end-of-chain GLUE compressor/saturator on its own, with NO effect engaged. *"You can use GLUE on its own as a bonus effect... turn on either channel and set its MODIFY knob at 12:00."* Useful to gently saturate/glue the stereo bus before the Microcosm's input. This is the **only** honest "utility" use of this pedal — L&F is a 12-mode multi-fx, not a utility, and is not run always-on here.

## How to play it
1. Turn **ONE** channel on, set its MODIFY knob to **EXACTLY NOON** (= no effect; the channel passes only GLUE).
2. Engage **GLUE** via the hidden option (hold **both** footswitches): low ~9–10:00 = gentle compression; higher = compresses, then overloads into saturation.
3. ROUTING is irrelevant (single channel).
4. Set MIX to noon for WET VOLUME unity (hold the right footswitch + turn MIX to trim).

## Notes
- This is the **only** honest "utility" use of this pedal — the brief's "character preamp/gate" framing was the rig's old incorrect shorthand.
- **MODIFY-at-noon = no effect** is the #1 "is it broken?" trap.
- Useful to gently saturate/glue the stereo bus before the Microcosm's input.
- Leave SPREAD / UNSYNC as the rig needs.
