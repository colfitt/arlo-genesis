# Broken-Console Screamed Crush (character)

## Concept
Car Seat Headrest blown-out, screamed-into-cheap-mic vocal CHARACTER, voiced onto a string source. The genuine broken-console fuzz — splatty, gated, self-compressing — is the overloaded cheap-mic clip. Voiced from the existing sheet patch "Broken Console Preamp Fuzz." There is no vocal path on the rig, so this delivers the same character on banjo / baritone / VG-800.

## How to play it
1. INST input, **HI** mode. STEP 5 (max) → PRE-VOL 5:00 (maxed).
2. **Set MASTER LAST** — pull WAY back to 9–10:00; this is the biggest volume jump on the pedal.
3. Start all bands noon/flat (pure preamp distortion), then push TREBLE + (~1:00) for the harsh screamed top.
4. Engage HI-PASS ~11:00.

## Notes
- 🟣 **designed-to-emulate** + **EXPLICIT APPROXIMATION flag** — Toledo's was a screamed VOCAL into a laptop mic; this is the same CHARACTER on a string source, not his actual chain.
- **WARNING:** biggest volume jump on the pedal — set MASTER last.
