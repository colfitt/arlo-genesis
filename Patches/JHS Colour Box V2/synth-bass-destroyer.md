# Synth-Bass Destroyer (self-gating crush)

## Concept
For VG-800 baritone-as-bass or synth bass — an almost noise-gate / bit-crusher self-gating breakup that delivers electronic / post-punk weight. This is the electronic cluster (sequenced/filtered synth) meeting the degraded aesthetic: the self-gating bit-crusher quality IS the recorded-wrong sound. The Bass Negus's downstream stack (octaver → resonant filter/detune → parallel comp) maps onto this rig's Middle/Texture board.

## How to play it
1. **CRITICAL: run a NOISE GATE BEFORE the Colour Box** to tame the cranked-gain hiss.
2. INST input (or XLR + PAD for the line-level VG-800). **HI** mode.
3. Gain-stage: STEP 2–3 clicks in → PRE-VOL ~3:00 (the "gain") → MASTER ~9:00 (post).
4. Boost MIDDLE around ≈1 kHz so it reads through the gate; support the BASS.
5. Engage HI-PASS ~10:00 to keep the crush tight.

## Notes
- Community-cited **exact** positions — not translated.
- Without the upstream noise gate the cranked gain hisses badly; the gate is mandatory, not optional.
