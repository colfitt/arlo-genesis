# Subtle Stereo Widener (On for the Split)

## Concept
The everyday "always-on for the split" Board-1 state — quiet, three-dimensional, subliminal in a mix. Establishes stereo for the whole downstream chain while the Somersault upstream does any broken character. This is the rig's default stereo-bed state for drone/shoegaze: subtle movement under everything.

## How to play it
1. Set MODE to Standard (switch left, CE-2 voicing).
2. RATE to 9:00 ("a crawl"), DEPTH to just shy of 12:00 (lowest-third zone = "subliminal in a multi-instrument mix").
3. Route OUTPUT A → Deco L (wet) and OUTPUT B → Deco R (dry).
4. Leave this one on — you stomp the Somersault upstream, not this.

## Notes
- ⭐ Signature: the rig's default stereo-bed state.
- Boss's 9:00 RATE is exact; DEPTH-just-shy-of-noon is the clock interpolation of Premier Guitar's "lowest third" range.
