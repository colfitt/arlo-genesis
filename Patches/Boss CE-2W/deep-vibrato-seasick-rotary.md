# Deep Vibrato (Seasick Rotary)

## Concept
True pitch-vibrato with the dry removed — seasick rotary-speaker wobble for drone/psych passages only (too dramatic for rhythm). Pairs with sustained VG-800 pads or Hizumitas walls upstream. CE-1 Vibrato is inherently dry-free (wet-only) with no patching trick needed.

## How to play it
1. Set MODE to CE-1 Vibrato (switch right — dry removed = inherently wet-only).
2. RATE to 12:00 (mid = "metallically tinged rotary speaker"), DEPTH to 1:00–2:00.
3. OUTPUT A alone is already 100% wet in this mode; A+B gives wet on A, dry on B for a wide spread.
4. Use for drone/psych passages only, not rhythm.

## Notes
- ⭐ Signature: art-rock detuned-modulation / post-punk brittle-motion lane.
- RATE/DEPTH interpolate Premier Guitar's "mid-range" description (community mode + range, DOUG-designed clock specifics).
- Pick this OR the Somersault for wobble — don't stack both wet at depth, or it turns to mush.
