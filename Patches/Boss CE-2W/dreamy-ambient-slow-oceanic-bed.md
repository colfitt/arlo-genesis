# Dreamy Ambient (Slow Oceanic Bed)

## Concept
Very slow, very deep CE-1 wash made to sit under delay/reverb — the slow oceanic drone-bed that feeds the Deco → Board 2 (Dark Star / Microcosm). The closest documented match to the shoegaze/drone goal.

## How to play it
1. Set MODE to CE-1 Chorus (middle).
2. RATE to 7:30–8:00 (very slow), DEPTH to 3:00 (very deep).
3. Route OUTPUT A+B wet/dry split into the Deco.
4. For an even slower variant, back RATE fully CCW with DEPTH 2:00+ in this same mode (TDPRI owners' recipe).

## Notes
- ⭐ Signature: drone/shoegaze/indie-folk bed.
- Clock positions translate Guitar Chalk's 0–10 ("Dreamy Ambient" RATE 2 / DEPTH 8 → ~7:45 / 3:00).
- Honest flag: a real 0–10 value of 8 lands deep on the CE-2W's range — that's intended for an under-the-wall wash, not a foreground chorus.
