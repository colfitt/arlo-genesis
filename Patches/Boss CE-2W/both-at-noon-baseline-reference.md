# Both at Noon (Baseline Reference)

## Concept
The canonical CE-2W reference tone — rich, chiming, sparkling classic Boss chorus. Both RATE and DEPTH at noon in Standard (CE-2) mode is your calibration starting point and the go-to for clean rhythm and arpeggios. Hear this first; it's the dial all the other patches are judged against.

## How to play it
1. Set MODE to Standard (switch left, CE-2 voicing).
2. RATE to 12:00, DEPTH to 12:00.
3. For the pure reference, use OUTPUT A only (mono — chorus and dry mixed).
4. For the wet/dry stereo spread, plug both A and B (A = wet/chorus only, B = dry only).
5. From here, back DEPTH/RATE down for the subtle states, push them up for the icy/agitated ones.

## Notes
- Not signature-weighted, but the reference everything else is measured against.
- Bypass is buffered — it always feeds both outs (dry), so the split survives bypass.
