# Forbidden Gated Fuzz

## Concept
The reference "is it broken?" Forbidden tone — Guitar Pedal X (Stefan Karlsson) dialed-in setting. Both gain toggles up (G1 up / G2 up) puts the Kilt in Forbidden mode: a bright, high-output, spitty/sputtery gated fuzz. This is the **only fully-quoted knob set on disk**, making it the most trustworthy concrete found patch. By design it does **NOT** clean up with guitar-volume.

## How to play it
- Set the grid to **Forbidden**: G1 up / G2 up (fuzz requires G1 engaged).
- Crank all three knobs: Volume 3:00, Gain 4:00, Tone 5:00.
- Low-Cut down/flat for the bright, woolly/beefy version.
- **Variant (Worship Tutorials):** same both-up fuzz but Low-Cut **up/cut** to lop the woolly/beefy low end instead of brightening.
- Park it BEFORE the Middle board's granular/smear pedals (Microcosm / Dark Star / Blooper) for a drone-doom degraded wall.
