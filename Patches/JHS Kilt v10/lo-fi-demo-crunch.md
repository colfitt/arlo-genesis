# Lo-fi Demo Crunch

## Concept
Designed against the taste lens (Lo-fi cluster — Car Seat Headrest, Modest Mouse — the "recorded-wrong" degrade chain). Re-purposes the Low-Cut "cut" (a documented critique that it reads "almost lo-fi-sounding") as the intentional lo-fi character knob, for deliberately thinned "cheap-amp / console-overload" demo crunch and brittle clean-ish jangle.

## How to play it
- Set the grid to **Crunch** (G1 down / G2 down) OR bump to **Overdrive** (G1 up) for more hair.
- Keep Gain low/mid (10:00–11:00).
- Push Tone bright (~1:00) for brittle jangle.
- Engage Low-Cut **up/cut** — the deliberately thin/telephone character (Eirik: "almost lo-fi-sounding"; Stu uses cut as a studio effect).
- Push into a clean amp/model so the warts stay audible.
- Stacks into PORTA424 / Generation Loss / Colour Box V2 in the rig's degrade chain.
