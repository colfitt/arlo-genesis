# Thom Yorke "Bodysnatchers" Distortion

## Concept
Thom Yorke (Radiohead) — Bixonic Expandora, 2008–2009 *In Rainbows* tours / *From the Basement* "Bodysnatchers". The Kilt is built on the Expandora circuit, so Thom's documented live Expandora DIP maps directly: "D1 Lo, D2 Hi, Ins B" = Distortion mode + bass-guitar preset. Per the Legendary Tones DIP chart, D2-Hi == Kilt G2-up and the "Ins B/bass" preset == Kilt Low-Cut **flat** (the fuller/beefier low end). For tracking the repo's Radiohead folders (Let Down, Ful Stop, Creep).

## How to play it
- Set the grid to **Distortion**: G1 down / G2 up. Low-Cut down/flat (the "Ins B / bass" preset = beefier low end).
- Push Gain (~12:00) for bloom; roll guitar-volume back to clean up — this mode DOES clean up.
- Keep Volume and Tone at noon as the designer-chosen starting point.

## Notes
- The toggle states are mapped from Thom's DOCUMENTED Expandora DIP via the canonical Legendary Tones chart, so the **mode is artist-cited (not fabricated)**; only the knob clocks are designer-chosen, and that's flagged.
- **HONEST FLAG:** no source confirms Jonny Greenwood used an Expandora — keep the tie on **Thom only**. The studio "Bodysnatchers" actually used an overdriven Motown desk; this captures the live/Basement tone.
- Art-rock cluster (Radiohead #1 all-time).
