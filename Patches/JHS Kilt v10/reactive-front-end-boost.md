# Reactive Front-End Boost

## Concept
The JHS-named "high-gain stage boost" mode (G1 down / G2 up), used as a reactive opto-FET front-end that tracks pick attack. It pushes an amp's front end OR stacks INTO a fuzz (MXR 108 Fuzz / EQD Hizumitas), giving touch-sensitivity a platform drive lacks. This is the purpose-built reason to keep the pedal near the rig — a reactive front-end that makes the 108/Hizumitas thicker and more sustained than the fuzz alone (fuzz sustain into the wall = post-punk saturated-drone-wall).

## How to play it
- Set the grid to G1 down / G2 up (the JHS "high-gain stage boost").
- Keep Gain low/mid (10:00–12:00) so the reactive opto-FET gain tracks pick attack.
- Use the new headroom: push Volume past noon (1:00–2:00) for boost.
- Run it **before** the fuzz / before Brothers AM — it should FEED a fuzz, not duplicate one.

## Notes
- Signature-fit. The reason to keep this pedal near the rig.
- **Do NOT** run its distortion parallel to Longsword/Oxford (redundant) — it should FEED a fuzz.
