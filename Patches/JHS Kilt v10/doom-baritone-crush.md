# Doom Baritone Crush

## Concept
Designed against the taste lens (Post-punk / Swans drone-wall cluster + baritone-weight aesthetic). Distortion mode (G1 down / G2 up) for baritone via GK-5 → VG-800 baritone model — low, heavy, sustained doom fundamentals that bloom rather than chug. The opto-FET reactive gain loves fat low strings; baritone is the source the Kilt sounds best on in this rig ("home territory").

## How to play it
- Set the grid to **Distortion**: G1 down / G2 up.
- Push Gain into saturated/sustaining territory (1:00–2:00) — the reactive gain loves fat low strings.
- Keep Tone dark/warm (11:00–12:00); the germanium-style clipping is already warm.
- Engage Low-Cut **up/cut** to keep the heavy low end DEFINED, not mud.
- Dig in for bloom; roll instrument volume back for dynamic builds (this mode cleans up).
- Feeds the Middle/End boards for sustained crescendo builds — pair the bloom with freeze/feedback downstream.
