# Brittle Jangle Crunch

## Concept
Designed-to-emulate (KLT-LF2) for the lo-fi cluster — Modest Mouse brittle bright edge-of-breakup jangle (blues-driver method). Crunch mode (G1 down / G2 down) using the now-usable first-25% gain (the V2→10 fix) with the single Tone tilt up for brittle/bright jangle. Cleans up beautifully with instrument volume = Brock's "basic pedals = a blues driver" dynamic.

## How to play it
- Set the grid to **Crunch**: G1 down / G2 down.
- Keep Gain low (9:00–10:00) — edge of breakup, exploiting the now-usable first 25%.
- Push Tone bright/brittle (1:00–2:00) — the single Tone tilt up for jangle.
- Engage Low-Cut **up/cut** to thin it toward the brittle Tele-into-Twin character.
- Clean up with instrument volume = the Brock roll-back-for-clean move.
- Banjo-as-lead carries the jangle; keep DRY.

## Notes
- 🟣 designed-to-emulate. No artist setting for this box. Uses the now-usable first-25% gain (the V2→10 fix).
