# Studio Re-amp All-Rounder

## Concept
Designed template (not a found setting) — the pedal's documented benched "real home." A single-box drive-coverage tool for studio re-amping a printed track via EHX Effects Interface + Radial X-Amp into Iridium / VG-800 amp models. From one box you get OD, distortion, or Forbidden fuzz, with the modeler-friendly Low-Cut taming mud into digital amp models.

## How to play it
- Start at noon across the board: Volume 12:00, Gain 12:00, Tone 12:00.
- Low-Cut **up/cut** to tame mud into a digital amp model; flip to flat for fuller re-amps.
- Set **toggles per target sound** via the four-mode grid:
  - both down = Crunch
  - G1 up = Overdrive
  - G2 up = Distortion
  - both up = Forbidden
- Use the **Red Remote** to flip G2 between takes for OD↔fuzz without re-patching.
- Dial each knob per track from the noon starting point.
