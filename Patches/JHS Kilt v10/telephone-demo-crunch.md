# Telephone Demo-Crunch

## Concept
Designed-to-emulate (KLT-LF1) for the lo-fi/prolific cluster — Car Seat Headrest / Modest Mouse thinned telephone/console-overload demo crunch. This is the existing sheet patch #6 (Lo-fi Demo Crunch) — the Kilt's best fit for this cluster — re-anchored explicitly to CSH/Modest Mouse. Re-purposes the documented Low-Cut critique as the intentional lo-fi knob.

## How to play it
- Set the grid to **Crunch** (G1 down / G2 down) or bump to **Overdrive** (G1 up) for more hair.
- Keep Gain low/mid (10:00–11:00).
- Push Tone bright (~1:00) for brittle jangle.
- Engage Low-Cut **up/cut** — Eirik's "almost lo-fi sounding" cut weaponized as the lo-fi character.
- Push into a clean amp/model so the warts stay audible.
- Stacks into PORTA424 / Gen Loss / Colour Box in the degrade chain.

## Notes
- 🟣 designed-to-emulate. No artist setting exists for this analog box; toggle logic + gain regions are grounded, exact clocks are designer-chosen.
