# Chaos Sound-On-Sound Wall

## Concept
A self-sustaining looped wall built from old-signal feedback — Chance pushed toward the documented "chaotic sound-on-sound looper" edge. Above noon, Chance climbs the probability that old output feeds back into itself, toward 100% sound-on-sound at max. For sustained drone-wall crescendos that build and consume. This IS the Swans-style sustained-wall crescendo translated to Parting — use it deliberately as a climax, not a default.

## How to play it
1. Set global defaults: Routing Stereo, Trails ON.
2. Push Chance to 2:00–3:00 (above noon) so old output feeds back into itself toward sound-on-sound.
3. Set Smear to max for lush diffusion into a wall; Dissolve at 1:00 adds light reverse movement; Time at 1:00 for a bigger bloom.
4. Slow Vibrato drift: Depth 9:00, Rate 9:00, Random Sine shape.
5. **Runaway reset:** turn the **Time** knob (silent by design) to instantly clear the buffer when feedback runs away.
