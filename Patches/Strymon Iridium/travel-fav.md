# Travel FAV (grab-and-go fly rig)

## Concept
The one-footswitch do-everything travel tone — the Iridium's stated reason to never leave the studio bag. A complete amp+cab+room+verb fly rig: Polytune 3 (tuner first) → Iridium (FAV) → UAFX Del-Verb → interface or FOH; any magnetic guitar, no real amp, headphone out for silent practice. A conservative noon-everything default so it's reliable cold in any room.

## How to play it
1. AMP round OR punch (pick the trip's core voice), CAB b; DRIVE noon, BASS noon, MIDDLE noon, TREBLE noon.
2. LEVEL to FOH unity; ROOM noon, SIZE medium.
3. INPUT LEVEL = Instrument, OUTPUT MODE = Normal (red), rear = MONO (OUT L) for a single FOH send, or STEREO if the venue takes two.
4. Store as the FAV snapshot — hold FAV until blue, press again.
5. Monitor on the front 1/8" headphone jack (latency-free) while OUT feeds the room.

## Notes
- DESIGNED patch — conservative noon-everything default so it lands cold in any room.
- Pairs with the Del-Verb (also a travel-only bench mate) and the Shiflett Deluxe (#3) for a full amp+verb gig.
- Power: dedicated 500 mA isolated slot — no daisy-chain.
