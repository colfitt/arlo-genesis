# Round Clean DI (SG / Jazzmaster cornerstone)

## Concept
The cleanest, most amp-like tone in the rig: a Fender Deluxe Reverb (Round) into the OwnHammer 1×12 open-back cab, straight into the 1 MΩ IN → Apollo x8 line in. This is the pristine "real amp" clean the VG-800 cannot serve — the non-GK analogue to the always-on CB Clean, and the dry source you stack everything else onto. Round has the most headroom of the three amps, so DRIVE can be pushed to wake it up and still stay clean under loud picking.

## How to play it
1. Set AMP to round, CAB to a; DRIVE 1:00, BASS noon, MIDDLE 10–10:30, TREBLE 1–2:00.
2. Set ROOM modest (knob ~10:00, SIZE small via HOLD-ON + ROOM knob) — let downstream verbs do the long tails.
3. INPUT LEVEL = Instrument, OUTPUT MODE = Normal (red), rear = MONO (OUT L), or STEREO if after a stereo split.
4. Push DRIVE to keep it lively but clean; back MIDDLE off noon for a slight scoop, toward MAX for tweedier grit + secondary power-stage gain.
5. Pull TREBLE toward noon to thicken if it's too bright.

## Notes
- Daily clean DI for any humbucker/single-coil guitar with no GK pickup — store as the everyday FAV.
- Alt voicing (Hayley Briasco / Guitar Center): CAB c (2×10 Vibrolux) for a deeper scoop.
- INPUT LEVEL = Instrument by default; any hot source (line out, hot preamp) needs Line or it clips.
- DRIVE = amp volume (non-master-volume), LEVEL = pedal output only.
