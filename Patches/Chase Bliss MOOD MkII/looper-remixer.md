# Looper Remixer (Slip on a routed micro-loop)

## Concept
Turns a captured banjo/baritone loop into a chromatic re-pitcher without re-playing it. Route a micro-loop through the Wet Channel, set Wet MODE = **Slip**, then punch Slip in/out while moving **MODIFY (semitone steps)** — you get "a rainbow of steppy glitches" and chromatic key changes.

## How to play it
1. Capture a micro-loop and set ROUTING so the loop runs through the Wet Channel (IN+micro-loop or micro-loop only).
2. Set Wet MODE = **Slip**.
3. Punch Slip in and out (LEFT footswitch) while sweeping **MODIFY** in its semitone steps.
4. Listen for steppy glitches and chromatic key changes over the looped material.

## Notes
- Re-pitches a captured loop chromatically without you re-playing the part.
