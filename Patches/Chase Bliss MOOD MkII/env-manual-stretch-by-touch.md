# Env Manual-Stretch-by-Touch (dynamic drone)

## Concept
Attack-controlled stutter/freeze where your picking carves into a held bed. Set Looper MODE = **Env**, **crank MODIFY (sensitivity MAX)**, and set LENGTH for the slice size. Capture a loop, then **gently tap the strings or send any quiet signal** — "the loop slows to a crawl … time-stretching by touch." **Go fully quiet** so Env freezes and repeats the last slice.

## How to play it
1. Set Looper MODE = **Env** and **crank MODIFY** for max sensitivity.
2. Set LENGTH for slice size: low/CCW for microscopic grains, high for longer phrases.
3. Capture a loop.
4. **Gently tap the strings / send a quiet signal** — the loop slows to a crawl, stretching by touch.
5. **Go fully quiet** to freeze and repeat the last slice. Lower CLOCK for aliasing.

## Notes
- The manual's own gesture behind Perry Frank's "Reverb + Env" dynamic drone (reverb-env-dynamic-interactive).
- Low MODIFY = less sensitive.
