# Reverb + Stretch — sprawling epic

## Concept
Reverb over a Stretch loop: a short phrase smeared into a sprawling grainy bed. A **mode pairing only** (Perry Frank's set has no published knob values) — dial the Stretch settings from Stretching 101 / Frozen Grain Wall and layer Reverb over the top.

## How to play it
1. Set Wet MODE = **Reverb**.
2. Set Looper MODE = **Stretch** and use the Stretching 101 / Frozen Grain Wall gesture (stretching-101-frozen-grain-wall).
3. Smear a short phrase into a sprawling grainy bed under the reverb.

## Notes
- The pairing behind **Slot L** (stretching-101-frozen-grain-wall + freeze-pad).
- No knob values published — pairing only.
