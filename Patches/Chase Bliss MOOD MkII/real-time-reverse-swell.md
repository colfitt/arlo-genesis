# Real-Time Reverse Swell (Daydreaming / Backdrifts)

## Concept
Radiohead's signature reverse-bloom — Thom reversing ringing glockenspiel notes so they "fade in" (Daydreaming), and Jonny's reversed guitar (Backdrifts). MOOD's Slip does it live, in real time, while you sustain. **MODIFY CCW of noon = REVERSE** (the Unperson-confirmed reverse); **TIME = sampling size** (low for instant reverse on each note, high for reversed phrases that follow behind you, the glockenspiel "fades in" feel).

## How to play it
1. Set Wet MODE = **Slip**, MODIFY CCW of noon for reverse.
2. Set TIME: low (~10–11:00) for instant reverse on each note, high (~1–2:00) for reversed phrases that trail you.
3. Play a ringing pluck (banjo / VG pad) — it returns reversed, swelling in.
4. Add the **CLASSIC** dip for aliased crunch. CLOCK noon (or down for grit), MIX to taste.
5. **Freeze** (hold LEFT FS) for a held reversed bed under a live lead.

## Notes
- All 🟣 designed-to-emulate — no MOOD artist setting is documented and Radiohead never used one; this chases the *result* (Slip = the reverse method). Clock-face numbers are inferred starting points, dial by ear.
- Radiohead's reverse is really a captured-then-reversed *sample* (per King of Gear); MOOD's Slip is the honest *live* approximation. For a committed reversed loop layer, use Blooper TB4 instead.
