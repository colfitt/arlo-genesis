# Getting Started

## Concept
From-scratch first sound: a few notes become an evolving textural bed (banjo/baritone or any input). This is the manual's literal "good default to do your own thing" — start neutral at noon across all six effect knobs, then drop CLOCK into the grit zone to add character. "You should be a texture now."

## How to play it
1. Start with BOTH channels bypassed and **all dips OFF**. Set all six effect knobs (Wet TIME/MODIFY, looper LENGTH/MODIFY) ~noon; CLOCK ~noon; MIX ~noon; ROUTING = **IN+micro-loop**.
2. Play a few notes, then turn ON the Micro-Looper.
3. Turn ON the Wet Channel to "smear it."
4. In order: roll **CLOCK down** to the lower-third (illustration ~6k) for grit; then move each MODIFY; finally bring up TIME and LENGTH.

## Notes
- This is the orientation patch — **don't store it**. It's neutral-noon as a launch pad, not a destination.
- The whole point is the gesture: noon everything, then drop CLOCK into the grit zone and start sculpting from there.
