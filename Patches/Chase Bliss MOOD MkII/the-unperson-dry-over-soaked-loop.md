# The Unperson — dry input over a reverb-soaked loop

## Concept
Playing dry over a drenched loop — "beautiful ambient setup," ideal with synth / OP-1 / VG-800 sources. Set ROUTING = **micro-loop only** so the Wet hits only the loop while your input stays dry, with Wet MODE = **Reverb** on the loop. Run a loop, then use **DIRECT MICRO-LOOP / FADE** (hidden) to push the loop into the background.

## How to play it
1. Set ROUTING = **micro-loop only** and Wet MODE = **Reverb**.
2. Capture a loop (synth / OP-1 / VG-800 source) — the Reverb soaks the loop only.
3. Play dry on top — your live input bypasses the Wet path.
4. Use **DIRECT MICRO-LOOP / FADE** (hidden) to push the soaked loop into the background under your dry playing.

## Notes
- Directly rig-relevant — VG-800/OP-1 are perfect looper food here.
- Demoed with Vong Replay synth, Mutable Rings, and OP-1 as sources.
