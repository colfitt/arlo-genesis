# Granular Stutter (rig patch c)

## Concept
Chopped, arcade-glitch texture; go quiet to freeze the last slice and stutter it. Env in small slices with high sensitivity, under a multi-tap Reverb, at a lo-fi clock and with CLASSIC for grit — the most "recorded-wrong" of the rig patches.

## How to play it
1. Set Looper MODE = **Env**, LENGTH 9:00 (small slices), MODIFY low (high sensitivity). Set Wet MODE = **Reverb**, MODIFY low (multi-tap clusters). CLOCK 10:00 (aliased/lo-fi).
2. Add the **CLASSIC** dip for extra grit.
3. Play, then **go quiet** so Env freezes the last slice and stutters it.
