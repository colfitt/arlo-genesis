# Delay + Tape — rhythmic / pulsing drone

## Concept
Piling delay over a Tape loop = a bed that **pulses** rather than sits still. A **mode pairing only** (no published knob values) — set the Delay to pile up (MODIFY high) over a Tape micro-loop. Appears 3× in Perry Frank's set with different feels.

## How to play it
1. Set Wet MODE = **Delay** with MODIFY high so repeats pile up (see self-stretch-delay-as-looper).
2. Set Looper MODE = **Tape**.
3. Run the Delay over the Tape loop — the bed pulses rather than sitting static.
