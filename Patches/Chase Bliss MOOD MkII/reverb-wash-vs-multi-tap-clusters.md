# Reverb Wash vs Multi-Tap Clusters

## Concept
The foundational reverb map. **MODIFY is smear**: at MIN you hear distinguishable multi-tap delay clusters; at MAX it washes out into reverb; in the middle you get "semisolid hybrids." A twist of MODIFY can also split the sound into flitty grains. **TIME** sets decay/size (CCW short, CW long). Everything frozen or rhythmic downstream depends on knowing where MODIFY lives.

## How to play it
1. Set Wet MODE = **Reverb**.
2. Set TIME for the decay/size you want (CCW = short, CW = long).
3. Sweep MODIFY: MIN for distinct multi-tap clusters, MAX for a washed reverb, mid for semisolid hybrids.
4. For granular textures, give MODIFY a twist to split the sound into flitty grains.

## Notes
- This is the reference map for every reverb-based patch — learn where MODIFY transitions from taps → wash → grains, because freeze and rhythmic moves build on it.
