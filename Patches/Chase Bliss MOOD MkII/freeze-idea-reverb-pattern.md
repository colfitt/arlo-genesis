# Freeze Idea: Reverb Pattern

## Concept
A percussive, rhythmic frozen-reverb wall from the multi-tap clusters. Set Wet MODE = **Reverb** with **MODIFY down (MIN)** so the taps stay distinct, then play and **HOLD LEFT FS to freeze** — the frozen reverb sounds like a percussive pattern. Once frozen you can turn **MODIFY up to smudge** the pattern, but this is **one-way** (you can't go back).

## How to play it
1. Set Wet MODE = **Reverb**, MODIFY at MIN so taps stay distinct; set TIME for decay/size.
2. Play, then **HOLD the LEFT footswitch** to freeze the multi-tap pattern into a rhythm.
3. Optionally turn **MODIFY up** to smudge the frozen pattern — remember this is one-way.
4. Lower **CLOCK** for grit.

## Notes
- Low Reverb MODIFY = distinguishable clusters that freeze into a rhythm. A named manual recipe.
