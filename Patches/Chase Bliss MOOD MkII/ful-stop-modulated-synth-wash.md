# Ful-Stop / Modulated Synth Wash (frozen Reverb pad)

## Concept
The modulated, moving reverb-wash side of the art-rock lens, and the Ful Stop second-half synth bed — a sustained, slightly-detuned-feeling wash to float arpeggios/leads over. Freeze a washed-out Reverb into an infinite pad, then slowly turn CLOCK down while frozen for the moving/detuning character that fakes the unison-detune *drift* defining the Ful Stop wash.

## How to play it
1. Set Wet MODE = **Reverb**, TIME up to sustain (~1:00), MODIFY CW (~1:00) for a washed-out reverb (smear, not distinct taps).
2. Play a sustained VG-800/OP-1 pad note (or banjo), then **HOLD the LEFT footswitch** to freeze into an infinite ambient pad (with **LATCH** so it holds hands-free).
3. While frozen, **slowly turn CLOCK down (CCW)** for the moving/detuning character (the Clock-Decay shift) — the downsample drift fakes the unison-detune drift.
4. Set MIX so the pad sits as a bed.

## Notes
- The Ful Stop wash is "multiple synths in unison, each voiced differently — the *drift* between voices is the sound." MOOD can't multi-voice, but the frozen-reverb-plus-CLOCK-drift gives the same *moving, detuning* quality on one source.
- Run the actual phasing-arpeggio line into Big Time TB2 on top of this pad.
- Designed-to-emulate; clock-face values are inferred starting points, dial by ear.
