# Classic Mode — degraded "recorded-wrong" base layer

## Concept
A global tone shift that restores all the MkI flaws — the most directly on-aesthetic toggle for drone/doom/shoegaze degradation. Flick the **CLASSIC** dip ON (controls are identical; sound/response differ): CLOCK is noise-full (turning down adds noise); micro-loops gradually deteriorate/distort over time; Reverb trails get resonant/ringy (mild feedback); Slip gets aliased/uneven crunch; Delay TIME adds rubbery pitch-bends; Tape LENGTH **chops** the loop instead of shortening it.

## How to play it
1. Flick the **CLASSIC** dip ON.
2. Run any recipe — the degraded character layers under it automatically.
3. For MkI "hold-to-record, release-to-play": CLASSIC on + **NO DUB** on + overdub engaged + **LATCH** off.
4. Flash **firmware 1.1** for the original cloudy/wispy Stretch inside Classic.

## Notes
- Not a single patch — a global "degraded" switch to layer under any recipe.
- **Dip settings save with presets**, so bake CLASSIC into both stored slots.
