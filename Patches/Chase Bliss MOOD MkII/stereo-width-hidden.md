# STEREO WIDTH — set the wet pan width

## Concept
Dial exactly how wide the Wet Channel's stereo image spreads when **SPREAD** is engaged — from squished/near-mono to fully glorious wide. This is a hidden option set with the **Wet TIME** knob: at minimum the wet image is "squished completely," at maximum it backs out "sounding glorious." Think of it as the precision companion to the SPREAD dip — SPREAD turns the spread on, STEREO WIDTH says how much. Default is mid.

## How to play it
1. Engage the **SPREAD** dip (and **MISO** first if your source is mono).
2. **Hold both footswitches** until the LEDs go green (hidden-options mode).
3. Turn the **Wet TIME** knob: CCW narrows the wet image toward mono, CW opens it up wide.
4. Exit the hidden menu; **Wet TIME** returns to its normal time/decay duty.

## Notes
- Has **no audible effect unless SPREAD is on**.
- Use a narrower width when a mono stage sits downstream so the collapse is less jarring; go wide only when a stereo stage follows.
- Default is mid.
