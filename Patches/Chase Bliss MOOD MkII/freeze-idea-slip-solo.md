# Freeze Idea: Slip Solo

## Concept
Bizarre chromatic melodies and arcade-style SFX from frozen Slip — a chromatic lead generator. Set Wet MODE = **Slip**, then **freeze (hold LEFT FS)** → "glitchy synth." Manipulate **MODIFY (semitone-stepped)** to generate chromatic melodies and arcade FX, and play under it. TIME = sample size (low = pitch-shifter, high = trailing harmonies).

## How to play it
1. Set Wet MODE = **Slip**.
2. Play, then **HOLD the LEFT footswitch** to freeze Slip into a glitchy synth.
3. Manipulate **MODIFY** through its semitone steps to generate chromatic melodies / arcade FX.
4. Play under the frozen Slip; set TIME for the sample size (low = pitch-shifter, high = trailing harmonies).

## Notes
- Glitchy more than droney, but the frozen Slip is a chromatic banjo-as-lead source.
- Add **CLASSIC** for aliased crunch. Dossier patch (b) (slip-reverb-smear) builds on this.
