# Stretching 101 + Frozen Grain Wall

## Concept
The single best sustained-wall move: classic grainy time-stretch, then one frozen grain held into an infinite drone. Set Looper MODE = **Stretch**, both MODIFY and LENGTH to MAX, then slowly rotate MODIFY toward NOON for "good ol' stretching." **FROZEN: MODIFY exactly at NOON** stops the progression and infinitely repeats the current slice; LENGTH then sizes the held grain.

## How to play it
1. Set Looper MODE = **Stretch**. Set **both MODIFY and LENGTH to MAX**.
2. **Slowly rotate MODIFY toward NOON** — once it's moving slowly, **gradually rotate LENGTH down to minimum** → "good ol' stretching."
3. To FREEZE: set **MODIFY exactly at NOON** — this stops progression and infinitely repeats the current slice.
4. Use **LENGTH** to size the held grain: high = clearer/repeating phrase, low = blurry/grainy.

## Notes
- ⚠️ **MODIFY at MAX outputs no effect** regardless of LENGTH.
- Lower CLOCK = grainier; firmware 1.1 restores MkI Stretch character inside CLASSIC.
- **Slot L = "Frozen Grain Wall"** (with Reverb freeze layered, CLASSIC on). Perry Frank's Reverb+Stretch "sprawling epic" lives here (reverb-stretch-sprawling-epic).
