# Reverb + Tape — endless decay

## Concept
A frozen or decaying reverb bed with an octave/speed-shifted Tape micro-loop underneath — Perry Frank's headline "endless decay," the sustained-wall workhorse and the default drone preset to chase. This is a **mode pairing only**: Perry Frank's set has no captions and no published knob values, so dial the actual settings from the per-mode recipes.

## How to play it
1. Set Wet MODE = **Reverb** (frozen or long — see Freeze Pad / freeze-pad).
2. Set Looper MODE = **Tape** with MODIFY octave/speed-shifted (see gritty-tape-loop / balance-beam-tape-loop).
3. Run the Tape loop under the long/frozen reverb for the endless-decay bed.

## Notes
- The default drone preset to chase. Layer **CLASSIC** for grit.
- No knob values published — pairing only; dial from the cited per-mode recipes.
