# Infinite shimmer drone (rig recipe a)

## Concept
Endless, rising, glassy drone: strum a baritone chord, freeze it, and let it ring under banjo leads. Glassy HPF shimmer (Filter ~1:00), both shifters +1 octave, near-infinite Decay, clean Crush. DOUG-designed numeric starting point for this banjo/baritone rig, Stereo routing and trails on.

## How to play it
1. Set Mix 2:00-3:00, Decay 4:00-5:00 (near infinite), Multiply 1:00-2:00.
2. Set Filter 1:00 (HPF, glassy), both Pitch knobs +1 oct (or Pitch 2 +12 right with Pitch 1 detuned left), Crush noon, Lag 11:00.
3. Assign Aux = Hold.
4. Strum a baritone chord, freeze, and let it ring under banjo leads.

## Notes
- DOUG-inferred numeric values for THIS rig — not quoted from any external source.
- Glassy HPF shimmer is more shoegaze than doom — fits the lens, but octave-up can ice-pick over banjo; switch to the LPF side if it does.
- Suggested onboard slot 2.
