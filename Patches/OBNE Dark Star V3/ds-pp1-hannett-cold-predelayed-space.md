# DS-PP1 — Hannett Cold Predelayed Artificial Space

## Concept
Joy Division / Hannett's cold, separated, predelayed AMS space: the dry source stays separate, a clinical glassy room arrives after a gap, with cavernous cross-panned width. This one resists the degrade tie-breaker — it is COLD and CLEAN. DOUG-designed for this rig, Stereo routing and trails on.

## How to play it
1. Set Mix noon (source stays dry/separate), Decay 11:00-12:00 (a room that then stops).
2. Set Lag 1:00 (LONG predelay — the Hannett separation), Filter 1:00 (HPF, cold/glassy).
3. Set Crush noon (clean digital), both Pitch knobs noon, Multiply 10:00 (tight tail).
4. Set Spread full CCW to invert the wet image for the cavernous cross-pan.

## Notes
- APPROXIMATION: the AMS RMX16 character (predelay + Nonlin gate + cold digital) — the rig has NO true non-linear gated-reverb mode.
- Keep Decay short or add a BF-3 gate downstream for a literal gated reverb.
