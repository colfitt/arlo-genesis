# Classic Verb (clean shoegaze baseline)

## Concept
A smooth, clean-ish shoegaze pad meant as a baseline to *dirty up from*. The source page ties it to Slowdive and Cocteau Twins. It is not this rig's default lean — the rig renders degraded — but it is a useful clean recall to A/B against.

## How to play it
1. Set Crush and Filter to noon (clean and flat), both Pitch knobs to noon (unity).
2. Set Decay and Mix to mid (~noon); leave Lag, Multiply, Spread neutral.
3. Play sustained chords for the smooth pad wash.
4. To put it on-aesthetic: pull Crush below noon for the "recorded-wrong" character and roll the Filter toward LPF (below noon) to seat it.

## Notes
- Stored on any of the 3 onboard slots as a "go clean" recall.
- This is a launchpad voice — the rig's lean is degraded, so this is mostly here to compare against the crushed/octave patches.
