# Octave-down doom pad (rig recipe b)

## Concept
Subterranean, heavy doom-pad weight on the baritone Jazzmaster — the numeric version of the community octave-down doom drone. Both shifters octave-down, dark LPF Filter, light bit-crush. DOUG-designed numeric starting point for this rig, Stereo routing and trails on.

## How to play it
1. Set Mix 1:00, Decay 3:00, Multiply 11:00, Filter 10:00 (LPF, dark).
2. Set Pitch 1 -1 oct, Pitch 2 -1 oct (or -2 oct on one side), Crush 11:00, Lag noon.
3. Pair with the Hizumitas wall (Board 1) + an octave-down VG-800 baritone patch.
