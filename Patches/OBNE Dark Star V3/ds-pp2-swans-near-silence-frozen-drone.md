# DS-PP2 — Swans Near-Silence Frozen Drone Bed

## Concept
The 'Bring the Sun' begin-from-a-drone architecture: freeze a baritone/banjo chord into a barely-audible static bed, then swell the live wall (HZ-PP1) up over it from near-silence. The "layer many sustained voices" move in one. DOUG-designed for this rig, Stereo routing and trails on.

## How to play it
1. Set Mix 1:00, Decay noon, Multiply noon, Filter 11:00-12:00 (slightly dark, warm bed).
2. Set Pitch 1 noon (or -1 oct for weight), Pitch 2 noon, Crush 11:00 (barely-there bit-crush), Lag 1:00.
3. Assign Aux = Hold.
4. Freeze a baritone/banjo chord into the static bed, then swell the live wall up over it from near-silence.

## Notes
- GOTCHA: Hold passes new notes BONE-DRY — solo over the frozen pad via the dry path + the live wall, or expression-swell in.
- APPROXIMATION: Swans layer real instruments over minutes; the freeze fakes the static "many sustained voices" bed.
- Store as a preset with Aux assigned to Hold.
