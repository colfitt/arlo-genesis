# DS-PP3 — BC,NR Octave-Down Droning Intro Pad

## Concept
The minute-long droning guitar intro before the BC,NR slab: subterranean, dark, slightly degraded. The existing octave-down doom pad cast as the pre-slab drone. Pair with a sustained Hizumitas/Longsword source upstream. DOUG-designed for this rig, Stereo routing and trails on.

## How to play it
1. Set Mix 1:00, Decay 3:00 (long), Multiply 11:00, Filter 10:00 (LPF, dark).
2. Set Pitch 1 -1 oct, Pitch 2 -1 oct, Crush 11:00 (light bit-crush), Lag noon.
3. Pair with a sustained Hizumitas/Longsword source upstream.
4. Let it occupy the pre-climax space before the slab.

## Notes
- The existing octave-down doom pad (rig recipe b) cast as the pre-slab drone.
- APPROXIMATION: BC,NR's intro is a sustained guitar drone into Orange amps; this builds the octave-down pad to occupy that pre-climax space.
