# AndyDemos core voice (huge space at low Decay)

## Concept
A teaching baseline from AndyDemos showing how *little* Decay you need for a big space — at roughly 10:00 the reverb is already a lively, cavernous hall. Clean and neutral: a launchpad voice before any effects, not a destination.

## How to play it
1. Set Filter, Pitch 1, Pitch 2, and Crush to noon.
2. Set Decay to roughly 10:00 — already a big, cavernous space.
3. Increase Lag for more reflections.
4. Brighten the Filter to change the "wall material."

## Notes
- One of the few sources giving an approximate clock position (Decay ~10:00) — treat it as approximate.
- Clean/neutral; use as a teaching baseline in live mode.
