# OBNE SOUND 3 — detuned shimmer regeneration

## Concept
OBNE's own demonstrated V3 build: a slight detune driven into self-feedback for a shimmer that regenerates but stays tamed. Pitch pulled only slightly off (into the Detune zone), Multiply engaged so the pitches start a shimmer (octave-up = shimmer; the detuned voice gives a chorusy "drop"), and the Filter on the LPF side to chill it out and tame the regenerating pitches.

## How to play it
1. Set Pitch to a slight detune — pull octave-down only slightly, into the Detune zone.
2. Engage Multiply so the pitches regenerate into a shimmer with a chorusy drop.
3. Roll Filter to LPF (below noon) to chill it out and tame the regenerating pitches.
4. Use Spread to move the pitches in the field.
