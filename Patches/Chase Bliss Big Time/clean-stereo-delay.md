# Clean Stereo Delay

## Concept
The "delay you can always reach for" — steady, transparent repeats locked to clock, with the limiter fully off (Digital STATE) for maximum cleanliness. This is the bearings-reset home base before sculpting a wild patch. It's essentially what holding the MODE button (panic reset) gives you live — Factory #0 NICE DELAY territory, but cleaner.

## How to play it
1. Tap TIME or slave to MIDI clock for steady repeats.
2. Keep COLOR low, CLUSTER at 0, TEXTURE/DIFFUSE at 0 for a transparent, stable delay.
3. To recall it live from anywhere, **hold the MODE button** — the panic reset lands you here (≈ Factory #0).

## Notes
- Off the core degraded aesthetic (not signature-tagged) but essential as the always-available base and the reset before a wild patch.
- Digital STATE = no limiter, so it stays clean and stable.
