# Jet-Flange Swell (Tap-Synced)

## Concept
A bigger, more pronounced sweeping rush on the doom wall — the dramatic crescendo jet. Deep DEPTH and high-ish RES make the sweep rise through a sustained wall, and the RATE can be set by hand (~9–10:00) or tap-synced so the swell lands exactly on the song's beat. For builds: let the wall sit, then let this rise through it.

## How to play it
1. Set MODE to Standard (or Ultra for more edge), MANUAL 12:00, RES 1:00, DEPTH 2:00, RATE ~9–10:00.
2. To tempo-sync: hold the pedal switch ≥2 s until the indicator blinks, then tap the beat. Touch the RATE knob to return to manual rate.
3. Build the wall, then engage so the jet swell rises through it as a crescendo.

## Notes
- Rig-tuned (designed) from the dossier's own starting points — not a found recipe.
- Higher RES = output spike; trim into the Somersault.
