# Opines — paper_keys (spring-verb keys)

## Concept

Spring-reverbed keys with value-LFO movement — the one Opines patch that uses reverb over delay. Chiptune-flavored keys, intimate when run into Porta tape. A REAL Cuckoo "Opines" patch, op1.fun-verified.

## How to play it

1. Load paper_keys (Dr Wave, Octave 0, spring reverb, value LFO).
2. Play key parts — the spring reverb gives it the chiptune-room feel.
3. Run into **Porta** tape for an intimate, degraded version.
