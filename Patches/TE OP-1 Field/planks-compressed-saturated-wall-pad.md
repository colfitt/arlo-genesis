# planks — compressed saturated wall-pad

## Concept

A lush-but-dirty pad bed for shoegaze/doom — the wall layer. Cluster + Nitro gives a saturated, compressed pad (the author simply calls it "Compressed pad"). 11.1k downloads. This is the default lush wall bed of the rig.

## How to play it

1. Load planks (Cluster, Octave 0, Nitro on).
2. Hold a chord with the **Hold** sequencer for a sustained saturated wall.
3. Stack the ±20-cent tape-double chorus (two-pass pan-to-tape) for width.

## Notes

- Cluster + Nitro is the saturated/compressed combo; value LFO present but off.
- Pairs with cluster_pad (clean) — clean foundation + saturated wall = depth.
