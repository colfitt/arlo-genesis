# Andreas Roman — one-note polyphonic sampler

## Concept

Turn ONE sustained note from any source — a VG-800 pad, banjo, or baritone — into a playable polyphonic instrument. Andreas Roman called the Field's one-shot→poly sampler "the best I've encountered." Set the loop and crossfade for seamless infinite sustain, and the single captured note becomes a full polyphonic pad voice. This is the rig's "broken synth choir" engine.

## How to play it

1. Select the **Synth Sampler** engine.
2. Sample a single sustained note (play C **while holding C** so the root maps correctly).
3. Play polyphonically across the keys.
4. Set **loop start/end** and use **`[Shift]+GRAY` crossfade** for seamless infinite sustain.
5. Remember: no time-stretch — pitching up shortens the note.

## Notes

- REAL cited artist method (Roman sampled a Prophet 12 C3 — rig swap = banjo / baritone / VG-800).
- Save as `banjo-poly` or `vg800-pad` in the `sampler/` folder.
