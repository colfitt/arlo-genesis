# cluster_pad — clean lush foundation

## Concept

A clean lush Cluster pad starting point, made to be degraded later. FX (filter) and LFO (element) are present but off, so out of the box it's a clean foundation — an indie-folk intimate-room bed when kept clean. 11k downloads.

## How to play it

1. Load cluster_pad (Cluster, Octave 0) — clean by default.
2. For the degraded version, add **Nitro** FX or print it through the ±20-cent tape-double chorus.
3. Keep it clean for an intimate indie-folk room bed; play sustained chords via Hold.

## Notes

- FX/LFO present but off — this is intentionally the clean base; planks is the saturated counterpart.
- The degrade path: Nitro on, or the ±20-cent pan-to-tape double for width + character.
