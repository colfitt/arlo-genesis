# Bare Synth Pluck (Thermae Fuel)

## Concept
A deliberately bare, dry, percussive synth pluck built to be ideal fuel for the Big Time's Thermae cascade. FX OFF (no onboard delay/reverb — the Big Time IS the delay and any baked-in echo smears the pitch-stepped ladder). A clean mono note with a short-ish decay so each pluck is a discrete, transient-rich event that fires the STEP/MOTION cascade cleanly. The OP-1 plays almost nothing on purpose; downstream the delay does the arranging.

## How to play it
1. Pick a pluck engine: Dr Wave/Digital for a clean pitched pluck, FM for a glassier bell-pluck (select via [Shift]+T1).
2. Set Octave 0 (or +1 to sit brighter), and a short envelope so each note is a clean, discrete pluck.
3. Turn FX OFF — no onboard delay/reverb; the Big Time is the delay.
4. Send the dry mono voice out the 1/4" line into the EHX Interface → Clean → Big Time IN-L.
5. Play sparse single notes; let the downstream Big Time cascade do the melody.

## Notes
- FX OFF is load-bearing: any onboard OP-1 delay/reverb smears the Big Time's pitch-stepped cascade.
- Consistent pluck velocity matters — the STEP/MOTION cascade fires most musically off uniform transients (Clean downstream evens this out).
- HONESTY FLAG: engine/octave are op1.fun-granularity; the envelope values are a dialable recipe, not a published TE patch — there are no per-knob values for this pluck. Dial by ear.
