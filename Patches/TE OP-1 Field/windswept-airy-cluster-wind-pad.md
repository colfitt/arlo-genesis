# windswept — airy Cluster wind-pad

## Concept

An airy Cluster pad / wind texture — a shoegaze wash layer. The author describes it as replicating "the sounds of a windy day." Ships with the CWO chorus FX already loaded in the slot but switched off.

## How to play it

1. Load the windswept patch (Cluster, Octave 0).
2. Turn the **CWO** FX **ON** for a wide shoegaze chorus wash — it's already loaded in the slot, just engage it.
3. Hold a chord with the **Hold** sequencer for a sustained wind-pad wash.

## Notes

- CWO and the value LFO are present but off by default; engaging CWO is the main move.
- For concrete CWO values, see the "CWO chorus" recipe patch (freq 0 / delay 28 / feedback 15).
