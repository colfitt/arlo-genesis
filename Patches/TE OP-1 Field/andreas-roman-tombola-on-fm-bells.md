# Andreas Roman — Tombola on FM bells

## Concept

Generative "raindrop" shimmer over a drone/texture bed — the Field's signature evolving-bell move, validated on a full Andreas Roman OP-1 Field album. Drop note-balls into the Tombola's spinning hexagon, set the rotation speed and open the walls so notes escape rhythmically, and you get a self-evolving bell shower. Art-rock / studio-as-instrument fit — Ondes-ish bell leads.

## How to play it

1. Load an **FM** bell-tuned patch into FM slot [1].
2. Switch the sequencer to **Tombola** and drop note-balls in.
3. Set the hexagon **rotation speed** (Blue) and **open the hexagon walls** (Grey) so notes escape.
4. **Release** note-balls via GRAY.
5. For manual control, `[Shift]+Blue` = hand-crank the hexagon.

## Notes

- REAL cited artist technique; engine + sequencer are concrete, but per-knob FM values are method-level (TE never published them).
- Layer this over a drone bed (drone1–6) for the classic raindrops-over-a-wall ambient texture.
