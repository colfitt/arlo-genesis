# Vinyl / SP-303 master crush (Drive + Punch)

## Concept

Lo-fi SP-303-style vinyl crunch on the master bus — the "recorded-wrong" finish on the whole mix. Concrete, community-verified values: Master Drive maxed, then the Punch FX (master) with power 99, punch 0, rounds 5.

## How to play it

1. Open the **mixer page** master FX slot.
2. Set **Master Drive = maxed**.
3. Set the master **Punch** FX: power **99**, punch **0**, rounds **5**.
4. Run the full mix through it for SP-303 vinyl crunch.

## Notes

- CONCRETE community-verified values on disk.
- Pairs with the generation-loss resample loop for stacked degradation; watch double-compression on sample-in.
