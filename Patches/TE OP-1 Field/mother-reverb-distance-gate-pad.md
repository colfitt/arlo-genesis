# Mother reverb — Distance + 'through a door' gate

## Concept

The Field-exclusive lush/gated stereo reverb the OG OP-1 never had. **Distance** places the source from small-studio proximity (0, right against the source) out to a vast hall (99); **Gate** only passes reverb above a threshold, simulating sound passing through a door before it reaches you. It's one master FX slot that prints room/doom space before the signal ever hits the boards. For a huge ambient pad: Distance ~70-99, Gate on for a muffled-door swell, long decay. TE pairs Mother with the Dimension engine in its own materials — the intended lush on-board combo.

## How to play it

1. Open the **master FX slot** (mixer page `[3]`) and select **Mother**.
2. Raise **DISTANCE** toward **99** for a longer tail / larger space.
3. Engage **GATE** so only loud hits bloom — the gated-reverb sound (good on drums/pads).
4. Lengthen **decay** for a sustained wash; set **MIX** / **COLOR** to taste.
5. Pair with the **Dimension** engine for TE's intended lush combo.

## Notes

- **Field-exclusive** — the Distance 0-99 scale and the Gate "through a door" behavior are documented (TE via MusicRadar, magazinmehatronika review, ADG walkthrough).
- **Values are a dialable recipe, not sourced settings** — the Distance scale and Gate behavior are published, but exact knob positions for decay/mix/color/timbre are not; treat the control block as a starting recipe and dial to taste.
- Gate + a percussive source = the gated-reverb sound Shane Becker reaches for.
- **Master Mother is stereo; the per-track reverb is mono** — use the Master slot for the widest result.
