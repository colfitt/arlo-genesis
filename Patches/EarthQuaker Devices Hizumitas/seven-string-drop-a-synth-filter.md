# 7-String Drop-A — Synth-Filter Decay

## Concept
Premier Guitar's extended-range example: a 7-string dropped to A through the Hizumitas gives a beefy stoner-rock bottom end, and because the pedal sustains so well you can use the Tone knob as an almost synth-like filter on the held note — letting drop-A/D riffs slowly burn and change shape as they decay. Set Sustain high for the long bloom, start Tone fat/bass-side, and sweep it through the decay so the voicing morphs fat → mid-forward as the note burns, like a filter sweep.

## How to play it
1. 7-string in drop A (or any heavy drop tuning); set Sustain high for the long bloom.
2. Hold a low riff/chord and physically sweep the Tone knob through the note's decay.
3. The voicing morphs fat → mid-forward as it burns, like a filter sweep.
4. Stand close to a loud amp for the feedback-driven sustain.
