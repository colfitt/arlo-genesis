# Watery Tape Chorus

## Concept
Lush warbled stereo tape-chorus for dreamy/lo-fi shoegaze beds — the woozy doubled width feeding the texture board. Lag past noon puts it in the chorus zone; heavy wobble and a Lag-favoring blend make it seasick-pretty. Values are on a 0–10 scale from Guitar Chalk.

## How to play it
1. Engage Tape Saturation (subtle) and Doubletracker.
2. Saturation 3/10, Volume 5/10, Tone 4/10 (dark-ish).
3. Doubletracker: Type sum, Lag Time 6/10 (chorus zone), Blend 7/10 (favor Lag Deck), Wobble 8/10 (heavy warble).
4. Hold Tape Saturation ON, turn BLEND to set Doubletracker Boost/Cut up (DT Vol ~7/10).

## Notes
- QC FLAG — Guitar Chalk reads SEO/AI-generated; values internally consistent but NOT from Strymon or a named player. A/B by ear.
- For a subtler "doubling than seasick" chorus, drop Wobble to ~4/10 and Blend toward 4–5/10 (Celi's "favor Reference Deck").
