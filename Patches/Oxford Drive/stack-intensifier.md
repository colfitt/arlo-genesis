# Stack Intensifier (momentary into BF-3)

## Concept
The honest job in the CURRENT 5th slot — uses the soft-touch HOLD=momentary feature as a build/dynamics tool (post-punk/Swans crescendo lens). Low gain + low Bass = focus + level into the BF-3, making the flanger more vocal/metallic over a focused brick. The defensible way to keep the Oxford in its wired slot.

## How to play it
1. Set it low and clean: Gain 9:00, Bass 8:00 — don't add saturation, add focus + level.
2. HOLD the footswitch for the burst to commit the whole fuzz front-end into a denser brick for a chorus or peak.
3. Release to drop back to the nuanced stack.

## Notes
- Volume 1–2:00 is a level/mid SHOVE into the downstream BF-3, not more dirt.
- Low Bass (8:00) so the BF-3 isn't sweeping mud; mid-focused Contour (10:00) makes the flange more vocal/metallic.
- Soft-touch footswitch: tap = latch, HOLD = momentary — this patch uses the momentary mode.
