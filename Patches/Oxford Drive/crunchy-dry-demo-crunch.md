# Crunchy-Dry Demo Crunch (DS-2 character)

## Concept
Designed-to-emulate the Car Seat Headrest crunchy-dry, mid-forward DS-2-style demo crunch. The Oxford is a ShredMaster, not a DS-2 — this chases the DS-2's mid-spiky crunchy-dry RESULT (mid-crunch, not a wall), never a quoted artist setting. No CSH Oxford knob setting exists.

## How to play it
1. Set Gain to mid-crunch (11–12:00) — the DS-2 is mid-gain, NOT a wall.
2. Contour mid-forward (9:00) for the DS-2's honk; Treble bright/dry at 1:00.
3. Relocate ahead of the fuzzes (or run guitar/banjo-only into the End-board print) so its EQ isn't collapsed.
4. Keep it DRY (no verb) for the "crunchy and dry" target; into PORTA424/424 for the recorded-quiet-then-boosted finish.

## Notes
- Designed-to-emulate, not an artist setting. No CSH Oxford setting exists for this box.
- Bass kept LOW (8:00) per the log-taper / stacked rule.
