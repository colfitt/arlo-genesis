# 16 Levels = Filter — Sweepable Texture

## Concept
Spreading one sample across all 16 pads at different filter-cutoff frequencies, played by hand — a sweepable degraded texture. loopop's Type list is velocities / filter cutoff / chromatic pitch; this is the filter-cutoff mode.

## How to play it
1. Pad Play → 16 Levels → Type = Filter → one sample spreads across all 16 pads at different filter-cutoff frequencies.
2. Play the pads by hand for a sweepable degraded texture.

- On-aesthetic for sweepable degraded textures.
