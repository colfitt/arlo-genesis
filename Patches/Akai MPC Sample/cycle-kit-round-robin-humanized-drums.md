# Cycle-Kit / Round-Robin — Humanized Drums

## Concept
Sampled drums that feel alive — no machine-gun effect — the way the official Sample-Edition packs do it. The AC50 has no native round-robin engine, so this is the manual fake: record multiple takes of the same hit and rotate them. Up to 4 round-robins per "voice" matches The Downbeat Kit's config.

## How to play it
1. Record 2–4 takes of the same hit (vary velocity/strike).
2. Spread them across adjacent pads in one Mute Group (Play page: Mute Group = same number 1–16 for all).
3. Trigger them in rotation while finger-drumming, OR assign one source to 16 Levels spread across all 16 pads.
4. Save the kit WITH a matching demo groove sequence in the same Project (the Sample-Edition packaging template).

## Notes
- The AC50 has no native round-robin engine — this is the manual fake.
- Counts are vendor-stated (Dirty Drummer = 60 cycle-kit projects; Downbeat = 17 kits, up to 4 RR/pad).
