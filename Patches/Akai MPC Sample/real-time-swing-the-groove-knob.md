# Real-Time Swing — The Groove Knob

## Concept
Finding the human pocket live and non-destructively. In Sequencer mode, Real-Time Swing = K3 (bottom-right of the screen) — ride it live. Roger Linn's engine math: 50% = straight, 66% = perfect triplet, 54% = subtle, 62% = obvious shuffle. Feel depends on tempo — dial while the loop plays and stop where it pockets.

## How to play it
1. In Sequencer mode, ride Real-Time Swing = K3 live.
2. Practical starting points: 54–58% subtle hip-hop pocket, 58% = the classic recognizable MPC feel, push to 62% for shuffle.
3. Feel depends on tempo — dial while the loop plays, stop where it pockets.
4. Time Correct → Swing (K3) is the destructive/global version (also governs Note Repeat feel).

## Notes
- Percentages are Roger Linn's own engine math + Audeobox's "58% classic".
- The K3 location is tubedigga-confirmed on the AC50.
