# Finger-Drum Pad Layout — Bank A Standard

## Concept
A fast, comfortable, muscle-memory finger-drum layout — the same map on every kit. Community consensus (mpc-samples/xpresspads) centers the hands on row 2 rather than the factory default, which puts kick/snare/CH/OH on the bottom row A01–A04. The #1 rule is consistency: pick one layout and keep it identical across every kit.

## How to play it
1. Map Bank A: Pad1 kick(alt), Pad2 perc, Pad3 hat, Pad4 kick(main); Pad5 tom/perc, Pad6 perc, Pad7 closed hat, Pad8 snare; Pad9 clap, Pad10 perc, Pad11 open hat, Pad12 snare(ghost); Pad13 FX/fill, Pad14 perc, Pad15 crash/ride, Pad16 melodic/bed.
2. Right hand plays the kick/snare/hat column; left hand plays ghosts/open hat/claps/rolls.
3. Set Mute Groups (closed+open hat choke) plus per-pad Vel Sens.
4. #1 rule: pick one layout and keep it identical across every kit.

## Notes
- Community consensus layout (mpc-samples/xpresspads).
- Factory default puts kick/snare/CH/OH on the bottom row A01–A04; community prefers centering hands on row 2.
