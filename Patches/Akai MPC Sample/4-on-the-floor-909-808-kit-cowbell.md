# 4-on-the-Floor 909/808 Kit + Cowbell

## Concept
The LCD / filter-house drum-machine backbone — a tuned four-on-the-floor kit with the cowbell staple. Designed-to-emulate (MPC-E1) the LCD / filter-house / DFA drum-machine backbone (TR-808/909, cowbell, shuffle), built on AC50 controls, not a documented artist patch. Tune/Swing/Transient ranges are AC50-confirmed (UG v1.3); the kit choices are designed.

## How to play it
1. Build the kit on one pad bank: kick (center), snare/clap, closed + open hat in one Mute Group (closed chokes open), cowbell, ride/perc.
2. Tune drums to song key (Tune page Semi −24..+24 / Fine ±90 ¢; kick=root, snare=minor 3rd) so they lock to the baritone sub-bed.
3. Program a four-on-the-floor kick + offbeat hats; Swing 54–58% (dirty-909 shuffle).
4. Transient Knob FX on the kick (Attack +30–60%, Sustain −20%) for snap.
5. Pan kick/snare center, hats/perc to the sides.
