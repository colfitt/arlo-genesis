# Vintage Emulator — MPC60 Dark Grit

## Concept
Dark, low-end-bumped, rolled-high character on chopped drums/banjo/fuzz. A/B'd against real hardware, the MPC60 flavor shaves the top and bumps the bottom — hi-hats cooler/less shiny, snare laid-back. Subtle alone — stack it with LoFi/Color or the End-board tape stage for the full broken sound. Top pick for the rig's dark degraded sound.

## How to play it
1. Knob FX → Vintage Emulator → K1 Type = MPC60.
2. Character: slight low-end bump + rolled-off highs ("shaves the top, bumps the bottom"); hi-hats cooler/less shiny, snare laid-back.
3. Apply per-pad (target the drums) or whole beat; bake via Resample.
4. Subtle alone — stack with LoFi/Color or the End-board tape stage for the full broken sound.

- Character verified by A/B against real hardware.
- HONEST: it's an EQ/flavor curve, not a converter clone.
