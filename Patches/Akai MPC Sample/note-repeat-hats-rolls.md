# Note Repeat Hats & Rolls

## Concept
Natural-sounding hi-hat and snare rolls with less skill. Hold Note Repeat, set the division, and vary finger pressure for a roll that breathes. Its feel follows the global Swing setting, and it pairs with the poly-aftertouch pads.

## How to play it
1. Hold Note Repeat → set division with B1–B3 (`Shift` = triplet divisions).
2. Vary finger pressure light → medium for a natural roll.
3. Its feel follows the global Swing setting.
4. Pairs with the poly-aftertouch pads.
