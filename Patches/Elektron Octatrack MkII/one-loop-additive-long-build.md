# One-Loop Additive Long Build (All My Friends)

## Concept
The LCD Soundsystem additive arrangement: lock ONE loop and never change it (the All My Friends rule), then build by ADDING tracks across minutes rather than sweeping a filter. The build IS the arrangement. Stage the explosion as a cued Part switch dropping in a distorted VG-800 saw arp and full kick on the downbeat (the Dance Yrself Clean withhold-then-drop). Keep the deliberate loop slur in.

## How to play it
1. Lock ONE loop (piano-ish / banjo / VG figure) on a Flex track, LOOP ON, and never change it (the All My Friends rule).
2. Build by ADDING tracks across minutes — percussion, bass, pad — each entering one at a time, muted/unmuted live with [FUNC]+[TRACK].
3. Stage the explosion as a cued Part switch (or Pattern change at the next boundary) dropping in a distorted VG-800 saw arp + full kick on the downbeat.
4. Build = arrangement, not a filter sweep. Keep the deliberate loop slur in.
