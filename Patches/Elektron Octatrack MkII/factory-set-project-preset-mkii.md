# Factory Set/Project — PRESET MKII

## Concept
The only official preset bank on the OT MkII — a Set/Project named PRESET MKII with 16 patterns × 4 parts × 16 scenes to reverse-engineer. A teardown/learning reference rather than an on-aesthetic sound. The one habit worth copying into every rig template: Scene 1 is empty by default, giving a clean fader home at the far-A position.

## How to play it
1. The official Octatrack Set folder is named **PRESET MKII**. Load via **[PROJ] → SET CHANGE**.
2. Structure: **1 Set, 1 Project. Samples 24-bit/44.1 kHz. 16 patterns (all in Bank 1, [PTN]+[TRIG 1-16]), 4 parts, 16 scenes per part. Content 6.06 GB.**
3. Demo patterns use BOTH Flex and Static slots. Sample folders: Origin Sound, Loopmasters, Sample Magic, Samples from Mars.
4. **Scene 1 is EMPTY by default** (clean far-A fader home — copy this convention into rig templates).
5. Scenes assigned [SCENE A]/[SCENE B]+[TRIG 1-16]; swap a machine's sample via double-press [TRACK] → slot list → [YES], or [RIGHT] for FILE BROWSER.
