# Cortini 32-Sounds-to-a-Whole-Set (sampler-as-synth)

## Concept
Turning a tiny bank of source recordings into a playable harmonic instrument: build many tracks from a few samples, using scenes plus PTCH locks for chord and pitch changes. Cortini built a whole set from 32 short Buchla 200e sounds across seven tracks. Maps directly to ~32 short fuzz/banjo/baritone/VG-800 fragments.

## How to play it
1. Load a small bank of short source recordings (Cortini: **32 short sounds** from a Buchla 200e, factory samples erased).
2. Use OT **scenes + PTCH locks** to change chords/pitch from one source.
3. Build multiple tracks (he built **seven**) from those 32 samples.
4. Crossfader = the main performance gesture.
5. *Rig map:* capture **32 short fuzz/banjo/baritone/VG-800 fragments**, scene-lock **PTCH** on the same source for chord changes without more samples.

## Notes
- A METHOD/approach, not a knob-by-knob preset — the OT stores no shareable presets.
- The 32-samples + scene-PTCH harmony trick is the concrete reproducible core. Real, cited, on-record.
