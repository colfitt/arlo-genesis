# Blown-Out Screamed-Mic Crush

## Concept
CSH blown-out screamed-into-cheap-mic vocal character, ported to a string source. There is no vocal path on this rig, so this ports the blown-out-cheap-mic CHARACTER to a string source — it is NOT Toledo's actual vocal chain. A character analogy, used as a moment. Part of the lo-fi-prolific-volume taste profile.

## How to play it
1. Max Input Trim (~3–4:00) for the overloaded-cheap-mic clip.
2. Push Treble spiky to 1:00 — a screamed source is harsh; Bass at noon.
3. Channel Fader ~90%, Master well down (it's loud).
4. Run at 9V for maximum trash and compression.
5. Place front of chain on banjo/baritone/VG-800 so downstream inherits the overload.

## Notes
- Designed-to-emulate with an EXPLICIT APPROXIMATION flag. Toledo's blown-out vocal was a bundled laptop mic overloaded by screaming — there is no vocal path in the target boxes, so this is a character analogy on a string source, not a vocal patch. Use as a moment.
- All clock/fader/voltage values are designed-to-spec.
