# Front-of-Chain Amp Replacement

## Concept
Run the PORTA424 right after the guitar/banjo input as the "amp" — thick, trashy lo-fi distortion as the primary tone-shaping stage, with modulation, delay, and reverb coming AFTER it. The literal Mk.gee Tascam-as-amp approach, used front-of-chain. Per K_YEON_J's documented "Dream Police"-style tone, this is the only documented artist using this exact box.

## How to play it
1. Place the PORTA424 FRONT of chain, right after the guitar/banjo line-in.
2. Set Input Trim high (~2–3 o'clock) for thick, trashy distortion as the primary stage.
3. Shelve Treble down to 10 o'clock for a dark, midrange-forward voice; Bass 12–1.
4. Channel Fader ~80%, Master to taste into the rest of the chain.
5. Run 9V for trashier breakup; bring modulation/delay/reverb after it.
