# Subtle Console-Warmth DI

## Concept
The plain "familiar DI tones" job — low Trim at the end of the chain as a clean console-warmth print over any source, with no grit. The essential/classic clean voicing for when you want the cassette character without the saturation. The cleanest of the prints: coloration only.

## How to play it
1. Set Input Trim to 9 o'clock — the clean DI zone, below the breakup gate. Set the character with Trim first.
2. Treble 11–12, Bass 12–1.
3. Channel Fader ~70%.
4. Match print level with Master to unity.
5. Run 18V for maximum headroom, with grit only at the very top of the Trim sweep. Mono-sum the bus; place at end of chain.

## Notes
- Set the character with Trim first, then match print level with Master.
- The cleanest of the prints — coloration only.
- All clock/fader/voltage values are designed-to-spec.
