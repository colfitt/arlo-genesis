# Brittle Jangle Print

## Concept
Modest Mouse bright/brittle clean jangle, printed lo-fi without warming the top away. This is the cluster's one PORTA424 patch that only partly accepts the degrade tie-breaker: keep it brittle, not pristine. It resists the dark default voicing on purpose so the jangle (the hook) survives. Part of the lo-fi-prolific-volume taste profile.

## How to play it
1. Set Input Trim gentle at 10–11:00 — keep the jangly attack, just add tape.
2. LEAVE Treble at 12:00 — do NOT shelve the brittle top away; the brittleness IS the sound.
3. Bass slightly down to 11–12:00 for thin/brittle, not warm.
4. Channel Fader ~72%, Master at unity.
5. Run 18V for headroom — coloration not destruction. Place at the end of chain.

## Notes
- Designed-to-emulate. Brock used a clean Fender Twin + single-coils, not this pedal.
- Resists the dark default voicing on purpose so the jangle survives.
- All clock/fader/voltage values are designed-to-spec.
