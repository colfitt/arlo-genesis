# Bedroom 4-Track Demo Print

## Concept
The CSH bedroom laptop-DAW + cassette degrade, with the warts left audible. This is the cassette half of Toledo's budget-DAW degrade — it chases the RESULT (a budget-source cassette demo print), not the method. Part of the lo-fi-prolific-volume taste profile (Car Seat Headrest / Modest Mouse).

## How to play it
1. Set Input Trim to 12–1:00 — the breakup gate, for audible-but-musical demo grit.
2. Shelve Treble down to 10:00 for laptop/cassette dullness; Bass 12–1:00.
3. Channel Fader ~80%, Master down to compensate.
4. Run at 9V for earlier, trashier breakup — the budget-source feel.
5. Place end of chain, or front-of-chain as the amp per K_YEON_J's documented placement. Pair with the Gen Loss GL-LF1 upstream for a double-cassette bounce.

## Notes
- Designed-to-emulate. No artist setting exists for this box; Toledo used Audacity/GarageBand on a laptop, not a Tascam channel strip. Chases the RESULT (budget-source cassette demo print), not the method.
- All clock/fader/voltage values are designed-to-spec.
