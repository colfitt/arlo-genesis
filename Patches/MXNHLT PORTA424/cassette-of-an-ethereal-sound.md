# Cassette-of-an-Ethereal-Sound

## Concept
Saturating AFTER the H90's wettest reverbs and delays so the wet tails glue into "a cassette of an ethereal sound" rather than "a clean reverb with a fuzz in front." Built for Radiohead/art-rock modulated-wash and shoegaze-wall material. The gentle Trim is the point — tape character ON the tails, not the tails fuzzed away.

## How to play it
1. Mono-sum the H90 wet stereo bus into the input (the deliberate collapse the rig is built around).
2. Keep Input Trim gentle at 10–11 o'clock so the tails pass through and only pick up tape.
3. Pull Treble down slightly to 11 o'clock for a dark tilt; Bass at 12–1.
4. Channel Fader ~72%, Master at unity into the JHS 424.
5. Place immediately after the H90, last before the JHS 424.

## Notes
- Gentle Trim is the point — add tape character to the tails, don't fuzz them away.
- 18V keeps tail-printing clean, with grit only on peaks.
- CrushedHall / BlackHole-type algorithms print gorgeously through it.
- All clock/fader/voltage values are designed-to-spec.
