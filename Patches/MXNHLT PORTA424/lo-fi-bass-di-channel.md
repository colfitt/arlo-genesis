# Lo-Fi Bass DI Channel

## Concept
Using the box as what it literally is — a 4-track bass DI channel. Run low Trim for clean console warmth on a tracked bass, or push it for a gritty lo-fi bass tone. Mono collapse is a non-issue for bass. A utility/classic use included for completeness as the literal channel-strip-on-bass job.

## How to play it
1. **Clean print:** Input Trim 9 o'clock, Treble 11–12, Bass 1 o'clock, Channel Fader ~70%, Master at unity, 18V.
2. **Gritty lo-fi:** Input Trim 1–2 o'clock, Treble 11, Bass 12 (don't over-thicken when it clips), Channel Fader ~80%, Master down, 18V for firm lows or 9V for trashier.
3. Place anywhere — front for an amped bass, end for a print.

## Notes
- Lower signature fit (not drone/doom-specific) but a documented capability.
- All clock/fader/voltage values are designed-to-spec.
