# Never-Repeating Drone Bed (Recipe A)

## Concept
A sustained, ever-reshuffling diatonic chord cloud that never resolves the same way — the rig's core drone aesthetic, set-and-forget. Drives a sustained source: a VG-800 pad model, a soft-synth pad, or a Microcosm-fed loop. The trick is Random direction + Scale-Snap: chaotic but always in-key.

## How to play it
1. ONE PART → your sustained source.
2. STEPS: place 4–6 long notes, Gate = 8–24 steps so they overlap into a chord wash; leave most steps empty (sparse is fine).
3. OPTIONS → PATTERN: Direction = Random, Sync Rate = 1/8 or 1/4 (slow), Start 1 / End 16.
4. OPTIONS → CHANCE: held notes at 40–70% (VARIED per step) so the voicing reshuffles every pass (0% = a step-mute).
5. SCALES (Shift+Scale to enable): On, Root = song key, Type = Natural Minor / Dorian / Phrygian, Mode = Snap (out-of-scale notes snap to nearest — keeps every chaotic note diatonic). Scale On for this Part.
6. CLOCK: SL as master (Tx On) if the downstream source has tempo-synced FX.
7. PERFORMANCE: Sequence Transpose from the pads (±11 semis, firmware 1.4) to shove the whole wall up/down for swells/drops; cued Session switch to jump arrangements at the next boundary.
