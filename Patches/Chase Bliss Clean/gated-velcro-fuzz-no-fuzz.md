# Gated Velcro Fuzz (No Fuzz Pedal)

## Concept
A character voice the community noted on Clean: the Mode + Physics switches, plus heavy comp and a low-cut EQ, conjure a "gated velcro fuzz" and sputtering lo-fi textures with no fuzz pedal in the chain. The compressor's instability and gating tear and sputter like a torn, gated-fuzz sound — set Physics RIGHT (twitchy) for the velcro sputter, use a Shifty or Modulated EQ for the gated character, and roll the EQ toward low-cut so the uncompressed signal goes dark and gates. Stack with Sag/Dusty to taste. This is on-brand stacked degradation, distinct from the smooth blooming Dusty overdrive — it's the torn/gated character from Mode + Physics + EQ, a vibe, not a transparent comp.

## How to play it
1. Set Dynamics into heavy comp/limiting (low Sensitivity so playing slams the comp).
2. Set Physics RIGHT (twitchy) for the velcro/sputter instability.
3. Use an EQ mode (Mode LEFT Shifty or RIGHT Modulated) and roll the EQ knob toward low-cut so the signal darkens and gates between notes.
4. Dig in: the comp tears and gates like a velcro/gated fuzz — no fuzz pedal involved.
5. Add Sag or the Dusty dip for more grit if desired.

## Notes
- **No published clock-face values** — controls above are a dialable recipe drawn from community framing and demos, not sourced exact settings. Set Dynamics / Sensitivity / EQ / Attack / Release by ear.
- Community framing (SixStringSensei): "gated velcro fuzz character without a fuzz," plus sagging/sputtering lo-fi textures.
- Distinct from the smooth blooming Dusty overdrive — this is the torn/gated character from Mode + Physics + EQ.
- A vibe/character voice, not a transparent comp — you wouldn't use these settings for 70s funk.
