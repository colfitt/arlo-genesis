# One-Stomp Feedback Climax (Big Time OVERLOAD → H90 Wormhole)  ⭐

A performative feedback climb — one footswitch detonates a held chord into a climbing, self-oscillating wall, release to collapse. The build/drop as a single gesture. Designed integration chasing the post-punk/doom pulverising crescendo.

## Signal path

Baritone JM / banjo (GK-5 → VG-800 #7 −7 STEP) → CB Clean #7 (huge fuzz feeder) → 108 #9 → Hizumitas #3 Doom Wall → (End board) → CB Generation Loss (light) → **CB Big Time #10 One-Stomp Climax (OVERLOAD ramp)** → **H90 #15 Wormhole Hyperspace Wall** (Warp on a latching footswitch) → tape.

## Per-unit

- **Big Time #10:** MODE **Long** (OVERLOAD only works in Long), STATE Saturated, **COLOR ~35% (headroom for the ramp), FEEDBACK ~55% base, TILT EQ pushed DOWN** (boost lows → longer/louder tails feeding the climb), **WET set conservatively** (the ramp is "scary loud").
- **H90 #15 Wormhole:** Warp Factor 10, Stability DOWN (more pitch-wobble), Length UP, **Warp assigned to a LATCHING footswitch** so you hold it and ring out.

## Routing

Stereo from the End board. **Safety:** the Big Time manual warns about volume twice — set WET low before this; **hold-MODE = panic-reset** to a clean delay out of the chaos.

## Sound

A held chord that ramps into a climbing, pitch-warping, self-oscillating wall and falls back when released. Taste-ref: post-punk crescendo / doom feedback climb (Swans pulverising crescendo).

## Play

Hold the chord, then **hold Big Time's RIGHT footswitch** → OVERLOAD ramps COLOR+FEEDBACK toward max into a wall; simultaneously stomp the H90 Wormhole Warp latch for the rising hyperspace swell; release Big Time to collapse. Hold-MODE bails you out.
