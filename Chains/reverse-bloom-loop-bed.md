# Reverse-Bloom Loop Bed — Blooper reverse → H90 hall ⭐

A backwards swelling bed — captured plucks/tails flipped so each note fades *in*, then diffused into a big hall. The granular/reverse-bed brief, built from a committed loop (not a swelling reverse-delay). This is a 🟣 DOUG-designed integration; no artist used this exact pedal chain, the Taste-ref names the sound it chases.

## Signal path

Banjo / VG pad → **CB Clean** → [Boards 1–2 light] → [Board 3] → **MOOD MkII** (real-time reverse, optional) → **Blooper** (reversed loop) → **Chroma Console** (mono→stereo re-expand) → **H90** (hall) → tape.

## Per-unit

- **VG-800 — #6 CRYSTAL Shimmer Pad** or **#5 VIO Bowed Swell** — ringing, decaying notes reverse best (the tail becomes the swell-in).
- **MOOD MkII — #10 Slip as Real-Time Reverse** (MODE Slip, MODIFY CCW of noon = reverse; TIME high = reversed *phrases* that follow you) for a live reversed entrance before you commit it.
- **Blooper — TB4 Reverse-Bloom Loop** (Stepped Speed B4, knob **CCW = reverse**; the decaying tail swells in) — or capture a forward loop in **#6 Clean drone loop** then overdub a reversed pass in ADD. Mono in/out — degrade and rebuild here.
- **Chroma Console — #13 Ambient Diffusion Bed** (DOUBLER + SPACE + LPF FILTER, Buffered+Trails, both outs connected) — its structural job is to **re-expand the mono Blooper loop to stereo** before the H90.
- **H90 — #1 Cavernous Blackhole Drone** at moderate Mix, or **#18 Cosmic Web Reverse-Grain Pad** (reverse-pitched grains into reverb) for a doubled reverse character. Print the tail to tape.

## Routing

Blooper is **mono in/out** (4th on Board 3) — let the **Chroma re-stereo it** (Buffered+Trails, both outs, re-calibrate when you swap banjo/baritone/VG-800). Optionally clock Blooper to the Digitakt (Blooper #27, CV CLOCK dip / MIDIbox) so the loop arms on the next pulse.

## Sound

Notes that arrive backwards — swelling in from nothing — looped into an evolving bed, then smeared wide and deep by the hall. Glassy, disorienting, oceanic.

## Play

Play a ringing phrase into MOOD Slip to hear it reverse live → capture the reversed pass to Blooper (left FS RECORD → ADD) → drop to playback and solo fresh banjo over the backwards bed. Feed the reversed blooms into the H90 hall so they diffuse.

**Taste-ref:** art-rock-studio-as-instrument — Radiohead "Daydreaming" reversed-glockenspiel / "Backdrifts" reversed guitar (the result; Blooper TB4 / MOOD TB1 chase it on this rig).

**Approximation flag:** the reverse here is a captured-then-reversed *sample* (the honest hardware match per Blooper TB4), not a swelling reverse-delay.
