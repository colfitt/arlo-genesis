# One-Stomp Build → Drop — OVERLOAD Climax Over a Swelled Banjo Lead ⭐

A whole dynamic arc on the centerpiece alone: a banjo lead swells in clean, the part builds, then **one footswitch detonates the climax into a wall** and release collapses it — the Swans/post-punk near-silence-to-overwhelming move with no extra pedals. A bowed-banjo lead over clean trails that, on the stomp, ramps COLOR+FEEDBACK toward max into a self-oscillating wall; release drops it back to the singing trails.

🟣 DOUG-designed integration. No artist used this exact chain — the per-unit patch refs carry their own provenance, and the Taste-ref names the sound it chases.

## Signal path

Banjo (GK-5) → VG-800 (#VG-E1 portamento lead, optional) → **Clean #12 Dynamic Volume Swell** (+ **Clean TB1 Ondes-Martenot Bowed-Swell**) → drive → **Big Time #10 One-Stomp Climax (OVERLOAD ramp)** → MOOD → Chroma → **H90 #6 Blackhole Freeze Drone** → tape.

## Per-unit

- **Clean #12 Dynamic Volume Swell** (or **TB1 Ondes-Martenot Bowed-Swell**) — the banjo pluck auto-swells in (violined attack, no volume pedal), so the lead enters like a bowed Ondes voice feeding the delay pre-swelled material. AUX footswitch engages it.
- **Big Time #10 One-Stomp Climax (OVERLOAD ramp)** — MODE **Long** (OVERLOAD only works in Long), STATE Saturated, COLOR ~35% (headroom for the ramp), TILT pushed DOWN (boost lows → longer/louder tails feeding the climb), FEEDBACK ~55% base (the ramp takes it to max), WET set **conservatively** (the manual warns twice it's "scary loud"). SPREAD widen.
- **H90 #6 Blackhole Freeze Drone** — catch the climax tail and lock it as a held wall downstream.

## Routing

Mono banjo → IN-L (MISO). Can run free or clocked — if clocked, keep the SL/DT2 master so the swell timing sits on the grid. **Safety net:** the OVERLOAD wall is loud — set WET low before performing, and **hold MODE to panic-reset** Big Time to a clean delay if it runs away.

## Sound

A bowed-banjo lead over clean trails that, on the stomp, ramps COLOR+FEEDBACK toward max into a self-oscillating wall; release drops it back to the singing trails.

**Taste-ref:** post-punk / experimental (Swans' near-silence → overwhelming crescendo); art-rock Ondes-Martenot lead voice on the banjo (Radiohead Kid A leads, via Big Time TB3 / Clean TB1).

## Play

Swell the banjo lead in (Clean AUX), build the part, then **hold Big Time's RIGHT footswitch → OVERLOAD ramps into the wall**; release to collapse. For an infinite ending, throw the tail into H90 #6 Freeze instead of releasing. **Hold MODE = panic reset** out of the chaos.
