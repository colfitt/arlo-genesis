# Vocal-Halo Banjo

The modern-folk signature — Bon Iver's Prismizer/Messina **vocal halo** (a pitched harmony cloud around one voice) recreated on the rig as a **banjo-as-lead** with a played diatonic harmony voice, a frozen pitched grain cloud around it, a dual-pitch shimmer pad, all kept **close and dry-forward** (halo, not hall) with intimate iPhone-cassette degrade — never doom fuzz. This is the live-rig **approximation** of the halo; the genuine article is the in-the-box Antares Harmony Engine (PRZ-TB1, index-only).

This is a **designed integration** — no artist ever used this exact chain. The real method = laptop + 2× Auto-Tune + Eventide H8000 (the Messina) / Antares Harmony Engine / Deering banjo / SM7 / iPhone — none of this rig. The **whole chain chases the RESULT, not the METHOD** (no MIDI-played, formant-shifted, polyphonic halo on the rig). 🟣 designed.

## Signal path

banjo (GK-5 → **VG-800** halo-harmony voice) → **CB Clean** (present, light comp) → *[fuzz wall bypassed]* → **Microcosm** (granular harmony onset-cloud) → **QI Etherealizer** (Prismizer-halo grain freeze) → **Dark Star V3** (dual-pitch shimmer pad, Board-2 terminus, stereo) → Board 3: **Chroma Console** (iPhone-cassette intimate degrade) → **H90** (faint room) → tape.

## Per-unit

- **VG-800** — **#31 Banjo Halo Harmony Drone** (recall GK SETTING #1 EBM5 BANJO · INST = E.GUITAR or VIO · `ALT TUNE MODE = HARMONY`, `HARMONY = +1 oct` or a USER-SCALE diatonic 3rd/5th, KEY set to the tune · `NORMAL MIX ~30%` so the dry banjo stays under). ⚠️ **Approximation:** ONE fixed played interval, not the 4-note MIDI-played formant-shifted cloud — the cloud comes downstream. For the lead voice itself use **#32 Sufjan Fingerpicked Banjo Lead** (5th string excluded).
- **CB Clean** — #2 *Transparent Enhancer* or **#5 Banjo Transient Tamer** (tames the violent banjo attack so notes behave like guitar). Keep the source present/dry-forward — the voice stays close.
- **Microcosm** — **#29 Glacial Halo Bed (Haze)** or **#30 Banjo Fingerpick Onset-Halo** (Strum B / Mosaic A): Filter ~noon (tame the 2–4 kHz), **Mix ~11:00 — modest, halo-not-hall** so the granular pitch-cloud sits *behind* the present banjo. Global Config = Stereo in, Line, Buffered+Trails. ⚠️ a granular pitch-cloud, not played harmony.
- **QI Etherealizer** — **#27 Prismizer Halo Freeze Cloud** (Flow Parallel · Grain Cloud, Playback **x2** = the bright prism-up layer · Chorus Tri, Depth ~11 = the long-glide slur stand-in · Space 10–11:00 modest · Tone ~1:00 to roll the ice-pick · play a chord → **short-press Freeze** = the cloud loops, play the lead over it). ⚠️ **RIG-CRITICAL: pad/reamp the VG-800 line level before the Qi** (no level switch — it clips hot; use the bench EHX Effects Interface). ⚠️ fixed pitched cloud, not MIDI-played 4-note harmony. For the "715 hit-it-harder" distortion, push the source hot into the padded input.
- **Dark Star V3** — **#29 Halo Shimmer Pad (dual-pitch glide)** (Stereo, trails on · Pitch 1 = +1 oct (L) · Pitch 2 = detune just off unity (R), seated with the "a in Dark" off-noon trick · **Crush ~12 clean** (intimate lane, not doom) · Filter HPF glassy so the up-voice floats · Mix mostly wet but UNDER the dry voice). ⚠️ two fixed pitched voices, not the played 4-note halo.
- **Chroma Console** — **#31 iPhone Voice-Memo Cassette Capture** (TEXTURE CASSETTE ~10–11:00 · CHARACTER SWEETEN ~9–10 so the close source still reads present · MOVEMENT VIBRATO + DRIFT ~9 = subtle "wrong machine" warble · MIX ~65–70%). **Pick this OR Gen Loss as "the tape" — don't double it;** Gen Loss stays bypassed/clean for this chain.
- **H90** — a low-Mix **Room/Plate** (Posen "Room" distant-mic role, Mix ~20%) for a faint halo, not a hall. The space is a halo, never a swallow.

## Routing / sync

Banjo mono into Board 1, kept **present/dry-forward** the whole way (CB Clean + the dry path carry the close voice; every texture box sits *behind* it). **Parallel/Mix-modest** on Microcosm, QI, and Dark Star so none takes over — this is the halo-not-hall discipline. **Dark Star Hold passes new notes bone-dry** — fine here: the dry banjo lead sits in front of the frozen halo. Run **Buffered+Trails on Chroma** with both outs connected for the mono→stereo re-expansion. Let ONE box own the big space (here Dark Star + a faint H90) — don't stack three reverbs (mud). No clock needed; this is a played, rubato chain.

## Sound

A close, dry-forward fingerpicked banjo lead haloed by a pitched harmony cloud and a bright shimmer pad, warbled by an iPhone-in-a-hotel-room cassette degrade, with a faint room around it. A choir-of-one-banjo — intimate, recorded-wrong in the *gentle* way, never doom.

## Play

Recall the VG-800 halo-harmony banjo voice; fingerpick rolling arpeggios. Play a chord → **short-press QI Freeze** to capture the pitched cloud → solo the banjo lead over the frozen halo. Use **Dark Star Aux = Hold** (or the live shimmer) for the static pad; lean on the dry path for the lead since Hold passes dry. Keep everything restrained — the voice stays close. For the *genuine* halo, run **Antares Harmony Engine (PRZ-TB1, index-only)** in the DAW on the banjo and let these pedals only approximate it live.
