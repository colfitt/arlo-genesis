# 108 → Big Time MISO Mono-Fuzz Stereo Fan ⭐

The "no-splitter" width trick. A splatty silicon Fuzz Face (108) gets smoothed into a Muff wall (Hizumitas), then a SINGLE mono cable into Big Time's IN-L auto-engages MISO (mono-in / stereo-out) — and the whole stacked-mono fuzz fans out into a wide stereo delay field. No external splitter, no stereo amp rig: one mono fuzz source becomes a panned, multi-tapped stereo bed off one pedal. This is the cleanest demonstration of Big Time's MISO mode widening a hot mono source.

🟣 DOUG-designed integration. No artist played this exact chain — the per-unit patch refs carry their own provenance, and the Taste-ref names the sound it chases. The only "real-artist signal order" invoked is the general **fuzz-before-delay** standard (a documented pedal-order best-practice), not an artist-specific path.

## Signal path

guitar → **MXR 108 — "108 → Hizumitas DOOM WALL"** (splat / grind / level pre-stage) → **EQD Hizumitas — "MXR 108 Front Gated Pre-Stage — splat into a smooth wall"** (smooths the 108 splat into a sustained Muff wall, mono) → **CB Big Time — "MISO Stereo Fan"** (IN-L only → auto-MISO mono→stereo, SPREAD gentle widen, CLUSTER multi-tap fan) → stereo out.

The 108 sits IN FRONT of the Hizumitas — the rig's documented (and deliberately inverted) fuzz architecture: the silicon Fuzz Face is the gated, splatty pre-stage; the Muff is what resolves it into an even wall. Then ONE mono cable from the Hizumitas into Big Time's IN-L is the whole magic trick — that single-input feed is what auto-engages MISO.

## Per-unit

- **MXR 108 — "108 → Hizumitas DOOM WALL"** — OUTPUT/VOLUME ~12:00 (to unity), FUZZ/INPUT 12:00–2:00, **BUFFER ON**. The 108 supplies the silicon splat, the grind, and the level the Hizumitas lacks; BUFFER ON tames the FF erraticism and pre-stages the level. If it sounds thin even buffer-on, swap to Hizumitas → 108 (the Muff's low output Z drives the FF cleanly). *Published-range controls; FUZZ exact position is a dialable window, not a single sourced value.*
- **EQD Hizumitas — "MXR 108 Front Gated Pre-Stage — splat into a smooth wall"** — Volume to unity, **Sustain 12:00–2:00** (a wall voice, not full-max chaos — we want the splat *smoothed*, still articulate enough for the multi-taps to read), Tone to taste (toward CW for body, but leave some upper mid so the stereo taps stay defined). **Honesty flag:** no published Hizumitas knob values exist for the 108-in-front stack — these positions are a dialable recipe (wall voicing), not a sourced setting.
- **CB Big Time — "MISO Stereo Fan"** *(new patch)* — **Chain / Input: Hizumitas (mono) → IN-L only → auto-MISO**. MODE **Short** (a delay *field*, not an endless drone), STATE **Saturated** (the 108-fuzz variant per the gain-staging dossier — keeps the spitty stabs articulate as they multiply), VOICING **Focus** (shaves highs+lows to keep the muff wall defined across the stereo image), **COLOR LOW ~25–35%** (the fuzz already supplies all the saturation — too much COLOR + a hot fuzz clamps the limiter to porridge), **CLUSTER ~50–60%** (the multi-tap engine that fans the mono stab into a scattered stereo pattern), **SPREAD widen** (gentle — not ping-pong; we want a *cohesive wide field*, not hard L/R bouncing), FEEDBACK ~35–40% (the field sustains and overlaps without runaway), WET ~40%, TILT EQ above noon (cut muff low mud so the width reads clearly). *Numeric faders are designed starting points; on recall the motorized faders fly to the saved positions and override these.*

## Routing

The entire chain is **mono until IN-L**. That is the point: a single mono cable into Big Time's LEFT input auto-engages **MISO** (mono-in / stereo-out) — the documented, splitter-free way to fan a mono fuzz into a stereo delay field. From Big Time onward you're stereo.

**Gain-staging is the whole game.** The 108 grinds and sets level → the Hizumitas saturates and smooths into a wall → Big Time's job is *width and movement*, not more dirt. Keep COLOR modest so the analog STATE limiter doesn't clamp the already-hot fuzz to porridge. Big Time's input is an analog stereo preamp (COLOR) feeding an analog limiter (STATE) — it's *built* to be slammed by a hot source, so a fuzz in front is exactly the intended drive. Use **Saturated** (not Compressed) here: the 108's spitty, slightly-gated decay reads as articulate stabs, and Saturated + higher CLUSTER lets the multi-taps spit them across the stereo image instead of gluing them flat. SPREAD stays on **widen** (not ping-pong) so the fan is a wide cohesive bed rather than a hard-panned echo. Favor **DRY KILL ON** (Big Time Options) if you want the wet stereo field to print clean without a centered mono dry note pinning the image.

## Sound

A splatty silicon fuzz, smoothed into a Muff wall, then opened up wide — one mono source spread into a panned, gently-overlapping stereo delay field with movement and air on both sides. Big, immersive, and stereo, from a single mono cable and no splitter.

**Taste-ref:** Boris/Wata sustaining-fuzz weight fanned into MBV/shoegaze stereo width — the wall, but wide. The "how is this stereo off one pedal" trick.

## Play

1. Set the 108 → Hizumitas wall first in mono — get the splat smoothing into an even, articulate wall (still picky enough to hear individual stabs).
2. Run the ONE mono cable into Big Time **IN-L** and confirm MISO engaged (you'll hear the source open into stereo).
3. Pick single stabs and short phrases — the CLUSTER multi-taps fan each one across the stereo field; let phrases overlap so the field fills in.
4. Ride **CLUSTER** for how scattered/dense the fan is, and **SPREAD widen ↔ off** to A/B how much width MISO is buying you.
5. Hold MODE to panic-reset Big Time to a clean delay if the field runs away.
