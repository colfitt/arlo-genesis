# Ruined-Cassette Drop

The blown-out climax of a degraded groove — the whole bus collapses into one ruined-cassette bounce as a moment, then collapses back. The most "recorded-wrong" stack in the file. A 🟣 DOUG-designed integration + **explicit approximation flag** (no vocal path).

## Signal path

groovebox bus + heavy source → JHS Colour Box V2 (or Longsword) → CB Generation Loss → CB Big Time → PORTA424 → JHS 424 → tape.

## Per-unit

- **JHS Colour Box V2** — **#16 Broken-Console Screamed Crush (CBX-LF2)** (HI · STEP 5 max · PRE-VOL maxed · EQ flat then TREBLE up) — the genuine broken-console fuzz; OR **Longsword #LSW-LF2 Glitchy 8-Bit Broken Slam** (Level/Drive/Boost MAX, silicon) for the squared/gated sputter.
- **CB Generation Loss** — **#9 Total Destruction (moment climax)** (Wow/Flutter 2:00+, Saturate + Failure maxed, VHS Model 1–3, Noise on) — a nauseating-on-purpose full meltdown, a *moment* not a tone.
- **CB Big Time** — **#10 One-Stomp Climax (OVERLOAD ramp)** (MODE Long, hold RIGHT footswitch → OVERLOAD ramps COLOR+FEEDBACK to a wall; WET set LOW first — "scary loud"; hold MODE = panic-reset out).
- **PORTA424** — **#14 Blown-Out Screamed-Mic Crush (P424-LF3)** (Trim MAXED, 9V) — the overloaded-cheap-mic crush terminus.
- **JHS 424** — **#4 Gated-Fuzz Monster** (Gain 2 4:00, PAST the gate) for total collapse, OR keep it light if PORTA424 is already maxed (don't double-destroy unless you want the full porridge for the single climax).

## Routing / sync

This is the one place the **don't-crank-both-boxes rule is deliberately broken** — for a single climax you *can* stack both PORTA424 maxed + 424 gated for total collapse (flagged: it goes to undifferentiated mush, so use it as one moment). Groovebox clock master; trigger the drop on a cued pattern boundary. **APPROXIMATION:** CBX-LF2 / P424-LF3 port the *screamed-into-a-cheap-mic vocal* character to a string/groovebox source — **this rig has no vocal path**, so it is a character analogy, not Toledo's actual vocal chain. Flagged.

## Sound

A speaker-ripping, gated, blown-out cassette meltdown that swells in on the OVERLOAD ramp and falls apart — the drop where the whole degraded groove eats itself, then snaps back to the verse.

## Play

Run the groove clean-ish into the verse; at the drop, foot the Colour Box (or Longsword) max-fuzz, hold the Big Time RIGHT footswitch for the OVERLOAD ramp, and let Gen Loss Total Destruction + the maxed tape stage detonate. Release / hold MODE to collapse back. Use once per song — the hiss + pitch-warp fatigue fast.
